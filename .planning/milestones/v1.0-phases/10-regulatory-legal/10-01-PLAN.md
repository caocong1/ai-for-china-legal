<objective>
将监管合规插件从 v0.1.0 的扁平浅骨架深化为深层子技能体系，建立全插件复用的法律引用脊柱、监管合规版实践配置契约，并**新建** regulatory-legal/CLAUDE.md 实践配置模板。

本计划交付四块内容：
1. **监管合规引用脊柱（citation spine）** — 一套所有 regulatory-legal 技能共享的法律依据引用规范（来源标签 + 法律/行政法规/部门规章/规范性文件/国家标准 与 案例/学说 分类 + 待验证 + **行政处罚法 2021 条号重排保守标注纪律** + **行业监管规章快速演进标注纪律** + 机构定位 + 监管合规立场定位），以及一个可复用的 行政处罚法 2021/行政许可法·行政强制法·行政复议法·行政诉讼法规则描述/各行业监管规章占位/企业合规管理体系国家标准 引用库。这是 CONTEXT「全面法律研究」决策的可复用基础设施，后续 10-02 / 10-03 直接引用，避免每个技能重复维护条文。
2. **监管合规版实践配置契约** — regulatory-legal/skills/_shared/practice-profile-schema.md：明确配置文件是什么、在哪（regulatory-legal/CLAUDE.md）、技能怎么读，以及访谈/立场/行业/监管机关/升级字段与 regulatory-legal/CLAUDE.md 章节的映射；**立场字段区分监管动态监控/行政处罚应对/合规体系建设/监管报告起草**（监管合规工作的核心变量），并含**所属行业与主管监管机关、监管关系、行刑衔接升级字段**。
3. **新建 regulatory-legal/CLAUDE.md 实践配置模板** — 监管合规语境的散文模板：执业角色与企业画像（含**所属行业与主管监管机关**）/ 谁在使用 / 可用集成（监管机关·行业协会·法规数据库资源）/ 合规与监管立场（监管动态监控/行政处罚应对/合规体系建设/监管报告起草 + 合规严格度 + 监管关系）/ 行政违法风险校准 / 升级矩阵（重大处罚/吊销许可/行刑衔接/人身自由风险强制升级律师）/ 文书风格 / 输出 / 共享护栏（监管合规立场护栏 / 行政程序时限护栏 / 行刑衔接升级护栏 / 律师审查护栏 / 地方性法规与裁量基准提示）。当前插件**尚无此文件**，本计划首次创建。
4. **监管动态监控拆分** — 按 CONTEXT「深层子技能拆分」决策，将 regulatory-monitoring（现 ~17 行浅骨架）拆为四个可独立触发的子技能：监管机关与规则源识别 / 动态抓取与分类 / 影响评估与合规差距 / 预警与台账。现有三步纲要（需求收集 / 法律分析 / 输出生成）拆分迁移并大幅深化，原 SKILL.md 改为编排入口。

Purpose: 关闭「CONTEXT envisions 深度子技能拆分 + 5 能力面，现实只有 3 个扁平浅骨架且无 CLAUDE.md」的核心 gap，并为监管合规语境建立可复用引用脊柱与配置真相来源。引用脊柱、配置契约与 CLAUDE.md 先行，保证三个计划的引用与配置读取一致。
Output: 1 个引用规范 + 1 个引用库 + 1 个配置契约 + 1 个新建 CLAUDE.md + 4 个 regulatory-monitoring 子技能 SKILL.md + 1 个改造后的编排入口。

中国化原则约束（来自 CLAUDE.md，必须遵守）：成文法为主（行政处罚法·行政许可法·行政强制法·行政复议法·行政诉讼法 为分析基础，各行业监管行政法规·部门规章·规范性文件·国家标准 为补充）；指导性案例作参考非判例法（最高院行政指导性案例·行政复议诉讼典型案例可参考，案号标待验证）；法院层级识别（基层/中院/高院/最高院四级 + 复议机关 上级行政机关·本级人民政府）；行业监管特色（市场监管总局·网信办·金融监管·药监·生态环境·应急管理 等中国特有监管体系）；执业环境适配（监管合规律师 / 企业合规官 / 企业法务 / 当事人本人 / 非律师但有律师可咨询）。命名约定：技能名 kebab-case；斜杠命令 /regulatory-legal:<skill>。文件格式：JSON 2 空格缩进、文件以换行符结尾、无尾部空格、markdown 表格列数一致。

法律内容硬性要求：引用按 法律/行政法规/部门规章/规范性文件/国家标准 与 案例/学说 分类，带来源标签。中度把握锚点（须标「已核实锚点（中度把握），建议复核」，**务必用行政处罚法 2021 修订后新条号**）：处罚法定原则=行政处罚法第4条；处罚种类=第9条；责令改正=第28条；一事不再罚（同一违法行为不得两次罚款）=第29条；从轻减轻处罚=第32条；不予处罚（无主观过错/初次违法危害轻微及时改正）=第33条；追责时效（一般2年；涉及公民生命健康安全·金融安全且有危害后果的5年）=第36条；当事人陈述申辩权=第44条、第45条；重大执法决定法制审核=第58条；听证=第63条。**行政许可法·行政强制法·行政复议法·行政诉讼法的具体条号（复议前置·起诉期限·举证责任倒置等）作者把握不足的，一律描述规则 + `[待验证]`，绝不臆造条号。** **各行业监管部门规章/规范性文件、行政处罚裁量基准 属规范性文件且快速演进、地方差异大，一律描述规则并标 `[待验证]`，不硬写文号、发文日期、具体条款编号、罚款数额区间。** 企业合规管理体系国家标准（GB/T）标准编号与版本须核实，描述规则 + `[待验证]`，不臆造标准号。司法解释具体条号、案例案号一律标 `[待验证]`，描述规则呈现，绝不臆造。
