<objective>
将婚姻家事插件从 v0.1.0 的扁平浅骨架深化为深层子技能体系，建立全插件复用的法律引用脊柱、婚姻家事版实践配置契约，并**新建** family-legal/CLAUDE.md 实践配置模板。

本计划交付四块内容：
1. **婚姻家事引用脊柱（citation spine）** — 一套所有 family-legal 技能共享的法律依据引用规范（来源标签 + 法条/司法解释 与 案例/学说 分类 + 待验证 + **司法解释具体条号保守标注** + **旧法条号→民法典新条号迁移警示** + 行业/机构定位），以及一个可复用的 民法典婚姻家庭编/继承编/妇女权益保障法/婚姻家庭编司法解释(一)/继承编司法解释(一) 引用库。这是 CONTEXT「全面法律研究」决策的可复用基础设施，后续 08-02 / 08-03 直接引用，避免每个技能重复维护条文。
2. **婚姻家事版实践配置契约** — family-legal/skills/_shared/practice-profile-schema.md：明确配置文件是什么、在哪（family-legal/CLAUDE.md）、技能怎么读，以及访谈/审查/敏感度字段与 family-legal/CLAUDE.md 章节的映射；**立场字段区分离婚一方侧/某继承人侧/遗嘱人侧/中立**（婚姻家事工作的核心变量），并含**敏感度与升级字段**（家暴/未成年人/情绪危机/强制升级）。
3. **新建 family-legal/CLAUDE.md 实践配置模板** — 婚姻家事语境的散文模板：执业角色与当事人画像 / 谁在使用 / 可用集成（法院·民政·公证·鉴定资源）/ 审查与代理立场（离婚一方/某继承人/遗嘱人 + 协议审查严格度 + 诉讼倾向）/ 敏感度校准（家暴·未成年人·情绪危机识别）/ 风险校准 / 升级矩阵（家暴/涉刑/重大财产/未成年人利益强制升级）/ 文书风格 / 输出 / 共享护栏（当事人情绪护栏 / 未成年人利益最大化 / 律师审查护栏 / 地方性规定提示）。当前插件**尚无此文件**，本计划首次创建。
4. **离婚协议起草拆分** — 按 CONTEXT「深层子技能拆分」决策，将 divorce-agreement（现 ~16 行浅骨架）拆为四个可独立触发的子技能：财产分割条款 / 子女抚养与探望 / 债务处理 / 协议生效与登记。现有三步纲要（需求收集 / 法律分析 / 输出生成）拆分迁移并大幅深化，原 SKILL.md 改为编排入口。

Purpose: 关闭「CONTEXT envisions 深度子技能拆分 + 5 能力面，现实只有 4 个扁平浅骨架且无 CLAUDE.md」的核心 gap，并为婚姻家事语境建立可复用引用脊柱与配置真相来源。引用脊柱、配置契约与 CLAUDE.md 先行，保证三个计划的引用与配置读取一致。
Output: 1 个引用规范 + 1 个引用库 + 1 个配置契约 + 1 个新建 CLAUDE.md + 4 个 divorce-agreement 子技能 SKILL.md + 1 个改造后的编排入口。

中国化原则约束（来自 CLAUDE.md，必须遵守）：成文法为主（民法典婚姻家庭编 第1040-1118条、继承编 第1119-1163条、妇女权益保障法 为分析基础，婚姻家庭编司法解释(一)、继承编司法解释(一) 为补充）；指导性案例作参考非判例法（最高院/婚姻家事典型案例可参考，案号标待验证）；法院层级识别（基层/中院/高院/最高院四级 + 家事审判庭与家事调解）；行业监管特色（民政婚姻登记机关、公证机构、妇联与反家暴庇护、不动产登记）；执业环境适配（婚姻家事律师 / 企业或个人法务 / 当事人本人 / 非律师但有律师可咨询）。命名约定：技能名 kebab-case；斜杠命令 /family-legal:<skill>。文件格式：JSON 2 空格缩进、文件以换行符结尾、无尾部空格、markdown 表格列数一致。

法律内容硬性要求：引用按 法条/司法解释 与 案例/学说 分类，带来源标签。中度把握锚点（须标「已核实锚点（中度把握），建议复核」）：夫妻共同财产=民法典第1062条；夫妻个人财产=第1063条；约定财产制=第1065条；协议离婚=第1076条；离婚冷静期30日=第1077条；诉讼离婚=第1079条；子女抚养=第1084条；家务劳动补偿=第1088条；离婚经济帮助=第1090条；离婚损害赔偿=第1091条；遗产范围=第1122条；法定继承顺序=第1127条；遗嘱形式=第1134-1139条；必留份=第1141条；遗赠扶养协议=第1158条。**严禁回退旧法条号（原婚姻法第17/18/32条、原继承法第10条等），一律使用民法典婚姻家庭编/继承编新条号。** 规则类（来自司法解释，描述规则不写裸条号）：夫妻共同债务认定（共债共签/家庭日常生活需要/超出部分债权人举证）；离婚损害赔偿适用细化；房产婚后还贷增值分割；彩礼返还。**司法解释（婚姻家庭编司法解释(一)、继承编司法解释(一)）具体条号、案例案号一律标 `[待验证]`，描述规则呈现，绝不臆造。** 离婚协议援引合同编通则锚点（解除/可撤销/欺诈胁迫意思表示）复用商事脊柱已核实结论，但**不得错套合同编条号到婚姻家庭编规则上**。
