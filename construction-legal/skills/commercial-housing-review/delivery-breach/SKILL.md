# 交付与违约

## 目的

系统审查商品房买卖合同中的交付条件、逾期交房违约责任、逾期办证违约责任与面积误差处理规则，特别是商品房买卖合同司法解释中的面积误差3%处理规则与违约金调整机制。按代理立场（购房人侧 vs 开发商侧）调整审查侧重。

本子技能聚焦于：
1. **逾期交房违约责任**（交付条件/逾期起算/违约金计算与调整/催告与解除）
2. **逾期办证违约责任**（协助办理产权登记义务/逾期办证违约金/解除条件）
3. **面积误差3%处理规则**（面积误差比 ≤3% vs >3% 的不同处理，退补规则）
4. **质量与保修**（交付条件中的质量要求/保修责任）

与 `commercial-housing-review` 其他子技能的分工：本技能专注于**合同履行阶段的交付与违约**；`presale-license-subject` 处理资质前置核查；`deposit-subscription` 处理认购书定金；`loan-termination-risk` 处理贷款解除与产权登记保护。

**可单独触发**：`/construction-legal:delivery-breach`，或由 `/construction-legal:commercial-housing-review` 编排入口作为第三步调用。

## 详细步骤

### 第一步：交付条件核查

商品房交付须满足一定条件（主要是竣工验收合格），须逐项核查合同约定与法律要求：

| 交付条件要素 | 法律要求（描述规则） | 合同审查要点 |
|------------|-----------------|------------|
| **竣工验收合格** | 民法典第799条（建议复核）规定建设工程竣工经验收合格后方可交付使用；商品房交付须以竣工验收合格为前提 | 合同是否明确约定「竣工验收合格」为交付前提条件；不得约定「主体结构验收合格即交付」等降低标准 |
| **取得竣工验收备案证明** | 竣工验收后须向主管部门备案 | 合同是否要求开发商取得竣工验收备案证明并告知购房人 |
| **消防、规划等专项验收** | 部分项目须专项验收（消防/人防/环保等） | 专项验收是否纳入交付条件 |
| **房屋外观与装修标准（精装房）** | 精装修合同须约定装修标准；与样板间不一致时的处理 | 合同约定装修标准是否清晰可执行；实物交付标准的约定 |

**逾期交房的触发条件**：合同约定的交付时间届满，开发商未满足上述交付条件，构成逾期交房。

### 第二步：逾期交房违约责任分析

#### 2.1 逾期交房违约金合同约定审查

| 审查要点 | 购房人侧关注 | 开发商侧关注 |
|---------|------------|------------|
| **违约金计算基数** | 核查是否以「已付购房款」或「合同总价款」作为基数（宜以合同总价款作为基数，对购房人更有利） | 确认基数约定是否清晰；避免争议 |
| **违约金计算比率/标准** | 合同约定违约金率是否过低（过低可请求按实际损失调整，民法典第585条第2款，建议复核） | 确认违约金率未被高估冒算 |
| **违约金封顶条款** | 购房人侧：任何封顶条款均应标红；封顶后实际违约金可能远低于损失 | 开发商侧：封顶条款可合理控制违约风险 |
| **免责条款** | 核查不可抗力免责范围是否过宽（一般不可抗力免责须经通知且事件直接导致延期） | 确认不可抗力条款合理 |
| **催告与解除机制** | 逾期超过一定时间购房人是否有权解除合同（须有明确约定或适用法定解除权） | 限制解除条件的约定是否合理 |

#### 2.2 逾期交房违约金调整规则

来自商品房买卖合同司法解释（描述规则，具体条号 `[待验证 — 司法解释条号易变，须核实现行版本]`）：

| 情形 | 调整方向 | 依据 |
|------|---------|------|
| **约定违约金过分高于实际损失** | 开发商可请求适当减少（民法典第585条第2款，建议复核） | 通则违约金调整规则；「过分高于」的认定标准须结合司法实践 |
| **约定违约金低于实际损失** | 购房人可请求按实际损失增加（民法典第585条第2款，建议复核） | 须举证实际损失（如租金损失、临时住宿费等） |
| **无违约金约定** | 购房人可按损失赔偿规则主张（民法典第583/584条，建议复核）；也可请求适用司法解释规定的参考标准（描述规则，条号 `[待验证]`） | 司法解释有关于无违约金约定时的参考计算方式（描述规则，条号 `[待验证]`）`[模型知识 — 需验证]` |

#### 2.3 逾期交房导致合同解除

购房人解除权触发条件（商品房买卖合同司法解释描述规则，具体条号 `[待验证]`）：

- 逾期交房**超过一定期限**（合同约定或法定期限，须核实 `[待验证具体期限]`）
- 购房人可选择：（1）继续要求履行并主张逾期交房违约金；或（2）解除合同并要求退款、利息及损失赔偿
- 解除合同后：开发商须返还已收取的购房款及利息，并可能承担额外损失赔偿

### 第三步：面积误差3%处理规则

**面积误差处理规则**来自商品房买卖合同司法解释（描述规则，具体条号 `[待验证 — 司法解释条号易变，须核实现行版本]`，规则相对稳定），是商品房审查的核心规则之一：

#### 3.1 面积误差比计算

```
面积误差比 = (产权登记面积 - 合同约定面积) ÷ 合同约定面积 × 100%
```

> **注意**：面积误差比以绝对值计算（正负均适用）；面积单位须统一（建筑面积 vs 套内面积须与合同约定对应）。

#### 3.2 面积误差比处理决策表

| 面积误差情形 | 规则描述 | 购房人选择 |
|------------|---------|----------|
| **面积误差比绝对值 ≤ 3%（含3%）** | 据实结算：合同约定面积与产权登记面积之差，按合同约定的单价据实结算（多退少补，但不超过3%范围内） | 按实结算，无权主张解除合同 |
| **面积误差比绝对值 > 3%** | 买受人可以请求解除合同（并取回已付房款及利息）；或买受人不要求解除合同的，有超出3%部分的特别处理规则（见下表） | **关键选择权**：解除 or 继续执行（选择对应规则） |
| **误差比 > 3%，买受人选择不解除** | 产权登记面积 > 合同约定面积：3%以内部分价款由买受人补足，超出3%部分的价款由出卖人承担，产权归买受人；产权登记面积 < 合同约定面积：3%以内部分价款由出卖人返还，超出3%部分的价款由出卖人**双倍返还**买受人（描述规则，具体条号 `[待验证]`）`[模型知识 — 需验证]` | 双倍返还的超出部分有较强的补偿性 |

**购房人侧操作建议（面积误差超3%）**：
1. 计算面积误差比，确认是否超过3%
2. 评估继续执行 vs 解除合同的利弊（解除可全额退款；继续执行可获双倍返还超出部分）
3. 明确告知购房人行使选择权的期限（以免超期丧失选择权，建议结合司法解释规定核实）

**开发商侧操作建议（面积误差超3%）**：
1. 主动排查建筑面积测量结果，提前评估误差风险
2. 在误差较大时考虑与购房人协商（补偿方案）
3. 避免合同中约定排除面积误差处理规则的格式条款（该类条款效力存疑）

#### 3.3 面积类型确认

- **建筑面积 vs 套内面积**：合同约定应明确以哪种面积为计价基础；两种面积的测量结果可能不同
- **公摊面积**：商品房公摊面积规则 `[地方 flagged — 各地计算规则有差异，需核实地方规定]`
- **车位/储藏室面积**：通常单独计价，不纳入主合同面积计算

### 第四步：逾期办证违约责任分析

逾期办证是指开发商未按约定时间协助购房人办理**不动产权证（产权登记）**：

| 审查要点 | 规则描述 |
|---------|---------|
| **办证义务的法律依据** | 开发商有协助购房人办理不动产初始登记、转移登记的义务；合同须明确约定协助办证期限 |
| **逾期办证违约金约定** | 商品房买卖合同司法解释有关于逾期办证违约金的规则（描述规则，具体条号 `[待验证 — 司法解释条号易变，须核实现行版本]`）`[模型知识 — 需验证]` |
| **逾期办证违约金调整** | 与逾期交房违约金类似，适用民法典第585条第2款（违约金调整规则，建议复核） |
| **逾期办证导致解除** | 逾期办证超过一定期限且购房人催告后仍未办理，购房人可解除合同（描述规则，具体条号 `[待验证]`）`[模型知识 — 需验证]` |
| **开发商原因 vs 购房人原因** | 须区分逾期办证的归责方：因开发商未缴清土地出让金、五证不齐等原因导致无法办证，归责于开发商；因购房人未配合提供材料等，须具体分析 |

#### 逾期办证违约金核查表

| 核查项 | 购房人侧关注 | 开发商侧关注 |
|--------|------------|------------|
| 办证违约金计算基数 | 须明确基数（总价款 vs 已付款） | 确认基数约定 |
| 办证违约金率 | 是否低于实际损失（如长期租房成本）；是否与逾期交房违约金一致 | 确认违约金率合理 |
| 办证期限起算点 | 从交付日起算 vs 从申请日起算（不同起算点影响计算） | 确认约定的起算点 |
| 是否有不可归责免责约定 | 免责范围不宜过宽 | 确认不可归责情形有明确约定 |

### 第五步：质量与保修

| 审查要点 | 规则描述 |
|---------|---------|
| **主体结构质量** | 若主体结构质量不合格，购房人可主张退房（解除合同）并要求赔偿（商品房买卖司法解释描述规则，具体条号 `[待验证]`）`[模型知识 — 需验证]` |
| **保修期限** | 建设工程保修制度（建设工程质量管理条例，具体条号 `[待验证]`）规定地基基础和主体结构保修期为设计文件规定的合理使用年限；屋面防水、供排水管道、供热供冷系统、电气管线等各有不同的保修期限 `[待验证具体期限]` |
| **精装修质量** | 精装修商品房的装修质量标准须与合同约定一致；与样板间不符须主张违约责任 |
| **质量问题诉讼时效** | 质量问题的诉讼时效起算须注意 `[待验证]` |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| **违约金过高（开发商主张调整）** | 适用民法典第585条第2款（建议复核）；「过分高于实际损失」的举证责任由主张方承担 |
| **套内面积 vs 建筑面积争议** | 确认合同约定的计价方式与面积口径；两种面积误差可能产生不同结论 |
| **精装修与交付标准争议（精装房）** | 精装标准须以合同约定为准；样板间展示不具备合同效力（除非合同特别约定） |
| **逾期交房期间涨价/跌价** | 房价上涨时购房人损失（含涨价损失）可能较大；房价下跌时开发商损失风险 `[待验证司法实践处理]` |
| **群体性退房（多户购房人同时主张）** | 须升级处理（见升级矩阵）；可能涉及行政监管介入 |
| **烂尾楼停工场景** | 须评估工程是否有重启可能；停贷风险见 `loan-termination-risk` |
| **因不可抗力（如疫情）导致逾期** | 分析不可归责于双方情形；适用情势变更（民法典第533条，建议复核）或不可抗力条款 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| **面积数据不完整（无产权登记面积）** | 基于合同约定面积进行分析框架；明确须取得产权登记面积后才能计算误差 |
| **合同无逾期交房违约金约定** | 说明可按实际损失主张（民法典第583/584条，建议复核）；或提示购房人主张司法解释参考标准 |
| **违约金标准争议较大** | 建议委托专业律师评估；提供双方主张的基础分析框架 |
