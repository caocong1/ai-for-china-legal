# 预售许可与主体核查

## 目的

核查商品房买卖合同的预售许可合法性与开发商主体资质，识别预售/现售阶段的程序瑕疵与效力风险，并按代理立场（购房人侧 vs 开发商侧）调整审查侧重与风险建议。

本子技能聚焦于：
1. **商品房预售许可证**的取得条件与效力判定
2. **五证核查**（国有土地使用证/建设用地规划许可证/建设工程规划许可证/建筑工程施工许可证/商品房预售许可证）
3. **开发商主体资质**（房地产开发企业资质/项目公司结构/失信与异常状态）
4. **现售条件**核查（现房销售的法定条件 vs 预售条件）

与 `commercial-housing-review` 其他子技能的分工：本技能专注于**合同签订前的资质与合法性核查**；`deposit-subscription` 处理认购书与定金法律性质；`delivery-breach` 处理交付与违约；`loan-termination-risk` 处理贷款解除与产权风险。

**可单独触发**：`/construction-legal:presale-license-subject`，或由 `/construction-legal:commercial-housing-review` 编排入口作为第一步调用。

## 详细步骤

### 第一步：预售/现售模式识别

确认合同为预售合同（期房）还是现售合同（现房），两者适用不同的合规条件和核查路径。

| 类型 | 判断依据 | 适用核查路径 |
|------|---------|------------|
| **预售合同（期房）** | 房屋尚未竣工验收，以商品房预售许可证为销售依据 | 重点：预售许可证核查 + 五证核查 + 开发商资质 |
| **现售合同（现房）** | 房屋已竣工验收合格，以现售房屋备案/不动产权证为基础 | 重点：现售条件核查 + 产权状态核查 |
| **认购书阶段** | 尚未签订正式买卖合同，认购书性质须结合 `deposit-subscription` 子技能判断 | 预售许可取得时间与认购书签订时间的关系 |

### 第二步：五证核查

五证是商品房合法销售的基础性证件，逐项核查：

| 证件名称 | 核查要点 | 主要风险 | 风险等级参考 |
|---------|---------|---------|------------|
| **国有土地使用证** （或不动产权证书） | 证载用途（住宅/商业/工业，须与合同约定一致）；使用权人与开发商主体一致性；出让年限与到期时间；是否已设抵押或被查封 | 土地性质与合同约定不符；土地被抵押且无充分担保；使用年限届满风险 | 🔴（被查封/用途严重不符）→ 🟠（设抵押/年限较短） |
| **建设用地规划许可证** | 用地范围与项目范围一致性；有效期状态 | 用地范围不含所购单元；许可已过期 | 🟠（范围不符/过期） |
| **建设工程规划许可证** | 许可的建设工程规模与实际项目一致；审批楼栋/户型涵盖所购单元 | 超规划建设；所购单元不在许可范围内 | 🟠（超规划建设）→ 🔴（违规严重） |
| **建筑工程施工许可证** | 是否已取得；与合同所购单元对应楼栋匹配 | 未取得施工许可即开工，影响竣工验收合法性 | 🟠（未取得或不对应） |
| **商品房预售许可证** | 是否已取得；许可证载明的销售范围（楼栋/楼层/套数）涵盖所购单元；有效期状态；发证机关为地方住建/房产主管部门 | 未取得预售许可而销售（内部认购/未证销售）；所购单元不在许可范围内；许可已失效 | 🔴（未取得许可的预售）→ 🟠（范围不完全覆盖） |

**五证核查获取方式**：
- 要求开发商出示原件并核对
- 通过当地住建/房产主管部门官网或窗口查询备案信息
- 通过全国建筑市场监管公共服务平台辅助核查
- 国有土地使用证/不动产权情况可通过当地不动产登记机构查询

### 第三步：预售许可效力判定

商品房预售合同效力与预售许可证的关系，规则来自城市房地产管理法及商品房买卖合同司法解释（具体条号 `[待验证]`，描述规则）：

| 情形 | 规则描述 | 效力结论 | 代理建议 |
|------|---------|---------|---------|
| **签订合同时已取得预售许可证** | 合规；预售许可为合法销售前提 | 预售合同一般有效（其他条件亦须满足） | 购房人侧：确认证件有效；开发商侧：确认合规 |
| **签订合同时未取得预售许可证，但在起诉前已取得** | 商品房买卖合同司法解释有规则：未取得预售许可证情形下，在当事人起诉前取得预售许可证明的，可以认定合同有效（描述规则，具体条号 `[待验证 — 司法解释条号易变，须核实现行版本]`） | 可认定有效（条件：起诉前取得） | 购房人侧：若开发商补办，评估能否主张预期赔偿或违约；开发商侧：尽快补正 |
| **起诉时仍未取得预售许可证** | 未取得预售许可证明而与买受人签订的商品房预售合同，一般应认定无效（描述规则，条号 `[待验证]`） | 一般无效；买受人可请求退款并赔偿损失 | 购房人侧：主张合同无效、退款加损失赔偿；开发商侧：评估补正可能性与赔偿风险 |
| **内部认购/「认筹」但无预售许可** | 不具备预售条件时的内部认购法律效力存疑；定金/认筹金的退还另见 `deposit-subscription` | 效力风险高 | 🔴 风险警示 |

> **注意**：预售许可效力判定规则来自最高法商品房买卖合同纠纷司法解释，司法解释历经修订，**具体条号一律 `[待验证 — 司法解释条号易变，须核实现行版本]`**，在引用时须核实现行版本。

### 第四步：预售许可证详细核查清单

核查预售许可证本身的内容要素：

- [ ] 许可证号（发证机关/年份/顺序号格式）
- [ ] 发证机关是否为有权机关（当地住建/房产主管部门）
- [ ] 建设项目名称与合同项目名称一致
- [ ] 开发商（预售人）名称与合同出卖人名称一致
- [ ] 预售范围明确涵盖所购楼栋/楼层/户型
- [ ] 有效期状态（部分地区预售许可证有期限，须核实是否在有效期内 `[地方 flagged — 需核实地方性规定]`）
- [ ] 是否有注记：分期预售/特殊限制条件

### 第五步：开发商主体资质核查

| 核查项 | 核查内容 | 获取渠道 | 风险信号 |
|--------|---------|---------|---------|
| **营业执照/统一社会信用代码** | 存续状态（非注销/吊销）；经营范围含房地产开发；注册资本与项目规模匹配 | 国家企业信用信息公示系统（https://www.gsxt.gov.cn） | 经营范围不含房地产；注册资本明显偏低 |
| **房地产开发企业资质证书** | 资质等级（一级/二级/三级/四级）；在有效期内；与项目规模相符（较大型项目须较高资质等级 `[待验证具体要求]`） | 当地住建/房产主管部门；全国建筑市场监管平台 | 无资质或资质不匹配 |
| **是否被列入经营异常名录** | 通过国家企业信用信息公示系统查询 | 国家企业信用信息公示系统 | 被列入异常 — 🟠 风险 |
| **是否被列入严重违法失信名单** | 法院失信被执行人名单（「老赖」名单）；住建系统建筑市场黑名单 | 中国执行信息公开网；全国建筑市场监管平台 | 被列入失信/黑名单 — 🔴 风险，强烈建议暂缓签约 |
| **股权穿透与关联风险** | 项目公司股东背景；是否存在多层嵌套或空壳架构 | 企查查/天眼查（辅助工具，需核实） | 多层壳公司/未实际出资 |
| **司法查封/抵押** | 土地与在建工程是否有未解除的查封或抵押 | 当地不动产登记机构；法院执行公告 | 整体抵押/查封 — 🔴 |

**与 `loan-termination-risk` 子技能的衔接**：土地/在建工程被查封或抵押的情况，在 `loan-termination-risk` 中进一步处理（网签备案保护、预告登记、一房二卖风险）。

### 第六步：现售条件核查（适用于现房合同）

现房销售须满足一定条件（城市房地产管理法及相关法规规定，具体条号 `[待验证]`，描述规则）：

| 现售条件（描述规则，具体条号待验证） | 核查要点 |
|----------------------------------|---------|
| 土地使用权出让金已全部缴清 | 向当地土地主管部门核实 |
| 持有国有土地使用证 | 确认证件原件 |
| 建设工程规划许可证符合 | 确认房屋实际建设符合规划批准范围 |
| 通过竣工验收 | 核查竣工验收备案证明 |
| 工程质量符合相关要求 | 工程质量验收报告 |
| 已办理不动产登记（或具备不动产初始登记条件） | 核查不动产登记状态 |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| **内部认购/「排号」无预售许可** | 🔴 风险警示：不具备预售条件的内部认购效力存疑；定金/认筹金返还见 `deposit-subscription` |
| **车位/储藏室** | 车位/储藏室可能不在预售许可范围内，须单独确认车位的产权性质（独立产权 vs 人防车位）及预售许可涵盖情况 |
| **小产权房** | 在集体土地上建设的「小产权房」，不具备商品房预售许可条件，无法办理不动产登记；购房人无法获得国家保护，🔴 风险，应明确建议不购买 |
| **商住两用/公寓** | 商用性质（办公/商业）土地年限（通常 40~50 年）短于住宅（70 年）；须核实土地用途，提示年限差异 `[待验证具体年限]` |
| **开发商已破产/进入破产重整** | 须核查破产重整进展、项目是否由管理人继续销售；预售许可状态是否维持 `[待验证破产程序中预售资质规则]` |
| **地方特殊预售规定** | 部分省市对预售许可取得条件（如形象进度）有地方性规定，须核实 `[地方 flagged]` |
| **分期预售项目** | 预售许可分期核发，须确认所购单元所在期次已取得预售许可 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| **开发商拒绝出示五证原件** | 通过官方渠道查询；告知购房人这是合理核查权利 `[购房人侧]`；评估拒绝本身是否为风险信号 |
| **预售许可证信息不完整（无法核实所购单元是否涵盖）** | 标注 🟠 风险，建议通过发证机关或主管部门补充核实 |
| **五证信息部分无法通过官方渠道核验** | 标注 `[信息来源：用户提供 — 需核实]`，建议自行核查；不将未经核验信息作为确定结论 |
| **开发商存在轻微程序瑕疵（如证件格式问题）** | 区分实质性合规问题与程序性格式问题；一般建议开发商补正 |
| **用户无法提供证件信息** | 说明五证核查的重要性，提供查询渠道，不预设五证齐全 |

## 法律依据

### 一、法条类

**（一）城市房地产管理法（商品房预售与现售核心依据）**

《中华人民共和国城市房地产管理法》（全国人大常委会）规定商品房预售须取得预售许可证，具体预售条件（包括缴清土地出让金、取得建设工程规划许可证、完成规定形象进度等）；现售条件另有规定。具体条号 `[待验证]` `[模型知识 — 需验证]`
