# 商品房买卖合同审查编排入口

## 目的

编排四个商品房买卖合同审查子技能，完成从预售许可合规到贷款风险与产权保护的全面商品房审查。律师/法务/购房人代理可以：
- **完整运行全部四步**（按预售许可与主体核查 → 定金与认购书 → 交付与违约 → 贷款解除与风险的顺序）
- **单独触发任一子技能**（如只审查面积误差处理、只分析定金返还、只核查五证）

## 整体审查流程概览

商品房买卖合同审查按以下四个阶段推进，每个阶段由独立子技能负责：

```
[第一步] 预售许可与主体核查
  → 五证是否齐全？开发商资质是否合规？预售许可效力？
  ↓
[第二步] 定金与认购书
  → 认购书性质（预约/本约）？定金vs订金/诚意金？定金罚则如何适用？
  ↓
[第三步] 交付与违约（含逾期交房、逾期办证、面积误差3%）
  → 交付条件是否合规？逾期违约金约定是否公平？面积误差如何处理？
  ↓
[第四步] 贷款解除与风险
  → 按揭不成可否无责解除？网签备案/预告登记是否保护产权？一房二卖风险？
```

## 入口级护栏

### 合同性质确认

开始审查前，确认合同类型：

- **预售合同（期房）**：开发商已取得预售许可证，买受人购买尚未竣工的房屋
- **现售合同（现房）**：房屋已竣工验收合格，买受人购买现房
- **认购书/认购协议**：正式合同前的预约/要约，须先通过 `deposit-subscription` 判定性质
- **二手房买卖合同**：非商品房初次销售，适用规则不同，须提示 `[超出本技能范围，二手房买卖另行分析]`

### 商品房审查立场加载（核心护栏）

> **商品房审查立场是本技能的核心变量。** 同一份商品房合同，购房人侧与开发商侧的审查侧重完全相反。

| 立场 | 审查方向 |
|------|---------|
| **购房人侧** | 重点识别：预售资质瑕疵/五证缺失烂尾风险；逾期交房/办证违约金是否被封顶限制；面积误差超3%时的退款/解除权；按揭不成无责解除条件；网签/预告登记保护 |
| **开发商侧** | 重点核查：确认五证合规；限制过高违约金约定；明确面积误差据实结算标准；约定贷款不成的可归责情形；确认网签备案合规 |
| **中立审查** | 双向识别合同中显失公平条款与双方潜在争议点，均衡呈现购房人与开发商风险 |

### 购房人保护提示

商品房买卖法律规则具有较强的消费者保护色彩，以下规则向购房人倾斜，无论代理立场均须呈现：

- **面积误差超3%**：购房人有权解除合同，且超出3%部分的价款由开发商双倍返还（超出部分），这是法定规则，不得通过格式条款排除
- **逾期交房/办证违约金**：违约金过低时购房人可请求按损失调整（民法典第585条第2款，建议复核）
- **按揭不成无责解除**：因不可归责于双方的事由导致贷款未能办理的，购房人可解除合同（司法解释规则，描述规则，条号待验证）
- **预售许可无效时的保护**：合同无效时买受人有权退款及损失赔偿

### 五证与网签备案核查提醒

每次商品房审查均须提醒：

- 五证核查建议通过官方渠道（当地住建/房产主管部门），不依赖开发商单方声明
- 网签备案应尽快完成（防止一房多售）
- 建议购房人在有条件时及时办理预告登记

### 目的地/特权检查

输出前检查目的地是否在律师-当事人特权圈内。面向对方当事人（开发商）、法院、仲裁机构、媒体等目的地，须提供特权版本（内部使用）和脱敏版本两套选项。

### 地方性规定提示

商品房预售许可管理、限购限贷政策、网签备案规则受地方住建/房产主管部门规章影响较大。当已知项目所在地时，标注：

```
[地方 flagged — 需核实 [省/市] 地方性规定，包括：预售许可条件/限购限贷政策/网签备案规则]
```

### 时效性提示

商品房买卖合同司法解释历经修订，面积误差3%规则/逾期交房办证违约金/按揭解除规则的具体条号易变。每次引用具体条号前须通过最高人民法院官网或法宝核实现行版本：

## 法律依据参考

完整商品房买卖合同审查涉及的核心法律依据（详见各子技能 `## 法律依据`）：

- 《城市房地产管理法》：预售许可/现售条件/网签备案（条号 `[待验证]`）
- 《民法典》建设工程合同章第799条（竣工验收，建议复核）；通则：定金=第586/587/588条/竞合=**第588条**（注意不是585条第2款）/违约金调整=第585条第2款/解除=第563条/情势变更=第533条（均复用商事脊柱已核实结论，建议复核）
- 最高法商品房买卖合同纠纷司法解释：面积误差3%规则/逾期交房办证违约责任/按揭不成解除/一房二卖/预售许可效力（**描述规则，所有具体条号一律 `[待验证 — 司法解释条号易变，须核实现行版本]`**）
- 《城市房地产开发经营管理条例》：开发商资质（条号 `[待验证]`）
