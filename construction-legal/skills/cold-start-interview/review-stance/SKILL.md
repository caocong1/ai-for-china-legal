# 审查与争议立场采集

## 目的

通过结构化访谈，采集用户的整体代理立场、合同审查严格度、争议处理倾向与商品房审查立场，并将结果填充到 `construction-legal/CLAUDE.md` 的 `## 审查与争议立场` 章节（含三个子章节）。

> **审查立场是建设工程审查的核心变量。** 同一份施工合同，发包人侧审查侧重控价/质量/违约追责，承包人侧审查侧重足额结算/签证/优先受偿权——立场不同，结论可能截然相反。强调代理立场是建设工程审查不同于一般合同审查的最重要特征之一。

**可单独触发**：`/construction-legal:review-stance`（只更新审查与争议立场部分），或由 `/construction-legal:cold-start-interview` 编排入口作为第二步调用。

## 访谈问题清单

### 第一部分：整体代理立场（核心问题）

> **代理立场说明**：建设工程审查中，代理立场决定了审查的侧重方向。请告知你在建设工程业务中的整体立场——这将影响所有建设工程技能的默认审查方向。

**Q1.1 — 整体代理立场**

> 你在建设工程合同审查与争议处理中的整体代理立场是什么？

```
选项：
A. 发包人侧（建设单位立场）
   → 关注：承包人资质核查 / 工期质量违约追责 / 结算高估冒算防控 / 合同无效风险防控
B. 承包人侧（施工方立场）
   → 关注：足额结算保障 / 工期顺延签证 / 无效合同折价补偿 / 优先受偿权行使时机
C. 中立审查（双向平衡）
   → 关注：识别合同显失公平条款 / 双向潜在争议点 / 合同缺口
D. 购房人侧（商品房买受方）
   → 关注：预售资质/五证 / 逾期交房违约救济 / 面积误差处理 / 贷款不成解除 / 网签保护
E. 多场景混合（不同项目有不同立场，请说明主要组合及比例）
```

*字段映射*：→ `## 审查与争议立场` > 「整体代理立场」

> 如有特别说明（例如：「绝大多数为承包人侧，偶尔承接发包人方尽职调查」），请说明：

### 第二部分：合同审查严格度

**Q2.1 — 审查严格度**

> 在合同审查中，你倾向于哪种严格度？

```
选项：
A. 保守型 — 任何可能对代理方产生不利后果的条款均标红，优先识别风险
B. 平衡型 — 重大风险标红，一般风险标橙，优先商务对话
C. 进取型 — 仅拦截明显不利条款，一般条款尽量放行
```

*字段映射*：→ `## 审查与争议立场 > ### 合同审查严格度` > 「审查严格度」

**Q2.2 — 各立场保守重点（可选填）**

> 在你的主要代理立场下，有哪些特别需要重点关注（保守标记）的条款类型？

```
发包人侧常见关注点（供参考）：
- 承包人资质核查要求
- 价款调整封口条款（固定总价、结算依据约定）
- 结算逾期答复条款
- 违约金基数与上限

承包人侧常见关注点（供参考）：
- 签证权限与期限约定
- 工期顺延事由的完整性
- 变更程序（口头指令的书面确认）
- 进度款支付条件（避免发包人单方把控）
- 质保金返还期限与条件

购房人侧常见关注点（供参考）：
- 逾期交房/逾期办证违约金封顶条款
- 面积误差条款（是否排除法定规则）
- 商品房预售资质核查
```

*字段映射*：→ `## 审查与争议立场 > ### 合同审查严格度` > 各立场保守重点段落

### 第三部分：争议处理倾向

**Q3.1 — 争议处理优先级**

> 在建设工程争议处理中，你倾向于哪种路径优先？

```
选项：
A. 协商优先 — 先通过书面函件/沟通推动解决
B. 造价鉴定优先 — 争议金额大或结算依据不足时优先申请鉴定
C. 仲裁优先 — 合同有仲裁条款且倾向于非公开程序
D. 诉讼优先 — 倾向于法院判决
E. 视情况 — 按争议类型与金额选择
```

*字段映射*：→ `## 审查与争议立场 > ### 争议处理倾向` > 「争议处理优先级」

**Q3.2 — 优先受偿权行使倾向**

> 对于承包人的工程价款优先受偿权（民法典第807条，建议复核），你的行使倾向是？

```
选项：
A. 积极行使 — 工程款拖欠即评估启动
B. 仅在金额重大时行使
C. 以启动作为催款手段（谈判筹码）
D. 不适用（我方通常是发包人侧）
```

> **提示**：优先受偿权有行使期限限制，具体期限须核实现行司法解释版本 `[待验证 — 司法解释条号易变，须核实现行版本]`。

*字段映射*：→ `## 审查与争议立场 > ### 争议处理倾向` > 「优先受偿权行使倾向」

**Q3.3 — 争议解决机构偏好**

> 你通常偏好哪类争议解决机构？

```
选项：
A. 北京仲裁委（BAC/BIAC）
B. 中国国际经济贸易仲裁委（CIETAC）
C. 当地仲裁机构（请注明：[地区]）
D. 人民法院（诉讼）
E. 视合同约定
```

*字段映射*：→ `## 审查与争议立场 > ### 争议处理倾向` > 「争议解决机构偏好」

### 第四部分：商品房审查立场（适用于含商品房买卖项目）

> **本部分仅在项目类型包含商品房买卖时生效。** 如果你没有商品房业务，可跳过。

**Q4.1 — 商品房审查代理方向**

> 在商品房买卖合同审查中，你主要代理哪一方？

```
选项：
A. 购房人侧（以购房人利益为先）
   → 关注：预售资质/五证合规 / 逾期交房/办证违约金不被封顶限制 / 面积误差超3%时退款/解除 / 按揭不成无责解除
B. 开发商侧（以开发商利益为先）
   → 关注：确认五证合规 / 限制过高违约金 / 明确面积误差据实结算 / 约定贷款不成可归责情形
C. 中立审查（双向均衡）
```

*字段映射*：→ `## 审查与争议立场 > ### 商品房审查立场` > 「商品房审查代理方向」

**Q4.2 — 逾期违约金容忍度**

> 在审查商品房合同的逾期交房/办证违约金条款时，你的容忍度是？

```
选项：
A. 购房人侧严格 — 任何封顶条款均标红
B. 开发商侧宽松 — 可接受约定上限条款
C. 视约定具体数额
```

*字段映射*：→ `## 审查与争议立场 > ### 商品房审查立场` > 「逾期交房/办证违约金容忍度」

**Q4.3 — 面积误差处理倾向**

> 在面积误差问题上，你的处理倾向是？

```
选项：
A. 购房人侧 — 超3%即评估追究（解除或双倍返还超出部分）
B. 开发商侧 — 尽量在误差范围内解决（3%以内据实结算）
C. 中立 — 按面积误差比实际数值分析
```

*字段映射*：→ `## 审查与争议立场 > ### 商品房审查立场` > 「面积误差处理倾向」

**Q4.4 — 退房条件容忍度**

> 对于商品房合同中的退房/解除条件约定，你的立场是？（可口头描述）

*字段映射*：→ `## 审查与争议立场 > ### 商品房审查立场` > 「退房条件容忍度」
