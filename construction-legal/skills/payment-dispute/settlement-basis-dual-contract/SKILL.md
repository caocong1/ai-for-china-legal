# 结算依据与黑白合同

> 可单独触发：`/construction-legal:settlement-basis-dual-contract`
> 也可由 payment-dispute 编排入口调用（工程款争议处理第二步）
> **本子技能须按代理立场（承包人侧争取足额结算 vs 发包人侧主张按约定控价结算）调整侧重**

---

## 目的

系统分析工程款争议中的结算依据认定与黑白合同处理，重点覆盖：

1. **结算依据认定**（合同约定结算方式/送审价与审定价/逾期不答复视为认可条款效力）
2. **黑白合同识别与处理**（备案中标合同 vs 实际履行合同的实质性内容差异识别，以备案中标合同结算）
3. **未签证变更与实际履行**（变更未签证时以实际施工/往来证据/鉴定确定）
4. **造价鉴定的启动与质证衔接**

本子技能迁移并深化了 payment-dispute 原浅骨架「需求收集 + 法律分析」中的结算依据维度。结算金额确定后，如需进一步主张工程价款优先受偿权，请触发 `priority-payment-right` 子技能（注意行使期限）。

## 详细步骤

### 第一步：收集结算相关信息

| 信息项 | 收集内容 | 备注 |
|--------|---------|------|
| 合同基本信息 | 合同价款/计价方式/是否经招标 | 判断是否涉及黑白合同 |
| 结算文件 | 承包人提交的结算报告/发包人审定意见 | 送审价与审定价的差异 |
| 黑白合同情况 | 是否存在备案中标合同与另行签订合同 | 核心判断点 |
| 变更签证 | 已签证的变更汇总/未签证的变更清单 | 承包人举证材料 |
| 逾期不答复条款 | 合同是否有「逾期不答复视为认可结算文件」约定 | 重要结算推进工具 |
| 结算争议金额 | 双方对结算金额的主张差距 | 评估是否需要造价鉴定 |
| 审价机构 | 是否已委托第三方造价机构审价 | 审价结论的证据效力 |

---

### 第二步：结算依据优先级分析

**结算依据优先级判定表：**

| 优先级 | 结算依据类型 | 适用条件 | 备注 |
|--------|---------|---------|------|
| 1 | **备案中标合同（白合同）** | 存在黑白合同情形时，以备案中标合同结算（施工合同司法解释(一) `[待验证]`）| 强制适用，不得用黑合同结算 |
| 2 | **有效施工合同约定结算方式** | 无黑白合同情形，直接适用合同约定（固定总价/单价/成本加酬金）| 主要路径 |
| 3 | **鉴定结论** | 合同约定不明或双方对工程量/价格有重大争议时 | 须申请法院委托鉴定机构 |
| 4 | **无效合同折价补偿** | 施工合同无效但工程经竣工验收合格的，参照合同约定折价补偿（司法解释规则 `[待验证]`）| 无效合同的特殊路径 |

---

### 第三步：黑白合同识别与处理

**黑白合同识别框架：**

> **黑白合同规则（描述规则）**：招标人和中标人就中标的建设工程另行签订的建设工程施工合同，约定的工程价款/工期/质量等实质性内容与备案的中标合同不一致的，以备案中标合同作为结算依据。来源：建设工程施工合同司法解释(一)，具体条号 `[待验证 — 司法解释条号易变，须核实现行版本]`。

**黑白合同识别要素：**

| 识别要素 | 判断标准 | 风险等级 |
|---------|---------|---------|
| **是否经过招标** | 工程属于必须招标范围且实际进行了招标，是黑白合同的前提条件 | - |
| **是否有备案中标合同** | 中标后是否将施工合同在相关主管部门备案（「白合同」）| 🔴 关键判断 |
| **是否另行签订合同** | 是否在备案合同之外另行签订协议/补充协议/内部协议（「黑合同」）| 🔴 关键判断 |
| **实质性内容是否不一致** | 两份合同的工程价款/工期/质量标准等实质性内容是否存在差异 | 🔴 关键（实质性内容认定）|
| **已实际履行哪份合同** | 施工现场实际按哪份合同执行（材料标准/付款节点/价款构成）| 🟠 高（事实认定）|

**「实质性内容」认定（描述规则）：**

> 实质性内容通常包括工程价款、工期和工程质量标准。对于不影响工程基本条款的约定（如付款方式/材料供应方式等细节），是否构成「实质性内容差异」司法实践存在争议 `[待验证]`。建议对一切可能被认定为实质性内容的差异均按黑白合同处理，以备案合同为结算依据。

**按代理立场区分策略：**

- **承包人侧**：若已实际履行价款更高的合同（黑合同），则因结算须以白合同为准，可能不利；应着重主张差价部分以变更签证方式补足，或主张黑合同价款高于白合同部分构成独立变更。
- **发包人侧**：若已实际履行价款较低的合同（实际履行的是黑合同价款更低），则黑白合同规则可能不利（仍以白合同结算）；若发现黑合同价款更高，可积极援引白合同结算规则降低结算金额。

---

### 第四步：逾期不答复视为认可条款分析

**条款效力分析（描述规则）：**

| 约定情形 | 效力规则 | 说明 |
|---------|---------|------|
| **合同明确约定逾期不答复视为认可** | 从其约定；司法解释规定该约定有效（具体规则须核实 `[待验证]`）| 承包人可援引此条款推进结算 |
| **无该约定，承包人提交结算发包人长期未答复** | 司法解释对该情形的处理须核实现行版本 `[待验证]`；法院实践存在差异 | 不能自动视为认可，但发包人拖延可作为结算迟延的证据 |
| **约定的期限已届满，发包人未回复** | 承包人可以送审价作为结算主张；发包人可能被认定为认可（须核实 `[待验证]`）| 操作上须保留发送记录（快递/邮件/签收）|

**操作要点（承包人侧）：**

1. 以书面方式正式提交结算文件（快递/挂号信，保留回执）
2. 在提交函件中明确援引逾期不答复条款，注明到期时间
3. 保留结算文件副本与所有往来记录
4. 期限届满后，及时以函件确认视为认可并主张付款

**发包人侧应对：**

1. 确认公司是否有人员负责在期限内回复结算文件（建立结算审查时限管理机制）
2. 如不认可结算，须在期限内书面提出异议及理由
3. 如合同无该约定，发包人应尽快审查结算避免被认定为拖延

---

### 第五步：未签证变更的实际履行举证

**未签证变更举证清单：**

| 证据类型 | 证明方向 | 实践操作要点 |
|---------|---------|------------|
| 施工图与竣工图对比 | 证明实际施工超出/改变原施工图范围 | 竣工图须经各方确认，差异须清晰标注 |
| 监理日志与施工日志 | 证明施工变更的实际发生 | 关键节点（发包人口头指令/现场勘查记录）须在日志中体现 |
| 往来函件/邮件/微信记录 | 证明发包人同意变更的意思表示 | 微信记录须完整截图，关键对话须有对方姓名与时间 |
| 会议纪要 | 证明各方就变更达成一致 | 须各方签字或盖章 |
| 材料采购凭证 | 证明实际采购的材料规格/数量超出合同清单 | 结合竣工图证明材料用于特定变更 |
| 发包人签字的付款凭证/请款单 | 证明发包人曾认可特定工程量或价款 | 即使未正式签证，付款行为可能视为认可 |
| 造价鉴定结论 | 通过鉴定证明工程量与造价 | 须通过法院委托，单方委托的参考价值有限 |

**举证困难时的处理：**

> 如证据不足以直接证明未签证变更，可在结算文件中完整列明变更内容与造价，并向法院申请委托造价鉴定。鉴定机构将通过现场勘查、资料核查确定工程量，承包人须积极配合并提供现有全部证据材料。

---

### 第六步：造价鉴定启动与质证

**造价鉴定启动条件与时机：**

| 条件 | 说明 |
|------|------|
| 结算争议金额重大 | 双方差距悬殊，无法通过协商解决 |
| 合同约定不明确或存在黑白合同 | 鉴定机构可确定工程量与合理价款 |
| 变更签证不完整 | 鉴定机构可通过现场核查补充 |
| 法院/仲裁庭要求 | 在诉讼/仲裁程序中，鉴定通常由法院/仲裁庭委托 |

**造价鉴定质证要点：**

- 核查鉴定机构资质（须为具有工程造价咨询资质的机构）
- 审查鉴定依据（鉴定机构采用的计价定额/鉴定基准日）
- 逐条质疑鉴定报告中不合理的工程量认定或单价采用
- 对鉴定结论有异议的，须书面提出并申请说明理由
- 读取 `construction-legal/CLAUDE.md` 中 `## 可用集成` 的造价鉴定机构信息，确认是否已委托

## 边界条件

| 边界情形 | 处理方式 |
|---------|---------|
| 多份合同并存（补充协议 + 备案合同 + 另行签订协议）| 逐份分析各合同的形成时间、是否通过招标、内容是否属于实质性内容，判断以哪份合同结算；如均无备案则判断最终有效合同 |
| 补充协议与备案合同的关系 | 补充协议如对备案合同实质性内容作了变更，是否属于黑白合同中「另行签订合同」须具体分析；价款调整类补充协议风险尤高 |
| 招标范围外工程的结算 | 超出招标范围的新增工程按另行签订合同结算，无黑白合同问题，但须确认合同授权范围 |
| 合同无效时的结算路径 | 无效合同参照合同约定折价补偿（需工程经竣工验收合格），鉴定在无效合同路径中更常见 |
| 发包人不配合结算（长期拖延审核）| 分析逾期不答复条款；如无约定，评估诉讼/仲裁路径，通过法院委托鉴定解决；保留催告记录 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 结算依据无法确定（合同文本矛盾或缺失）| 标🔴/🟠风险，建议申请造价鉴定，告知鉴定所依据的规则 |
| 黑白合同情形下以黑合同结算会不利于当事人 | 解释法律规则（须以白合同结算），提出差价弥补策略（变更签证/独立工程款主张）|
| 结算文件提交未留证（无快递/签收记录）| 告知举证风险，建议通过公证处留证重新提交，或以邮件方式重新正式提交 |
| 代理立场未明确 | 显示临时模式提示，以中立模式继续分析 |
