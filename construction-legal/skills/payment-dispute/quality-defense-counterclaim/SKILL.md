# 质量抗辩与反索赔

> 可单独触发：`/construction-legal:quality-defense-counterclaim`
> 也可由 payment-dispute 编排入口调用（工程款争议处理第三步）
> **本子技能须按代理立场（发包人侧组织质量抗辩与反索赔 vs 承包人侧反驳质量抗辩并主张工程款）调整侧重**

---

## 目的

系统分析工程款争议中的质量抗辩与反索赔机制，重点覆盖：

1. **发包人质量抗辩**（以工程质量不合格拒付或少付工程款的成立条件与举证）
2. **质量不合格工程款处理**（经验收不合格的修复/减价/不予支付规则）
3. **反索赔**（发包人对承包人的工期延误违约金、质量缺陷修复费、第三方损失等）
4. **质量责任举证**（谁主张谁举证原则 + 鉴定的启动与运用）

本子技能迁移并深化了 payment-dispute 原浅骨架「法律分析」中的质量与反请求维度。如发现质量问题导致工程价款实质减少，应与 `settlement-basis-dual-contract` 联动（调整结算依据），并评估是否影响 `priority-payment-right` 中优先受偿权的主张金额。

## 详细步骤

### 第一步：收集质量与反索赔信息

| 信息项 | 收集内容 | 备注 |
|--------|---------|------|
| 质量问题描述 | 缺陷类型/部位/程度（地基基础/主体结构/装饰装修/设备安装等）| 判断质量缺陷严重程度 |
| 验收情况 | 是否经竣工验收/分部分项验收/是否出具验收合格证明 | 影响质量抗辩成立 |
| 修复情况 | 是否已修复/修复后是否合格/修复费用 | 修复后合格的，扣减修复费用 |
| 使用情况 | 发包人是否已擅自使用工程 | 发包人提前使用可能视为认可质量 |
| 质量鉴定 | 是否已进行质量鉴定/鉴定机构/鉴定结论 | 质量争议时鉴定为关键证据 |
| 反索赔项目 | 工期延误违约金/质量缺陷修复费/第三方损失/停工损失等 | 发包人反索赔项目 |
| 工期延误 | 是否有工期延误/延误原因（承包人责任/发包人原因/不可抗力）| 区分责任归属 |

---

### 第二步：质量抗辩成立要件分析

**质量抗辩成立要件核对表：**

| 要件 | 判断标准 | 满足情况 | 说明 |
|------|---------|---------|------|
| **质量问题客观存在** | 须有客观的质量缺陷，而非主观不满；建议经质量鉴定证明 | 待核查 | 举证责任在主张质量抗辩的一方（通常为发包人）|
| **质量缺陷系承包人原因** | 缺陷系承包人施工原因造成，而非设计问题/发包人提供材料问题/不可抗力 | 待核查 | 责任归属须专项分析 |
| **工程未验收合格或验收不合格** | 未经验收即主张工程款的质量抗辩，或验收时已标注质量问题 | 待核查 | 已通过竣工验收后质量抗辩成立门槛更高 |
| **发包人未擅自使用** | 发包人未在验收前擅自使用工程（否则不得以质量问题主张工程款抗辩）| 待核查 | 民法典第799条建议复核 |

**特别注意**：

> 根据民法典第799条（建议复核），发包人提前使用工程的，不得以该理由主张工程质量问题。因此发包人在行使质量抗辩时，须确认自身未擅自提前使用工程。

---

### 第三步：质量不合格工程款处理规则

**质量不合格工程款处理决策表（描述规则）：**

| 情形 | 处理规则（描述规则） | 依据 | 验证状态 |
|------|-----------------|------|---------|
| **竣工验收不合格，承包人拒绝修复** | 发包人可就此抗辩工程款请求；不予支付至修复完毕 | 施工合同司法解释(一) | `[待验证]` |
| **竣工验收不合格，修复后合格** | 可就修复费用从工程款中扣减；承包人可获得工程款减修复费后的余额 | 施工合同司法解释(一) | `[待验证]` |
| **竣工验收不合格，修复后仍不合格** | 发包人可拒绝支付工程款（或仅支付对应可用价值的工程款）| 施工合同司法解释(一) | `[待验证]` |
| **已通过竣工验收后发现质量问题** | 属于保修责任范畴（须区分保修与工程款抗辩）；发包人可主张修复费用赔偿，但通常不能以此拒付工程款 | 民法典建设工程合同章 + 施工合同司法解释 | `[待验证]` |
| **验收时已标注质量问题，承包人认可整改** | 可将整改费用与工程款结算中扣抵 | 合同约定 + 司法解释规则 | `[待验证]` |

**与保修责任的区分：**

> **重要区分**：在竣工验收合格并交付后发现的质量问题，属于保修责任（承包人有义务在保修期内修复，但发包人不得以此拒付工程款）。工程款质量抗辩主要适用于**竣工验收阶段**发现的质量问题，而非交付使用后的保修事项。

---

### 第四步：反索赔分析

**发包人反索赔项目清单与抵销表：**

| 反索赔项目 | 成立要件 | 计算方式 | 与工程款的关系 |
|---------|---------|---------|------------|
| **工期延误违约金** | 须证明工期延误系承包人原因（排除发包人原因、签证工期顺延、不可抗力）| 合同约定的工期延误违约金/日 × 延误天数 | 可与工程款主张相抵销 |
| **质量缺陷修复费** | 须证明缺陷系承包人责任，已实际发生修复费用 | 实际修复费用或鉴定费用 | 可从工程款中扣减 |
| **第三方损失** | 因承包人质量/工期问题导致发包人向第三方赔偿的损失（如逾期交付商品房等）| 实际赔偿金额，须有裁判文书或协议 | 可反索赔，但须经诉讼/仲裁确认 |
| **停工损失** | 因承包人原因造成发包人停工损失（较少见）| 具体核算 | `[待验证]` |
| **保证金/质保金扣除** | 在质保期内承包人拒绝修复的质量缺陷修复费用 | 从质保金中扣除 | 合同约定 |

**工期延误责任归属分析：**

| 延误事由类型 | 责任归属 | 承包人可否主张工期顺延 |
|------------|---------|-----------------|
| 承包人原因（施工效率/材料供应不及时等）| 承包人 | 否，发包人可主张延误违约金 |
| 发包人原因（图纸缺陷/付款拖延/甲供材延迟/变更频繁）| 发包人 | 是，承包人可主张工期顺延 |
| 不可抗力（自然灾害/疫情/战争等）| 双方免责 | 是，工期顺延 |
| 政府原因（政策调整/停工令等）| 须具体分析 | 通常可主张顺延 |
| 监理方问题 | 须具体分析（监理由发包人委托，其原因通常视为发包人原因）| 通常可主张顺延 |

---

### 第五步：质量举证与鉴定

**质量鉴定启动条件：**

| 条件 | 说明 |
|------|------|
| 双方对质量问题存在争议 | 发包人主张有质量缺陷，承包人否认 |
| 质量问题已客观存在 | 已显现，可进行现场勘查 |
| 鉴定对结案有实质意义 | 质量认定影响工程款计算 |

**质量举证责任分配：**

| 情形 | 举证责任 | 实践要点 |
|------|---------|---------|
| 发包人主张质量抗辩（拒付/少付工程款）| **发包人举证**质量缺陷的存在与承包人责任 | 须提供质量检测报告/鉴定结论 |
| 承包人主张已通过验收、质量合格 | 承包人举证验收记录/合格证明 | 竣工验收报告/监理验收意见 |
| 双方对质量问题成因有争议 | 法院通常委托鉴定机构确定 | 积极配合鉴定，提供相关材料 |

**鉴定机构资质核查：**

- 工程质量鉴定须委托具备相应资质的检测机构
- 单方委托鉴定的结论仅有参考价值，须经对方质证才有证据效力
- 法院/仲裁庭委托的鉴定结论通常具有较高证据效力

---

### 第六步：按代理立场区分结论

**发包人侧（组织质量抗辩与反索赔）：**

1. 全面收集质量问题证据（检测报告/监理记录/整改通知）
2. 核查自身是否存在提前使用工程的情形（避免丧失质量抗辩权）
3. 区分验收阶段质量抗辩与交付后保修责任，准确适用
4. 计算工期延误违约金（排除非承包人原因的延误天数）
5. 整理反索赔金额与工程款主张的抵销关系
6. 如质量缺陷严重，评估诉讼/仲裁中申请造价+质量联合鉴定的可行性

**承包人侧（反驳质量抗辩，维护工程款权益）：**

1. 核查竣工验收记录：如已通过验收，发包人提出质量抗辩的成立难度较大
2. 分析质量问题责任归属（设计缺陷/发包人提供材料质量/正常使用损耗）
3. 积极推进修复（修复后合格可主张修复前的全部工程款减修复费）
4. 核查发包人是否存在提前使用（存在则发包人丧失质量抗辩权）
5. 准备工期顺延证据（反驳工期延误违约金主张）
6. 评估优先受偿权在质量争议时的行使策略（注意行使期限）

## 边界条件

| 边界情形 | 处理方式 |
|---------|---------|
| 已交付使用视为认可质量 | 分析发包人是否提前使用（民法典第799条建议复核）；如提前使用则发包人丧失质量抗辩权，但保修责任不受影响 |
| 地基基础与主体结构终身责任 | 即使在保修期外，地基基础与主体结构质量问题仍涉及终身责任（建设工程质量管理条例 `[待验证]`），影响责任认定但不影响工程款质量抗辩的时效 |
| 发包人提供材料或施工设备导致的质量问题 | 责任归属于发包人，承包人可以此反驳质量抗辩 |
| 设计缺陷导致的质量问题 | 责任归属于设计单位（或代发包人委托设计的责任），承包人可反驳质量抗辩；发包人须向设计单位另行主张 |
| 质量问题与工期延误并发 | 须分别核算；注意工期延误违约金计算是否已排除质量缺陷修复时间 |
| 保修期内承包人拒绝修复 | 发包人可委托第三方修复并从质保金中扣除；超出质保金的部分可向承包人主张赔偿 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 质量鉴定结论与合同约定标准不一致 | 分析合同约定标准是否高于或低于国家标准；合同约定低于国家强制标准的无效 |
| 发包人与承包人均主张对方原因导致质量问题 | 建议启动第三方联合鉴定；诉讼/仲裁程序中法院通常委托鉴定 |
| 反索赔金额超过工程款 | 超出部分承包人须另行赔偿，评估承包人实际偿付能力与执行风险 |
| 代理立场未明确 | 显示临时模式提示，以中立模式提供双向分析 |
