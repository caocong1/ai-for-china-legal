# 法律依据引用规范（建设工程版）

> 本文件定义 construction-legal 插件所有技能统一使用的法律依据引用规范。
> 后续计划（07-02、07-03）直接引用本规范，避免各技能重复维护。
> **建设工程施工合同司法解释经历(一)(二)整合与历次修订，具体条号变更频繁，本规范特别强调司法解释条号保守标注原则。**

---

## 一、来源分类

所有法律依据必须归入以下类别之一，并在引用时明确标注类别标签。

### 1. 法律（全国人大及其常委会制定）

指全国人民代表大会及其常务委员会制定的法律，建设工程语境中主要包括：

- 《中华人民共和国民法典》（2020年5月28日通过，2021年1月1日施行）合同编第三编第十八章「建设工程合同」第788-808条
- 《中华人民共和国民法典》合同编通则部分（违约责任/解除/担保等）
- 《中华人民共和国建筑法》（全国人大常委会制定，历经修正）
- 《中华人民共和国招标投标法》（全国人大常委会制定，历经修正）
- 《中华人民共和国城市房地产管理法》（全国人大常委会制定，历经修正）

**法律层级最高，是建设工程法律分析的核心基础。**

> **建设工程法律引用特别提示（必须遵守）**：民法典建设工程合同章（第788-808条）为法典化成文法，整章条文相对稳定；建筑法、招标投标法、城市房地产管理法均经历历次修正，**凡涉及上述法律中具体分则条号（除已核实中度把握锚点外），一律标 `[待验证]`，描述规则呈现，不臆造条号。**

**格式要求（法律最小引用格式）：**

```
《[法律全名]》第[X]条[第X款]  [来源标签]  [验证状态]
```

示例（中度把握锚点）：
```
《中华人民共和国民法典》第788条  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
```

示例（待验证条号）：
```
《中华人民共和国建筑法》第[X]条  `[模型知识 — 需验证]`  `[待验证]`
```

### 2. 司法解释（最高人民法院发布）

指最高人民法院发布的解释性文件，建设工程语境中主要包括：

- 《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释(一)》（经历与解释(二)整合，历次修订，**具体条号一律标 `[待验证]`**）
- 《最高人民法院关于审理商品房买卖合同纠纷案件适用法律若干问题的解释》（历次修订，**具体条号一律标 `[待验证]`**）
- 《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》（2023年发布，**具体条号标 `[待验证]`**）
- 《最高人民法院关于适用〈中华人民共和国民法典〉有关担保制度的解释》（**具体条号标 `[待验证]`**）

**核心纪律（必须遵守）**：

> **司法解释的具体条号一律标 `[待验证]`，以「描述规则 + 待验证」呈现，不写裸条号。** 规则内容（如黑白合同备案结算、无效合同折价补偿、优先受偿权范围与期限、面积误差3%处理规则）可完整描述，但来源条号须待验证。

**格式要求：**

```
《[司法解释全名]》（最高人民法院）规定：[描述规则]  [来源标签]  `[待验证]`
```

示例：
```
《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释(一)》（最高人民法院）规定：实际履行的施工合同与备案中标合同实质性内容不一致的，以备案中标合同作为结算依据  `[模型知识 — 需验证]`  `[待验证 — 司法解释条号易变，须核实现行版本]`
```

### 3. 行政法规（国务院制定）

指国务院制定发布的行政法规，建设工程语境中主要包括：

- 《建设工程质量管理条例》（国务院发布，**具体条号 `[待验证]`**）
- 《建设工程安全生产管理条例》（国务院发布，**具体条号 `[待验证]`**）
- 《招标投标法实施条例》（国务院发布，**具体条号及规模标准 `[待验证]`**）
- 《城市房地产开发经营管理条例》（国务院发布，**具体条号 `[待验证]`**）

**核心纪律（必须遵守）**：

> **行政法规的具体条号、规模标准数值（如必须招标的金额阈值）、期限阈值，一律标 `[待验证]`，以描述规则呈现，绝不硬写裸条号或具体数字。**

**格式要求：**

```
《[行政法规全名]》（国务院，[发布年份]）[描述规则]  [来源标签]  `[待验证]`
```

### 4. 部门规章与规范性文件（住建部/发改委等制定）

指国务院各部门制定的规章或规范性文件，建设工程语境中主要包括：

- 《必须招标的工程建设项目规定》（国家发展改革委发布，**具体规模标准 `[待验证]`**）
- 工程总承包管理相关规章（住房和城乡建设部，**具体条号 `[待验证]`**）
- 建筑业企业资质管理相关规章（住房和城乡建设部，**具体资质等级标准 `[待验证]`**）
- 建筑工程施工许可管理办法（住房和城乡建设部，**具体条号 `[待验证]`**）

**核心纪律（必须遵守）**：

> **部门规章的具体条号、具体数值、具体期限，一律标 `[待验证]`，以描述规则呈现。**

**格式要求：**

```
《[规章全名]》（[制定机关]，[年份]）[描述规则]  [来源标签]  `[待验证]`
```

### 5. 案例（指导性案例与建设工程典型案例）

建设工程领域可参考的案例类型：

- **最高院指导性案例**：最高人民法院发布，对下级法院有参照作用，仅作参考，**不具判例法约束力**
- **最高院建设工程典型案例**：最高人民法院或高院发布的典型案例，实践参考价值较高，**非判例法**
- **中院/高院建设工程裁判案例**：各地中院或高院的建设工程判决，参考价值有限

**核心纪律（必须遵守）**：

> **所有案例案号一律标 `[待验证]`，未经检索核验不得移除该标注。** 中国采成文法体系，案例不具判例法约束力，须明确说明。

**格式要求：**

```
[案例名称或案号 `[待验证]`]  [裁判要点摘要]  [法院层级]  参考，不具判例法约束力  `[待验证]`
```

### 6. 学说（学术观点）

学说为辅助参考，存在争议性，引用时须标注来源和争议性：

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息]  "[观点摘要]"  [是否有争议：有/无]  `[待验证]`
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用五个标准标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站（全国人大/住建部/最高法） | 官网原始文件 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查；司法解释条号风险尤为突出 |
| `[用户提供]` | 来自用户提供的文件 | 合同文本、施工许可证、招标文件等 |

**标签使用规则：**

1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 来源在依赖前**必须**对照原始来源检查——建设工程司法解释条号变更频繁，风险尤为突出
4. 来源标签不替代 `[待验证]` 标注——两者可同时出现在一行

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` 的条号 | 模型回忆的条文号风险高 | 国家法律法规数据库、住建部官网 |
| **司法解释的任何具体条号** | 建设工程施工合同司法解释经历(一)(二)整合与历次修订，条号易变 | 最高人民法院官网、法宝核实现行版本 |
| 行政法规具体条号及规模标准金额 | 行政法规修订后条号与数值可能变化 | 国家法律法规数据库 |
| 部门规章具体条号、资质标准数值 | 部门规章更新频繁 | 住建部官网、国家发改委官网 |
| 建筑法/招标投标法/城市房地产管理法分则条号（除已核实中度把握锚点外） | 上述法律历次修正 | 国家法律法规数据库 flk.npc.gov.cn |
| 距最后更新超过 6 个月的法规 | 法规可能已修订或废止 | 查询官方数据库最新版本 |
| 用户口述的条款号、期限阈值、金额 | 可能记忆偏差或引用旧版 | 核查官方法规全文 |
| 所有案例引用（默认） | 建设工程诉讼案例多，案号极易错误回忆 | 中国裁判文书网、最高法官网 |

**待验证标记格式（示例）：**

```
《最高人民法院关于审理建设工程施工合同纠纷案件适用法律问题的解释(一)》第[X]条  `[模型知识 — 需验证]`  `[待验证 — 司法解释条号易变，须核实现行版本]`
```

## 四、司法解释具体条号保守标注

> **这是建设工程语境最重要的纪律之一，必须严格遵守。**

### 建设工程司法解释修订说明

| 司法解释 | 修订情况 | 条号风险 |
|---------|---------|---------|
| 《最高法建设工程施工合同纠纷司法解释》 | 原(一)(二)两部整合，历次修订，现行版本条号与旧版差异大 | 极高 — 条号易变，引用必须核实现行版本 |

**中度把握锚点（须标「已核实锚点（中度把握），建议复核」）：**

以下条号系中度把握，引用时须明确标注「已核实锚点（中度把握），建议复核」，且不得以此为由臆造其他条号：

| 锚点 | 法律依据 | 规则摘要 |
|------|---------|---------|

| 可预见性损失范围 | 《民法典》**第584条** | 损失赔偿范围限于合同订立时应预见到的损失 |
| 法定解除权 | 《民法典》**第563条** | 不可抗力、预期违约、迟延履行、根本违约等情形下的法定解除权 |
| 情势变更 | 《民法典》**第533条** | 合同成立后客观情况发生重大变化，继续履行明显不公平，可请求变更或解除 |

| 连带责任保证 | 《民法典》**第686条** | 保证方式约定不明时，按一般保证处理；连带责任保证须明确约定 |

**规则类（来自司法解释，描述规则即可，不写裸条号）：**

| 规则 | 规则摘要 | 条号状态 |
|------|---------|---------|
| 黑白合同备案结算 | 实际履行的施工合同与备案中标合同实质性内容不一致的，以备案中标合同作为结算依据 | 描述规则，条号标 `[待验证]` |
| 无效合同折价补偿 | 施工合同无效但工程经竣工验收合格的，可参照合同约定折价补偿承包人 | 描述规则，条号标 `[待验证]` |
| 优先受偿权范围与期限 | 优先受偿权范围以工程造价为限（不含违约金等）；行使期限自合同约定的竣工日期或实际竣工日期起算，具体期限须核实现行版本 | 描述规则，期限数值标 `[待验证]` |
| 面积误差3%规则 | 面积误差比绝对值在3%以内的据实结算；超出3%的，买受人可解除合同，超出3%部分的价款有特别处理规则 | 描述规则，条号标 `[待验证]` |
| 逾期交房/办证违约金调整 | 逾期交房或逾期办证的违约金适用调整规则 | 描述规则，具体条号标 `[待验证]` |

**保守原则（硬性要求）：**

> **不确定即标待验证，不臆造。** 除上述中度把握锚点外，建设工程合同分则（第789-808条逐条细则）、建筑法/招标投标法/城市房地产管理法分则条号，以及所有司法解释具体条号，**一律以「描述规则 + 待验证」呈现，绝不硬写无把握的条号。** 建设工程争议金额大，条号错误会直接误导优先受偿权行使期限、黑白合同结算依据等关键法律结论。

## 五、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：建设工程法律分析的所有结论，必须首先援引现行成文法（民法典建设工程合同章/建筑法/招标投标法/城市房地产管理法及相关法规）作为规范依据
2. **不得以案例替代法条**：即使存在明确的典型案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下司法解释/典型案例/学说」

### 案例的辅助定位

1. **最高院指导性案例**：仅作参考，标注「最高院指导，参照意义强，不具判例法约束力」
2. **最高院建设工程典型案例**：实践参考价值较高，标注「建设工程典型案例，参照意义强，不具判例法约束力」
3. **中院/高院建设工程案例**：参考价值有限，标注「地方法院案例，参照」
4. **明确说明非判例法**：中国采成文法体系，案例不具判例法约束力

## 六、法院层级与机构定位

### 法院层级（建设工程案件）

| 层级 | 说明 | 建设工程特点 |
|------|------|------------|
| 最高人民法院 | 全国最高审判机关 | 指导性案例与司法解释发布 |
| 高级人民法院 | 省级，建设工程纠纷二审 | 部分高院有建设工程专项指导意见 |
| 中级人民法院 | 地市级，**建设工程争议金额大，通常由中院一审** | 建设工程合同纠纷、工程款纠纷一审主力法院 |
| 基层人民法院 | 县区级，小额建设工程纠纷 | 金额较小或简易程序案件 |
| 建设工程仲裁机构 | 中国国际经济贸易仲裁委员会等 | **建设工程纠纷仲裁亦常见，约定仲裁条款者须走仲裁程序** |

| 机构 | 主要职能 | 建设工程业务 |
|------|---------|------------|
| 住房和城乡建设部（住建部） | 建筑业综合管理 | 施工许可制度、建筑业企业资质管理、工程质量与安全监管 |
| 地方住建主管部门 | 地方建筑业管理 | 地方施工许可、资质核发与撤销、工程质量监督 |
| 工程质量监督机构 | 工程质量监督 | 工程质量强制性标准执行监督 |
