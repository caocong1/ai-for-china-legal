# 安装后配置（post-install-configuration）

> **所属编排**：`legal-builder-hub/skills/installation/SKILL.md`（安装向导）
> **分工说明**：本子技能负责安装后的首次配置引导。安装执行由 `installation-execution` 前置处理；兼容性检查由 `compatibility-check` 处理；更新卸载由 `update-uninstall` 处理。
> **可单独触发**：`/legal-builder-hub:post-install-configuration`

---

## 目的

在技能安装成功后，引导用户完成技能的首次配置，包括：API 密钥或凭证配置、技能偏好设置、外部集成连接、功能验证测试，确保技能可以正常运行。

---

## 前置

1. **技能已成功安装** — 确认 `installation-execution` 已完成且验证通过
2. **读取技能配置说明** — 从 SKILL.md 或技能附带的 CONFIG.md 获取配置要求
3. **读取 `legal-builder-hub/CLAUDE.md`** — 获取：用户的集成工具状态、技术背景
4. 本子技能产出是配置完成的技能，可以正常调用

---

## 配置流程

### 步骤一：配置需求识别

读取技能的配置要求清单：

```markdown
### 配置需求

**技能**：[名称] v[版本]

**必需配置**（不配置无法运行）：
- [ ] [配置项名称]：[说明]
- [ ] [配置项名称]：[说明]

**可选配置**（影响功能范围）：
- [ ] [配置项名称]：[说明]（默认值：[默认]）

**集成连接**（扩展功能）：
- [ ] [集成名称]：[说明]
```

### 步骤二：逐项配置引导

对于每个配置项，提供：

1. **说明**：这个配置项是什么，为什么需要
2. **获取方式**：如何获取配置值（如 API 密钥的申请地址）
3. **输入**：接受用户输入的配置值
4. **验证**：验证配置值是否有效
5. **存储**：将配置值写入安全存储

**配置安全规则**：

- API 密钥等敏感信息不得明文写入日志
- 配置值存储在加密的本地配置文件中
- 配置变更须记录审计日志（不含敏感值本身）

### 步骤三：集成连接测试

对于需要外部集成的配置：

```markdown
### 集成连接测试

| 集成 | 连接状态 | 测试结果 | 说明 |
|------|---------|---------|------|
| [集成名称] | [已连接/未连接] | [✅ 成功/❌ 失败] | [说明] |
```

连接失败时的排查建议：
- 检查网络连接
- 检查凭证有效性
- 检查防火墙/代理设置
- 检查集成服务端状态

### 步骤四：功能验证

运行技能的基础功能测试：

```markdown
### 功能验证

| 测试项 | 结果 | 说明 |
|-------|------|------|
| 技能加载 | ✅ / ❌ | 技能是否可以正确加载 |
| 基础功能 | ✅ / ❌ | 核心功能是否正常运行 |
| 集成功能 | ✅ / ❌ | 集成连接是否正常工作 |
| 输出格式 | ✅ / ❌ | 输出格式是否符合预期 |
```

### 步骤五：配置完成确认

```markdown
## 配置完成

**技能**：[名称] v[版本]
**配置日期**：[日期]

### 配置摘要

| 配置项 | 状态 |
|-------|------|
| [配置项] | ✅ 已配置 |
| [配置项] | ✅ 已配置（使用默认值） |
| [配置项] | ⏭️ 已跳过（可稍后配置） |

### 集成状态

| 集成 | 状态 |
|------|------|
| [集成名称] | ✅ 已连接 / ⏭️ 未连接 |

### 功能验证

全部通过：[✅ 是 / ❌ 否]

### 下一步

1. **开始使用** — 技能已就绪，可通过 `/legal-builder-hub:[skill-name]` 调用
2. **修改配置** — 重新运行本子技能修改配置
3. **查看文档** — 查看技能的详细使用文档
```

## 配置安全

| 安全措施 | 说明 |
|---------|------|
| 敏感信息加密存储 | API 密钥等凭证不得明文存储 |
| 配置审计日志 | 记录配置变更时间和操作人，不记录值本身 |
| 配置备份 | 配置变更前自动备份旧配置 |
| 权限控制 | 配置文件权限设为仅当前用户可读写 |

---

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 技能无需配置 | 跳过配置步骤；直接进行功能验证 |
| 配置值无效 | 提示错误原因；引导重新输入 |
| 集成连接失败 | 提供排查建议；允许跳过集成（标注功能受限） |
| 用户暂时不想配置 | 保存配置进度；下次可继续配置 |
