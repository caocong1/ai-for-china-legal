# 数据隐私评估（data-privacy-assessment）

> **所属编排**：`legal-builder-hub/skills/security-review/SKILL.md`（安全审查）
> **分工说明**：本子技能负责数据处理行为的隐私合规评估。静态分析由 `static-analysis-scan` 处理；权限审查由 `permission-scope-review` 处理；报告认证由 `review-report-certification` 处理。
> **可单独触发**：`/legal-builder-hub:data-privacy-assessment`

---

## 目的

评估技能在运行过程中的数据处理行为是否符合《个人信息保护法》（PIPL）、《数据安全法》（DSL）等法律要求，包括：数据收集范围、处理目的和方式、数据存储和保留、数据外传行为、用户权利保障。

---

## 前置

1. **获取技能文档** — SKILL.md、隐私政策（如有）、数据处理声明（如有）
2. **静态分析结果** — 从 `static-analysis-scan` 获取代码中的数据流分析线索
3. **权限审查结果** — 从 `permission-scope-review` 获取权限范围信息
4. **读取 `legal-builder-hub/CLAUDE.md`** — 获取：数据隐私要求、敏感权限零容忍列表
5. 本子技能产出是数据隐私评估报告，**不修改技能代码**

---

## 评估维度

### 1. 数据收集评估

| 评估项 | 检查内容 | 合规标准 |
|-------|---------|---------|
| 收集范围 | 技能收集哪些数据 | 须明确声明收集的数据类型 |
| 收集目的 | 为什么收集这些数据 | 目的须具体、明确、合理 |
| 最小必要 | 收集范围是否超出功能所需 | 仅收集实现功能所必需的数据 |
| 合法性基础 | 收集的合法性基础是什么 | PIPL 第13条七类基础之一（已核实） |

### 2. 数据处理评估

| 评估项 | 检查内容 | 合规标准 |
|-------|---------|---------|
| 处理方式 | 如何处理收集的数据 | 处理方式须与声明目的相符 |
| 处理范围 | 数据处理是否超出声明范围 | 不得超出声明的处理范围 |
| 自动化决策 | 是否使用数据进行自动化决策 | PIPL 第24条：须保证透明度和公平性（已核实） |
| 敏感数据处理 | 是否处理敏感个人信息 | PIPL 第28-29条：须单独同意（已核实） |

### 3. 数据存储评估

| 评估项 | 检查内容 | 合规标准 |
|-------|---------|---------|
| 存储位置 | 数据存储在何处 | 须声明存储位置（本地/云端/第三方） |
| 存储期限 | 数据保留多长时间 | 须声明保留期限，到期须删除 |
| 存储安全 | 存储过程的安全措施 | 加密/访问控制/审计日志 |
| 数据本地化 | 是否涉及数据本地化义务 | CIIO 和达量处理者须境内存储（PIPL 第40条，已核实） |

### 4. 数据外传评估

| 评估项 | 检查内容 | 合规标准 |
|-------|---------|---------|
| 外传行为 | 是否向外部发送数据 | 须明确声明外传行为和目标 |
| 外传对象 | 数据发送给谁 | 须声明接收方身份 |
| 外传目的 | 为什么外传数据 | 目的须具体、合理 |
| 跨境传输 | 是否涉及跨境数据传输 | PIPL 第38-39条：须走合规路径+单独同意（已核实） |

### 5. 用户权利保障评估

| 评估项 | 检查内容 | 合规标准 |
|-------|---------|---------|
| 知情权 | 是否告知用户数据收集情况 | PIPL 第44条（已核实） |
| 查阅/复制权 | 用户能否查阅自己的数据 | PIPL 第45条（已核实） |
| 更正权 | 用户能否更正不准确的数据 | PIPL 第46条（已核实） |
| 删除权 | 用户能否请求删除数据 | PIPL 第47条（已核实） |
| 撤回同意 | 用户能否撤回同意 | PIPL 第14条（已核实） |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 技能无隐私政策 | 标注为 🟠 高风险；须要求开发者提供隐私声明 |
| 数据流向不明确 | 结合静态分析结果推断数据流；标注不确定项 |
| 第三方 SDK 数据处理 | 须评估第三方 SDK 的数据处理行为 |
| 加密数据传输 | 加密传输仍须评估接收方合规性 |

## 法律依据

| 法律依据 | 核心规则 | 适用场景 |
|---------|---------|---------|
| 《个人信息保护法》（2021年）第13-14条 | 合法性基础与同意规则 | 数据收集合法性 |
| 《个人信息保护法》（2021年）第28-29条 | 敏感个人信息保护 | 敏感数据处理 |
| 《个人信息保护法》（2021年）第38-39条 | 跨境提供合规路径 | 数据跨境评估 |
| 《个人信息保护法》（2021年）第44-47条 | 个人权利保障 | 用户权利评估 |
| 《个人信息保护法》（2021年）第55-56条 | PIA 义务 | 影响评估 |
| 《数据安全法》（2021年）第21条 | 数据分类分级 | 数据分级评估 |
