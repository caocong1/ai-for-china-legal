# 技能搜索与浏览（skill-search-browse）

> **所属编排**：`legal-builder-hub/skills/skill-discovery/SKILL.md`（技能发现）
> **分工说明**：本子技能负责执行搜索查询和返回结果列表。分类筛选由 `category-tag-filtering` 处理；排序推荐由 `recommendation-ranking` 处理；详情评估由 `skill-detail-evaluation` 处理。
> **可单独触发**：`/legal-builder-hub:skill-search-browse`

---

## 目的

在技能注册表中执行搜索查询，支持关键词搜索和分类浏览两种发现模式，返回匹配技能列表。搜索结果须标注安全审查状态、许可证类型和兼容性信息，帮助用户快速定位候选技能。

---

## 前置

1. **读取 `legal-builder-hub/CLAUDE.md`** — 获取：关注的法律领域、是否接受未审查技能、推荐排序偏好
2. **读取 `legal-builder-hub/skills/_shared/practice-profile-schema.md`** — 字段映射
3. **连接注册表** — 确定注册表类型（本地/组织/公共）
4. 本子技能产出是搜索结果列表，**不执行安装或修改操作**

---

## 搜索模式

### 模式一：关键词搜索

用户输入关键词或自然语言描述，在以下维度中匹配：

| 搜索维度 | 权重 | 说明 |
|---------|------|------|
| 技能名称 | 高 | 精确匹配或模糊匹配技能名称 |
| 技能描述 | 高 | 匹配 SKILL.md 中的 description 字段 |
| 法律领域标签 | 中 | 匹配技能标注的法律领域分类 |
| 能力类型标签 | 中 | 匹配技能标注的能力类型分类 |
| 功能关键词 | 低 | 匹配 SKILL.md 正文中的关键词 |

### 模式二：分类浏览

用户按分类层级逐级浏览：

```
法律领域
├── 商事合同
│   ├── 合同审查
│   ├── 合同起草
│   └── 履约管理
├── 诉讼仲裁
│   ├── 案件管理
│   ├── 证据整理
│   └── 文书起草
├── 数据合规
│   ├── PIA 生成
│   ├── 出境评估
│   └── 合规检查
├── 知识产权
│   ├── 商标查询
│   ├── 专利分析
│   └── 著作权
└── ...（更多分类）
```

## 搜索结果格式

每个搜索结果须包含以下信息：

```markdown
### [技能名称] v[版本号]

**描述**：[技能简述]
**法律领域**：[领域标签] | **能力类型**：[类型标签]
**安全状态**：[🟢 已审查通过 / 🟡 有条件通过 / 🔴 未通过 / ⚪ 未审查]
**许可证**：[MIT / Apache 2.0 / GPL / ...]
**兼容性**：[✅ 兼容当前平台 / ⚠️ 需确认 / ❌ 不兼容]
**来源**：[社区贡献 / 官方 / 组织内部]
**独立命令**：`/legal-builder-hub:[skill-name]`
```

## 搜索过滤规则

### 安全审查状态过滤

根据用户配置决定搜索结果是否包含未审查技能：

| 用户配置 | 搜索结果行为 |
|---------|------------|
| 仅展示已审查技能 | 过滤掉所有未审查/未通过的技能 |
| 接受未审查技能（标注状态） | 展示所有技能，未审查技能标注 `[未审查 — 安装前须完成安全审查]` |
| 临时模式（未配置） | 默认仅展示已审查技能 |

### 兼容性过滤

- 默认隐藏与当前平台版本不兼容的技能
- 用户可选择显示不兼容技能（标注 `[兼容性 ⚠️ — 须确认平台版本]`）

| ... | ... | ... | ... | ... | ... |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 搜索结果为空 | 建议使用更宽泛的关键词或浏览分类；推荐相近领域的技能 |
| 注册表不可达 | 提示检查网络连接和注册表地址配置；提供本地缓存搜索结果 |
| 搜索词过于宽泛 | 建议缩小范围；按安全状态或许可证类型进一步过滤 |
| 大量结果（>50） | 分页展示；建议增加筛选条件 |

---

## 错误处理

| 错误类型 | 触发情形 | 处理方式 |
|---------|---------|---------|
| 注册表连接失败 | 远程注册表不可达 | 回退到本地缓存；提示检查网络 |
| 搜索语法错误 | 不支持的搜索语法 | 提示支持的搜索语法；自动纠正常见错误 |
| 索引损坏 | 本地搜索索引异常 | 建议重建索引 |
