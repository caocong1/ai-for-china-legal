# 技能详情与评价（skill-detail-evaluation）

> **所属编排**：`legal-builder-hub/skills/skill-discovery/SKILL.md`（技能发现）
> **分工说明**：本子技能展示单个技能的完整详情和评价信息。搜索浏览由 `skill-search-browse` 处理；筛选由 `category-tag-filtering` 处理；排序由 `recommendation-ranking` 处理。
> **可单独触发**：`/legal-builder-hub:skill-detail-evaluation`

---

## 目的

展示候选技能的完整信息，帮助用户全面了解技能的功能、安全性、兼容性和社区口碑，为安装决策提供充分依据。

---

## 前置

1. **读取 `legal-builder-hub/CLAUDE.md`** — 获取：安全审查偏好、技术背景、整体技能管理偏好
2. **确定目标技能** — 从技能名称、搜索结果编号或技能路径获取
3. 本子技能产出是技能详情展示，**不执行安装或修改操作**

---

## 详情展示结构

### 1. 基本信息

```markdown
## [技能名称]

**版本**：[当前版本号]
**开发者**：[开发者/组织名称]
**许可证**：[许可证类型 + 简要说明]
**最后更新**：[日期]
**发布时间**：[首次发布日期]
```

### 2. 功能说明

从 SKILL.md 提取：
- **目的**：技能解决什么问题
- **工作流**：技能的核心工作流程
- **输出**：技能生成什么产物
- **子技能**：如为编排入口，列出子技能清单
- **法律领域**：适用的法律领域标签
- **能力类型**：提供的能力类型标签

### 3. 安全审查状态

```markdown
### 安全审查

**审查状态**：[🟢 已审查通过 / 🟡 有条件通过 / 🔴 未通过 / ⚪ 未审查]
**审查日期**：[日期]
**审查版本**：[审查时的版本号]
**审查级别**：[完整审查 / 标准审查 / 快速检查]

**安全发现摘要**：
- 🔴 关键：[N] 项
- 🟠 高：[N] 项
- 🟡 中等：[N] 项
- 🟢 低：[N] 项

**审查报告**：[查看完整审查报告 → `/legal-builder-hub:review-report-certification`]
```

> **安全提示**：若审查状态为「未审查」，须显示：
> ⚠️ 此技能尚未通过安全审查。安装前须先完成安全审查（→ `/legal-builder-hub:security-review`）。

### 4. 兼容性信息

```markdown
### 兼容性

**平台要求**：[最低平台版本]
**依赖项**：[依赖技能/库清单及版本要求]
**系统要求**：[操作系统/运行时要求]
**当前环境兼容性**：[✅ 兼容 / ⚠️ 部分兼容 / ❌ 不兼容]
```

### 5. 社区评价

```markdown
### 社区评价

**综合评分**：[X.X / 5.0]（[N] 条评价）

| 评价维度 | 评分 |
|---------|------|
| 功能完整性 | [X.X] |
| 易用性 | [X.X] |
| 文档质量 | [X.X] |
| 响应速度 | [X.X] |
| 安全可信度 | [X.X] |

**近期评价**：
- [用户名] [评分] [日期]：[评价摘要]
- ...
```

### 6. 版本历史

```markdown
### 版本历史

| 版本 | 发布日期 | 变更摘要 |
|------|---------|---------|
| [版本] | [日期] | [变更说明] |
| ... | ... | ... |
```

## 安装决策辅助

根据用户配置和安全审查结果，提供安装决策建议：

| 审查状态 | 建议 |
|---------|------|
| 🟢 通过 | ✅ 建议安装。可直接进入安装流程 |
| 🟡 有条件通过 | ⚠️ 建议修复指定问题后安装。查看修复清单 |
| 🔴 未通过 | ❌ 不建议安装。存在安全隐患 |
| ⚪ 未审查 | ⚠️ 须先完成安全审查。进入安全审查流程 |

---

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 技能详情不完整 | 展示已有信息；标注缺失项；提示开发者补充 |
| 安全审查报告过期 | 审查版本与当前版本不一致；建议重新审查 |
| 无社区评价 | 显示「暂无评价」；标注为新技能 |
| 技能已下架/废弃 | 显示废弃原因；推荐替代技能 |
