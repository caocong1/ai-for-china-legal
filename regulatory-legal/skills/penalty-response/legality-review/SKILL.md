> **可单独触发**：`/regulatory-legal:legality-review`

## 目的

对已收到的行政处罚决定（或拟作出的处罚告知）进行五维度合法性审查：

1. **主体与职权**：处罚机关是否具有法定职权，是否存在超越职权或管辖错误
2. **事实与证据**：违法事实认定是否清楚，证据是否充分、合法、关联
3. **法律适用**：处罚所依据的法律规范是否正确，是否存在适用错误或依据不足
4. **程序合法性**：处罚程序是否符合法定要求（告知·陈述申辩·听证·法制审核·送达等）
5. **裁量合理性**：处罚种类与幅度是否与违法情节相当，是否符合裁量基准

### 步骤二：主体与职权审查

| 审查项 | 审查要点 | 法律依据 | 审查结论 |
|-------|---------|---------|---------|
| 处罚机关是否为法定有权机关 | 该机关是否依法具有对该类违法行为的行政处罚权 | 行政处罚法第4条（处罚法定，2021修订，建议复核） | [合法/违法/存疑] |
| 是否存在超越职权 | 处罚机关是否超越法定权限（如跨区域执法·超越事项管辖） | 行政处罚法第4条（处罚法定，2021修订，建议复核） | [合法/违法/存疑] |
| 管辖是否正确 | 是否存在管辖错误（级别管辖·地域管辖·事项管辖） | 行政处罚法相关规定 `[待验证]` | [合法/违法/存疑] |
| 执法人员资格 | 执法人员是否具备执法资格，是否出示执法证件 | 行政处罚法相关规定 `[待验证]` | [合法/违法/存疑] |
| 是否存在委托处罚 | 如为委托处罚，委托是否符合法定条件 | 行政处罚法相关规定 `[待验证]` | [合法/违法/存疑/不适用] |

### 步骤三：事实与证据审查

| 审查项 | 审查要点 | 审查结论 |
|-------|---------|---------|
| 违法事实是否清楚 | 处罚决定书载明的违法事实是否具体·明确·完整 | [清楚/不清楚/部分不清楚] |
| 证据是否充分 | 据以认定事实证据是否形成完整证据链 | [充分/不充分/存疑] |
| 证据合法性 | 证据收集是否符合法定程序（是否存在非法取证） | [合法/违法/存疑] |
| 证据关联性 | 证据与待证事实之间是否具有关联 | [有关联/无关联/部分关联] |
| 当事人陈述与申辩是否记录 | 当事人的陈述申辩意见是否在案卷中体现 | [已记录/未记录/不知] |

### 步骤四：法律适用审查

| 审查项 | 审查要点 | 法律依据 | 审查结论 |
|-------|---------|---------|---------|
| 处罚依据是否存在 | 处罚决定是否引用了具体的法律规范条文 | 行政处罚法第4条（处罚法定，2021修订，建议复核） | [有依据/无依据/依据不足] |
| 处罚依据是否有效 | 引用的法律规范是否现行有效（是否已废止或修订） | 行政处罚法第4条（处罚法定，2021修订，建议复核） | [有效/已废止/已修订] |
| 处罚依据是否适格 | 引用的规章是否超越法律·行政法规授权 | 行政处罚法第4条（处罚法定，2021修订，建议复核） | [适格/越权/存疑] |
| 处罚种类是否法定 | 处罚种类是否在行政处罚法第9条规定的法定种类范围内 | 行政处罚法第9条（处罚种类，2021修订，建议复核） | [法定/超种类/存疑] |
| 是否违反一事不再罚 | 是否存在对同一违法行为重复罚款的情形 | 行政处罚法第29条（一事不再罚，2021修订，建议复核） | [未违反/涉嫌违反/需核实] |
| 条号是否为 2021 修订后新条号 | 处罚决定引用的行政处罚法条号是否为新条号 | 行政处罚法 2021 修订条号重排 | [新条号/旧条号/无法判断] |

### 步骤五：程序合法性审查

| 审查项 | 审查要点 | 法律依据 | 审查结论 |
|-------|---------|---------|---------|
| 处罚前是否告知 | 作出处罚决定前是否告知拟处罚内容及事实·理由·依据 | 行政处罚法第44条（陈述申辩权，2021修订，建议复核） | [已告知/未告知/不知] |
| 是否告知陈述申辩权 | 是否告知当事人依法享有陈述·申辩的权利 | 行政处罚法第44条（陈述申辩权，2021修订，建议复核） | [已告知/未告知/不知] |

| 是否进行法制审核 | 重大处罚决定作出前是否进行法制审核 | 行政处罚法第58条（法制审核，2021修订，建议复核） | [已审核/未审核/不知] |
| 处罚决定书是否载明法定事项 | 处罚决定书是否载明法定要素 | 行政处罚法相关规定 `[待验证]` | [完备/不完备/不知] |
| 送达是否合法 | 送达方式与期限是否符合法定要求 | 行政处罚法相关规定 `[待验证]` | [合法/违法/不知] |

## 违法点-救济方向对照表

| 违法点 | 对应审查维度 | 可能的救济方向 | 救济强度 |
|-------|------------|-------------|---------|
| 处罚机关无职权或超越职权 | 主体与职权 | 复议申请撤销 / 诉讼请求撤销 | 强（程序硬伤） |
| 违法事实认定不清·证据不足 | 事实与证据 | 复议申请撤销或变更 / 诉讼请求撤销 | 强（事实基础不牢） |
| 处罚依据错误或依据不足 | 法律适用 | 复议申请撤销 / 诉讼请求撤销 | 强（无法定依据则无效） |
| 违反一事不再罚 | 法律适用 | 主张重复罚款部分无效 | 强（行政处罚法第29条，建议复核） |
| 未告知处罚事实·理由·依据 | 程序合法性 | 复议申请撤销 / 诉讼请求撤销 | 强（程序违法） |
| 未告知或剥夺陈述申辩权 | 程序合法性 | 复议申请撤销 / 诉讼请求撤销 | 强（程序违法） |
| 较重处罚未告知听证权 | 程序合法性 | 复议申请撤销 / 诉讼请求撤销 | 强（行政处罚法第63条，建议复核） |
| 未经法制审核 | 程序合法性 | 复议申请撤销 / 诉讼请求撤销 | 中强（行政处罚法第58条，建议复核） |
| 处罚过重·不符合裁量基准 | 裁量合理性 | 复议申请变更 / 诉讼请求变更 | 中（裁量权审查空间） |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| **处罚决定书不完整**：用户仅收到口头告知或未收到正式文书 | 标 `[前提 flagged — 处罚决定书不完整，审查结论可能有遗漏]`；建议用户提供正式处罚决定书后再行完整审查 |
| **处罚依据引用旧版条号**：处罚决定书引用的行政处罚法条号为旧版 | 标 `[时效性 flagged — 处罚决定引用旧版条号，须以 2021 修订后新条号为准]`；对照新旧条号后重新审查法律适用 |
| **行业监管规章快速演进**：处罚依据的规章可能已修订 | 标 `[时效性 flagged — 行业监管规章快速演进，须核实现行版本]`；通过监管机关官网核实现行有效版本 |
| **重大处罚识别**：处罚种类为吊销许可·责令停产停业·较大数额罚款·限制从业 | 立即触发 `[强制升级 — 重大处罚]`；提示联系执业律师；本技能可提供初步分析供律师参考 |
| **行刑衔接识别**：违法情节可能达到刑事追诉标准 | 立即触发 `[强制升级 — 行刑衔接]`；提示与 criminal-compliance 插件衔接 |
| **一事不再罚疑点**：多个机关对同一行为分别处罚 | 标 `[一事不再罚 — 须核实是否属于同一违法行为]`；提示「同一违法行为」认定边界存在争议 `[待验证]` |
| **地方裁量基准差异**：处罚幅度与地方裁量基准的关系 | 标 `[地方 flagged — 需核实地方裁量基准]`；以当地现行裁量基准为准 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| **无法读取配置文件** | 标 `[临时模式]`；审查按保守型严格度执行；建议运行 `/regulatory-legal:cold-start-interview` |
| **处罚决定书关键信息缺失** | 标 `[前提 flagged — 信息不完整]`；列出需补充的信息清单；不静默假设缺失内容 |
| **行刑衔接升级**：识别到行政违法可能升格刑事 | 立即标 `[强制升级 — 行刑衔接]`；暂停常规审查；提示联系执业律师；与 criminal-compliance 插件衔接 |
| **监管合规边界触发**：用户要求伪造证据·隐匿事实·规避检查 | 标 `[合规边界 — 拒绝]`；拒绝提供相关建议；提示依法合规·如实申报 |
| **行政程序时限临近**：陈述申辩期·听证申请期·复议申请期临近 | 强标时限提示；升级给执业律师确认；不得静默放任时限流逝 |
| **条号混用**：审查过程中混淆新旧条号 | 立即核实并统一为 2021 修订后新条号；标「建议复核」 |
| **审查范围超出本技能**：用户需要陈述申辩书起草或复议申请 | 标注「详细文书起草须衔接对应子技能」；推荐 `/regulatory-legal:statement-defense-hearing` 或 `/regulatory-legal:review-litigation-bridge` |
