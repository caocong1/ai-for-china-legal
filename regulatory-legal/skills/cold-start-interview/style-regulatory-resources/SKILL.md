# 文书风格与监管资源访谈 — 监管合规

## 目的

通过结构化问卷采集用户的文书风格偏好、可用监管资源和地方性法规差异信息，写入 `regulatory-legal/CLAUDE.md` 的 `## 文书风格`、`## 可用集成` 和 `## 共享护栏`（`### 地方性法规与裁量基准提示`）三个章节。

## 前置

- 已完成身份与企业画像访谈，所属行业与主管监管机关已确定。
- 已完成合规立场与服务范围访谈，服务立场已确定。
- 读取 CLAUDE.md 当前 `## 文书风格`、`## 可用集成`、`## 共享护栏` 章节内容。

## 访谈问卷

### 第一部分：文书风格偏好

**Q1. 监管动态简报的风格偏好？**

- 追问：期望的格式——分类列示·重点提示·待办清单？还是其他格式？
- 追问：是否需要按效力层级（法律/行政法规/部门规章/规范性文件/地方性法规）分类？
- 追问：是否需要标注生效日期与过渡期？
- 默认：简洁·按效力层级与行业分类·标生效日期与过渡期·高亮重大新规

**Q2. 行政处罚应对文书的风格偏好？**（陈述申辩书·听证陈述·复议申请书·起诉状）

- 追问：是否需要遵循特定法院或复议机关的格式要求？
- 追问：证据引用格式偏好——脚注式·文内引用·附件清单？
- 默认：客观·有据·程序合规·以事实与证据为基础；避免情绪化表达；关注合法性论点

**Q3. 合规制度文本的风格偏好？**

- 追问：制度条文结构偏好——章节式·条目式·流程图+条目？
- 追问：是否需要包含审批权限表与责任追究条款？
- 追问：是否需要包含操作指引与禁止性清单？
- 默认：规范·可落地；制度条文明确；操作指引具体；禁止性清单清晰；含审批权限与责任追究

**Q4. 监管报告的风格偏好？**（定期报告·专项报告·重大事项报告·整改报告·自查报告）

- 追问：是否有特定监管机关的报告模板需要遵循？
- 追问：数据口径偏好——财务数据/运营数据/合规数据的统计口径是否已统一？
- 追问：合规声明是否需要逐条引用法规依据？
- 默认：客观准确·不夸大·如实陈述；术语规范；数据口径一致；合规声明有据可查

**Q5. 工作产物存放位置？**

- 默认：`[PLACEHOLDER — 如：内部系统路径 / 共享文件夹]`

### 第二部分：可用监管资源

**Q6. 主管监管机关（中央级）是否已确认？**

- 追问：官网地址是什么？（如 samr.gov.cn / cac.gov.cn / nmpa.gov.cn 等）
- 默认：根据身份访谈中确认的监管机关填写

**Q7. 主管监管机关（地方级·日常执法）是否已确认？**

- 追问：地方监管机关名称与联系方式？
- 默认：根据身份访谈中确认的地方监管机关填写，或标记不适用

**Q8. 行政复议机关是否已确认？**

- 追问：对行政处罚不服时，向哪个机关申请行政复议？（可能是本级政府或上级主管部门）
- 追问：是否存在复议前置的情形？（特定行业或处罚类型要求先复议再诉讼）
- 默认：`[待验证 — 复议机关认定须核实]`

> **复议机关提示**：行政复议机关的认定须结合具体处罚机关与处罚类型核实，部分情形复议前置。

**Q9. 行政审判法院是否已确认？**

- 追问：管辖法院是哪一级？（基层法院/中级法院/专门法院）
- 追问：是否有特定的行政审判庭或集中管辖安排？
- 默认：`[待验证 — 管辖法院须核实]`

**Q10. 是否有行业协会资源？**

- 追问：协会名称、是否提供行业合规指引与培训？
- 默认：未配置

**Q11. 法规数据库接入情况？**

- 追问：是否已接入北大法宝？
- 追问：是否使用其他法规数据库？（如威科先行·万律·国家法律法规数据库 flk.npc.gov.cn）
- 默认：未接入

**Q12. 是否有可委托的司法鉴定/检测/审计机构？**

- 追问：机构名称与业务范围？（如产品质量鉴定·环境检测·财务审计）
- 默认：可根据需要委托

**Q13. 文档存储与企业通讯工具？**

- 追问：文档存储路径？
- 追问：是否使用企业微信或钉钉？
- 默认：未配置

### 第三部分：地方性法规与裁量基准差异

**Q14. 企业所在地（省/市）？**

- 追问：是否有跨地区经营？如有，列出所有涉及地区。
- 默认：根据身份访谈信息填写

**Q15. 是否已了解所在地的地方性法规与裁量基准差异？**

- 追问：所在省/市是否有特定的行政处罚裁量基准需要关注？
- 追问：是否有特定的地方性法规对行业有更严格要求？
- 追问：地方监管口径是否有特别之处？（如某地对数据安全执法偏严、某地对环保处罚裁量较重等）
- 默认：标记 `[地方 flagged — 需核实地方性法规与地方裁量基准]`

> **地方差异提示**：各地地方性法规·行政处罚裁量基准·监管口径存在地区差异。具体罚款数额·情节阶次·许可条件等须以当地现行版本为准。

## 边界条件

- 用户未确认复议机关或管辖法院：写入 `[待验证]` 标记，不阻塞后续。
- 用户无行业协会资源：写入「未配置」。
- 用户跨地区经营：在地方性法规提示中列出所有涉及地区。
- 用户无特定文书风格偏好：采用 CLAUDE.md 现有默认值。
- 增量更新模式：仅替换用户确认修改的字段，其余保留原值。

## 错误处理

- 前置访谈未完成：基于行业默认值推荐监管机关与法规数据库，用户确认后写入。
- 用户口述具体地方裁量基准数额：标 `[待验证 — 地方 flagged]`，不写入裸数字。
- 用户口述地方性法规文号：标 `[待验证]`，提示核实现行版本。
- 用户提供的监管机关官网无法确认：标 `[待验证 — 官网须核实]`，列出已知官网供选择。
