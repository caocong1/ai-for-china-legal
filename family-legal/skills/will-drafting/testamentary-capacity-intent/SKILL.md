# 遗嘱能力与意思表示（testamentary-capacity-intent）

> **可独立触发**：本技能可由 `/family-legal:will-drafting` 编排入口调用，也可由律师/法务/立遗嘱人单独触发（`/family-legal:testamentary-capacity-intent`），无需运行整个遗嘱起草流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载代理立场（遗嘱人侧）、敏感度（能力存疑/受胁迫识别）、升级矩阵。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1143条遗嘱无效情形）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）。
>
> **核心目标**：保障遗嘱人真实意愿与遗嘱有效性。遗嘱能力与意思真实是遗嘱有效性的根本要件——能力不足或意思不真实的遗嘱无效，无论形式多么完备。
>
> **分工说明**：本技能专注遗嘱能力评估与意思真实保障；遗嘱形式选择见 `will-form-selection`；遗嘱内容与必留份见 `content-reserved-share`；见证人资格与生效要件见 `witnessing-effectiveness`。

---

## 目的

在遗嘱起草前后，系统评估立遗嘱人的民事行为能力状态，核查遗嘱是否表示遗嘱人真实意思，识别欺诈、胁迫、伪造、篡改等导致遗嘱无效的情形，并提供能力留痕的最佳实践建议。当能力存疑或疑似受胁迫时，**必须暂停并强制升级**。

覆盖核心分析：

1. 遗嘱能力要件（完全民事行为能力）
2. 立遗嘱时能力的时间节点判断
3. 真实意思表示保障措施
4. 欺诈胁迫遗嘱的识别与处理
5. 伪造篡改遗嘱的效力
6. 能力存疑或受胁迫情形的强制升级流程

## 详细步骤

### 步骤一：遗嘱能力要件

#### 1.1 遗嘱能力的法律要求

依据民法典第1143条（中度把握，建议复核）：

| 要件 | 具体要求 |
|------|---------|
| **完全民事行为能力人** | 须为完全民事行为能力人（年满18周岁且精神状态正常，或年满16周岁且以自己劳动收入为主要生活来源） |
| **无民事行为能力人** | 所立遗嘱无效（如：无行为能力的精神病患者、未满8周岁的未成年人） |
| **限制民事行为能力人** | 所立遗嘱无效（如：8-18周岁未成年人、精神状态时好时坏的间歇性精神病患者处于发作状态时） |

#### 1.2 能力的时间节点

**关键原则**：遗嘱能力在**立遗嘱时**判断，不以立遗嘱后的状态推算。

| 时间节点情形 | 处理规则 |
|------------|---------|
| 立遗嘱时有完全民事行为能力，其后丧失 | **不影响遗嘱效力**——遗嘱仍有效 |
| 立遗嘱时无完全民事行为能力（如发作期间），其后恢复 | **遗嘱仍无效**——效力以立遗嘱时为准 |
| 间歇性精神障碍者在清醒期（缓解期）立遗嘱 | **遗嘱可能有效**——须证明立遗嘱时处于完全行为能力状态（医疗证明+见证人证词） |

#### 1.3 高龄遗嘱人的能力核查

高龄遗嘱人（特别是70岁以上，或有认知症迹象者）须额外关注：

| 核查方面 | 具体措施 |
|---------|---------|
| 认知状态评估 | 建议在立遗嘱前进行认知功能评估（可由医院神经内科或老年科出具意见） |
| 独立沟通测试 | 在无家庭成员陪同的情况下，与遗嘱人独立沟通，验证其对遗嘱内容的理解与意愿 |
| 医疗记录留痕 | 取得近期医院诊疗记录，证明立遗嘱时的精神状态 |
| 录音录像辅证 | 对高龄遗嘱人，建议采用录音录像遗嘱或录音录像记录立遗嘱过程，以备将来争议时使用 |

### 步骤二：真实意思表示保障

#### 2.1 意思真实的法律要求

依据民法典第1143条（中度把握，建议复核）：遗嘱必须表示遗嘱人的**真实意思**，受欺诈、胁迫所立的遗嘱无效。

| 要件 | 具体含义 |
|------|---------|
| **自主意思** | 遗嘱内容完全按遗嘱人本人意愿表达，不受他人不当干预 |
| **无欺诈** | 无他人通过虚假陈述诱使遗嘱人作出不真实的意思表示 |
| **无胁迫** | 无他人通过威胁或强迫手段影响遗嘱人作出违背意愿的表示 |

#### 2.2 欺诈与胁迫的识别信号

| 信号类型 | 具体表现 | 风险等级 |
|---------|---------|---------|
| **欺诈信号** | | |
| 虚假财产信息 | 他人向遗嘱人虚报某继承人的财务状况、品德，诱导遗嘱人将更多财产留给该人 | 🔴 高度可疑 |
| 遗嘱内容出现异常偏差 | 遗嘱内容与遗嘱人平时表达的意愿差异很大（如突然将全部财产留给非传统受益人） | 🟠 需关注 |
| 立遗嘱后内容不知情 | 遗嘱人不知道或不理解自己签署的遗嘱内容 | 🔴 高度可疑 |
| **胁迫信号** | | |
| 陪同控制 | 某人全程陪同立遗嘱，不允许遗嘱人独立与律师沟通 | 🔴 高度可疑 |
| 遗嘱人行为异常 | 遗嘱人在陪同者离开后表达与遗嘱内容不同的意愿 | 🔴 高度可疑 |
| 遗嘱人表达恐惧 | 遗嘱人在讨论遗嘱内容时明显紧张、顾虑、眼神求助 | 🔴 高度可疑 |

#### 2.3 保障真实意思的最佳实践

| 最佳实践 | 说明 |
|---------|------|
| **独立沟通** | 在无继承人/受遗赠人在场的情况下，律师或公证员与遗嘱人单独沟通，确认遗嘱内容符合其真实意愿 |
| **理解测试** | 请遗嘱人用自己的语言复述遗嘱的主要内容，确认遗嘱人真正理解并认可 |
| **录音录像记录** | 建议对立遗嘱过程录音录像，记录遗嘱人主动表达遗嘱内容的过程，作为意思真实的证据 |
| **见证人独立核查** | 见证人须独立判断遗嘱人的意思表示是否真实，不可成为胁迫的从犯 |
| **多次确认** | 若有能力疑虑，建议隔日或多次独立沟通确认，避免单次情绪化表达影响判断 |
| **医疗证明** | 若有能力疑虑，立遗嘱前取得当日或近日的医疗诊断证明 |

### 步骤三：伪造与篡改遗嘱

#### 3.1 伪造遗嘱的效力（第1143条，建议复核）

**伪造的遗嘱无效**。

| 伪造情形 | 法律后果 |
|---------|---------|
| 完全非遗嘱人所写的「自书遗嘱」 | 整份遗嘱无效 |
| 遗嘱人签名系伪造（非本人签名） | 整份遗嘱无效 |
| 遗嘱人死亡后伪造遗嘱 | 整份遗嘱无效；伪造者可能丧失继承权（第1125条，建议复核） |

**识别方法**：笔迹鉴定（法院委托司法鉴定机构）、公证档案比对、目击人证词。

#### 3.2 篡改遗嘱的效力（第1143条，建议复核）

**遗嘱被篡改的，篡改的内容无效。**

| 篡改情形 | 法律后果 |
|---------|---------|
| 修改了财产分配比例 | 修改部分无效，原内容仍有效（若能识别原内容） |
| 添加了非遗嘱人所写的条款 | 添加部分无效 |
| 删除了遗嘱人所写的条款 | 视具体情况，可能导致遗嘱部分无效或影响整体解释 |

#### 3.3 丧失继承权的关联

依据民法典第1125条（中度把握，建议复核）：伪造、篡改、隐匿或者销毁遗嘱，情节严重的，丧失继承权。

**程序说明**：丧失继承权须通过法院诉讼程序认定，AI 工具不得自行认定。

### 步骤四：能力存疑或受胁迫时的强制升级流程

**识别到以下任一情形，须立即暂停并强制升级**：

```
⚠️ 识别到遗嘱能力存疑或疑似受胁迫情形
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
触发情形：[描述具体情形]

强制升级原因：
遗嘱能力是遗嘱有效性的根本要件。如果立遗嘱人不具备完全民事行为能力，
或遗嘱系在欺诈胁迫下立，则遗嘱无论形式多么完备均无效。
AI 工具无法进行能力鉴定，必须由具有执业资格的律师或医疗专业人员介入。

建议措施：
1. 暂停遗嘱起草
2. 由律师独立与遗嘱人进行能力评估沟通（无家庭成员陪同）
3. 如有疑虑，建议前往医院进行认知功能评估，取得近期医疗证明
4. 若疑似受胁迫，律师须单独与遗嘱人确认其意愿

恢复条件：确认遗嘱人具备完全民事行为能力，且意思表示真实自由，再继续遗嘱起草。
```

## 边界条件

| 边界情形 | 处理方式 | 风险等级 |
|---------|---------|---------|
| 间歇性精神障碍者立遗嘱 | 须证明立遗嘱时处于清醒缓解期；建议立遗嘱当日取得医疗证明 | 🟠 高 |
| 高龄遗嘱人（70岁以上，无明显认知问题） | 建议录音录像留痕；独立沟通确认意愿；取得近期医疗证明（预防性措施） | 🟡 中等 |
| 遗嘱人在住院或医疗机构中立遗嘱 | 建议要求医疗机构记录立遗嘱时的精神状态评估 | 🟡 中等 |
| 近亲属全程在场 | 须确保遗嘱人能在无亲属在场的情况下独立表达意愿；若亲属拒绝离开，🔴 胁迫嫌疑，强制升级 | 🟠 高 |
| 遗嘱内容与立遗嘱人平时表达差异极大 | 🔴 欺诈/胁迫嫌疑，必须独立核实；提示「是否表达过这样的意愿？可以解释原因吗？」 | 🔴 关键 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 立遗嘱人无法配合独立沟通（一直有人陪同控制） | **立即暂停**，标注 🔴 关键，强制升级律师；律师须通过其他方式（如律师函、到访公证机构）创造独立沟通条件 |
| 立遗嘱人明确表示遗嘱内容是「被迫的」 | **立即暂停全部遗嘱起草工作**，标注 🔴 关键，强制升级；记录遗嘱人陈述并保密 |
| 发现遗嘱疑似伪造（签名与其他文件不符） | 标注 🔴 关键，建议笔迹鉴定（法院委托或自行委托司法鉴定机构）；不得在鉴定前依据该遗嘱进行继承分配 |
| 无法取得医疗能力证明 | 提示替代措施（律师独立评估记录、公证员记录、见证人证词）；在无证明情形下注明 `⚠️ 能力未经医疗证明确认` |

## 法律依据

### 法条（来源：民法典继承编）

**第1143条 — 遗嘱无效情形（含能力与意思真实）**
- 《中华人民共和国民法典》**第1143条** `[模型知识 — 需验证]` 中度把握，建议复核
- 无民事行为能力人或限制民事行为能力人所立遗嘱无效；遗嘱须表示遗嘱人真实意思，受欺诈胁迫无效；伪造遗嘱无效；被篡改的内容无效

**第1125条 — 丧失继承权（含伪造篡改遗嘱）**
- 《中华人民共和国民法典》**第1125条** `[模型知识 — 需验证]` 中度把握，建议复核
- 伪造、篡改、隐匿或销毁遗嘱情节严重，丧失继承权

### 司法解释（描述规则，条号待验证）
