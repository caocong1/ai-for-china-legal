# 遗嘱起草（编排入口）

> **迁移说明**：原遗嘱起草深度内容（遗嘱形式选择、遗嘱能力与意思表示、遗嘱内容与必留份、见证与生效）已拆分迁移至四个子技能，每个子技能可独立触发。本文件为编排入口，负责前置护栏、代理立场加载与子技能调用顺序。子技能路径：`family-legal/skills/will-drafting/<子技能名>/SKILL.md`。

---

## 目的

遗嘱起草是遗嘱人将其遗产分配意愿以符合法律形式的方式固定下来的过程。本技能以**遗嘱人侧**为核心立场，确保遗嘱在形式上合法有效、在内容上符合遗嘱人真实意愿、在必留份要求上符合法律强制性规定，并安排妥当的见证与保管。

**主要法律框架**：民法典继承编（第1119-1163条），特别是第1134-1145条（遗嘱各形式要件、见证人资格、必留份、附义务、遗产管理人），及最高法继承编司法解释(一)（具体条号 `[待验证]`）。

## 入口级护栏

### 文书风格加载

遗嘱文书须符合以下风格标准：

| 要素 | 要求 |
|------|------|
| **严谨准确** | 财产描述须精确（含不动产权证书号/银行账户等）；受益人身份须明确（姓名/身份证号）|
| **无歧义** | 避免模糊表述（如「我的孩子们」「其余财产」等）；每项财产归属须单独列明 |
| **形式要件明确** | 遗嘱形式选择须在文本中体现（如「本遗嘱为自书遗嘱，系本人亲笔书写」）|
| **语言克制** | 不含对继承人的情绪化评价；不含可能引发纠纷的人身攻击性内容 |

### 遗嘱能力与真实意思核查提示（强制优先）

在进入具体子技能前，须做以下核查：

| 核查项 | 触发护栏 |
|--------|---------|
| 立遗嘱人年龄是否满18周岁（或16周岁以劳动收入为主）| 未满足：🔴 关键，须升级律师评估 |
| 是否有认知障碍/精神疾病/近期手术麻醉等能力疑虑 | 有疑虑：🔴 关键，**强制升级律师**，建议医疗机构出具能力证明 |
| 是否有家庭成员全程陪同且控制沟通的迹象（胁迫嫌疑）| 有迹象：🔴 关键，**强制升级律师**，须创造独立沟通条件 |
| 遗嘱内容是否与立遗嘱人平时表达的意愿明显不符 | 明显偏差：🟠 高风险，须独立核实 |

**遗嘱能力与意思真实是遗嘱有效性的根本要件**。识别到任何能力存疑或受胁迫信号，须**立即暂停遗嘱起草，强制升级律师**，不得继续推进形式起草。

### 必留份与处分范围提示

在起草遗嘱内容前，须提示：

1. **先析产**：遗嘱只能处分遗嘱人的个人财产（含夫妻共同财产中属其的一半），不得处分配偶或他人财产
2. **必留份**：若继承人中有「缺乏劳动能力又无生活来源」者，遗嘱须为其保留必要份额（民法典第1141条，建议复核）；违反必留份的部分无效

### 公证路径提示

根据遗嘱人情况，提示以下公证路径（不强制，但建议考虑）：

| 情形 | 公证路径建议 |
|------|------------|
| 财产复杂（多处房产/股权/海外资产） | 建议办理公证遗嘱，由公证机构留档，减少继承争议 |
| 担心遗嘱能力事后被质疑 | 立遗嘱时同步办理公证，公证员可记录立遗嘱时的精神状态 |
| 遗嘱内容重要且不希望更改 | 公证遗嘱可提供强有力的留痕，但注意民法典下公证遗嘱**不再优先**，仍可通过其他形式修改 |

**公证遗嘱不再优先的重要提示**：民法典取消公证遗嘱效力优先规则，数份遗嘱以**最后订立的**为准（民法典第1142条，建议复核），与是否公证无关。

### 目的地与特权检查

遗嘱文本的目的地：

- **律师工作草稿（内部）**：标题为「律师工作产物 — 受律师-当事人特权保护 / 保密」
- **遗嘱正式文本（对外）**：遗嘱人签署的遗嘱是对外文件；草稿与正式文本须区分
- **非律师角色（当事人本人）**：遗嘱定稿前须暂停并升级律师确认形式与内容合规性


遗嘱形式新规（民法典2021年施行）、继承编司法解释条号经修订整合，具体规则须通过最高法官网（https://www.court.gov.cn）核实现行版本。

## 子技能编排顺序

律师/法务/立遗嘱人可单独触发任一子技能，也可通过本入口运行完整遗嘱起草流程：

| 顺序 | 子技能 | 职责 | 子技能路径 | 单独触发命令 |
|------|-------|------|---------|-----------|
| 1 | **testamentary-capacity-intent** | 遗嘱能力与意思表示：完全民事行为能力核查、真实意思表示保障、欺诈胁迫识别、能力存疑强制升级 | `skills/will-drafting/testamentary-capacity-intent/SKILL.md` | `/family-legal:testamentary-capacity-intent` |
| 2 | **will-form-selection** | 遗嘱形式选择：六种形式要件对照、形式选择决策、公证遗嘱不再优先修订说明、各形式无效陷阱 | `skills/will-drafting/will-form-selection/SKILL.md` | `/family-legal:will-form-selection` |
| 3 | **content-reserved-share** | 遗嘱内容与必留份：先析产确定处分范围、必留份合规、附义务遗嘱条款、遗赠与遗嘱继承区分 | `skills/will-drafting/content-reserved-share/SKILL.md` | `/family-legal:content-reserved-share` |
| 4 | **witnessing-effectiveness** | 见证与生效：见证人资格核查、各形式生效要件逐项核对（打印遗嘱每页签名等）、遗嘱执行人指定、保管建议 | `skills/will-drafting/witnessing-effectiveness/SKILL.md` | `/family-legal:witnessing-effectiveness` |

**顺序说明**：能力与意思真实在前（根本要件，能力不足则任何形式均无效）→ 形式选择第二（确定起草框架）→ 内容与必留份第三（实体合法性）→ 见证与生效最后（形式完成与生效安排）。

**单独触发场景**：律师可仅针对单一问题调用子技能。例如：

- 仅需核查某份遗嘱的见证人是否适格：`/family-legal:witnessing-effectiveness`
- 仅需核查打印遗嘱每页签名要件：`/family-legal:witnessing-effectiveness`
- 仅需确认必留份是否满足：`/family-legal:content-reserved-share`
- 仅需选择适合的遗嘱形式：`/family-legal:will-form-selection`
- 仅需评估立遗嘱人的行为能力：`/family-legal:testamentary-capacity-intent`

## 输出格式

遗嘱起草完成后，输出须包含：

1. **立遗嘱人情况摘要**：代理立场（遗嘱人侧）、财产概况、受益人信息
2. **遗嘱形式确认**：选定形式及关键要件核查结果
3. **遗嘱文本草稿**：（如已完成所有步骤）严谨的遗嘱正文
4. **必留份合规确认**：是否满足必留份要求
5. **见证安排**：见证人名单与资格核查、签名安排
6. **保管建议**：原件保管方式
7. **后续步骤**：公证建议、遗嘱执行人告知、继承人通知建议

**工作产物标题**（律师角色）：

```
律师工作产物 — 受律师-当事人特权保护 / 保密
```

**工作产物标题**（非律师角色）：

```
研究笔记 — 非法律建议，供律师审查
遗嘱定稿前须升级律师确认形式与内容合规性
```
