# 见证与生效（witnessing-effectiveness）

> **可独立触发**：本技能可由 `/family-legal:will-drafting` 编排入口调用，也可由律师/法务/立遗嘱人单独触发（`/family-legal:witnessing-effectiveness`），无需运行整个遗嘱起草流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载代理立场（遗嘱人侧）、文书风格、敏感度。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1140条见证人资格、第1145条遗产管理人、第1134-1139条各形式要件）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）。
>
> **核心目标**：保障遗嘱人真实意愿与遗嘱有效性。见证人不适格或形式要件缺失，是遗嘱无效的最常见原因之一——即使内容和意愿都正确，一旦见证人不适格，遗嘱可能无效。
>
> **分工说明**：本技能专注见证人资格核查与遗嘱生效要件；遗嘱形式选择见 `will-form-selection`；遗嘱能力与意思表示见 `testamentary-capacity-intent`；遗嘱内容与必留份见 `content-reserved-share`。

## 目的

系统核查遗嘱见证人的资格合规性，逐项检查所选遗嘱形式的生效要件（签名/日期/每页签名等），处理遗嘱执行人与遗产管理人的指定，提供遗嘱保管的最佳实践建议，防止因见证人不适格或形式要件缺失导致遗嘱无效。

覆盖核心分析：

1. 见证人资格排除清单
2. 各遗嘱形式见证人要求与签名日期要件
3. 见证人「在场见证」的要求
4. 遗嘱执行人与遗产管理人的区分
5. 遗嘱保管与生效时间
6. 见证形式缺失的强制复核与风险提示

## 详细步骤

### 步骤一：见证人资格排除清单

依据民法典第1140条（中度把握，建议复核）：

**以下人员不能作为遗嘱见证人：**

| 排除类型 | 具体情形 | 原因 |
|---------|---------|------|
| **无民事行为能力人** | 未满8周岁的未成年人；无行为能力的精神障碍患者 | 无见证能力 |
| **限制民事行为能力人** | 8-18周岁未成年人；精神障碍处于发作状态 | 见证能力受限 |
| **其他不具有见证能力的人** | 视听障碍等导致无法有效见证（描述规则，具体认定 `[待验证]`） | 无法有效见证 |
| **继承人** | 遗嘱指定的继承人（无论其最终是否获益） | 利益冲突 |
| **受遗赠人** | 遗嘱指定的遗赠受益人 | 利益冲突 |
| **与继承人有利害关系的人** | 继承人/受遗赠人的配偶、父母、子女、兄弟姐妹等近亲属 | 间接利益冲突 |
| **与受遗赠人有利害关系的人** | 受遗赠人的近亲属等 | 间接利益冲突 |

> **「利害关系」的范围**：以「与继承人或受遗赠人有共同利益关系」为判断标准，具体认定由法院裁量（描述规则，条号 `[待验证]`）。实践中，继承人/受遗赠人的配偶及直系亲属通常被认定为有利害关系。

#### 合格见证人的正向标准

| 合格条件 | 说明 |
|---------|------|
| 完全民事行为能力 | 年满18周岁且精神状态正常（或年满16周岁以自己劳动收入为主） |
| 与继承/受遗赠人无利害关系 | 最好选择与遗嘱人家庭无关联的社区成员、同事、朋友等 |
| 了解立遗嘱过程 | 全程在场，亲眼见证遗嘱人的立遗嘱行为 |

### 步骤二：各遗嘱形式见证与生效要件核对表

| 遗嘱形式 | 见证人数 | 见证方式 | 遗嘱人签名 | 见证人签名 | 日期要求 | 其他要件 |
|---------|---------|---------|---------|---------|---------|---------|
| **自书遗嘱** | 无需见证人 | — | 遗嘱人亲笔签名 | — | 遗嘱人注明年、月、日 | 全文须亲笔书写 |
| **代书遗嘱** | ≥2人（含代书人） | 全程在场见证 | 遗嘱人签名 | 代书人+全部见证人均须签名 | 注明年月日 | 一人代书，见证人须独立见证（不可仅代书人完成所有步骤） |
| **打印遗嘱** | ≥2人 | 全程在场见证 | **每一页**遗嘱人签名 | **每一页**全部见证人签名 | 注明年月日（每页须有） | 见证人须独立签名（非仅在末页） |
| **录音录像遗嘱** | ≥2人 | 在录音录像中记录 | 录音录像中遗嘱人说出/展示姓名 | 录音录像中见证人说出/展示姓名或肖像 | 录音录像中记录年月日 | 见证人须在录音录像中记录其姓名或肖像；录制时间须明确 |
| **口头遗嘱** | ≥2人 | 全程在场见证 | — | 见证人记录并签名（以书面方式记录） | 见证人记录立遗嘱日期 | **仅限危急情况**；危急解除后失效 |
| **公证遗嘱** | 公证员（代替见证人） | 公证机构办理 | 在公证机构签名 | 公证员签署 | 公证机构记录 | 无需额外见证人；公证机构存档 |

> **关键提示——打印遗嘱「每一页」签名**：第1136条要求遗嘱人和见证人在遗嘱**每一页**上签名。遗嘱共3页则须3页均有签名，不得仅在末页签名。这是打印遗嘱最常见的无效陷阱之一。

### 步骤三：「在场见证」要件解析

#### 3.1 「在场见证」的含义

「两名以上见证人在场见证」不仅指见证人在同一场所，还要求见证人：

| 要求 | 说明 |
|------|------|
| **全程在场** | 从遗嘱人开始代书/打印/录音录像/口头表达，到完成签名/日期注明，须全程在场 |
| **亲眼见证** | 见证人须实际目击遗嘱人表达遗嘱内容的过程及签名行为 |
| **独立见证** | 见证人须独立判断，不得仅凭代书人/律师事后告知而签名确认 |

#### 3.2 远程见证的争议

目前司法实践对**视频远程见证**（如视频通话方式的见证）是否满足「在场见证」要求存在争议（描述规则，`[待验证]`）。建议谨慎对待远程见证方式，优先选择同一场所的现场见证。

### 步骤四：遗嘱执行人与遗产管理人

#### 4.1 遗嘱执行人

遗嘱人可在遗嘱中**指定遗嘱执行人**（无法律强制要求，但指定可减少继承争议）。

| 方面 | 说明 |
|------|------|
| 指定方式 | 在遗嘱中明确指定姓名及联系方式 |
| 职责 | 执行遗嘱内容：管理遗产、分配遗产给继承人/受遗赠人、处理遗产债务 |
| 身份限制 | 无法律限制（可以是继承人本人，但可能有利益冲突风险） |
| 未指定时 | 由继承人推选遗产管理人，或由继承人共同担任 |

**遗嘱中指定遗嘱执行人的建议条款**：

```
遗嘱执行人：本人指定 [姓名]（身份证号：[XXXXXXXXXX]，联系方式：[XXXXXXXXXX]）
为本遗嘱的遗嘱执行人，授权其依照本遗嘱内容管理和分配本人遗产。
```

#### 4.2 遗产管理人（民法典第1145条，建议复核）

遗产管理人与遗嘱执行人的区别：

| 对比维度 | 遗嘱执行人 | 遗产管理人 |
|---------|---------|---------|
| **产生方式** | 遗嘱人指定 | 继承人推选（无遗嘱执行人时）；或由法定来源产生 |
| **适用情形** | 有指定遗嘱执行人的情形 | 无遗嘱执行人时（继承人推选、共同担任、民政部门/村委会） |
| **法律地位** | 第1145条：遗嘱执行人为遗产管理人（即遗嘱执行人同时履行遗产管理人职责） | 有遗嘱执行人时，其为遗产管理人；无时，按继承人推选规则 |
| **职责** | 执行遗嘱内容 | 管理遗产、清偿债务、分配遗产 |

依据民法典第1145条（中度把握，建议复核）：

> 继承开始后，遗嘱执行人为遗产管理人；没有遗嘱执行人的，继承人应当及时推选遗产管理人；继承人未能推选的，由继承人共同担任遗产管理人；没有继承人或者继承人均放弃继承的，由被继承人生前住所地的民政部门或者村民委员会担任遗产管理人。

### 步骤五：遗嘱保管与生效

#### 5.1 遗嘱生效时间

**遗嘱在遗嘱人死亡时生效**——遗嘱的法律效力在遗嘱人死亡时才发生，遗嘱人生前可随时撤回或修改。

| 规则 | 说明 |
|------|------|
| 生效时间 | 遗嘱人死亡时生效 |
| 有效期 | 遗嘱无有效期（只要未被撤回/修改，自立遗嘱时起均有效） |
| 撤回修改 | 遗嘱人生前任何时候可撤回或修改遗嘱（民法典第1142条，建议复核） |

#### 5.2 遗嘱原件保管建议

| 保管方式 | 说明 | 优缺点 |
|---------|------|-------|
| **遗嘱人自行保管** | 存放于保险箱/私人存储 | 优：保密；缺：死亡后可能难以找到 |
| **律师/律所保管** | 委托律师保管原件 | 优：专业、死亡后可及时通知；缺：律所关闭风险 |
| **公证机构存档** | 公证遗嘱由公证机构档案存档 | 优：可核查；缺：修改须通过公证机构 |
| **告知受信任的人** | 告知至少一名受信任的人遗嘱原件存放位置 | 重要：防止原件无法找到 |
| **多份副本** | 多份副本存放不同地点 | 注意：副本须与原件内容完全一致 |

## 边界条件

| 边界情形 | 处理方式 | 风险等级 |
|---------|---------|---------|
| 见证人临时回避（立遗嘱中途离开又返回） | 须全程在场，中途离开可能影响「在场见证」要件认定，建议重新完成见证 | 🟠 高 |
| 远程见证效力存疑（视频见证） | 建议谨慎，远程见证「在场」要件不明确，可能被质疑无效 `[待验证]` | 🟠 高 |
| 遗嘱原件丢失（仅有副本） | 遗嘱效力认定困难，须通过其他证据证明内容；无原件时遗嘱继承执行难度大 | 🟠 高 |
| 见证人签名为手印而非书写签名 | 各形式要件中均要求「签名」，手印是否等同于签名存在争议（描述规则，`[待验证]`）；建议优先使用书写签名 | 🟡 中等 |
| 见证人死亡（无法作证） | 见证人在遗嘱生效前死亡，应通过其他方式（录音录像/留存见证记录）证明见证事实 | 🟡 中等 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 见证人为继承人/受遗赠人 | **立即标注 🔴 关键**，该见证人不适格，须更换合格的见证人重新见证；不合格见证人参与的遗嘱形式要件不满足 |
| 打印遗嘱某页漏签 | **立即标注 🔴 关键**，须在遗嘱人和全部见证人在场时补签，或重新立遗嘱（补签是否有效存争议，建议重新立）|
| 口头遗嘱危急情况已解除但未转书面 | **立即标注 🔴 关键**，提示口头遗嘱已无效，须立即安排书面/录音录像形式立遗嘱 |
| 见证人只有一名（不足两名） | 标注 🔴 关键，须增加至少一名合格见证人重新见证 |
| 未注明完整日期（仅年月无日） | 标注 🟠 高风险，日期不完整的遗嘱可能被认定为形式要件不满足；建议补注日（可通过重新立遗嘱确保） |
| 见证人不确定是否与利害关系人相关 | 提示核实见证人与各继承人/受遗赠人的关系，不确定时建议更换无疑虑的见证人 |

## 法律依据

### 法条（来源：民法典继承编，中度把握锚点）

**第1140条 — 见证人资格限制**
- 《中华人民共和国民法典》**第1140条** `[模型知识 — 需验证]` 中度把握，建议复核
- 无民事行为能力人、限制民事行为能力人及不具有见证能力的人不能作见证人；继承人、受遗赠人不能作见证人；与继承人、受遗赠人有利害关系的人不能作见证人

**第1145条 — 遗产管理人**
- 《中华人民共和国民法典》**第1145条** `[模型知识 — 需验证]` 中度把握，建议复核
