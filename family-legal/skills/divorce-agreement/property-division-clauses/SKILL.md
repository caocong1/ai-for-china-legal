# 财产分割条款（property-division-clauses）

> **可独立触发**：本技能可由 `/family-legal:divorce-agreement` 编排入口调用，也可由律师/法务/当事人单独触发（`/family-legal:property-division-clauses`），无需运行整个离婚协议流程。
> **前置依赖**：本技能读取 `family-legal/CLAUDE.md` 实践配置（代理立场·协议审查严格度·敏感度校准），引用 `family-legal/skills/_shared/family-law-citations.md` 引用库与 `family-legal/skills/_shared/legal-basis-conventions.md` 引用规范。
> **分工说明**：本技能专注财产分割条款起草；子女抚养探望由 `child-custody-visitation` 处理；债务处理由 `debt-handling` 处理；协议生效与登记由 `effectiveness-registration` 处理；复杂特殊财产（婚前房产婚后增值、股权评估、公司股权、知识产权收益归属争议）参见 `property-division` 主技能（08-03）。

---

## 目的

起草离婚协议中的财产分割条款，覆盖夫妻共同财产范围认定、个人财产剥离确认、各类财产归属与过户安排、折价补偿计算、隐瞒财产救济保留。确保条款清晰可执行、按代理立场保护当事人财产权益、保留未来救济渠道。

### 2. 代理立场区分

| 立场 | 审查侧重 |
|------|---------|
| **主张方（要求分割一方）** | 关注共同财产足额分割、追查隐匿财产（第1092条）、婚后经营收益认定为共同财产、折价补偿不低于市价 |
| **被主张方（被要求分割一方）** | 关注个人财产范围明确（婚前证明）、婚前贡献析出、婚后自然增值不归共同财产、合理折价 |
| **中立审查** | 双向识别显失公平条款、财产清单遗漏、条款约定不明风险 |

### 3. 敏感情形检查

- 识别到财产隐匿转移迹象（异常过户、转账、资产低估）：标注 🔴 关键，提示财产保全路径，强制升级
- 识别到家庭暴力情形（影响财产分割谈判能力）：标注 `[强制升级 — 家暴敏感情形]`，参见 `family-legal/CLAUDE.md ## 共享护栏`

## 详细步骤

### 步骤一：财产清单梳理

**1.1 夫妻共同财产清单**（依据民法典第1062条，建议复核）

引导当事人逐类梳理：

| 财产类型 | 典型示例 | 权属性质判断要点 | 证明材料 |
|---------|---------|----------------|---------|
| 不动产（房产） | 婚后购买的房产 | 购买时间（婚前/婚后）、出资来源（个人/共同/父母）、登记名称 | 房产证/不动产权证/购房合同/网签备案/还贷记录 |
| 银行存款 | 婚后共同账户存款 | 工资及经营收入一般为共同财产 | 银行流水（婚后开始）/定期存款证明 |
| 车辆 | 婚后购买的机动车 | 购买时间（婚前/婚后）、出资来源 | 行驶证/购车发票/登记证书 |
| 股权与公司份额 | 婚后出资设立或增资的公司股权 | 出资来源（婚前/婚后）、股权评估价值 `[待评估]` | 工商登记信息/出资证明/审计报告 |
| 股票/基金/债券 | 婚后购买的证券投资 | 购买时间与资金来源 | 证券账户/基金账户截图（日期确认） |
| 知识产权收益 | 婚后取得的著作权/专利许可费 | **注意**：知识产权本身可能为个人，但婚后收益归共同财产 | 合同/许可协议/收款记录 |
| 保险现金价值 | 婚后缴费的寿险/年金保险 | 婚后缴纳部分的现金价值 | 保险合同/缴费记录/现金价值证明 |
| 对他人债权 | 婚后共同借出的款项 | 出借时间与资金来源 | 借条/转账记录/法院判决 |

**1.2 夫妻个人财产清单**（依据民法典第1063条，建议复核）

| 财产类型 | 认定要点 | 证明材料 |
|---------|---------|---------|
| 婚前房产 | 购买于婚前，或婚前签订合同/付款、婚后取得登记 | 房产证（登记日期）/购房合同（签订日期）/付款记录 |
| 婚前存款 | 明确为婚前积累的存款 | 银行流水（婚前开始，金额对应） |
| 人身损害赔偿 | 一方因人身损害获得的赔偿/补偿（含残疾赔偿金/医疗费/精神损害抚慰金） | 法院判决/调解协议/赔偿协议 |
| 遗嘱/赠与明确归一方 | 遗嘱或赠与合同明确只归一方的财产 | 遗嘱文件/赠与合同（明确「只归一方」） |
| 一方专用生活用品 | 个人日常使用的生活用品（手表/首饰等一般不超过个人日常消费范围） | 购买凭证 |

**特别注意——婚前房产婚后增值**：婚前个人购买，婚后价值增长，增值部分性质存在争议。司法实践中，自然增值（市场行情增长）一般仍属个人财产，主动增值（婚后共同装修/改扩建）部分可能涉及补偿。具体处理须结合司法解释（**描述规则，具体条号 `[待验证]`**）和当地裁判口径（`[地方 flagged — 需核实当地裁判口径]`）。

### 步骤二：各类财产分割条款拟定

**2.1 房产分割条款要点**

| 场景 | 分割方案选项 | 条款要点 |
|------|------------|---------|
| 共同购买，双方都想要 | 折价补偿（一方支付另一方市值一半） | 评估价值确定方式（双方协议/评估机构）、补偿金额、支付期限、过户时间 |
| 共同购买，一方不想要 | 直接归属（归一方，支付折价） | 同上 |
| 一方婚前购买，婚后共同还贷 | 归产权方，另一方获还贷补偿 | 婚后还贷总额、增值分摊（描述规则，条号 `[待验证]`），补偿金额须明确计算依据 |
| 父母出资 | 按出资性质（赠与/借款）确定 | 须明确父母出资属于对哪方的赠与，或双方均等赠与，影响共同财产认定 |

**房产分割条款模板结构**：
```
第X条 [房产描述（地址、不动产权证号）] 归 [甲/乙] 方所有，
[甲/乙] 方于本协议生效后 [X] 个工作日内协助办理产权过户手续，
过户所需税费由 [甲/乙/双方按比例] 承担。
[另一方] 在 [日期] 前一次性支付 [金额] 元作为折价补偿。
房产上现有抵押贷款余额由 [甲/乙/双方] 负责还清。
```

**2.2 存款分割条款要点**

- 明确分割的账户（银行+账号后四位）和金额
- 注意：婚后共同积累的存款为共同财产，婚前存款须提供证明
- 条款须写明划转方式（转账/提现）和时间

**2.3 股权/公司份额分割条款要点**

- 股权价值须通过评估确定（`[待评估]`，建议聘请评估机构），不得单方低价定价
- 分割方式：折价补偿（股权归一方，支付等值补偿）或股权转让（须公司章程/其他股东审查）
- **注意**：公司股权分割须防止影响公司正常经营，条款须明确过渡安排

**2.4 投资类资产分割条款（股票/基金/债券）**

- 以协议签署日或约定日期的市值为准（明确取值时间点防争议）
- 证券账户直接归属的，明确账户归属与转移方式
- 不能直接分割的，一方折价补偿另一方

**2.5 知识产权收益分割**

- 婚后已发生的收益归共同财产，须列明
- **注意**：知识产权本身如专利/著作权，如系婚后取得但具有人身属性，归属处理须谨慎
- 婚后尚未支付但已到期的许可费、版税，确认归属

### 步骤三：隐瞒财产救济保留条款

**强烈建议在协议中加入以下救济保留条款**（依据民法典第1092条，建议复核）：

```
第X条（财产如实申报与后续救济）
双方承诺本协议所列财产清单系夫妻共同财产与个人财产的完整申报，
未存在隐藏、转移、变卖、毁损或挥霍夫妻共同财产，
也未伪造共同债务的行为。

如一方日后发现另一方在协议签署前存在上述行为，
受损害方保留向人民法院提起诉讼、请求再次分割的权利，
不受本协议财产分割条款约束。
（法律依据：民法典第1092条，建议复核）
```

**主张方重点**：此条款保障日后发现隐匿财产时可再次主张；建议同时在协议中写入「对方有如实申报义务，如日后发现隐瞒财产，按隐瞒价值的一定比例赔偿」（具体约定，须律师审核合法性）。

**被主张方重点**：如对方声称发现未申报财产，须有书面证据支持，不能凭口头主张撤销已签协议；协议中明确确认「本协议为双方真实意思表示，对财产清单已充分知情」以防对方事后滥用第1092条。

## 边界条件

| 情形 | 处理方式 | 升级/标注 |
|------|---------|---------|
| 婚前房产婚后增值自然增值 vs 主动增值 | 区分处理，自然增值通常仍归个人，主动增值可能涉及补偿；须结合司法解释（`[待验证]`）和当地口径 | `[地方 flagged — 需核实当地裁判口径]` |
| 约定财产制（第1065条）与共同财产认定的关系 | 约定优先适用，但约定仅对夫妻双方有效，不当然对抗善意第三人 | 须律师核实约定书面形式与效力 |
| 复杂特殊财产（公司股权/知识产权/海外资产） | 超出本子技能范围，参见 property-division 主技能（08-03）子技能 | 强烈建议律师专项处理 |
| 双方对财产价值存在重大分歧 | 建议聘请第三方评估机构，双方确认评估价值后再起草条款 | — |
| 一方不配合提供财产清单 | 提示通过诉讼路径申请调查令/财产保全；本子技能暂时记录已知财产 | 升级律师 |
| 房产有未偿还贷款 | 条款须明确贷款清偿责任（继续还贷方与贷款行的三方协议可能须另行办理）；银行通常不承认离婚协议内部分担约定 | `[待验证 — 需核实银行要求]` |

## 错误处理

| 错误类型 | 识别信号 | 处理方式 |
|---------|---------|---------|
| 财产清单明显不完整 | 描述的财产种类/数量与婚姻年限/生活水平不符 | 提示可能存在遗漏，逐类追问；不得以不完整清单起草条款 |
| 疑似财产隐匿（🔴关键） | 资产异常低价转让、离婚前大额转账/赠与第三方、异常收益未申报 | 标注 🔴 关键，提示财产保全申请，**强制升级律师** |
| 家暴影响财产谈判能力（🔴关键） | 识别到家暴情形，当事人被迫接受不利条款 | **强制升级 — 家暴敏感情形**，参见 `family-legal/CLAUDE.md ## 共享护栏` |
| 条款约定明显显失公平 | 一方所得财产价值远低于应得份额 | 标注 🟠 高风险，提示当事人该条款可能被认定可撤销（须律师评估） |
| 旧法条号引用（如原婚姻法第17条） | 用户或材料引用旧法条号 | 自动纠正为民法典第1062/1063条（建议复核），说明旧法已废止 |

## 法律依据

### 法条（法律）

**夫妻共同财产范围 — 民法典第1062条**（已核实锚点，中度把握，建议复核）
- 《中华人民共和国民法典》**第1062条**  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
- 工资奖金/生产经营投资收益/知识产权收益/继承赠与所得（除明确归一方外）为共同财产

**夫妻个人财产 — 民法典第1063条**（已核实锚点，中度把握，建议复核）
- 《中华人民共和国民法典》**第1063条**  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
- 婚前财产/人身损害赔偿/遗嘱赠与明确归一方/一方专用生活用品为个人财产

**约定财产制 — 民法典第1065条**（已核实锚点，中度把握，建议复核）
