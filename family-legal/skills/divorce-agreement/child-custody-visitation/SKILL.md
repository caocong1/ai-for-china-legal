# 子女抚养与探望（child-custody-visitation）

> **未成年人利益最大化护栏贯穿本技能全程。** 本技能涉及未成年子女的核心权益，须以「最有利于未成年子女」为首要原则（民法典第1084条，建议复核），不得仅按父母博弈结论处理。**识别到家庭暴力、抢夺藏匿子女、当事人情绪危机等敏感情形须立即强制升级。**
>
> **可独立触发**：本技能可由 `/family-legal:divorce-agreement` 编排入口调用，也可单独触发（`/family-legal:child-custody-visitation`）。
> **前置依赖**：读取 `family-legal/CLAUDE.md` 实践配置（代理立场·敏感度校准·未成年人利益护栏），引用 `family-legal/skills/_shared/family-law-citations.md` 引用库。
> **分工说明**：本技能专注子女抚养与探望；财产分割由 `property-division-clauses` 处理；协议生效与登记由 `effectiveness-registration` 处理。

---

## 目的

分析并起草离婚协议中的子女抚养权归属条款、抚养费标准与支付安排、探望权方式与时间安排，以及抚养关系变更情形约定。核心原则：**最有利于未成年子女**，在父母双方的立场差异中守住子女利益底线。

## 前置

### 1. 读取实践配置

读取 `family-legal/CLAUDE.md`，重点检查：

- **整体代理立场**：离婚一方侧（主张抚养方/被主张方）/ 中立审查
- **涉未成年子女事项**（`## 敏感度校准`）：是否有涉未成年人特殊情形
- **家暴识别等级**：是否有家暴或虐待子女情形

如果配置缺失，以临时模式继续，**敏感度默认最高，未成年人利益最大化护栏强制开启**。

### 2. 未成年人利益最大化护栏（贯穿全程）

> **本护栏不得被任何代理立场覆盖。**

无论代理哪方，以下原则须始终贯穿：

- 子女有独立的利益主体地位，其利益不等同于父母任何一方的利益
- 任何抚养权安排须以「最有利于未成年子女的生活成长」为判断标准，不是以「父母哪方更强势」为判断标准
- 已满8周岁子女的真实意愿须被尊重，不得替代子女表达意见
- 即使为被代理方，若某安排损害子女利益，须明确标注并建议调整

### 3. 代理立场区分

| 立场 | 审查侧重 |
|------|---------|
| **主张抚养方（争取抚养权一方）** | 关注抚养权归属论证（年龄/生活环境/抚养能力）、抚养费标准不低于合理水平、探望权安排合理不过度 |
| **被主张方（可能失去抚养权一方）** | 关注探望权充分保障、探望时间/地点/方式明确不受阻挠、抚养费标准合理 |
| **中立审查** | 双向识别：双方能力综合评估、子女意愿、探望安排可执行性 |

### 4. 敏感情形识别

- **家庭暴力/虐待子女**（🔴关键）：若一方有对子女实施暴力/虐待的情形，须**强制升级**，家暴情形下不宜将抚养权判给施暴方
- **抢夺藏匿子女**（🔴关键）：识别到一方在离婚过程中擅自带走/藏匿子女，标注 🔴 关键，**强制升级律师**，提示申请人身安全保护令及扣押子女令
- **当事人情绪极度激动**（🟠高）：确保分析不激化对立，措辞中立克制

## 详细步骤

### 步骤一：子女基本情况核查

收集以下信息：

| 信息项 | 内容 | 对抚养权判断的影响 |
|--------|------|-----------------|
| 子女年龄 | 出生年月 | 不满2周岁：母亲原则；2-8周岁：综合考量；8周岁以上：参考子女意愿 |
| 就读情况 | 学校/幼儿园/所在城市 | 影响抚养权归属（通常以不变动子女生活学习环境为佳） |
| 与父母同住情况 | 目前随哪方生活 | 已稳定同住的一方通常有优势 |
| 子女健康状况 | 有无特殊疾病/残障/心理状况 | 特殊需求可能要求特定医疗资源或照护能力 |
| 子女意愿（8周岁以上） | 子女希望跟谁生活 | 须尊重（第1084条），但不等于绝对服从；须核实意愿的真实性与稳定性 |

### 步骤二：抚养权归属分析

**2.1 法定抚养权原则**（依据民法典第1084条，建议复核）

| 子女年龄段 | 法定原则 |
|-----------|---------|
| 不满2周岁 | 以由**母亲直接抚养为原则**；但母亲有重大疾病/不利于子女成长的特殊情形除外（具体情形描述规则，条号 `[待验证]`） |
| 2-8周岁 | 按**最有利于未成年子女的原则**判定，综合考量双方条件 |
| 8周岁以上 | 应**尊重子女的真实意愿** |

**2.2 抚养权归属综合考量因素**

| 考量因素 | 说明 |
|---------|------|
| 稳定的生活成长环境 | 哪方能为子女提供稳定住所、学校不变化 |
| 抚养能力与日常照料 | 哪方实际承担日常照料（接送上学/辅导作业/医疗护理） |
| 经济条件 | 哪方能提供充足的经济保障（注意：经济条件不是唯一因素） |
| 品德与教育能力 | 双方的品行、教育理念与子女的契合度 |
| 工作时间与陪伴能力 | 哪方有更多时间陪伴子女 |
| 与子女的感情基础 | 子女与哪方的感情更深厚（注意家暴/虐待情形须过滤） |
| **家庭暴力情形** | 有家暴/虐待记录的一方通常不应获得直接抚养权 |
| 是否有意阻断另一方与子女联系 | 抢夺藏匿子女、阻止探望的一方抚养能力存疑 |

**2.3 主要抚养权协议条款拟定**

```
第X条（子女抚养权）
[子女姓名]（出生日期：[日期]），离婚后由[甲/乙]方直接抚养。
[对方]仍享有探望权，双方均有继续履行对子女抚养、教育、保护的义务。
```

**注意**：「直接抚养」（与子女同住、日常照料）与「法定监护权」（父母双方均自然享有）须区分，离婚不改变父母均为监护人的法律地位。

### 步骤三：抚养费标准与支付安排

**3.1 抚养费标准参考**（司法实践惯例，具体数字须核实当地裁判口径）

司法实践中一般参考月均收入比例确定，常见标准（**描述规则，具体比例须核实现行当地裁判口径 `[地方 flagged — 需核实当地裁判口径]`**）：

- 一个子女：月均收入的一定比例（各地惯例有差异）
- 多个子女：按子女数量适当调整比例
- 固定收入参考工资流水；非固定收入（经营/自由职业）参考年均收入核算月均

**3.2 抚养费支付条款要素**

| 条款要素 | 说明 |
|---------|------|
| 支付金额 | 明确每月金额（如约定调整机制也须明确触发条件） |
| 支付方式 | 银行转账到指定账户（保留流水）；避免现金以防争议 |
| 支付时间 | 每月X日前到账 |
| 支付期限 | 至子女年满18周岁（或完成大学教育，须明确上限） |
| 大额费用分担 | 教育（择校费/学杂费）/医疗（重大疾病/手术）/其他重大支出如何分担（建议另行协商或按比例） |
| 调整机制 | 一方收入变化超过一定比例时，可申请法院变更抚养费（无需写入协议，法律已有规定，但可约定双方协商优先） |

**3.3 抚养费条款模板**：

```
第X条（抚养费）
[甲/乙]方每月向另一方支付[子女姓名]的抚养费人民币 [金额] 元，
于每月[X]日前转入[甲/乙]方名下银行账户（账号：[XXXX]）。
上述抚养费支付至[子女姓名]年满十八周岁之日止，
或在此之前完成[大学本科]教育止（以较晚者为准，双方另行协商）。

以下费用由双方各承担50%（或约定比例）：
- 每学年学杂费（学费/住宿费）超过 [金额] 元的部分
- 重大疾病（单次医疗费超过 [金额] 元）的医疗费用
其他重大支出经双方协商决定。
```

### 步骤四：探望权安排条款

**4.1 探望权的法律基础**（民法典第1086条，中度把握，建议复核）

不直接抚养方有探望权，另一方有协助义务。探望不利于子女身心健康的，可申请法院中止。

**4.2 探望权安排要素**

| 安排要素 | 常见方案示例 | 注意事项 |
|---------|------------|---------|
| 常规探望频率 | 每两周一次（周末）；或每周X、X | 须具体明确，避免「适时探望」等模糊表述 |
| 探望时长 | 每次 [X] 天（如周六上午至周日晚）| 注意学习日安排对子女的影响 |
| 探望地点 | 接送地点（如学校/约定地点）；是否可带子女过夜 | 注意异地探望的交通安排 |
| 假期探望 | 寒暑假轮流/平分；法定节假日安排 | 明确哪一年哪方优先 |
| 接送安排 | 由哪方负责接送，或自行安排 | 异地须明确交通费承担 |
| 通讯联系 | 非探望期间电话/视频联系频率 | 建议写入「每周不少于X次电话/视频」 |

**4.3 探望权条款模板**：

```
第X条（探望权）
[未直接抚养方（甲/乙）]享有探望[子女姓名]的权利，
[直接抚养方]有协助义务，不得无故阻止探望。

常规探望安排：
（一）每[两周/月]一次，时间为[周六]上午[10:00]起至[周日]晚[18:00]止；
（二）在[指定地点/学校]接送；
（三）寒暑假及法定长假期间，双方另行协商安排（原则上各一半时间）；
（四）非探望期间，[甲/乙]方可通过电话/视频与子女保持联系，每周不少于[X]次。

如上述探望安排因子女升学、工作变动等重大情况需要调整，
双方协商一致后可变更，如协商不成可申请法院调整。
```

**未成年人保护提示**：探望安排须以子女的生活秩序和心理健康为首要考量；如探望方有家暴或虐待行为，另一方可向法院申请中止探望（须证据支持）。

### 步骤五：抚养关系变更情形约定

**法律允许在以下情形变更抚养权**（条号 `[待验证]`，描述规则）：

| 变更情形 | 说明 |
|---------|------|
| 直接抚养方患重大疾病 | 无力继续抚养子女 |
| 直接抚养方虐待、放弃抚养 | 严重损害子女利益 |
| 子女已满8周岁明确表示愿与另一方生活 | 尊重子女真实意愿 |
| 直接抚养方就学/工作安排明显不利于子女成长 | 须综合评估 |

**建议在协议中加入**：

```
第X条（抚养关系变更）
双方同意，如发生以下情形之一，另一方可要求变更直接抚养关系，
双方优先协商；协商不成可申请人民法院调整：
（一）直接抚养方患重大疾病或无力继续抚养；
（二）直接抚养方虐待子女或严重损害子女身心健康；
（三）[子女姓名]年满八周岁后明确表示愿与另一方生活。
```
