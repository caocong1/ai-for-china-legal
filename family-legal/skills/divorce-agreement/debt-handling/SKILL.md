# 债务处理（debt-handling）

> **可独立触发**：本技能可由 `/family-legal:divorce-agreement` 编排入口调用，也可单独触发（`/family-legal:debt-handling`）。
> **前置依赖**：读取 `family-legal/CLAUDE.md` 实践配置（代理立场·协议审查严格度·敏感度校准），引用 `family-legal/skills/_shared/family-law-citations.md` 引用库。
> **分工说明**：本技能专注债务处理；财产分割由 `property-division-clauses` 处理；协议生效与登记由 `effectiveness-registration` 处理。
> **关键纪律**：夫妻共同债务认定规则来自司法解释，**描述规则，不写裸条号，具体条号 `[待验证]`**。

---

## 目的

分析离婚协议中的债务处理条款，覆盖夫妻共同债务与个人债务的认定、债务清单梳理、对外清偿责任与内部分担约定、债务条款的对外效力局限。确保债务处理条款合法合理，保护当事人不承担非共同债务，同时明确善意第三人（债权人）的权利边界。

### 2. 代理立场区分

| 立场 | 审查侧重 |
|------|---------|
| **主张方（希望债务定性为对方个人债务）** | 关注举证责任分配（债权人须证明该债务用于共同生活/经营）、认定为个人债务后本方无清偿义务 |
| **被主张方（对方主张本方承担共同债务）** | 关注共同债务认定要件（是否共同签名/追认/是否用于家庭日常生活）、个人债务剥离、虚构债务识别 |
| **中立审查** | 双向识别共同债务与个人债务的认定争议点 |

### 3. 敏感情形检查

- **虚构债务（🔴关键）**：识别到一方在离婚前虚构债务，试图侵占另一方财产——标注 🔴 关键，强制升级律师，提示作为第1092条隐匿/伪造债务处理
- **大额经营债务（🟠高）**：一方以个人名义举借大额经营性债务，涉及债权人权益——提示需要律师评估对外效力

## 详细步骤

### 步骤一：债务清单梳理

**1.1 债务类型识别与分类**

| 债务类型 | 典型示例 | 初步归类 | 证明材料 |
|---------|---------|---------|---------|
| 住房抵押贷款 | 婚后共同购房的房贷 | 通常为共同债务 | 借款合同/房贷余额证明 |
| 消费类信贷/信用卡 | 婚后的信用卡分期/消费贷 | 视用途而定（家庭日常消费一般为共同）| 信用卡账单/贷款合同 |
| 亲友借款 | 向父母/亲友借款 | 须核实：共同签名？用于家庭生活？ | 借条/转账记录 |
| 经营性债务 | 一方以个人名义为经营活动借款 | 须核实：是否超出家庭日常生活需要？债权人是否知情 | 借款合同/资金用途记录 |
| 担保债务 | 一方为第三方债务提供担保 | 个人债务（未共同签名/追认）| 担保合同/借款合同 |
| 购车贷款 | 婚后共同购买的车辆贷款 | 通常为共同债务 | 车辆贷款合同/还款记录 |

### 步骤二：共同债务认定分析

**2.1 共同债务认定规则**（最高法婚姻家庭编司法解释(一)，**描述规则，条号 `[待验证]`**）

司法解释确立了以下共同债务认定路径：

**路径一：共债共签**
夫妻**双方共同签名**，或夫妻一方所负债务经另一方**事后追认**，为夫妻共同债务。
- 适用场景：购房借款（双方签名）、大额家庭消费（双方签名）
- 举证要求：债权人举证双方签名或追认的证明

**路径二：家庭日常生活需要**
夫妻一方在婚姻关系存续期间以**个人名义**为**家庭日常生活需要**所负的债务，为夫妻共同债务。
- 适用场景：日常生活费用、子女教育费用、家庭医疗费用、日常消费类信用卡
- 「家庭日常生活需要」的范围由法院根据双方收入水平、生活消费习惯等综合判断（无固定标准，`[地方 flagged]`）

**路径三：超出家庭日常生活需要须债权人举证**
夫妻一方在婚姻关系存续期间以个人名义**超出家庭日常生活需要**所负的债务，**债权人须举证**证明该债务用于夫妻共同生活、共同生产经营，或基于夫妻双方共同的意思表示，否则不认定为夫妻共同债务。
- 适用场景：大额经营性贷款、为他人担保债务、超出家庭消费能力的大额债务
- 举证责任在债权人而非债务人配偶

**2.2 个人债务情形**

| 个人债务情形 | 认定依据 |
|------------|---------|
| 婚前以个人名义所负的债务 | 婚前债务，配偶未共同签名 |
| 婚姻关系存续期间以个人名义负债，但与家庭无关 | 个人挥霍、违法犯罪、赌博等未用于家庭生活 |
| 未共同签名且超出家庭日常生活需要、债权人无法举证用于共同生活 | 依据路径三 |
| 担保债务（未经另一方签名/追认） | 个人单方行为 |

### 步骤三：对外清偿与内部分担

**3.1 核心原则（必须理解）**

> **离婚协议约定债务由一方承担，不能对抗善意债权人（对外效力局限）。**
>
> 内部约定（如「房贷由甲方承担」）仅对双方有约束力，**善意债权人依然可以向双方主张还款**，一方代替另一方还款后，可向另一方追偿。

这一原则必须在条款中明确告知当事人，防止误解「协议约定谁承担谁就不用管」。

**3.2 对外清偿与内部分担条款结构**

对于夫妻共同债务，协议通常包含两层约定：

**第一层：内部分担（对夫妻双方有效）**
```
第X条（共同债务内部分担）
以下债务系夫妻共同债务，离婚后由[甲/乙]方负责对外清偿：
（一）[债务名称，如：XX银行住房抵押贷款]，余额约 [金额] 元，由[甲/乙]方负责按期还款；
（二）[其他债务]……

双方同意，如上述债务由[另一方]代为清偿，[代偿方]有权向[应还方]追偿全额还款及合理费用。
```

**第二层：免除连带责任说明（仅供内部参考，不能对抗债权人）**
```
（以上内部约定仅在双方之间有效。上述债务的债权人依法有权向任何一方主张还款，
本约定不影响债权人对本合同任何一方的债权主张，但应还方应向代偿方偿还。）
```

**3.3 个人债务确认条款**

```
第X条（个人债务确认）
以下债务由债务人（[甲/乙]方）在婚姻关系存续期间以个人名义所负，
不属于夫妻共同债务，由债务人自行清偿，另一方不承担连带清偿责任：
（一）[债务名称]，余额约 [金额] 元；
（二）[其他债务]……

如债权人就上述个人债务主张另一方承担连带清偿责任，
另一方有权提出异议并要求债权人出示共同债务的证明。
```

**3.4 担保债务处理**

如一方在婚姻关系存续期间未经另一方知情或同意为他人提供担保：

```
第X条（担保债务）
[甲/乙]方在婚姻关系存续期间为[债务人名称]担保的债务，
系[甲/乙]方个人行为，另一方对此不知情/未签名/未追认，
不属于夫妻共同债务，由[甲/乙]方自行承担担保责任。
```

### 步骤四：虚构债务识别与处理

**识别信号**（任一出现须标注并提示）：

- 离婚前短期内突然出现大额新债务
- 债权人为亲属/密友，缺乏真实交易背景
- 债务金额与家庭收入/消费水平严重不符
- 无法提供资金流向证明（钱用于哪里）
- 对方拒绝提供借条/转账记录/用途说明

**处理方式**：

标注 🔴 关键（第1092条，建议复核）——虚构债务属于「伪造夫妻共同债务企图侵占另一方财产」，可主张少分或不分财产，并提示当事人保留追诉权：

```
第X条（虚构债务追责）
[甲/乙]方承诺本协议所列债务系真实债务，未存在虚构、伪造债务以侵占另一方财产的行为。
如日后发现存在虚构债务行为，另一方有权依据民法典第1092条请求再次分割财产，
并保留追究[甲/乙]方相应法律责任的权利。
```

## 边界条件

| 情形 | 处理方式 | 升级/标注 |
|------|---------|---------|
| 大额举债的共同债务认定争议（路径三） | 须评估债权人能否举证资金用于共同生活；建议律师评估诉讼风险 | 建议律师介入 |
| 虚构债务（🔴关键） | 标注 🔴，强制升级；第1092条追偿路径 | **强制升级** |
| 对方拒绝披露债务 | 提示可能存在未披露债务；建议通过征信查询/法院调查令核实 | 升级律师 |
| 离婚前已产生的诉讼债务（法院判决） | 须明确该判决债务的性质（共同/个人）和清偿安排 | 建议律师确认 |
| 一方为公司债务承担连带责任 | 公司债务与个人债务的区分（公司法人格独立原则）须律师专项评估 | 升级律师 |

## 错误处理

| 错误类型 | 识别信号 | 处理方式 |
|---------|---------|---------|
| 虚构债务（🔴关键） | 大额债务突然出现/资金流向不明/债权人为亲属 | **强制升级**，提示第1092条伪造债务处理 |
| 混淆「协议约定」与「债权人无法追偿」| 当事人以为协议写明谁还款债权人就只能找那人 | 明确说明协议内部分担不对抗善意债权人 |
| 超出家庭日常需要的债务被错误认定为共同债务 | 大额经营性贷款未经共同签名被主张为共同债务 | 提示路径三：债权人举证责任，建议律师评估 |
