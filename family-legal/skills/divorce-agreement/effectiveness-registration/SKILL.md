# 协议生效与登记（effectiveness-registration）

> **可独立触发**：本技能可由 `/family-legal:divorce-agreement` 编排入口调用，也可单独触发（`/family-legal:effectiveness-registration`）。
> **前置依赖**：读取 `family-legal/CLAUDE.md` 实践配置（代理立场·协议审查严格度·敏感度校准），引用 `family-legal/skills/_shared/family-law-citations.md` 引用库。
> **分工说明**：本技能专注协议生效程序与冷静期管理；财产分割条款由 `property-division-clauses` 处理；子女抚养由 `child-custody-visitation` 处理；债务处理由 `debt-handling` 处理。
> **关键纪律**：冷静期（第1077条）、协议离婚（第1076条）为中度把握锚点；协议可撤销情形复用合同编结构参照，不得将合同编条号直接套用为婚姻家庭编规则。

## 目的

分析离婚协议的生效程序、冷静期规则、协议反悔与可撤销情形、财产分割协议与离婚登记生效的关联效力，以及协议不成时的诉讼路径衔接。确保当事人理解协议从签署到生效的全过程，及时行使冷静期撤回权利，或在协议显失公平/存在意思表示瑕疵时获得救济。

## 详细步骤

### 步骤一：协议离婚程序概览

**1.1 协议离婚法定要素**（民法典第1076条，建议复核）

协议离婚须同时满足以下三个要素，缺一不可：

| 要素 | 具体要求 | 备注 |
|------|---------|------|
| **双方自愿** | 双方均须真实自愿，不存在欺诈、胁迫 | 意思表示须真实，否则协议可撤销 |
| **书面离婚协议** | 须签订书面离婚协议，协议须载明：双方自愿离婚的意思表示 + 子女抚养安排 + 财产处理 + 债务处理 | 四项内容均须涵盖，无缺项 |
| **民政登记** | 双方须**亲自**到婚姻登记机关申请离婚登记 | 不可代理；须携带有效身份证件 |

**1.2 离婚协议必备条款核查清单**

在申请登记前，核查协议是否包含以下必备内容：

- [ ] 双方自愿离婚的意思表示
- [ ] 子女直接抚养权归属（双方均确认）
- [ ] 子女抚养费金额与支付方式（或明确无抚养费约定的说明）
- [ ] 财产处理安排（共同财产归属/个人财产确认）
- [ ] 债务处理安排（共同债务分担/个人债务确认）
- [ ] 双方签名（确认为本人亲笔签名）
- [ ] 签署日期

**重要**：婚姻登记机关审查协议是否涵盖上述要素，但通常不就实质公平性进行司法审查——这意味着条款显失公平的协议也可能通过登记，但可在登记后申请撤销。

### 步骤二：离婚冷静期规则（民法典第1077条，建议复核）

**2.1 冷静期时间轴**

```
申请登记之日 ─ [30日] ─ 冷静期结束 ─ [30日内] ─ 须共同申请发证 ─ 获发离婚证
                 ↑                           ↑
          任何一方可单方撤回          期满不申请=视为撤回
```

**2.2 冷静期关键规则**

| 规则 | 内容 |
|------|------|
| 撤回权 | 自婚姻登记机关收到离婚登记申请之日起**30日内**，任何一方均可向登记机关撤回离婚登记申请，无需说明理由 |
| 单方撤回 | 只需一方撤回即可终止该次申请，无需对方同意 |
| 期满申请 | 冷静期届满后30日内，双方须**共同亲自**向登记机关申请发给离婚证 |
| 视为撤回 | 期满后30日内未共同申请的，视为撤回（本次申请作废，如仍要离婚须重新提交） |
| 期间可补签协议 | 冷静期内若双方协商修改协议，可重新提交修改后的协议（须与登记机关确认操作流程 `[地方 flagged — 需核实当地民政局操作要求]`） |

**2.3 冷静期与诉讼离婚的关系**

> **重要区分**：协议离婚有冷静期，**诉讼离婚无冷静期**。
> 若双方已申请协议离婚且进入冷静期，一方在冷静期内撤回后，另一方可直接转向诉讼离婚（向法院提起离婚诉讼），无需等待冷静期结束。

**2.4 家暴情形的特别说明**

> 家庭暴力受害方**无需等待冷静期**，可直接向法院起诉离婚（家暴是诉讼离婚的法定准予条件，民法典第1079条）。协议离婚冷静期制度不阻断诉讼权利。同时，受害方可在冷静期内向法院申请人身安全保护令。
> 如识别到家暴情形，须**强制升级**，优先告知上述权利。

### 步骤三：协议反悔与可撤销情形

**3.1 协议离婚后反悔财产分割**

离婚协议生效后（离婚登记完成），一方对协议中**财产分割**部分反悔，须满足**可撤销情形**方可申请变更或撤销：

**可撤销的法定情形**（复用合同编意思表示瑕疵结构，**标「复用商事合同脊柱结论，建议复核」；不套合同编条号到婚姻家庭编实体规则**）：

| 可撤销情形 | 说明 | 举证要点 |
|-----------|------|---------|
| **欺诈** | 一方故意虚构事实/隐瞒真相，使对方在错误认识下签署协议（如虚报财产清单） | 须证明对方故意虚构，且受骗方在非欺诈情况下不会签署该条款 |
| **胁迫** | 一方以威胁手段迫使对方签署协议（家暴胁迫、扣押子女等） | 须证明胁迫行为与签署协议的因果关系 |
| **重大误解** | 对协议重大条款存在根本性误解（如误解财产范围） | 较难证明，须区分「自愿放弃」与「误解」 |
| **显失公平** | 协议签署时一方有优势地位，内容严重不公平 | 须证明签署时存在显失公平情形，仅事后悔改不构成 |

**3.2 申请撤销的期限**（描述规则，具体期限须核实现行法律规定 `[待验证]`）

申请撤销须在法定期限内（自知道或应当知道可撤销情形之日起计算），超过期限丧失撤销权。具体期限须核实《中华人民共和国民法典》合同编相关规定或婚姻家庭编司法解释。

**3.3 仅就财产分割条款反悔（不涉及离婚本身）**

> **重要**：协议离婚已完成（取得离婚证），一方仅对财产分割部分反悔，不影响离婚的效力——即不能以此为由「撤销」整个离婚。可撤销的仅为财产分割条款，离婚本身已成立。

### 步骤四：财产分割协议与离婚登记的关联效力

**4.1 关联效力原则**

> **财产分割协议以离婚登记完成为生效条件（附解除条件/停止条件）。**
>
> 若双方签署了包含财产分割约定的离婚协议，但最终**未办理离婚登记**（冷静期撤回/一方不愿），财产分割协议不独立生效——因为该协议是以「离婚」为前提缔结的，离婚未成立，财产分割协议也不产生分割效力。

**4.2 实践中的关联效力问题**

| 场景 | 效力结论 |
|------|---------|
| 已签协议，已完成离婚登记 | 协议（含财产分割）全面生效 |
| 已签协议，冷静期内撤回 | 协议不生效（无论是哪方撤回），双方重新谈判 |
| 已签协议，一方在冷静期后不去发证 | 视为撤回，协议不生效 |
| 已签协议，但未提交登记即发生纠纷 | 协议效力存在争议，须律师评估（已签订但未生效） |

**4.3 协议内容须完整处理方可申请登记**

- 登记机关通常不接受遗漏子女抚养安排、财产处理或债务处理的协议
- 如当前无子女/无共同财产/无共同债务，须在协议中明确写明「双方无共同子女/无共同财产/无共同债务」而非跳过该条款

### 步骤五：协议不成时转诉讼路径提示

如双方就离婚协议条款（特别是财产分割、子女抚养）无法达成一致，可转诉讼离婚路径：

**诉讼离婚路径**（民法典第1079条，建议复核）：

- 一方向有管辖权的基层人民法院（家事审判庭）提起离婚诉讼
- 法院须先行调解；调解不成且感情确已破裂的，应准予离婚
- 法庭判决抚养权与财产分割，无需双方同意（但须法院审理）
- **诉讼离婚无冷静期**，但通常审理周期较长（数月至一年以上）
- 初次起诉法院判决不准离婚，经过一年分居后再次起诉须准予（第1079条第5款，建议复核）

**建议提示当事人**：

```
协议离婚与诉讼离婚对比说明：

协议离婚：双方自愿+协议+登记（通常更快，成本低，但须双方全程配合）
诉讼离婚：一方起诉+法院调解/审理+判决（可单方推进，但周期较长，须律师代理）

如协议谈判陷入僵局，尤其是财产分割或子女抚养存在重大分歧，建议：
1. 考虑聘请家事调解机构协助调解（成本低于诉讼）
2. 若调解失败或一方有家暴等紧急情形，直接启动诉讼离婚程序
```

## 边界条件

| 情形 | 处理方式 | 升级/标注 |
|------|---------|---------|
| 家暴情形下冷静期 | 明确告知：无需等待冷静期，可直接诉讼，且可申请人身安全保护令 | **强制升级** |
| 冷静期内一方撤回后另一方转诉讼 | 说明：撤回协议离婚申请后，另一方可立即提起诉讼离婚，无需等待 | — |
| 协议附条件（如「子女考上大学后离婚」） | 附条件离婚协议的效力存在争议；婚姻登记不接受附条件申请 | 须律师评估 + `[待验证]` |
| 外籍当事人 | 涉外离婚有特殊程序（如在华登记的须回登记地办理）；须核实 `[待验证 — 涉外婚姻须核实特殊程序]` | 建议律师专项处理 |

## 错误处理

| 错误类型 | 识别信号 | 处理方式 |
|---------|---------|---------|
| 家暴情形误以为须等冷静期（🔴关键） | 当事人表示「还有X天冷静期结束才能离婚」但有家暴 | **强制升级**，立即告知诉讼路径与人身安全保护令 |
| 协议遗漏子女/财产/债务任一条款 | 提交登记前发现缺项 | 提示补充，缺项协议不符合第1076条要求 |
| 当事人误解「协议约定债务由一方还」即可免责 | 以为对方不找自己还了 | 说明协议内部分担不对抗善意债权人 |
| 混淆冷静期与协议撤销期限 | 以为冷静期就是撤销期 | 明确区分：冷静期是撤回登记申请权利；协议可撤销须满足欺诈/胁迫等情形并在期限内申请 |
| 旧法条号引用 | 引用原婚姻法第31/32条等 | 自动纠正为民法典第1076/1077/1079条（建议复核） |
