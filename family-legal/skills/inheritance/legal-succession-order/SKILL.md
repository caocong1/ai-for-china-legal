# 法定继承顺序与份额（legal-succession-order）

> **可独立触发**：本技能可由 `/family-legal:inheritance` 编排入口调用，也可由律师/法务/继承人单独触发（`/family-legal:legal-succession-order`），无需运行整个继承权分析流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载整体代理立场（`## 审查与代理立场` → 某继承人侧/中立）、敏感度校准、风险校准定义。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1127条法定继承顺序、第1129条丧偶儿媳女婿、第1130条同顺序份额均等与照顾酌情）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约，代理立场字段）。
>
> **分工说明**：本技能专注法定继承顺序与份额（仅限无有效遗嘱或遗嘱无效时）；遗嘱继承效力见 `testate-succession-validity`；遗产范围与债务清偿见 `estate-scope-debt`；代位继承与转继承见 `subrogation-transmission`。

---

## 目的

在无有效遗嘱（或遗嘱无效、仅部分有效）时，依法确定法定继承人资格与继承顺序，计算各继承人应继份额，识别酌情多分或少分情形，判断丧偶儿媳女婿的第一顺序继承地位，并按代理立场调整份额主张侧重。

本技能覆盖以下核心分析：

1. 法定继承启动条件确认（无遗嘱 / 遗嘱无效 / 部分法定继承）
2. 第一顺序继承人范围核查（配偶、子女、父母的扩大定义）
3. 第二顺序继承人范围核查（仅第一顺序无人继承时启用）
4. 同顺序继承份额计算（一般均等原则 + 三类酌情情形）
5. 丧偶儿媳/女婿的第一顺序继承地位判定
6. 按代理立场区分份额主张侧重

### 2. 代理立场区分

法定继承分析的代理立场决定份额主张侧重：

| 立场 | 审查侧重 |
|------|---------|
| **代理第一顺序继承人（如配偶/子女）** | 关注其继承资格确立、同顺序多分情形（尽主要扶养义务、与被继承人共同生活）、丧偶儿媳女婿资格排除或确立 |
| **代理第二顺序继承人（如兄弟姐妹）** | 关注第一顺序继承人是否全部缺失或放弃，第二顺序启用条件 |
| **代理丧偶儿媳/女婿** | 关注尽主要赡养义务的证据材料、第一顺序继承地位确立 |
| **中立审查** | 双向列示各继承人份额、识别可能争议的多分少分情形 |

### 3. 敏感情形检查

- 识别到继承人纠纷激化迹象（如遗嘱真实性争议、继承人间肢体冲突风险）：标注 🟠 高风险，提示冷静分析
- 识别到涉刑情形（如继承人故意杀害被继承人、以欺诈胁迫手段争夺遗产）：标注 🔴 关键，**强制升级给刑事律师**，参见 `family-legal/CLAUDE.md ## 共享护栏`
- 识别到丧失继承权争议（第1125条）：明确提示须通过法院认定程序，不得自行宣布

## 详细步骤

### 步骤一：法定继承启动条件确认

#### 1.1 法定继承的适用场景

| 场景 | 说明 | 分析路径 |
|------|------|---------|
| 无遗嘱 | 被继承人未立遗嘱，全部遗产依法定继承分配 | 直接进行法定继承顺序与份额分析 |
| 遗嘱全部无效 | 遗嘱因形式或实质要件瑕疵全部无效 | 视同无遗嘱，全部法定继承 |
| 遗嘱部分无效 | 遗嘱部分有效，无效部分按法定继承分配 | 先确认遗嘱有效范围（调用 `testate-succession-validity`），再对无效部分法定继承 |
| 遗嘱继承人放弃继承 | 遗嘱指定继承人放弃，该部分按法定继承处理 | 确认放弃声明的效力，余额法定继承 |
| 无人受遗赠 | 受遗赠人放弃或先于遗嘱人死亡，遗赠财产归法定继承 | 遗赠部分按法定继承分配 |

**关键判断**：确认是否存在**遗赠扶养协议**（民法典第1158条，建议复核）。遗赠扶养协议效力优先于遗嘱和法定继承，须先扣除遗赠扶养协议部分，剩余遗产再按法定继承处理。

#### 1.2 先析产后继承原则

在进行法定继承前，须先完成遗产析产：

1. 被继承人遗产中若含夫妻共同财产，先分出配偶应得的一半
2. 被继承人遗产中若含其他共有财产（如与他人合伙财产），先完成析产
3. 析产后属于被继承人的个人部分才是法定继承的遗产

> **析产步骤**：具体遗产范围认定与析产计算见 `estate-scope-debt` 子技能。本技能以析产完成后的遗产为基础进行继承顺序与份额分析。

### 步骤二：第一顺序继承人核查

#### 2.1 第一顺序继承人范围

依据民法典第1127条（已核实锚点，中度把握，建议复核），第一顺序继承人包括：

**配偶**（须满足以下条件之一）：

| 资格条件 | 说明 |
|---------|------|
| 合法婚姻存续 | 被继承人死亡时，双方处于有效婚姻关系中（已登记结婚，未协议离婚/未诉讼离婚判决生效） |
| 注意：无效婚姻/可撤销婚姻 | 若婚姻被认定为无效或已被撤销，配偶继承资格丧失——须核实婚姻效力 |
| 注意：事实婚姻 | 1994年2月1日前已符合婚姻实质要件的事实婚姻，可能具有继承资格（描述规则，条号 `[待验证]`） |

**子女**（包含以下类型，均具有平等继承权）：

| 类型 | 说明 | 注意事项 |
|------|------|---------|
| 婚生子女 | 父母婚内所生 | 无特殊要求 |
| 非婚生子女 | 父母未婚所生 | 与婚生子女继承权平等，须证明亲子关系 |
| 养子女 | 依法形成收养关系的子女 | 须有合法收养手续（民政登记或法院认定） |
| 有扶养关系的继子女 | 继父母与继子女之间形成了事实扶养关系 | **须证明扶养关系**；无扶养关系的继子女不继承继父母遗产（反之亦然） |

**父母**（包含以下类型）：

| 类型 | 说明 | 注意事项 |
|------|------|---------|
| 生父母 | 亲生父母（含离婚后的父母） | 离婚不影响对子女的继承权 |
| 养父母 | 依法形成收养关系的父母 | 须有合法收养手续 |
| 有扶养关系的继父母 | 继父或继母与被继承人之间形成了事实扶养关系 | **须证明扶养关系**；无扶养关系的继父母不具备继承资格 |

> **继承资格核查清单（第一顺序）**：
>
> - [ ] 配偶：婚姻状态确认（死亡证明日期 vs 离婚证/判决书生效日期）
> - [ ] 子女：亲子关系证明（户口本/出生证明/亲子鉴定/收养手续）
> - [ ] 父母：扶养关系确认（养父母/继父母须证明扶养关系）
> - [ ] 各继承人是否存在丧失继承权情形（第1125条，建议复核，见步骤五）
> - [ ] 各继承人是否已放弃继承（须书面声明，并在遗产分割前作出）

#### 2.2 丧偶儿媳/女婿的第一顺序继承地位

依据民法典第1129条（中度把握，建议复核）：

**适用条件**：

| 条件 | 具体要求 |
|------|---------|
| 身份要求 | 丧偶儿媳（丈夫已死亡）或丧偶女婿（妻子已死亡） |
| 亲属关系 | 丧偶儿媳对公婆尽了主要赡养义务；丧偶女婿对岳父母尽了主要赡养义务 |
| 「主要赡养义务」 | 须由法院综合认定（无明确条件定义，通常考量：共同居住时间、实际照料程度、经济支持情况） |

**证明材料建议**：

- 共同居住证明（户籍同址、物业记录、邻居证言）
- 经济支持记录（生活费转账记录、购物票据）
- 医疗照料证明（医院陪护记录、护工证言）
- 其他家庭成员证言（兄弟姐妹对赡养贡献的认可声明）

**代理侧重**：
- 代理丧偶儿媳/女婿：收集上述证明，主张第一顺序地位
- 代理其他第一顺序继承人抗辩：质疑「主要」赡养义务认定，举反证（其他继承人实际尽了更多义务）

### 步骤三：第二顺序继承人核查

#### 3.1 第二顺序继承人范围

依据民法典第1127条（已核实锚点，中度把握，建议复核），第二顺序继承人包括：

| 类型 | 扩大定义 |
|------|---------|
| 兄弟姐妹 | 包括同父母的兄弟姐妹、同父异母或同母异父的兄弟姐妹、养兄弟姐妹、有扶养关系的继兄弟姐妹 |
| 祖父母 | 父方祖父母 |
| 外祖父母 | 母方外祖父母 |

**第二顺序启动条件**：

第二顺序继承人仅在**无第一顺序继承人**时继承。具体情形包括：

- 第一顺序继承人全部先于被继承人死亡（且无代位继承人）
- 第一顺序继承人全部丧失继承权（第1125条）
- 第一顺序继承人全部放弃继承（须书面声明）
- 被继承人无配偶、无子女、无父母（含养父母、有扶养关系继父母）

> **注意**：只要有**任何一名**有效的第一顺序继承人，第二顺序继承人即不继承。

### 步骤四：同顺序继承份额计算

#### 4.1 一般均等原则

依据民法典第1130条（中度把握，建议复核）：

同一顺序继承人继承遗产的份额，**一般应当均等**。

**示例**：被继承人死亡，遗产（析产后）为 120 万元，无有效遗嘱。第一顺序继承人：配偶甲、子女乙、父母丙丁，共 4 人。一般均等份额：每人 30 万元。

#### 4.2 酌情照顾多分情形

| 情形 | 法律依据 | 说明 | 举证方向 |
|------|---------|------|---------|
| 对生活有特殊困难且缺乏劳动能力的继承人 | 第1130条（建议复核） | **须同时满足**「特殊困难」和「缺乏劳动能力」两个条件；「应当予以照顾」为强制性规定 | 残疾证明、无业证明、生活困难陈述、医疗诊断书 |
| 对被继承人尽了主要扶养义务或与被继承人共同生活 | 第1130条（建议复核） | 「可以多分」为授权性规定（非强制，由继承人协商或法院裁量）；须证明「主要」扶养义务 | 共同居住证明、赡养费记录、照料证明 |

> **重要区分**：
> - 「缺乏劳动能力又有特殊困难」的照顾是**强制性**的（「应当」）
> - 「尽主要扶养义务」的多分是**授权性**的（「可以」）——可由继承人协商，协商不成由法院酌情判决

#### 4.3 酌情不分或少分情形

| 情形 | 法律依据 | 说明 |
|------|---------|------|
| 有扶养能力和有扶养条件但不尽扶养义务 | 第1130条（建议复核） | 「应当不分或者少分」为强制性规定；须同时满足「有能力」「有条件」「不尽义务」三要素 |

**认定说明**：
- 「有扶养能力」：有经济来源、劳动能力
- 「有扶养条件」：居住距离合理、无客观障碍
- 「不尽扶养义务」：长期不探视、不提供生活费、不协助医疗照料等

**代理侧重**：
- 代理尽了主要义务一方：收集多分证据，主张多分情形
- 代理被主张不尽义务一方：举反证（如客观原因导致无法尽义务、实际已尽部分义务）

#### 4.4 继承人协商一致的不均等分配

依据第1130条（建议复核），继承人协商同意的，也可以不均等分配。协商结果须在遗产分割协议中明确记载。

### 步骤五：丧失继承权审查

依据民法典第1125条（中度把握，建议复核），继承人有下列情形之一的，丧失继承权：

| 情形 | 说明 | 涉刑风险 |
|------|------|---------|
| 故意杀害被继承人 | 刑事定罪或民事法院认定 | 🔴 强制升级刑事律师 |
| 为争夺遗产杀害其他继承人 | 刑事定罪或民事认定 | 🔴 强制升级刑事律师 |
| 遗弃被继承人或虐待情节严重 | 情节严重的认定由法院裁量 | 🟠 可能涉刑，需评估 |
| 伪造、篡改、隐匿或销毁遗嘱情节严重 | 情节严重的认定由法院裁量 | 🟠 高风险 |
| 以欺诈胁迫手段迫使或妨碍被继承人设立、变更或撤回遗嘱情节严重 | 情节严重由法院裁量 | 🟠 高风险 |

**丧失继承权的恢复**：被继承人对上述部分情形（如虐待后有悔改表现）可予宽恕，须被继承人明示宽恕（建议在遗嘱中体现）；被继承人死后不得宽恕。

**程序说明**：丧失继承权不由继承人自行主张，须通过法院诉讼程序认定。AI 工具不得自行宣布某继承人丧失继承权，仅提供风险提示，**强制升级给律师**后由律师决定是否提起相关诉讼。

### 第一顺序继承人

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 继承人资格文件不齐全（如无出生证明、无收养手续） | 提示需要补充的材料清单；核实前在结论中标注 `⚠️ 资格未经完整文件确认` |
| 被继承人婚姻状态不明确（如疑似事实婚姻、离婚协议真实性存疑） | 暂停继承份额计算，提示须先通过法院确认婚姻效力；标注 🟠 高风险 |
| 发现涉刑情形（故意杀害被继承人等） | **立即暂停**所有继承份额分析，标注 🔴 关键，**强制升级刑事律师** |
| 丧失继承权争议（如有人主张某继承人应丧失继承权） | 提示丧失继承权须经法院认定程序，AI工具不得自行认定；标注 🟠 高风险，升级律师处理诉讼 |
| 无法确定遗产范围（析产尚未完成） | 先触发 `estate-scope-debt` 子技能完成析产，再回到本技能进行继承分析 |
| 继承人情绪危机（丧亲悲痛，无法冷静分析） | 以中立克制措辞呈现，暂停分析复杂争议情形，提示寻求专业支持（法律援助/心理支持） |
| 多份遗嘱并存且效力不明 | 先触发 `testate-succession-validity` 子技能确认遗嘱效力范围，再确定法定继承的启动范围 |

## 法律依据

### 法条（来源：民法典继承编，中度把握锚点）

**第1127条 — 法定继承顺序**
- 《中华人民共和国民法典》**第1127条** `[模型知识 — 需验证]` 已核实锚点（中度把握），建议复核
- 第一顺序：配偶、子女（含婚生/非婚生/养/有扶养关系继子女）、父母（含生父母/养父母/有扶养关系继父母）
- 第二顺序：兄弟姐妹（含同父母/半血/养/有扶养关系继兄弟姐妹）、祖父母、外祖父母
- 继承开始后，由第一顺序继承人继承，无第一顺序继承人时由第二顺序继承人继承

**第1129条 — 丧偶儿媳女婿继承地位**
- 《中华人民共和国民法典》**第1129条** `[模型知识 — 需验证]` 中度把握，建议复核
- 丧偶儿媳对公婆、丧偶女婿对岳父母，尽了主要赡养义务的，作为第一顺序继承人

**第1130条 — 同顺序份额均等与酌情多分少分**
- 《中华人民共和国民法典》**第1130条** `[模型知识 — 需验证]` 中度把握，建议复核
- 同顺序继承人一般均等；对生活有特殊困难且缺乏劳动能力的应予照顾（强制性）；尽主要扶养义务或共同生活的可以多分（授权性）；有能力有条件不尽扶养义务的应不分或少分（强制性）

**第1125条 — 丧失继承权**
- 《中华人民共和国民法典》**第1125条** `[模型知识 — 需验证]` 中度把握，建议复核
- 故意杀害被继承人/为争夺遗产杀害其他继承人/遗弃虐待情节严重/伪造篡改隐匿销毁遗嘱情节严重/欺诈胁迫影响遗嘱情节严重，丧失继承权；部分情形经被继承人明示宽恕可恢复

**第1158条 — 遗赠扶养协议**
- 《中华人民共和国民法典》**第1158条** `[模型知识 — 需验证]` 已核实锚点（中度把握），建议复核
- 遗赠扶养协议优先于遗嘱和法定继承
