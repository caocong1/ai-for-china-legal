# 代位继承与转继承（subrogation-transmission）

> **可独立触发**：本技能可由 `/family-legal:inheritance` 编排入口调用，也可由律师/法务/继承人单独触发（`/family-legal:subrogation-transmission`），无需运行整个继承权分析流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载整体代理立场（代位/转继承人侧/其他继承人侧/中立）、敏感度校准。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1128条代位继承、第1125条丧失继承权）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）。
>
> **分工说明**：本技能专注代位继承与转继承的区分分析及丧失继承权审查；法定继承顺序与份额见 `legal-succession-order`；遗嘱继承效力见 `testate-succession-validity`；遗产范围与析产见 `estate-scope-debt`。

## 目的

在被继承人的继承人（子女或兄弟姐妹）先于被继承人死亡或在遗产分割前死亡时，准确判断代位继承或转继承是否适用、范围如何、份额多少，防止因概念混淆导致继承份额错误。同时审查继承人丧失继承权的情形，为代位/转继承人的份额主张提供法律依据。

覆盖核心分析：

1. 代位继承的触发条件与范围（被继承人子女/兄弟姐妹先死）
2. 代位继承的份额计算（代位继承被代位人有权继承的份额）
3. 转继承的触发条件与范围（继承人在遗产分割前死亡）
4. 代位继承与转继承的本质区别（时间点/主体/份额/性质）
5. 丧失继承权的情形认定（第1125条）
6. 按代理立场区分份额主张侧重

### 2. 代理立场区分

| 立场 | 审查侧重 |
|------|---------|
| **代理代位继承人** | 关注代位资格确立（被代位人先于被继承人死亡 + 晚辈直系血亲关系）、被代位份额的确认、丧失继承权情形排除（被代位人是否有丧失继承权情形）、代位人数量与份额分配 |
| **代理转继承人** | 关注转继承人资格（继承人在遗产分割前死亡且未放弃继承）、转继承时应继份的确认、转继承人的继承人范围 |
| **代理其他法定继承人（抗辩代位/转继承人份额）** | 质疑代位资格（如亲属关系存疑）或转继承有效性，主张丧失继承权情形 |
| **中立** | 双向列示代位/转继承适用情形与各方份额 |

### 3. 敏感情形检查

- 涉及丧失继承权争议（如被代位人曾故意伤害被继承人）：标注 🟠 高风险，提示须通过法院程序认定，强制升级律师
- 涉刑情形（故意杀害）：标注 🔴 关键，强制升级刑事律师

## 详细步骤

### 步骤一：代位继承分析

#### 1.1 代位继承的触发条件（民法典第1128条，建议复核）

代位继承须同时满足以下条件：

| 条件 | 具体说明 |
|------|---------|
| **时间要件** | 被代位继承人**先于被继承人死亡**（在被继承人死亡前已死亡） |
| **身份要件** | 被代位继承人须为：(1) 被继承人的子女，或 (2) 被继承人的兄弟姐妹 |
| **代位人身份** | (1) 子女先死：由子女的**晚辈直系血亲**代位；(2) 兄弟姐妹先死：由兄弟姐妹的**子女**代位 |
| **份额限制** | 代位继承人只能继承被代位继承人**有权继承**的遗产份额（不能超过被代位人应继份） |

#### 1.2 代位继承的适用场景

| 情形 | 是否触发代位继承 |
|------|--------------|
| 被继承人子女先于被继承人死亡 | ✅ 是 — 由该子女的晚辈直系血亲（孙子女/外孙子女等）代位 |
| 被继承人兄弟姐妹先于被继承人死亡 | ✅ 是 — 由该兄弟姐妹的子女代位（仅一代，不延伸至孙辈） |
| 被继承人父母先于被继承人死亡 | ❌ 否 — 父母先死不适用代位继承（父母的上辈无代位权） |
| 继承人先于被继承人死亡但被继承人已放弃让其继承 | ❌ 无代位份额可代位 |
| 被代位继承人丧失继承权 | ⚠️ 存在争议——主流认为丧失继承权者的晚辈直系血亲仍可代位（描述规则，`[待验证]`） |

#### 1.3 代位继承人范围与份额

**子女先死的代位继承人范围**：

- **晚辈直系血亲**：包括孙子女、外孙子女（及更低辈）
- 代位继承人可以是多人（如先死子女留有多名子女），则由多名代位继承人均等分配被代位人的应继份
- 代位继承人的子女（曾孙等）在特定情形下是否可再次代位，存在争议（描述规则，`[待验证]`）

**兄弟姐妹先死的代位继承人范围**：

- 仅限该兄弟姐妹的**子女**（一代，不延伸至孙辈）
- 民法典第1128条（建议复核）

#### 1.4 份额计算示例

**示例**：被继承人甲（无遗嘱），遗产 120 万元（析产后）。第一顺序继承人：配偶乙、父母丙丁、子女戊（戊已先于甲死亡，戊有子女己庚）。

```
法定继承人（含代位）：乙（配偶）、丙（父）、丁（母）、己（代位戊的份额）、庚（代位戊的份额）
均等顺序计算：应为 4 人（乙+丙+丁+戊代位份）均等 = 每份 30 万元
戊的应继份 30 万元由己、庚均等代位：己 15 万元 + 庚 15 万元
```

### 步骤二：转继承分析

#### 2.1 转继承的触发条件

转继承须同时满足以下条件：

| 条件 | 具体说明 |
|------|---------|
| **时间要件** | 继承人在**继承开始后**（被继承人死亡后）、**遗产分割前**死亡 |
| **态度要件** | 该继承人在死亡前**未放弃继承**（若已放弃则应继份不存在） |
| **转移主体** | 该继承人死亡时其已确定的应继份，转移给其**自身的继承人**继承（按该继承人自己的继承规则处理） |

#### 2.2 转继承的性质

转继承本质上是继承人的**「应继份」**在该继承人死亡时转为其遗产，由其继承人按该继承人自身的继承规则继承。

| 特点 | 代位继承 | 转继承 |
|------|---------|-------|
| 发生时间 | 被代位人先于被继承人死亡（被继承人死亡前） | 继承开始后（被继承人死亡后）、遗产分割前 |
| 法律基础 | 民法典第1128条（法定代位规则） | 应继份作为继承人遗产的转移（继承人的继承人） |
| 代位/转移主体 | 晚辈直系血亲（子女先死）或兄弟姐妹的子女 | 转继承人的全部继承人（法定或遗嘱） |
| 份额限制 | 代位继承被代位人有权继承的份额 | 转继承人的应继份全额转移（不限于晚辈） |
| 遗嘱的影响 | 代位不受遗嘱影响（代位法定规则，除非遗嘱排除） | 转继承按转继承人自身的继承规则（可有遗嘱） |

### 步骤三：代位继承与转继承对比表

| 维度 | 代位继承 | 转继承 |
|------|---------|-------|
| **触发时间** | 被代位人先于被继承人死亡 | 继承人在被继承人死亡后、遗产分割前死亡 |
| **法律条文** | 第1128条（建议复核） | 无专门条文（描述规则，`[待验证]`） |
| **代位/转移主体** | 晚辈直系血亲（子女）；兄弟姐妹的子女 | 转继承人的全体继承人（法定或遗嘱指定） |
| **份额** | 限于被代位人有权继承的份额 | 转继承人的全部应继份 |
| **放弃的影响** | 被代位人的放弃不影响代位继承（仅在生前放弃有争议，`[待验证]`） | 转继承人在继承开始后死亡前放弃的，无转继承份额 |
| **适用范围** | 法定继承（遗嘱继承一般不适用，`[待验证]`） | 法定继承与遗嘱继承均可适用 |

### 步骤四：丧失继承权审查（民法典第1125条，建议复核）

#### 4.1 丧失继承权的情形

| 情形 | 认定难度 | 涉刑风险 | 处理方式 |
|------|---------|---------|---------|
| 故意杀害被继承人 | 刑事判决或民事认定 | 🔴 极高 — 强制升级刑事律师 | 依刑事判决/民事法院认定 |
| 为争夺遗产杀害其他继承人 | 刑事判决或民事认定 | 🔴 极高 — 强制升级刑事律师 | 依刑事判决/民事法院认定 |
| 遗弃/虐待被继承人情节严重 | 情节严重由法院裁量 | 🟠 可能涉刑（遗弃罪/虐待罪） | 须通过法院诉讼认定 |
| 伪造/篡改/隐匿/销毁遗嘱情节严重 | 情节严重由法院裁量 | 🟠 高风险 | 须通过法院诉讼认定 |
| 欺诈胁迫影响遗嘱情节严重 | 情节严重由法院裁量 | 🟠 高风险 | 须通过法院诉讼认定 |

#### 4.2 丧失继承权的法律后果

| 后果 | 说明 |
|------|------|
| 丧失继承权的继承人不参与继承 | 视为自始不具有继承资格 |
| 其晚辈直系血亲是否可代位 | 主流观点：丧失继承权不影响代位继承（描述规则，`[待验证]`）——晚辈直系血亲仍可代位 |
| 宽恕后恢复继承权 | 被继承人可对虐待/遗弃（情节较轻者）明示宽恕，须在遗嘱中体现；死后不得宽恕 |

#### 4.3 丧失继承权的程序

**重要程序说明**：

> **丧失继承权须经法院诉讼认定程序**。AI 工具不得自行宣布某继承人丧失继承权，仅提供风险提示。识别到疑似丧失继承权情形时，**必须标注并强制升级给律师**，由律师决定是否通过诉讼程序主张。

程序路径：

1. 有争议的继承人提起确认继承权（或否认继承权）之诉
2. 或在遗产分割纠纷诉讼中作为附属争议处理
3. 法院认定后出具判决，明确该继承人的继承地位

```

代位继承：
[ ] 1. 确认被继承人子女或兄弟姐妹中是否存在先于被继承人死亡的情形
[ ] 2. 核实先死继承人的死亡时间（须在被继承人死亡前）
```

## 边界条件

| 边界情形 | 处理方式 | 风险等级 |
|---------|---------|---------|
| 双重代位（代位继承人也先于被继承人死亡） | 子女先死的代位继承人自身是否可再代位存在争议（描述规则，`[待验证]`）；兄弟姐妹的子女代位仅限一代 | 🟡 中等 |
| 被代位人丧失继承权 | 主流观点其晚辈直系血亲仍可代位，但具体规则须核实（`[待验证]`） | 🟡 中等 |
| 被代位人生前自愿放弃继承 | 主流观点认为若被代位人生前主动放弃，代位继承是否仍适用存在争议（`[待验证]`） | 🟡 中等 |
| 转继承人在被继承人死亡前放弃继承 | 继承开始前放弃的视为不继承（无转继承份额）；须区分「放弃时间」——死亡前放弃有效 | 🟡 中等 |
| 继承人与被继承人同时死亡（共同事故） | 无法确定死亡先后顺序时，推定互不继承（描述规则，条号 `[待验证]`） | 🟠 高 |
| 代位人认定关系存疑（如亲子关系不明） | 须通过亲子鉴定或法院认定；标注 🟠 高风险 | 🟠 高 |
| 遗嘱继承是否适用代位继承 | 主流观点遗嘱继承不适用代位继承（遗嘱指定的继承人先死，该遗嘱条款失效）——描述规则，`[待验证]` | 🟡 中等 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 先死亡继承人的死亡时间不明确 | 提示须取得死亡证明（医院记录/民政局死亡登记），明确死亡时间是判断代位还是转继承的关键 |
| 代位继承人身份关系证明不足 | 提示须取得亲属关系证明（户口本/出生证明/遗嘱等）；核实前标注 `⚠️ 亲属关系未经完整文件确认` |
| 涉刑情形（故意杀害） | 立即标注 🔴 关键，停止继承份额分析，强制升级刑事律师 |
| 丧失继承权争议（无法院认定） | 提示须通过法院诉讼认定程序；在认定完成前，争议继承人暂时保留继承候选人地位 |
| 转继承人自身也存在多重继承关系 | 按转继承人自身的继承规则处理其应继份；可能涉及多层继承，提示须律师梳理 |
| 丧亲情绪危机（多名亲属相继死亡） | 特别注意情绪护栏，以中立克制措辞呈现，避免在悲伤气氛中推动复杂份额争议 |

### 法条（来源：民法典继承编，中度把握锚点）

**第1128条 — 代位继承**
- 《中华人民共和国民法典》**第1128条** `[模型知识 — 需验证]` 中度把握，建议复核

### 司法解释（描述规则，条号待验证）

**关于代位继承范围的司法解释规则**（最高法继承编司法解释(一)）
- 规则描述：代位继承人只能代位继承被代位继承人有权继承的遗产份额，不能超过被代位继承人本应继承的份额 `[模型知识 — 需验证]` `[待验证]`

**关于转继承的规则**（描述规则）
- 规则描述：继承人在继承开始后遗产分割前死亡且未放弃继承的，其应继份作为其遗产，由其继承人按其继承规则继承——民法典继承编无专门条文，以继承法理与司法解释规则处理，条号 `[待验证]` `[模型知识 — 需验证]`

**关于丧失继承权者的晚辈直系血亲代位的规则**（描述规则）
- 规则描述：丧失继承权不影响其晚辈直系血亲的代位继承权（主流观点），具体规则须核实 `[模型知识 — 需验证]` `[待验证]`

**关于同时死亡推定规则**（描述规则）
- 规则描述：自然人同时发生死亡且无法确定死亡先后顺序的，推定同时死亡，互不继承各自遗产；具体规则须核实 `[待验证]` `[模型知识 — 需验证]`
