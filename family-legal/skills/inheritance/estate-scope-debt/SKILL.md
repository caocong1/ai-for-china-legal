# 遗产范围与债务清偿（estate-scope-debt）

> **可独立触发**：本技能可由 `/family-legal:inheritance` 编排入口调用，也可由律师/法务/继承人/债权人单独触发（`/family-legal:estate-scope-debt`），无需运行整个继承权分析流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载整体代理立场（某继承人侧/债权人侧/中立）、敏感度校准。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1122条遗产范围、第1161条债务限额清偿、第1160条无人继承归属）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）。
>
> **分工说明**：本技能专注遗产范围认定与债务清偿；法定继承顺序与份额见 `legal-succession-order`；遗嘱继承效力见 `testate-succession-validity`；代位继承与转继承见 `subrogation-transmission`。析产结果为继承顺序与份额分析的前提。

## 目的

在继承开始后，准确划定被继承人的个人遗产范围，处理夫妻共同财产析产，识别不属于遗产的财产，计算遗产债务的限额清偿，处理无人继承遗产归属。确保遗产分配在法律框架内进行，保护继承人的有限清偿原则，同时保障债权人的正当债权。

覆盖核心分析：

1. 遗产范围的正向认定（属于遗产的财产）
2. 遗产范围的反向排除（不属于遗产的财产）
3. 夫妻共同财产先析产后继承
4. 遗产债务的限额清偿原则
5. 遗产债务清偿顺序
6. 放弃继承与债务清偿的关系
7. 无人继承又无人受遗赠时的遗产归属

## 详细步骤

### 步骤一：遗产范围正向认定

#### 1.1 遗产的基本定义（民法典第1122条，建议复核）

> **遗产是自然人死亡时遗留的个人合法财产。**
> 依照法律规定或者根据其性质不得继承的财产，不得作为遗产。

核心判断要素：

| 要素 | 说明 |
|------|------|
| **「自然人死亡时」** | 继承开始时间点（自然死亡或宣告死亡的特定时间）——此后新增的财产不属于遗产 |
| **「个人合法财产」** | 须属于被继承人个人所有（夫妻共同财产须先析产），且合法取得 |
| **「不得继承的财产」** | 专属人身性权利（如肖像权本身）、法律规定不得转让的权利，不得作为遗产 |

#### 1.2 属于遗产的财产范围

| 财产类型 | 说明 | 注意事项 |
|---------|------|---------|
| **不动产（房产/土地使用权）** | 登记在被继承人名下的个人份额（若为共同共有须析产） | 析产后被继承人个人部分方为遗产 |
| **银行存款/现金** | 被继承人名下个人账户余额 | 共同账户须析产；大额存款来源须核实合法性 |
| **车辆/动产** | 登记或实际属于被继承人的动产 | |
| **股权/公司份额** | 被继承人持有的有限公司股权、合伙份额（转让限制须审查） | 股权继承须核查公司章程和合伙协议的限制 |
| **股票/基金/证券投资** | 被继承人证券账户内的金融资产 | |
| **债权/借款债权** | 被继承人对他人的借款、合同债权 | 须有证明文件（借条/合同） |
| **知识产权中的财产性权利** | 著作权中的财产权（使用权/收益权）、专利权的财产权、商标权 | **知识产权中的人身权（署名权）不属于遗产**；财产权可继承 |
| **已获赔偿但未实际支付的债权** | 如人身损害赔偿判决已生效但未实际收到的款项 | 赔偿金以被继承人为权利人的部分属遗产 |
| **经营资产** | 个体工商户的营业资产、农村承包经营权中的财产性利益 | |

#### 1.3 继承开始前须清缴税款和债务

注意：遗产分配前须**先清偿**被继承人依法应缴纳的税款和债务（见步骤三），剩余部分才是可分配遗产。

---

### 步骤二：遗产范围反向排除（不属于遗产的财产）

以下财产**不属于遗产**，须从遗产清单中排除：

| 财产类型 | 不属于遗产的理由 | 归属 |
|---------|--------------|------|
| **夫妻共同财产中属配偶的一半** | 须先析产，配偶应得份额不属于遗产 | 归配偶所有 |
| **其他共有财产中属共有人的份额** | 如合伙财产中他人份额 | 归其他共有人 |
| **指定受益人的人身保险金** | 有指定受益人时，保险金直接归受益人，不进入遗产 | 归指定受益人 |
| **依法不得转让的权利** | 如劳动报酬请求权（部分）、人身损害赔偿中的精神损害抚慰金（专属性）——存在争议 | 归权利主体本人 |
| **养老金/工龄补偿等个人权利** | 依社会保障法规定属于个人专属权益的部分（描述规则，`[待验证]`） | 依法处理 |
| **知识产权中的人身权利** | 著作权中的署名权、发表权（一定程度上可由继承人保护但本身不属遗产） | 特殊法律保护 |
| **特定用途的赔偿金** | 如残疾赔偿金、医疗费赔偿（专属人身补偿，归权利人本身，不属遗产——存在争议，`[待验证]`） | 个案处理 |

### 步骤三：夫妻共同财产先析产后继承

#### 3.1 析产原则

被继承人死亡时若处于婚姻关系存续状态（或存在未完成析产的财产），须按以下步骤处理：

```
被继承人死亡时全部财产
    ↓
Step 1: 识别夫妻共同财产范围（第1062条，建议复核）
    ↓
Step 2: 配偶析出一半夫妻共同财产（归配偶个人所有）
    ↓
Step 3: 其余夫妻共同财产的另一半 + 被继承人个人财产 = 被继承人遗产
    ↓
Step 4: 遗产先清偿税款和债务（第1161条）
    ↓
Step 5: 剩余遗产按遗嘱/法定继承分配
```

#### 3.2 夫妻共同财产范围确认

| 财产类型 | 共同财产认定（第1062条，建议复核） | 个人财产认定（第1063条，建议复核） |
|---------|--------------------------------|--------------------------------|
| 婚后工资收入 | ✅ 共同财产 | — |
| 婚后生产经营投资收益 | ✅ 共同财产 | — |
| 婚后知识产权收益 | ✅ 共同财产（收益部分） | — |
| 婚后继承赠与所得 | ✅ 共同财产（除非明确归一方） | ❌ 若遗嘱/赠与合同明确归一方则为个人 |
| 婚前个人财产 | — | ✅ 个人财产 |
| 婚后婚前财产的自然增值 | 描述规则，`[待验证]` | 一般倾向个人（自然增值，`[待验证]`） |
| 人身损害赔偿 | — | ✅ 个人财产 |
| 一方专用生活用品 | — | ✅ 个人财产 |

#### 3.3 析产实操注意事项

| 情形 | 处理说明 |
|------|---------|
| 房产等不动产登记为夫妻一方名下 | 不影响夫妻共同财产的认定；须审查购买时间和资金来源 |
| 约定财产制（第1065条，建议复核） | 若有书面约定财产制，按约定析产 |
| 析产争议 | 继承人与配偶对析产比例有争议，须通过诉讼解决；AI工具提示析产必要性，不独立计算 |

### 步骤四：遗产债务清偿

#### 4.1 限额清偿原则（民法典第1161条，建议复核）

> **继承人以所得遗产实际价值为限清偿被继承人依法应当缴纳的税款和债务。超过遗产实际价值部分，继承人自愿偿还的不在此限。放弃继承的，对被继承人依法应当缴纳的税款和债务可以不负清偿责任。**

核心规则：

| 规则要点 | 说明 |
|---------|------|
| **限额清偿** | 继承人的清偿义务上限 = 所得遗产实际价值（不超过、不承担超额部分） |
| **自愿超额偿还** | 继承人可自愿超额清偿，但法律不强制 |
| **放弃继承免债** | 放弃继承的，不负清偿责任（一般不能选择性继承财产、拒绝债务） |
| **不得以放弃逃债** | 虚假放弃继承逃避债务，债权人可主张撤销（描述规则，`[待验证]`） |

#### 4.2 遗产债务清偿顺序

依据司法解释（描述规则，条号 `[待验证]`）和民法典继承编规则，遗产债务清偿顺序为：

| 清偿顺序 | 内容 | 说明 |
|---------|------|------|
| 1 | 先析产（配偶析出应得份额） | 析产后方确定遗产范围 |
| 2 | 清偿被继承人应缴税款 | 税款优先（第1161条，建议复核） |
| 3 | 清偿被继承人债务 | 在遗产价值范围内按比例清偿（遗产不足则按比例） |
| 4 | 清偿遗赠扶养协议（优先于遗嘱/法定继承的受益） | 遗赠扶养协议的受益人优先 |
| 5 | 按遗嘱/法定继承分配剩余遗产 | 剩余遗产分配给遗嘱指定人或法定继承人 |

**遗产不足以清偿全部债务**：按比例清偿（同等顺序债务按各债权比例清偿）——描述规则，具体规则 `[待验证]`。

#### 4.3 继承人份额与债务分担

| 情形 | 处理方式 |
|------|---------|
| 多个继承人 | 各继承人在各自继承的遗产份额范围内清偿（限额为各自实际所得份额价值） |
| 继承人分割遗产后清偿债务 | 债权人可对各继承人主张在其所得份额内清偿 |
| 继承人代为偿还超额部分 | 须向其他继承人追偿（内部分担） |

---

### 步骤五：放弃继承与债务清偿

#### 5.1 放弃继承的法律效果

| 方面 | 放弃继承的影响 |
|------|--------------|
| 遗产继承 | 完全不继承，视为自始未参与继承 |
| 债务清偿 | 对被继承人的税款和债务**可以不负清偿责任**（第1161条，建议复核） |
| 放弃声明要求 | 须书面声明，须在遗产分割前作出（描述规则，具体要求 `[待验证]`） |

#### 5.2 虚假放弃继承逃债的处理

债权人利益保护：

- 若继承人以虚假放弃继承的方式逃避清偿被继承人债务，债权人可主张该放弃行为无效（撤销权）——描述规则，具体规则 `[待验证]`
- 法院可能认定放弃无效，要求继承人在实际所得份额内清偿
- 代理债权人时须核查继承人是否有放弃动机与财产保全的必要性

---

### 步骤六：无人继承遗产归属（民法典第1160条，建议复核）

当无人继承又无人受遗赠时：

| 被继承人身份 | 遗产归属 |
|------------|---------|
| 一般公民 | 归国家所有，用于公益事业 |
| 集体所有制组织成员 | 归所在集体所有制组织所有 |

**触发条件**：
- 无法定继承人（或全部丧失继承权/全部放弃继承）
- 无有效受遗赠人（受遗赠人放弃/先死亡）
- 无有效遗赠扶养协议

## 边界条件

| 边界情形 | 处理方式 | 风险等级 |
|---------|---------|---------|
| 遗产与个人财产混同（如家庭资金混入公司账户） | 提示先进行审计/清账，明确财产归属；专业财务鉴定可能需要 | 🟠 高 |
| 遗产分割前的债务（遗产管理期间产生的费用） | 遗产管理费用（如丧葬费、遗产管理人报酬）优先从遗产中扣除——描述规则，`[待验证]` | 🟡 中等 |
| 虚假放弃继承逃债 | 债权人可申请撤销；标注 🟠 高风险，建议申请财产保全 | 🟠 高 |
| 遗产含不动产（房产）须过户 | 继承房产须办理不动产登记过户（继承权公证+不动产登记）；遗产债务清偿完毕前房产可能被查封 | 🟡 中等 |
| 遗产含公司股权 | 股权继承须查公司章程（其他股东同意权）；股权价值评估须专业鉴定 | 🟠 高 |
| 遗产为负资产（债务超过遗产价值） | 继承人可放弃继承；若接受继承，只需清偿至遗产实际价值为限 | 🟡 中等 |
| 被继承人有未了结的诉讼 | 继承人可能继承诉讼当事人地位（财产权益类）；继续参与诉讼须律师主导 | 🟠 高 |
| 跨境遗产（境外房产/账户） | 可能适用境外法律和跨境继承程序，超出本技能范围 `[地方 flagged — 需核实境外法律]` | 🔴 关键 — 需律师 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 财产清单不完整（被继承人有隐藏财产） | 提示调查路径（不动产查询/股权工商查询/银行账户查询）；代理继承人时提示向法院申请调查令 |
| 夫妻共同财产析产争议 | 提示该争议须通过析产诉讼或协商解决；AI工具不独立计算存在争议的析产比例，标注 🟠 高风险 |
| 遗产债务真实性存疑（如被继承人死后出现大额债权人） | 提示债权人须提供证明文件；标注 🟠 高风险，建议核实债务真实性再清偿 |
| 继承人之间对遗产范围争议 | 提示须通过诉讼或公证处确认，暂不计算争议部分 |
| 股权价值无法确定 | 提示需要专业评估机构出具评估报告；以市场公允价值为准 |
| 丧亲情绪危机 | 中立克制措辞，提示专业支持，暂缓财产争议分析 |
