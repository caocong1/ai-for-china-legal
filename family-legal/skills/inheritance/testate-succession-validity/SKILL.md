# 遗嘱继承效力（testate-succession-validity）

> **可独立触发**：本技能可由 `/family-legal:inheritance` 编排入口调用，也可由律师/法务/继承人单独触发（`/family-legal:testate-succession-validity`），无需运行整个继承权分析流程。
>
> **前置依赖**：
> 1. 读取 `family-legal/CLAUDE.md`（实践配置文件），加载整体代理立场（主张遗嘱有效方/主张无效方/中立）、敏感度校准。
> 2. 读取 `family-legal/skills/_shared/legal-basis-conventions.md`（婚姻家事引用规范）。
> 3. 读取 `family-legal/skills/_shared/family-law-citations.md`（引用库，重点：第1142条数份遗嘱以最后为准、第1158条遗赠扶养协议优先、第1143条遗嘱无效情形）。
> 4. 读取 `family-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）。
>
> **分工说明**：本技能专注遗嘱效力认定（遗嘱继承优先、有效要件、数份遗嘱冲突、遗赠扶养协议顺位）；遗嘱具体形式要件（自书/代书/打印/录音录像/口头/公证六种形式各自要件）见 `will-drafting` 主技能及其子技能；法定继承顺序与份额见 `legal-succession-order`；遗产范围见 `estate-scope-debt`；代位继承与转继承见 `subrogation-transmission`。

---

## 目的

审查遗嘱继承的效力，确定被继承人遗产应按遗嘱继承还是法定继承（或两者结合）处理。覆盖以下核心分析：

1. 遗嘱继承与法定继承的优先顺位确认
2. 遗嘱有效要件综合审查（形式 + 实质）
3. 数份遗嘱冲突时的最后顺位规则（公证遗嘱不再优先的重大修订）
4. 遗赠与遗赠扶养协议优先顺位
5. 遗嘱部分无效的处理
6. 按代理立场区分举证策略

### 2. 代理立场区分

| 立场 | 审查侧重 |
|------|---------|
| **主张遗嘱有效方** | 关注形式要件齐备证明、遗嘱能力确认、无欺诈胁迫证明、最后遗嘱认定（时间序列清晰）、遗嘱人意思表示真实的留痕证据 |
| **主张遗嘱无效方** | 关注形式要件瑕疵（见证人不适格/签名日期缺失）、遗嘱能力存疑（立遗嘱时精神状态）、受欺诈胁迫证据、数份遗嘱冲突时认为更早遗嘱有效的主张 |
| **中立审查** | 双向列示效力支持与挑战因素，识别争议风险 |

### 3. 敏感情形检查

- 遗嘱效力争议激化（继承人之间对立明显）：提示冷静分析，以中立克制措辞呈现
- 涉及遗嘱能力争议（如高龄、精神疾病）：提示可能需要精神状态鉴定，🟠 高风险
- 涉及欺诈胁迫遗嘱指控：提示可能涉刑，🟠 高风险，评估是否需强制升级

## 详细步骤

### 步骤一：继承顺位体系确认

#### 1.1 四层继承/受益优先顺位

在遗产分配前，须先确认以下四层优先顺位（从高到低）：

| 优先层级 | 内容 | 法律依据 |
|---------|------|---------|
| **第一层：遗赠扶养协议** | 扶养人依协议受遗赠，优先于遗嘱和法定继承 | 第1158条（建议复核） |
| **第二层：遗嘱继承/遗赠** | 有效遗嘱指定继承人继承；遗赠给非法定继承人 | 遗嘱优先原则 |
| **第三层：法定继承** | 无有效遗嘱，或遗嘱未涵盖的遗产 | 第1127条（建议复核） |
| **（先于所有层级）** | 清偿被继承人税款和债务（以遗产实际价值为限） | 第1161条（建议复核） |

**关键要点**：

- 遗嘱继承**优先于**法定继承——被继承人生前立有有效遗嘱的，按遗嘱分配遗嘱涵盖的遗产
- 遗嘱未覆盖的遗产（如遗嘱成立后新取得的财产、遗嘱内容无效部分）才适用法定继承
- 遗赠扶养协议效力**高于**遗嘱，须先处理遗赠扶养协议部分

#### 1.2 遗赠扶养协议核查

**什么是遗赠扶养协议**（民法典第1158条，已核实锚点，建议复核）：

自然人与继承人以外的组织或个人签订协议，由扶养方承担受扶养人的生养死葬义务，受扶养人将其遗产遗赠给扶养方。

核查要点：

| 要点 | 说明 |
|------|------|
| 协议主体 | 扶养方须为「继承人以外」的组织或个人（不能是法定继承人） |
| 义务履行 | 扶养方须实际履行生养死葬义务；未履行或部分履行的，可能影响受遗赠权利 |
| 效力优先 | 遗赠扶养协议部分须先从遗产中扣除，剩余部分再按遗嘱/法定继承处理 |
| 与遗嘱并存 | 遗嘱人同时存在遗嘱和遗赠扶养协议时，协议优先；遗嘱不得处分协议已覆盖的遗赠部分 |

**受遗赠人的接受时限**（描述规则，期限条号 `[待验证]`）：受遗赠人须在知道受遗赠后的法定期限内作出接受表示，逾期未表示的，视为放弃受遗赠。具体期限须核实民法典或司法解释现行规定 `[待验证]`。

### 步骤二：遗嘱有效要件审查

#### 2.1 遗嘱有效要件体系

有效遗嘱须同时满足：

| 要件类别 | 具体内容 | 对应子技能 |
|---------|---------|---------|
| 形式合法 | 六种法定形式之一，形式要件完整（签名/日期/见证人等） | `will-drafting` 主技能及 `will-form-selection`/`witnessing-effectiveness` 子技能（详见） |
| 能力适格 | 立遗嘱时须为完全民事行为能力人 | `testamentary-capacity-intent` 子技能（详见） |
| 意思真实 | 遗嘱须表示遗嘱人真实意思，受欺诈胁迫无效 | 第1143条（建议复核） |
| 内容合法 | 不违反法律强制性规定；须为必留份继承人保留必要份额 | `content-reserved-share` 子技能（详见） |

> 本技能对形式要件、能力与意思表示进行**概括性审查和效力认定**；详细要件核查请调用对应 `will-drafting` 子技能。

#### 2.2 遗嘱无效情形（第1143条，建议复核）

依据民法典第1143条（中度把握，建议复核）：

| 无效情形 | 说明 | 法律后果 |
|---------|------|---------|
| 无民事行为能力人所立 | 如精神障碍、认知症患者等 | 遗嘱全部无效 |
| 限制民事行为能力人所立 | 如8-18周岁未成年人 | 遗嘱全部无效 |
| 受欺诈所立 | 遗嘱人被他人虚假陈述所诱导 | 受欺诈所立的遗嘱无效 |
| 受胁迫所立 | 遗嘱人被威胁强迫 | 受胁迫所立的遗嘱无效 |
| 伪造的遗嘱 | 遗嘱并非遗嘱人所立，而是他人伪造 | 伪造的遗嘱无效 |
| 遗嘱被篡改 | 遗嘱内容被他人擅自修改 | 篡改的**内容**无效（其余内容可能仍有效） |

**部分无效的处理**：
- 遗嘱部分无效（如遗嘱被篡改，仅篡改内容无效）：有效部分按遗嘱执行，无效部分依法定继承
- 遗嘱超出遗嘱人个人财产范围（如处分夫妻共同财产的全部）：析产后超出个人份额的部分无效

### 步骤三：数份遗嘱冲突处理

#### 3.1 「最后遗嘱为准」规则（民法典重大修订）

**民法典第1142条**（中度把握，建议复核）：

立有数份遗嘱，内容相抵触的，**以最后的遗嘱为准**。

**关键修订说明**：

民法典取消了原继承法下「公证遗嘱效力优先」的规则。原继承法规定，公证遗嘱效力高于其他形式遗嘱。民法典施行后（2021年1月1日起），**公证遗嘱不再具有优先效力**，所有形式的遗嘱一律按立遗嘱时间排序，以**最后订立的**遗嘱为准，无论其形式是否为公证遗嘱。

| 规则对比 | 原继承法 | 民法典（2021年起） |
|---------|---------|-----------------|
| 数份遗嘱冲突 | 公证遗嘱优先，无公证遗嘱时以最后为准 | **一律以最后订立的遗嘱为准，无论形式** |
| 公证遗嘱地位 | 优先于其他形式遗嘱 | **与其他形式遗嘱效力相同** |

**适用时间节点**：该修订自2021年1月1日起适用。2021年1月1日前已立的遗嘱，其效力认定应适用2021年1月1日起施行的民法典规定（存在过渡期争议，描述规则，条号 `[待验证]`）。

#### 3.2 数份遗嘱顺位判定流程

```
识别全部遗嘱
    ↓
逐份审查每份遗嘱的有效性（形式+实质）
    ↓
剔除无效遗嘱
    ↓
对有效遗嘱按立遗嘱时间排序
    ↓
识别内容相抵触的条款
    ↓
以最后订立的遗嘱中的内容为准
（最后遗嘱未覆盖的财产，由更早遗嘱或法定继承处理）
```

**时间确认要点**：

| 遗嘱形式 | 时间确认方式 |
|---------|------------|
| 自书遗嘱 | 遗嘱人亲笔注明的年月日 |
| 代书遗嘱 | 遗嘱人与见证人签名注明的年月日 |
| 打印遗嘱 | 每一页上遗嘱人与见证人签名注明的年月日 |
| 录音录像遗嘱 | 录音录像中记录的年月日 |
| 口头遗嘱 | 见证人记录的年月日（危急情况消除后即失效） |
| 公证遗嘱 | 公证机构办理公证的日期 |

### 步骤四：遗嘱内容与法定继承的衔接

#### 4.1 遗嘱未覆盖遗产的处理

| 情形 | 处理方式 |
|------|---------|
| 遗嘱成立后新取得的财产 | 遗嘱未提及的，按法定继承处理 |
| 遗嘱处分他人财产（无权处分部分） | 无权处分部分无效，该部分按法定继承 |
| 遗嘱继承人先于遗嘱人死亡 | 该部分遗嘱失效，归法定继承（代位继承规则不适用于遗嘱继承——描述规则，具体规则 `[待验证]`） |
| 遗嘱继承人放弃继承 | 放弃部分归法定继承 |

#### 4.2 遗赠的特殊规则

**遗赠**（给法定继承人以外的人或组织）：

| 规则 | 说明 |
|------|------|
| 受遗赠人身份 | 须为法定继承人以外的个人或组织 |
| 接受表示时限 | 须在知道受遗赠后的法定期限内明示接受，否则视为放弃（期限条号 `[待验证]`） |
| 受遗赠人先死亡 | 受遗赠人先于遗嘱人死亡的，遗赠失效，该部分归法定继承 |

## 边界条件

| 边界情形 | 处理方式 | 风险等级 |
|---------|---------|---------|
| 遗嘱部分无效（被篡改内容无效） | 有效部分按遗嘱，无效部分法定继承；须识别篡改范围 | 🟠 高 |
| 遗嘱与遗赠扶养协议并存 | 先处理遗赠扶养协议（效力优先），剩余部分按遗嘱 | 🟡 中等 |
| 受遗赠人先于遗嘱人死亡 | 遗赠失效，该部分归法定继承（注意与代位继承规则区分） | 🟡 中等 |
| 遗嘱继承人先于遗嘱人死亡 | 该遗嘱指定内容失效，归法定继承（遗嘱继承无代位继承——描述规则，`[待验证]`） | 🟡 中等 |
| 遗嘱仅处分部分遗产 | 遗嘱处分部分按遗嘱，未处分部分法定继承 | 🟢 低 |
| 公证遗嘱与非公证遗嘱并存 | 民法典施行后以最后订立者为准，无论是否公证；2021年1月1日前立的公证遗嘱过渡期争议 `[待验证]` | 🟡 中等 |
| 遗嘱无效但遗赠扶养协议有效 | 遗赠扶养协议部分仍执行，剩余法定继承 | 🟡 中等 |
| 受遗赠人超期未表示接受 | 视为放弃，该部分归法定继承；接受时限须核实 `[待验证]` | 🟡 中等 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 无法取得遗嘱原件审查 | 提示须取得原件（或公证机构档案）；依据描述审查效力时须在结论标注 `⚠️ 仅基于描述，未见原件` |
| 遗嘱人立遗嘱时的民事行为能力存疑 | 提示可申请精神状态鉴定；标注 🟠 高风险；能力认定时点为立遗嘱时，不可推定 |
| 涉及欺诈胁迫遗嘱指控 | 评估证据、提示举证路径；可能涉刑（欺诈胁迫遗嘱情节严重可能触发丧失继承权），🟠 高风险，升级律师 |
| 伪造遗嘱指控 | 提示笔迹鉴定/公证档案核查路径；标注 🔴 关键，强制升级律师 |
| 遗嘱日期不明确（仅写年月无日） | 标注形式要件瑕疵，可能影响遗嘱效力，须律师评估 |
| 数份遗嘱日期相同 | 无法依时间顺序确定最后遗嘱，须律师通过其他方式（如其他文件佐证）确认 |
| 继承人情绪危机 | 中立克制措辞，暂停复杂效力争议分析，提示专业支持 |

## 法律依据

### 法条（来源：民法典继承编，中度把握锚点）

**第1142条 — 遗嘱撤回变更与数份遗嘱最后为准**
- 《中华人民共和国民法典》**第1142条** `[模型知识 — 需验证]` 中度把握，建议复核
- 遗嘱人可撤回、变更自己所立的遗嘱；立有数份遗嘱内容相抵触的，以最后的遗嘱为准
- **公证遗嘱不再具有优先效力**（与原继承法相比的重大修订）

**第1143条 — 遗嘱无效情形**
- 《中华人民共和国民法典》**第1143条** `[模型知识 — 需验证]` 中度把握，建议复核
- 无行为能力人或限制行为能力人所立遗嘱无效；受欺诈胁迫所立遗嘱无效；伪造的遗嘱无效；被篡改的内容无效

**第1158条 — 遗赠扶养协议优先**
- 《中华人民共和国民法典》**第1158条** `[模型知识 — 需验证]` 已核实锚点（中度把握），建议复核
- 遗赠扶养协议效力优先于遗嘱和法定继承

**第1141条 — 必留份**
- 《中华人民共和国民法典》**第1141条** `[模型知识 — 需验证]` 已核实锚点（中度把握），建议复核
- 遗嘱须为缺乏劳动能力又无生活来源的继承人保留必要遗产份额

**第1125条 — 丧失继承权（含遗嘱相关情形）**
- 《中华人民共和国民法典》**第1125条** `[模型知识 — 需验证]` 中度把握，建议复核
- 伪造、篡改、隐匿或销毁遗嘱情节严重，或欺诈胁迫影响遗嘱情节严重，丧失继承权

### 司法解释（描述规则，条号待验证）

**关于受遗赠人接受时限的规则**（继承编司法解释，描述规则）
- 规则描述：受遗赠人须在知道受遗赠后一定期限内作出接受表示，具体期限须核实民法典或司法解释现行规定 `[待验证]`

**关于遗嘱继承无代位继承的规则**（描述规则）
- 规则描述：遗嘱指定的继承人先于遗嘱人死亡的，该遗嘱指定内容失效，该部分归法定继承（无法代位）；此点与法定继承的代位继承规则不同 `[模型知识 — 需验证]` `[待验证]`
