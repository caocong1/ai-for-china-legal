# 法律依据引用规范（婚姻家事版）

> 本文件定义 family-legal 插件所有技能统一使用的法律依据引用规范。
> 后续计划（08-02、08-03）直接引用本规范，避免各技能重复维护。
> **婚姻家庭编/继承编是 2021 年民法典施行后新编排的条号体系，取代原《婚姻法》《继承法》；本规范特别强调旧法条号迁移警示与司法解释条号保守标注原则。**

---

## 一、来源分类

所有法律依据必须归入以下类别之一，并在引用时明确标注类别标签。

### 1. 法律（全国人大及其常委会制定）

指全国人民代表大会及其常务委员会制定的法律，婚姻家事语境中主要包括：

- 《中华人民共和国民法典》（2020年5月28日通过，2021年1月1日施行）第四编「婚姻家庭」（第1040-1118条）
- 《中华人民共和国民法典》第六编「继承」（第1119-1163条）
- 《中华人民共和国民法典》第三编合同编通则部分（仅用于离婚协议本质为合同的可撤销/解除结构参照）
- 《中华人民共和国妇女权益保障法》（全国人大常委会制定，历经修正）
- 《中华人民共和国未成年人保护法》（全国人大常委会制定，历经修正）
- 《中华人民共和国反家庭暴力法》（全国人大常委会制定）

**法律层级最高，是婚姻家事法律分析的核心基础。**

> **婚姻家事法律引用特别提示（必须遵守）**：民法典婚姻家庭编（第1040-1118条）与继承编（第1119-1163条）为法典化成文法，中度把握锚点列于本规范第四章；婚姻家庭编/继承编以外的具体分则条号，以及妇女权益保障法/未成年人保护法/反家庭暴力法分则条号，**凡把握不足，一律标 `[待验证]`，描述规则呈现，不臆造条号。**

**格式要求（法律最小引用格式）：**

```
《[法律全名]》第[X]条[第X款]  [来源标签]  [验证状态]
```

示例（中度把握锚点）：
```
《中华人民共和国民法典》第1062条  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
```

示例（待验证条号）：
```
《中华人民共和国妇女权益保障法》第[X]条  `[待验证]`  `[模型知识 — 需验证]`
```

### 2. 司法解释（最高人民法院发布）

指最高人民法院发布的解释性文件，婚姻家事语境中主要包括：

- 《最高人民法院关于适用〈中华人民共和国民法典〉婚姻家庭编的解释(一)》（民法典施行后整合修订，**具体条号一律标 `[待验证]`**）
- 《最高人民法院关于适用〈中华人民共和国民法典〉继承编的解释(一)》（民法典施行后整合修订，**具体条号一律标 `[待验证]`**）

**核心纪律（必须遵守）**：

> **司法解释的具体条号一律标 `[待验证]`，以「描述规则 + 待验证」呈现，不写裸条号。** 规则内容（如夫妻共同债务认定、离婚损害赔偿适用、房产婚后还贷增值分割、彩礼返还情形、必留份计算方法）可完整描述，但来源条号须待验证。

**格式要求：**

```
《[司法解释全名]》（最高人民法院）规定：[描述规则]  [来源标签]  `[待验证]`
```

示例：
```
最高法婚姻家庭编司法解释(一)规定：夫妻双方共同签名或一方事后追认等共同意思表示所负的债务，应当认定为夫妻共同债务  `[模型知识 — 需验证]`  `[待验证 — 司法解释条号易变，须核实现行版本]`
```

### 3. 案例（指导性案例与婚姻家事典型案例）

婚姻家事领域可参考的案例类型：

- **最高院指导性案例**：最高人民法院发布，对下级法院有参照作用，仅作参考，**不具判例法约束力**
- **最高院婚姻家事典型案例**：最高人民法院或高院发布的典型案例，实践参考价值较高，**非判例法**
- **各地高院/中院婚姻家事判决**：参考价值有限，地域差异大

**核心纪律（必须遵守）**：

> **所有案例案号一律标 `[待验证]`，未经检索核验不得移除该标注。** 中国采成文法体系，案例不具判例法约束力，须明确说明。

**格式要求：**

```
[案例名称或案号 `[待验证]`]  [裁判要点摘要]  [法院层级]  参考，不具判例法约束力  `[待验证]`
```

### 4. 学说（学术观点）

学说为辅助参考，存在争议性，引用时须标注来源和争议性：

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息]  "[观点摘要]"  [是否有争议：有/无]  `[待验证]`
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用五个标准标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站（全国人大/民政部/最高法） | 官网原始文件 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查；婚姻家庭编/继承编司法解释条号风险尤为突出 |
| `[用户提供]` | 来自用户提供的文件 | 离婚协议文本、遗嘱、财产清单等 |

**标签使用规则：**

1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 来源在依赖前**必须**对照原始来源检查——婚姻家庭编/继承编司法解释经民法典施行后修订，条号风险突出
4. 来源标签不替代 `[待验证]` 标注——两者可同时出现在一行

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` 的条号 | 模型回忆的条文号风险高 | 国家法律法规数据库、最高法官网 |
| **司法解释的任何具体条号** | 婚姻家庭编/继承编司法解释经民法典施行后修订整合，条号易变 | 最高人民法院官网 court.gov.cn、北大法宝核实现行版本 |
| 妇女权益保障法/未成年人保护法/反家庭暴力法分则条号 | 上述法律历次修正 | 国家法律法规数据库 flk.npc.gov.cn |
| 抚养费比例/彩礼返还数额/必留份份额等数值 | 数值往往来自司法解释或地方裁判口径，极易变化 | 核实现行司法解释与当地裁判口径 |
| 距最后更新超过 6 个月的法规 | 法规可能已修订或废止 | 查询官方数据库最新版本 |
| 用户口述的条款号、期限、数额、份额 | 可能记忆偏差或引用旧版 | 核查官方法规全文 |
| 所有案例引用（默认） | 婚姻家事案件繁多，案号极易错误回忆 | 中国裁判文书网、最高法官网 |

**待验证标记格式（示例）：**

```
最高法婚姻家庭编司法解释(一)第[X]条  `[模型知识 — 需验证]`  `[待验证 — 司法解释条号易变，须核实现行版本]`
```

## 五、旧法条号→民法典新条号迁移警示

> **这是婚姻家事引用最高风险之一，必须严格遵守。**

### 迁移背景

婚姻家庭编/继承编是 2021 年民法典施行后**新编排的条号体系**，取代了原《中华人民共和国婚姻法》和原《中华人民共和国继承法》。旧法已废止，旧法条号不得再引用。

**严禁回退旧法条号**：不得出现原婚姻法（第17/18/32条等）或原继承法（第3/10/19条等）条号；发现旧法条号即视为错误，须立即纠正。

### 旧法条号→民法典新条号对照（仅供纠错参考，勿据此臆造其他条号）

| 旧法条号 | 旧法规则摘要 | 民法典新条号 | 迁移说明 |
|---------|------------|------------|---------|
| 原婚姻法 第17条 | 夫妻共同财产范围 | **民法典第1062条** | 已核实，建议复核 |
| 原婚姻法 第18条 | 夫妻个人财产 | **民法典第1063条** | 已核实，建议复核 |
| 原婚姻法 第32条 | 诉讼离婚感情破裂 | **民法典第1079条** | 已核实，建议复核 |
| 原婚姻法 第37条 | 子女抚养费（含规则）| **民法典第1085条** `[待验证]` | 把握不足，描述规则 |
| 原婚姻法 第38条 | 探望权 | **民法典第1086条** `[待验证 建议复核]` | 中度把握 |
| 原继承法 第3条 | 遗产范围 | **民法典第1122条** | 已核实，建议复核 |
| 原继承法 第10条 | 法定继承顺序 | **民法典第1127条** | 已核实，建议复核 |
| 原继承法 第19条 | 必留份 | **民法典第1141条** | 已核实，建议复核 |

**迁移纪律**：

> **凡引用婚姻家事条文一律使用民法典新条号**。发现旧法条号（原婚姻法/继承法任何条号）即视为错误，须纠正为民法典婚姻家庭编（第1040-1118条）或继承编（第1119-1163条）对应条号；把握不足时描述规则并标 `[待验证]`，不臆造新条号。

## 六、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：婚姻家事法律分析的所有结论，必须首先援引现行成文法（民法典婚姻家庭编/继承编及相关法律）作为规范依据
2. **不得以案例替代法条**：即使存在明确的典型案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下司法解释/典型案例/学说」

### 案例的辅助定位

1. **最高院指导性案例**：仅作参考，标注「最高院指导，参照意义强，不具判例法约束力」
2. **最高院婚姻家事典型案例**：实践参考价值较高，标注「婚姻家事典型案例，参照意义强，不具判例法约束力」
3. **高院/中院婚姻家事案例**：参考价值有限，标注「地方法院案例，参照，不具判例法约束力」
4. **明确说明非判例法**：中国采成文法体系，案例不具判例法约束力

## 七、法院层级与机构定位

### 法院层级（婚姻家事案件）

| 层级 | 说明 | 婚姻家事特点 |
|------|------|------------|
| 最高人民法院 | 全国最高审判机关 | 指导性案例与司法解释发布；婚姻家事典型案例 |
| 高级人民法院 | 省级，婚姻家事纠纷二审 | 部分高院有婚姻家事专项审判指导意见 |
| 中级人民法院 | 地市级，婚姻家事纠纷二审；重大财产分割诉讼 | 财产较大或复杂继承纠纷有时由中院一审 |
| 基层人民法院 | 县区级，**婚姻家事案件通常由基层法院一审** | 离婚诉讼、继承纠纷、财产分割诉讼一审主力；**设有家事审判庭与家事调解员，离婚案件须先行调解** |

**婚姻家事审判特色**：家事审判庭设立、家事调解前置、未成年子女最佳利益调查制度、家暴禁止令（人身安全保护令）程序。

### 行政机构与专业机构定位

| 机构 | 主要职能 | 婚姻家事业务 |
|------|---------|------------|
| 民政局婚姻登记机关 | 协议离婚登记、冷静期管理 | 办理协议离婚登记、30日冷静期申请撤回与期满发证 |
| 公证机构 | 公证服务 | 遗嘱公证、财产分割协议公证、继承权公证 |
| 不动产登记机构 | 房产登记 | 离婚后房产过户、析产登记、继承房产过户 |
| 司法鉴定机构 | 专业鉴定 | 亲子鉴定、遗嘱人精神状态鉴定、财产评估鉴定 |
| 妇联与反家暴庇护机构 | 妇女权益保护 | 家暴庇护、人身安全保护令协助申请 |
| 基层法院家事调解 | 婚姻家事调解 | 离婚调解、财产调解、抚养纠纷调解 |
