# 共同财产认定（community-property-identification）

> **可单独触发**：`/family-legal:community-property-identification`
> 也可由 `property-division` 编排入口作为第一步调用。
>
> **与 divorce-agreement/property-division-clauses 的分工**：本子技能侧重复杂财产的法律性质认定与分析（共同财产 vs 个人财产 vs 约定财产，婚后增值性质）；property-division-clauses 侧重在离婚协议文本中拟定具体财产分割条款。认定结果可直接输入协议起草流程。

---

## 目的

系统梳理离婚财产分割前的基础工作：识别哪些财产属于夫妻共同财产、哪些属于一方个人财产、约定财产制是否适用，以及婚前财产婚后增值的性质判断。认定结论直接影响后续特殊财产分割（special-asset-division）和分割方案设计的起点。

### 2. 代理立场对共同财产认定的影响

**代理立场是分析侧重的核心变量**：

| 代理立场 | 认定侧重方向 |
|---------|-----------|
| **离婚一方侧（主张方）** | 尽量扩大共同财产范围：婚后取得的财产推定为共同财产；婚后增值主张转化为共同财产或要求补偿；约定财产制效力审慎核查（对方是否真正知悉）；婚前出资记录有瑕疵时主张共同性 |
| **离婚一方侧（被主张方）** | 尽量剥离个人财产：举证婚前来源（房产/存款出资时间线/父母赠与协议）；主张自然增值属个人；约定财产制有效对抗主张方；争取个人专用生活用品单独列示 |
| **中立审查** | 依法律规则客观认定，双向识别争议点，平衡呈现双方可能主张 |

**不得被立场覆盖的护栏**：涉及未成年子女的财产安排以最有利于未成年子女为首要原则；家暴情形下强制升级，不因立场影响。

### 步骤二：按财产性质认定分类

**2.1 夫妻共同财产认定（民法典第1062条，建议复核）**

以下财产为夫妻共同财产：

| 来源类别 | 典型财产 | 认定要点 | 争议点 |
|---------|---------|---------|------|
| 工资、奖金、劳务报酬 | 婚后工资/年终奖/绩效/劳务费 | 婚姻关系存续期间取得即为共同财产，无需另行证明 | 婚前应得但婚后发放的奖金（应以取得时间还是应得时间认定）`[待验证]` |
| 生产、经营、投资收益 | 婚后开办公司/个体经营/投资股票基金所得 | 婚后收益归共同所有，婚前个人经营收益属个人 | 婚前投入、婚后获得收益的投资（见步骤三婚后增值部分） |
| 知识产权收益 | 婚后版权稿酬/专利许可费/商标许可费 | 注意：是「收益」而非「知识产权本身」，婚姻关系存续期间实现的收益为共同财产 | 婚前创作、婚后取得的著作权收益（司法解释描述规则，`[待验证]`） |
| 继承或受赠所得 | 婚后继承的遗产、婚后受赠的财产 | 遗嘱或赠与合同明确只归一方的除外；未明确时为共同财产 | 父母出资赠与的房产：是赠与双方还是只赠一方（须看赠与合同或明示表述） |
| 其他应归共同所有 | 婚后共同贷款所得、共同事业积累 | 兜底条款 | — |

**核查清单（共同财产认定）**：

- [ ] 财产取得时间是否确为婚姻关系存续期间
- [ ] 是否有遗嘱/赠与合同明确说明只归一方（如有则不属共同财产）
- [ ] 知识产权的「收益」与「权利本身」是否已区分
- [ ] 婚后工资/奖金的金融账户是否清晰（混同账户须清分）
- [ ] 经营实体（公司/个体户）的收益分配记录是否完整

**2.2 夫妻个人财产认定（民法典第1063条，建议复核）**

以下财产为一方个人财产：

| 类别 | 典型财产 | 认定要点 | 举证方向 |
|-----|---------|---------|---------|
| 婚前财产 | 婚前购买的房产、婚前存款、婚前车辆 | 婚姻登记前取得，婚后增值问题另行处理 | 购房合同/首付时间/存款开户时间/交付凭证 |
| 人身损害赔偿/补偿 | 交通事故/工伤/人身侵权赔偿金 | 无论婚前婚后取得，均为个人财产（与人身相关） | 判决书/调解书/银行入账记录 |
| 遗嘱/赠与合同明确归一方的财产 | 父母立遗嘱明确留给本方的遗产；赠与合同明确只归一方的赠与 | 须有书面遗嘱或赠与合同明确记载 | 遗嘱/赠与合同原件 |
| 一方专用生活用品 | 婚后购买的一方专用衣物/日常用品（价值较低） | 仅限一方专用、价值较小；高价值奢侈品不适用 | — |
| 其他应归一方 | 兜底条款 | 须有法律依据 | — |

**核查清单（个人财产认定）**：

- [ ] 婚前财产有无足够时间线证据（合同/首付时间/付款记录）
- [ ] 人身损害赔偿金是否与精神损害抚慰金一并归一方（确认）
- [ ] 父母赠与或遗嘱是否有书面记载明确只归一方
- [ ] 「专用生活用品」是否仅限低价值日常用品（高价值奢侈品须核查）

**2.3 约定财产制认定（民法典第1065条，建议复核）**

约定财产制须满足以下要件：

| 要件 | 内容 | 核查重点 |
|-----|------|---------|
| 书面形式 | 约定须采用书面形式（口头约定无效） | 核查是否有书面婚前/婚内财产协议 |
| 约定内容明确 | 约定各自所有 / 共同所有 / 部分各自部分共同（三种模式之一） | 约定内容是否覆盖争议财产类型 |
| 双方均知悉 | 约定对双方有约束力；对第三人须证明第三人知道该约定 | 核查第三人是否知情（影响对外债务清偿） |
| 意思表示真实 | 约定须为双方真实意思表示（无欺诈/胁迫/重大误解） | 若存在意思表示瑕疵，约定可被主张撤销（合同编参照结构，`[待验证]`） |

约定财产制有效时的效力分析：

- **对内效力**：双方受约定约束，不适用第1062/1063条默认规则
- **对外效力**：债权人仅在知道该约定的情形下，方可以夫或妻一方的个人财产主张清偿
- **约定的边界**：约定不得规避法律强制性规定（如不得约定免除赡养义务、不得损害债权人利益）

### 步骤三：婚前财产婚后增值的性质判断

**自然增值 vs 主动增值的区分**（司法解释规则，描述规则，`[待验证]`）：

| 增值类型 | 典型例子 | 性质认定 | 说明 |
|---------|---------|---------|-----|
| **自然增值** | 婚前购房因市场行情上涨；婚前存款利息；婚前债权到期实现 | 通常认定为一方**个人财产**（增值属于原物的自然延伸） | 无需配偶任何投入，仅因市场或时间自然产生 |
| **主动经营增值** | 婚前购房婚后共同装修/还贷/增值；婚前公司婚后双方共同经营获利 | 增值部分**可能转化为共同财产或产生补偿权** | 共同劳务投入、共同资金投入是转化的关键因素 |
| **混合型** | 婚前存款婚后用于共同投资，收益与本金混同 | 须依账务记录清分 | 混同时举证责任对主张方有利 |

**房产婚前购买婚后增值**（最典型争议）：

最高法婚姻家庭编司法解释(一)规定（描述规则，条号 `[待验证]`）：一方婚前以个人财产购买、登记为其名下的房产，婚后双方共同还贷的，离婚时归产权登记方，但婚后共同还贷的数额及对应房产增值部分，由产权登记方对另一方进行补偿。

具体分析见 `skills/property-division/special-asset-division/SKILL.md`（房产专项）。

**知识产权婚前创作婚后取得收益**（司法解释规则，描述规则，`[待验证]`）：

最高法婚姻家庭编司法解释(一)规定（描述规则）：婚姻关系存续期间，实际取得或者明确可以取得的知识产权收益属于夫妻共同财产。婚前创作婚后取得的收益是否属于共同财产，须结合取得时间认定。

## 检查清单

财产性质认定完成前，逐项核查：

- [ ] 财产清单已覆盖全部已知资产类型（不动产/动产/金融资产/知识产权/收益权等）
- [ ] 每项财产的取得时间已明确标注（婚前/婚后，精确到月份有利于争议时间点核查）
- [ ] 共同财产认定（第1062条）已逐项核查来源
- [ ] 个人财产认定（第1063条）已有对应凭证或核查路径
- [ ] 约定财产制有无——有则核查书面形式、意思表示真实性、内容覆盖范围
- [ ] 婚前财产婚后增值已区分自然增值与主动经营增值
- [ ] 父母出资的财产已核查是赠与一方还是双方还是借款（三种不同法律性质）
- [ ] 争议财产已标注举证方向与证据缺口
- [ ] 代理立场已加载，认定侧重方向已调整
- [ ] 家暴/涉刑/未成年人敏感情形已识别（无则继续；有则强制升级）

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 父母出资购房（全额出资） | 核查是否有书面赠与合同/公证；明确赠与一方还是双方还是借款——三种性质法律结果完全不同；无书面记录时存在争议 `[待验证]` |
| 父母出资购房（部分出资/首付）| 部分出资是赠与一方还是双方须有书面证明；剩余部分为婚后共同还贷（婚后还贷增值见 special-asset-division） |
| 公积金与养老金 | 公积金余额通常认定为夫妻共同财产（婚后缴存部分）；养老金在婚姻关系存续期间的积累部分属共同财产——具体认定须核实当地实践 `[待验证]` `[地方 flagged]` |
| 约定财产制对抗债权人 | 第三人不知道约定时，债权人可向夫妻双方追索；知道时按约定处理 |
| 知识产权收益婚前创作 | 婚前创作婚后持续取得的版税等收益，以收益实际取得时间为准（司法解释，描述规则，`[待验证]`） |
| 财产来源无法证明 | 无法证明来源时，推定为夫妻共同财产（举证责任在主张个人财产一方） |
| 婚姻无效/可撤销情形 | 婚姻被确认无效或撤销时，财产处理另有规则，不适用共同财产分割（须单独分析） |
| 涉外财产 | 涉及境外财产/跨境投资，准据法认定与财产调查路径须核实 `[待验证]` |

## 错误处理

| 错误/敏感情形 | 识别信号 | 处理方式 |
