# 特殊财产分割（special-asset-division）

> **可单独触发**：`/family-legal:special-asset-division`
> 也可由 `property-division` 编排入口作为第二步调用（在共同财产认定之后）。
>
> **与 divorce-agreement/property-division-clauses 的分工**：本子技能侧重复杂特殊财产的法律性质认定、估值方法与分割方案设计（含补偿计算思路）；property-division-clauses 侧重在离婚协议文本中具体拟定条款语言。本技能的分析结果直接为协议条款起草提供依据。

---

## 目的

针对离婚财产分割中最复杂、争议最大的特殊财产类型，提供分类别的法律性质认定、分割路径、估值参考与风险提示。覆盖范围：房产（最核心争议）、股权与有限公司份额、知识产权收益权、股票基金等投资性资产、养老金与保险现金价值。

### 2. 代理立场对特殊财产分割的影响

| 代理立场 | 分析侧重方向 |
|---------|-----------|
| **主张方** | 争取婚后共同还贷增值补偿最大化；主张股权/收益归共同财产；主张知识产权收益足额分割；养老金婚内积累部分争取公平分割 |
| **被主张方** | 主张婚前来源与个人出资；合理折价（股权公允价值而非账面）；主张知识产权本身为个人财产；争取公司经营连续性优先（折价补偿而非强制分割股权） |
| **中立审查** | 识别各类财产争议点，平衡呈现双方主张，注明估值分歧 |

## 详细步骤

### 一、房产分割

房产是离婚财产分割中争议最大的类别，须逐情形分析。

**房产基本信息采集**：

| 字段 | 内容 |
|-----|------|
| 产权登记情况 | 一方名下 / 双方共有 / 父母名下 |
| 购房时间 | 婚前 / 婚后 |
| 首付资金来源 | 一方婚前存款 / 婚后共同存款 / 一方父母出资 / 双方父母共同出资 |
| 贷款情况 | 无贷款（全款）/ 有按揭（婚前/婚后还贷） |
| 贷款还款方式 | 婚前一人还款 / 婚后共同还款 / 婚后一方独自还款 |
| 房产现值估计 | [市场评估价 / 网签备案价 / 鉴定评估价] |
| 婚后增值情况 | 估计增值额 |

**房产分割决策表**（依据最高法婚姻家庭编司法解释(一)，描述规则，`[待验证]`）：

| 情形 | 房产归属 | 补偿计算方向 | 主要争议点 |
|-----|---------|-----------|----------|
| **婚前一方全款购买，登记在一方名下** | 个人财产（第1063条，建议复核）；归产权登记方 | 无补偿（纯个人财产） | 婚后装修/翻新支出是否应补偿（较小） |
| **婚前一方购房，婚后双方共同还贷** | 产权归登记方（司法解释规则，`[待验证]`）；但婚后共同还贷本金+对应增值须补偿另一方 | 补偿额 = 婚后共同还贷本金 × 2 ÷ 2 + 增值部分 × （婚后还贷/总贷款比例）× 1/2，具体计算公式须司法实践确认 `[待验证]` | 增值比例计算方法；还贷来源证明（是否真正共同还贷） |
| **婚前一方父母全额出资，产权登记在子女名下** | 视为对该子女的个人赠与（司法解释，描述规则，`[待验证]`）；个人财产 | 无补偿（个人财产） | 是否有书面赠与合同；登记情况（登记在双方名下则不同） |
| **婚前一方父母部分出资（首付），子女婚后还贷** | 父母出资部分视为赠与一方（个人财产基础）；婚后还贷+增值须补偿 | 父母出资=个人财产；婚后还贷增值部分按共同财产处理 `[待验证]` | 父母出资是赠与还是借款须书面证明 |
| **婚后双方父母共同出资** | 若有书面说明各自赠与比例，按比例认定；无书面记录通常视为赠与双方 `[待验证]` | 依赠与比例认定各自份额 | 父母出资性质（赠与/借款）及各方份额比例 |
| **婚后共同出资购房，登记在一方名下** | 属夫妻共同财产，登记仅是形式（第1062条，建议复核）；分割时各占50%或按约定 | 产权登记方取得房产须向另一方支付折价补偿 | 折价补偿额（市场价还是评估价）；剩余贷款承担方式 |
| **婚后加名（原个人财产加另一方名）** | 婚后加名通常视为赠与（转变为共同共有）`[待验证]` | 加名后按共同财产分割 | 加名时是否有附带条件；赠与是否可撤销 |
| **婚后共同购房，登记在双方名下** | 共同财产，按份共有或共同共有依登记情况 | 各持份额或各50% | 剩余贷款承担比例 |

**按揭房产剩余贷款处理**：

- 离婚时剩余贷款通常随取得房产方承担；另一方已还本金须获得补偿
- 银行贷款变更需银行同意（实践中银行审批可能有限制）
- 如不能变更贷款，须约定内部分担责任并设置违约责任条款

**房产估值路径**：

| 方法 | 适用场景 | 优缺点 |
|-----|---------|------|
| 协议约定房价 | 双方自愿协商 | 简便，但可能一方不满意 |
| 委托评估机构出具评估报告 | 争议较大时 | 客观，有成本；评估基准日影响结果 |
| 参考近期同小区成交价 | 市场信息透明时 | 便捷，争议时对方可能不认可 |
| 申请法院委托评估 | 诉讼程序中 | 权威，诉讼成本较高 |

### 二、股权与公司份额分割

**有限责任公司股权分割**：

股权分割面临双重障碍：夫妻财产分割规则（民法典）与公司法对股东变更的限制。

**股权性质认定**：

| 情形 | 股权性质 | 分割路径 |
|-----|---------|---------|
| 婚前持有股权 | 股权本身为个人财产（第1063条）；婚后股权增值/分红可能为共同财产 | 认定婚后增值/收益性质 |
| 婚后出资设立公司 | 出资资金来源于共同财产时，股权属共同财产（第1062条，建议复核） | 分割股权或折价补偿 |
| 婚后以个人财产出资 | 个人财产出资，股权属个人财产（须举证出资来源） | 举证出资来源 |

**股权分割的三条路径**：

| 路径 | 内容 | 适用条件 | 利弊 |
|-----|------|---------|-----|
| **折价补偿** | 一方取得股权，向另一方支付相当于股权价值半数的补偿款 | 另一方不适合持股或双方同意 | 保持公司控制权完整；须确定公允股权价值 |
| **股权转让给非配偶** | 一方将共同财产部分股权转让给第三方，获得对价后分割对价 | 其他股东同意行使优先购买权放弃后 | 需其他股东配合；公司法对有限公司股权转让有规定 `[待验证]` |
| **强制分割股权** | 法院判决将配偶列为股东 | 其他股东不行使优先购买权时 | 干扰公司治理；实践中其他股东通常反对 |

**股权估值**：

| 方法 | 适用场景 | 注意事项 |
|-----|---------|---------|
| 净资产法 | 账面净资产清晰 | 不反映未来盈利能力和商誉 |
| 收益法（市盈率） | 盈利稳定 | 预测假设敏感 |
| 市场比较法 | 有可比公司 | 非上市公司难找可比对象 |
| 司法评估 | 争议较大，诉讼程序中 | 委托具有资质的司法鉴定机构 |

**其他注意事项**：

- **未实缴出资**：认缴未实缴的出资，实际价值与认缴值有差异，须区分认缴额与实际出资额
- **公司控制权**：大股东一方分割后可能影响公司经营决策，折价补偿是更常见选择
- **竞业限制与商业秘密**：分割股权后，前配偶成为小股东可能涉及公司商业秘密保护问题

### 三、知识产权收益分割

**知识产权本身 vs 知识产权收益**（重要区分）：

| 类别 | 性质 | 分割规则 |
|-----|------|---------|
| **知识产权本身（著作权/专利权/商标权）** | 与权利人人身密切相关，通常不分割权利本身 | 不强制分割权利归属 |
| **知识产权收益（版税/专利许可费/商标许可费）** | 婚姻关系存续期间实际取得的收益为共同财产（第1062条，建议复核） | 已实现收益按共同财产分割 |
| **尚未取得的预期收益（未到期版税/待实现专利许可费）** | 性质认定有争议（司法解释，描述规则，`[待验证]`） | 可约定未来实现时支付补偿；或折现现值分割 |

**知识产权收益认定的实务要点**：

- 收益已入账：有银行流水、合同对账单 → 认定为共同财产，可直接分割
- 收益未到账但合同已签：可约定将来实现时支付；须注意对方可能主张折现分割
- 持续性创作（作者婚后继续创作获得收益）：婚后创作对应的婚后收益为共同财产

### 四、股票、基金等投资性资产

| 资产类型 | 分割要点 | 估值时点 | 注意事项 |
|---------|---------|---------|---------|
| **股票（A股）** | 婚后买入的股票为共同财产；婚前持有的婚后增值可能涉及性质认定 | 离婚时市值；注意市值波动对公平性的影响 | 账户挂账的股票须明确持仓数量；停牌股票估值困难 |
| **基金（公募/私募）** | 婚后投入资金对应份额为共同财产 | 净值日估值 | 私募基金流动性差，赎回有限制；锁定期内分割须协商 |
| **债券/大额存单** | 婚后购入为共同财产 | 按面值或市场价 | 到期兑付时间影响现金流 |
| **互联网理财产品** | 婚后投入为共同财产 | 账面余额 | 注意关联账户归属 |

**估值时点的争议**：应以离婚诉讼立案时、庭审时还是判决时为准，各地实践不同 `[地方 flagged — 需核实地方裁判口径]`。

### 五、养老金与保险现金价值

**养老金（职工基本养老保险）**：

最高法婚姻家庭编司法解释(一)（描述规则，`[待验证]`）：婚姻关系存续期间缴交的养老保险费对应的权益属夫妻共同财产，离婚时一方可主张分割婚内缴交部分对应的养老金账户价值；具体分割方式因各地社保实践差异较大  `[地方 flagged — 需核实地方社保实践]`。

**人身保险现金价值**：

- 整存式两全保险/终身寿险的现金价值，婚后缴交保费对应的现金价值属共同财产
- 保险权益须注意受益人变更与保单归属
- 分红型保险的分红收益（婚后取得）为共同财产

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 父母出资性质争议（赠与 vs 借款） | 有书面借条且有还款记录的倾向认定为借款（共同债务）；无书面凭证的赠与认定须举证 `[待验证]` |
| 公司经营连续性 | 法院通常倾向折价补偿而非强制分割股权，以保护公司正常经营 |
| 跨境资产（境外房产/境外账户） | 准据法、资产调查、评估与执行路径均有特殊性，须专业律师介入 `[待验证]` |
| 房产被查封/限制交易 | 须先解除查封方可分割，诉讼保全可能影响分割程序 |
| 股票处于跌停/停牌 | 估值困难；可约定于正常交易日按收盘价分割或折算 |
| 期权/限制性股票（上市公司） | 未行权期权的现值认定有争议；已行权部分按收益认定 `[待验证]` |

## 错误处理

| 错误/敏感情形 | 识别信号 | 处理方式 |
|------------|---------|---------|
| 对方隐匿股权/公司 | 财产清单中缺少明知的公司股权；工商查询显示对方持股但未申报 | 🔴 调用 `/family-legal:concealed-asset-remedy`；申请工商调查令 |
| 房产单方秘密处置 | 离婚前夕突然房屋过户/抵押/出售 | 🔴 强制升级——申请不动产登记限制；撤销权主张（须律师确认 `[待验证]`） |
| 家暴影响房产分割 | 当事人因家暴被迫同意对方取得房产 | 🔴 强制升级——家暴情形协议可被主张撤销（意思表示受胁迫，合同编参照结构，`[待验证]`）；优先人身安全 |
| 财产评估争议重大 | 双方对房产或股权估值分歧超过30% | 建议委托司法鉴定机构，法院认可度更高 |
| 涉刑情形（职务侵占/挪用） | 疑似公司财产来源不明 | 🔴 强制升级刑事律师 |

## 法律依据

### 一、法律（法条）
