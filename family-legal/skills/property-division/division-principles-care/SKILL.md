# 分割原则与照顾（division-principles-care）

> **可单独触发**：`/family-legal:division-principles-care`
> 也可由 `property-division` 编排入口作为第三步调用（在特殊财产分割之后）。
>
> **与其他子技能的分工**：本子技能负责离婚分割中的照顾性调整、补偿请求权与损害赔偿；共同财产认定与特殊财产分割见前两个子技能；隐匿转移财产救济见 `concealed-asset-remedy`。
>
> **⚠️ 涉家暴情形特别提示**：本技能涉及家庭暴力导致的离婚损害赔偿（第1091条）。一旦识别到家庭暴力情形，**强制升级——优先保障人身安全**，在律师介入前不得推进损害赔偿数额谈判。

---

## 目的

分析离婚财产分割中的照顾原则与特殊请求权：

1. **离婚财产分割照顾原则**（第1087条）：照顾子女、女方和无过错方权益
2. **家务劳动补偿请求权**（第1088条）：负担较多家务义务的一方可请求补偿
3. **经济帮助请求权**（第1090条）：离婚后生活困难一方可请求适当帮助
4. **离婚损害赔偿**（第1091条）：无过错方在有法定过错情形时可请求损害赔偿

## 详细步骤

### 一、离婚财产分割照顾原则（第1087条，建议复核）

**基本规则**：离婚时，夫妻共同财产由双方协议；协议不成时，由法院根据财产具体情况，按照照顾子女、女方和无过错方权益的原则判决。

**照顾原则的适用分析**：

| 照顾情形 | 构成要件 | 实践中的影响程度 | 举证方向 |
|---------|---------|------------|---------|
| **照顾子女** | 直接抚养未成年子女一方，因承担日常抚育支出、子女有稳定居住需求 | 较大——通常倾向将有子女居住的房产判给抚养方，或优先考虑抚养方的居住需求 | 提供直接抚养证据（照顾记录/学校接送/医疗记录）；子女居住稳定性说明 |
| **照顾女方** | 女性一方处于明显不利地位（如家庭主妇/经济来源单一/年龄较大再就业困难） | 中等——具体比例酌定，不是硬性50%以上 | 收入差距证明；年龄与再就业困难说明；家庭贡献记录 |
| **照顾无过错方** | 另一方存在法定过错（重婚/同居/家暴/虐待遗弃/其他重大过错=第1091条法定情形）且导致离婚 | 中等——可影响分割比例（酌情增加无过错方份额），但具体比例由法院裁量 | 过错情形的证据（见损害赔偿章节举证清单） |

**照顾原则与损害赔偿的关系**：

- 照顾原则（第1087条）和损害赔偿（第1091条）是**两个独立请求权**，可同时主张
- 照顾分割影响的是共同财产分割比例（不均等分割）
- 损害赔偿是单独的赔偿请求权（可另行主张物质损害和精神损害抚慰金）

**核查清单（照顾原则）**：

- [ ] 是否有直接抚养未成年子女（照顾子女情形）
- [ ] 女方经济状况与男方是否有较大差距（照顾女方情形）
- [ ] 对方是否存在第1091条法定过错情形（照顾无过错方）
- [ ] 照顾原则主张的比例增加是否合理（须法院最终裁量，无法保证具体比例）

### 二、家务劳动补偿请求权（第1088条，建议复核）

**基本规则**：夫妻一方因抚育子女、照料老年人、协助另一方工作等负担较多义务的，离婚时有权向另一方请求补偿，另一方应当给予补偿。

**民法典修订说明**：民法典修订后，家务劳动补偿请求权**不再限于约定分别财产制**，所有夫妻均可主张（取消了原婚姻法司法解释的前提条件限制）。

**家务劳动补偿请求权分析**：

| 分析维度 | 内容 | 核查要点 |
|---------|------|---------|
| **「负担较多义务」的认定** | 须证明一方实际承担了远超另一方的家务劳动（抚育子女/照料老人/协助另一方工作） | 家务承担比例；是否为家庭主妇/主夫；照料老人的记录；协助另一方事业的具体贡献 |
| **补偿数额的计算基础** | 无明确计算公式（司法解释和司法实践中数额差异较大，`[待验证]`）；参考因素：家务年限、市场家政价格、工资损失、家庭规模 | 收集婚姻年限/家务承担年数/当地家政市场价格参考；法院最终酌定 |
| **举证方式** | 家务承担证据（证人证言/照片/医疗记录/学校接送记录/老人护理记录） | 证据碎片化，须系统收集 |
| **补偿时机** | 离婚时主张；协议离婚和诉讼离婚均可主张 | 注意：协议中如未约定，离婚后主张是否有时效问题须核实 `[待验证]` |

**家务补偿数额考量因素（实践参考，非法定标准）**：

| 因素 | 说明 |
|-----|------|
| 婚姻年限 | 婚龄越长，家务贡献积累越大 |
| 家务承担比例 | 主要承担全部家务 vs 承担部分家务，比例影响数额 |
| 对方收入水平 | 法院通常参考对方因获得支持而节省的成本 |
| 市场价格参考 | 当地家政服务市场价格（`[地方 flagged — 需核实地方价格水平]`） |
| 子女/老人数量 | 照料对象越多，负担越重 |
| 牺牲的职业发展机会 | 为家庭放弃职业发展的量化较难 |

### 三、经济帮助请求权（第1090条，建议复核）

**基本规则**：离婚时，如果一方生活困难，有负担能力的另一方应当给予适当帮助。

**请求权构成要件**：

| 要件 | 说明 | 举证要点 |
|-----|------|---------|
| **「一方生活困难」** | 须证明离婚后确实面临基本生活困难（无住房/无收入/重大疾病等），而非一般经济差距 | 收入证明/住房情况/健康状况/就业能力证明 |
| **「另一方有负担能力」** | 另一方须有实际经济能力给予帮助 | 另一方收入证明/财产状况 |
| **「适当帮助」** | 帮助程度以解决基本生活困难为限，不是无限制的扶养 | 法院酌定，无固定标准 |

**帮助方式**：

| 方式 | 说明 |
|-----|------|
| 住房方面 | 允许困难方在共同房产中居住一段时间，或给予其他住房安排 |
| 金钱帮助 | 一次性支付或定期支付（具体期限由双方协议或法院裁量） |
| 其他适当形式 | 由当事人协议或法院酌定 |

### 四、离婚损害赔偿（第1091条，建议复核）

**基本规则**：有下列情形之一导致离婚的，无过错方有权请求损害赔偿：（一）重婚；（二）与他人同居；（三）实施家庭暴力；（四）虐待、遗弃家庭成员；（五）有其他重大过错。

**⚠️ 家暴情形特别提示**：本条第三项涉及家庭暴力。识别到家暴情形时，**立即触发强制升级**（见前置章节），损害赔偿数额分析须在安全环境与律师指导下进行。

**各过错情形的认定与举证**：

| 过错情形 | 认定标准（描述规则，`[待验证]`） | 举证清单 | 举证难度 |
|---------|---------|---------|---------|
| **重婚** | 有配偶者与他人登记结婚 | 婚姻登记记录；公安户籍证明 | 相对明确（有登记记录） |
| **与他人同居** | 最高法司法解释认定：有配偶者与他人持续稳定地共同居住（描述规则，`[待验证]`）；非偶发性 | 同居地址证明；物业/门卫证人；通讯记录；照片/视频；共同消费记录 | 较高——须证明持续稳定性 |
| **实施家庭暴力** | 反家庭暴力法对家庭暴力的定义（描述规则，`[待验证]`）；包括身体暴力、精神控制等 | 医院伤情鉴定/诊断记录；报警记录/派出所处理记录；人身安全保护令申请记录；证人证言；伤情照片；通讯记录（威胁内容） | 较高——受害方往往因恐惧而未及时保留证据 |
| **虐待、遗弃家庭成员** | 持续性伤害行为（虐待）或拒绝履行扶养义务（遗弃） | 诊断记录；生活记录（对方未承担家庭责任的证明）；证人证言 | 较高 |
| **其他重大过错** | 由法院认定；实践中包括：长期赌博挥霍家庭财产、吸毒、刑事犯罪等 | 证据视具体情形而定 | 视情形 |

**损害赔偿的范围与数额**：

- 损害赔偿包括**物质损害赔偿**（财产损失）和**精神损害抚慰金**
- 精神损害抚慰金数额由法院酌定，无明确计算公式
- 实践中数额差异较大，受地方法院裁判口径影响  `[地方 flagged — 需核实地方裁判口径]`

**损害赔偿与照顾原则的关系（再强调）**：两个请求权独立，可并行主张；照顾原则影响共同财产比例，损害赔偿是独立赔偿金。

**损害赔偿的程序要点**：

- 提起时机：（1）提起离婚诉讼时一并主张；（2）离婚后在一定期限内单独主张（具体期限须核实 `[待验证]`）
- 诉讼形式：通常附随离婚诉讼提起，也可单独起诉
- 举证责任：无过错方须举证证明过错情形存在

**家暴情形的完整救济路径（强制升级后律师处理）**：

| 路径 | 内容 | 注意事项 |
|-----|------|---------|
| 人身安全保护令 | 向法院申请禁止被申请人实施家暴（反家庭暴力法，具体程序 `[待验证]`） | 紧急情况下优先申请；须提供家暴证据 |
| 离婚损害赔偿 | 第1091条第三项，家暴是法定请求权基础 | 须与离婚诉讼同时提起或离婚后一定期限内主张 |
| 刑事追责 | 家暴致人重伤/死亡构成故意伤害罪/故意杀人罪；持续性虐待可能构成虐待罪 | 须刑事律师介入 |
| 抚养权保护 | 家暴方通常不利于取得主要抚养权（最有利于未成年子女原则，第1084条，建议复核） | 须提供家暴影响子女的证据 |

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 家暴举证困难（未保留证据） | 律师指导补救性证据收集（邻居/家人证言/医疗记录补充）；申请调取报警记录 `[待验证]` |
| 过错认定与分割比例的关系 | 过错不自动确定具体分割比例（由法院酌定）；主张方不宜对比例有过高预期 |
| 损害赔偿请求与刑事程序衔接 | 重婚/虐待罪等刑事路径优先交刑事律师；民事损害赔偿与刑事追责可并行 |
| 经济帮助期限 | 经济帮助持续期限由双方协议或法院酌定，无法定期限（须核实 `[待验证]`） |
| 当事人情绪化要求高额赔偿 | 中立克制——说明法院裁量的不确定性；避免强化不切实际的期望 |
