# 隐藏转移财产救济（concealed-asset-remedy）

> **可单独触发**：`/family-legal:concealed-asset-remedy`
> 也可由 `property-division` 编排入口作为第四步调用，或在识别到财产异常时由其他技能触发跳转。
>
> **与其他子技能的分工**：本子技能专注于**主动不法行为的救济**（隐藏、转移、变卖、毁损、挥霍共同财产，伪造共同债务）；共同财产的认定见 community-property-identification；分割方案与补偿见 special-asset-division；分割原则见 division-principles-care。
>
> **⚠️ 紧迫性提示**：财产隐匿转移往往在离婚诉讼启动前或初期发生，时间窗口短暂。一旦识别到转移迹象，**应尽快申请诉前/诉中财产保全**，律师主导为宜。

---

## 目的

提供夫妻一方隐匿、转移、变卖、毁损、挥霍共同财产，或伪造共同债务情形的**完整救济路径**：

1. **识别隐匿转移行为**：常见情形、识别信号、财产线索收集
2. **少分或不分救济**（第1092条前半段）：离婚分割时对该方少分或不分
3. **离婚后再次分割**（第1092条后半段，描述规则）：离婚后发现可诉请再次分割
4. **举证策略**：调查令/工商查询/银行流水/司法鉴定取证路径
5. **财产保全**：诉前/诉中保全申请要点

## 详细步骤

### 一、隐匿转移财产的情形识别

**第1092条规定的五种不法行为**（建议复核）：

| 行为类型 | 常见手段 | 典型识别信号 |
|---------|---------|-----------|
| **隐藏** | 隐瞒财产存在（不申报存款账户/股权/贵重物品）；将财产转移至父母兄弟名下但实际控制 | 申报财产明显少于已知生活水平；工商查询显示隐性股权；社交媒体显示高消费但申报收入低 |
| **转移** | 将共同财产赠与或低价转让给第三人（父母/情人/朋友）；资金大额转出无说明；不动产无对价转让 | 离婚前夕大额转账给亲属；不动产突然过户；银行流水有大额出金无说明 |
| **变卖** | 低价处置共同财产（不动产/车辆/贵重物品）并截留价款 | 财产明显低价处置（明显低于市价）；短期内多项资产「消失」 |
| **毁损** | 故意损毁共同财产（车辆/贵重物品/生产设备） | 财产在离婚期间突然毁损/报废 |
| **挥霍** | 将共同财产用于赌博/吸毒/包养情人等挥霍行为 | 银行流水显示大额现金支取/博彩/娱乐；资产快速减少 |
| **伪造共同债务** | 伪造借条/虚构债务与亲属勾结（「夫妻共同债务」名义） | 离婚前突然出现大额债务；债主为亲友；无正当用途说明 |

**识别信号汇总表**：

| 信号类别 | 具体信号 | 紧急程度 |
|---------|---------|---------|
| 财产账目异常 | 银行流水有无法解释的大额转出/现金支取；账户余额与收入明显不符 | 🔴 高 |
| 资产快速消失 | 已知的房产/车辆/股权在离婚前夕突然「不见了」 | 🔴 高 |
| 关联方交易 | 大量资金流向亲属/朋友；资产以明显低价转让给熟人 | 🔴 高 |
| 突然新增债务 | 离婚前夕突然出现大额借款/债务，债主为亲友 | 🔴 高 |
| 工商信息变化 | 工商登记显示隐性股权/公司；或股权在离婚前被转走 | 🟠 较高 |
| 申报财产与生活水平明显不符 | 申报财产偏少但实际有明显高消费迹象 | 🟠 较高 |
| 涉赌博/吸毒/包养 | 银行流水有大额博彩/娱乐支出 | 🟠 较高 |

### 二、举证策略

**举证原则**：主张隐匿转移一方（通常为受损方）承担举证责任，但司法解释中有举证责任减轻规则（描述规则，`[待验证]`）——对方隐匿财产的证据明显难以获取时，法院可要求对方说明。

**举证路径**：

| 证据来源 | 可获取信息 | 获取方式 |
|---------|---------|---------|
| **银行流水** | 全部账户交易记录、大额出账方向、现金支取记录 | 申请法院调查令或当事人自行申请（部分银行支持），律师协助申请更有效 |
| **工商登记信息** | 对方名下公司/股权/任职情况（含隐性股权） | 通过国家企业信用信息公示系统查询；如信息不全，申请调查令 |
| **不动产登记信息** | 对方名下房产情况（含可能的低价转让/抵押记录） | 通过当地不动产登记中心查询；离婚诉讼立案后可申请法院调查 |
| **税务记录** | 对方收入（工资/经营收入/股权转让）缴税情况 | 法院调查令（需立案后） |
| **车辆登记** | 对方名下车辆及转让情况 | 公安交管信息查询；律师调查令 |
| **社保缴纳记录** | 对方真实收入水平 | 本人申请或律师调查令 |
| **第三方证人** | 亲属/朋友关于财产处置的知情证言 | 当事人收集；公证录音/录像（须合法取证） |
| **社交媒体/消费记录** | 高消费证据（奢侈品/出行/娱乐）与申报不符 | 截图保存；注意取证合法性 |

**调查令申请**：

离婚诉讼立案后，当事人可申请法院向金融机构/工商部门/不动产登记机构等发出调查令，获取对方财产信息。调查令是应对财产隐匿的重要工具，建议尽早申请。

**紧急举证措施**：在提起诉讼前，对已发现的可疑处置行为及时保全证据（截图/公证/拍照），避免证据灭失。

### 三、少分或不分救济（第1092条前半段，建议复核）

**规则**：一方隐藏、转移、变卖、毁损、挥霍夫妻共同财产，或伪造夫妻共同债务企图侵占另一方财产的，在离婚分割夫妻共同财产时，对该方可以少分或者不分。

**救济路径**：

| 路径 | 说明 | 适用条件 |
|-----|------|---------|
| **少分** | 降低过错方在共同财产中的分割比例（如30%而非50%）| 已有证据证明不法行为，但程度较轻 |
| **不分** | 过错方在某项财产上不获分配 | 不法行为严重，证据确凿 |
| **恢复价值折算** | 对已变卖/毁损的财产，按市场价格计算后在其他财产中扣除 | 财产已被处置无法恢复 |
| **伪造债务不认定** | 伪造的夫妻共同债务不予认定为共同债务 | 有证据证明债务系伪造 |

**证明标准**：少分不分须达到证明隐匿转移行为存在的标准；法院综合证据认定。

### 四、离婚后再次分割（第1092条后半段，描述规则）

最高法婚姻家庭编相关规则（描述规则，条号 `[待验证]`）：

> 离婚后，另一方发现有隐藏、转移、变卖、毁损、挥霍共同财产或伪造共同债务的行为的，可以向人民法院提起诉讼，请求再次分割夫妻共同财产。

**再次分割的重要注意事项**：

| 事项 | 说明 |
|-----|------|
| **诉讼时效** | 具体诉讼时效期限须核实（一般民事诉讼时效为3年，自知道或应当知道权利受损之日起算；但具体适用须核实 `[待验证]`） |
| **举证重点** | 须证明：（1）有隐匿转移不法行为；（2）该行为在离婚时未被发现/未被处理；（3）损失数额 |
| **适用范围** | 仅适用于不法行为（第1092条规定的情形）；不适用于正当财产处置 |
| **程序** | 重新起诉，属独立诉讼；非申请执行已生效判决 |

**协议离婚后再次分割**：

协议离婚后发现对方隐匿财产，同样可以依据第1092条主张再次分割，不因协议离婚程序的完成而丧失权利。

### 五、诉前/诉中财产保全

**财产保全的重要性**：财产保全是防止对方在诉讼期间继续转移财产、保障判决可执行性的关键措施。

**保全类型**：

| 类型 | 适用时机 | 主要条件 |
|-----|---------|---------|
| **诉前财产保全** | 尚未提起诉讼，但紧急情况下需保全 | 须缴纳担保金（一般为保全金额的一定比例）；须在保全后30日内提起诉讼 `[待验证]` |
| **诉中财产保全** | 离婚诉讼立案后提出 | 相对较易，可随时申请；提供担保后法院较快裁定 |
| **行为保全** | 需要禁止特定行为（如禁止转让不动产） | 须证明行为会造成难以弥补的损害 |

**保全对象**：

| 财产类型 | 保全方式 |
|---------|---------|
| 银行存款 | 冻结账户（法院向银行发出冻结令） |
| 不动产 | 查封/限制处分登记（法院向不动产登记机构发出协助执行通知） |
| 股权 | 冻结股权/禁止转让（法院向工商登记机构发出协助执行通知） |
| 车辆 | 扣押或限制转让（通过公安交管配合） |
| 到期债权 | 由法院通知第三方（借款人/承租人）向法院支付 |

**保全申请要点**：

1. 提交申请书（说明被保全财产、保全理由、担保安排）
2. 提供担保（现金/担保公司保函）
3. 法院裁定后立即执行（紧急情况下48小时内）`[待验证]`

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| **正常生活消费与挥霍的界限** | 正常合理的日常消费（餐饮/衣物/旅游）不构成挥霍；须证明支出明显超出正常生活水平或用于不正当目的 |
| **再次分割的诉讼时效** | 具体期限须核实（民事诉讼时效一般3年，从知道或应当知道权利受损之日起算，具体适用须核实）`[待验证]` |
| **虚构债务的认定** | 须区分真实债务与虚构债务；有借条但无资金流向的债务存疑；债主为亲属且无合理说明的存疑 |
| **善意第三人受让** | 对方转让给不知情第三人的财产，第三人受保护（物权法规则）；受损方只能主张转让价款的分配 |
| **保全损失赔偿** | 申请保全错误（对方财产系正当处置）须赔偿对方保全期间损失；申请前须充分评估证据强度 |
| **境外财产取证** | 境外银行账户/资产调查路径特殊，须借助司法协助或境外律师，时间成本高 `[待验证]` |

## 错误处理

| 错误/敏感情形 | 识别信号 | 处理方式 |
|------------|---------|---------|
| **财产隐匿与家暴同时存在** | 当事人描述在胁迫下不敢调查财产；财产信息完全被对方控制 | 🔴 **强制升级**——优先人身安全；家暴情形须律师主导，财产保全路径在安全环境下推进 |
| **虚构债务涉嫌刑事** | 伪造证据（假借条/虚假公证）；涉嫌诈骗/合同诈骗 | 🔴 **强制升级刑事律师**；保全证据避免灭失 |
| **当事人情绪危机** | 因财产损失导致极度绝望/危机信号 | **暂停分析**——提示心理援助；标注 `[暂停 — 情绪危机]` |
| **紧急转移进行中** | 对方正在过户/大额转账中 | 🔴 **立即申请诉前财产保全**——须律师主导，最快48小时内申请 |
| **举证严重不足** | 仅有怀疑无任何证据 | 告知举证责任风险；规划调查令申请计划；不基于无证据主张出具风险报告 |

## 法律依据

### 一、法律（法条）

**核心锚点（中度把握，建议复核）：**

- 《中华人民共和国民法典》**第1092条**（隐藏转移变卖毁损挥霍共同财产/伪造共同债务，少分或不分；离婚后发现可再次分割）`[模型知识 — 需验证]`  中度把握，建议复核

### 二、司法解释（描述规则，条号待验证）
