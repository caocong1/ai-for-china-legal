# 访谈子技能三：敏感度与升级（sensitivity-escalation）

> **可单独触发**：`/family-legal:sensitivity-escalation`
> 也可由 `cold-start-interview` 编排入口作为第三步顺序调用。
>
> **填充目标**：本访谈子技能的结果写入 `family-legal/CLAUDE.md` 的以下章节：
> - `## 敏感度校准`（家庭暴力识别/未成年子女事项/情绪危机/涉刑风险/敏感度等级）
> - `## 风险校准`（风险等级定义表）
> - `## 升级矩阵`（角色权限表/强制升级触发条件/诉讼触发表）
>
> **非独立 YAML**：输出直接写入 `family-legal/CLAUDE.md` 对应章节，不生成独立 YAML 文件。
>
> **⚠️ 本技能的特殊重要性**：婚姻家事是本项目所有领域中敏感度最高的之一——涉及当事人情绪（离婚冲突/丧亲悲痛/家暴恐惧）、未成年人利益与人身安全。本访谈子技能的目的是**固化敏感护栏**，使后续所有技能自动按配置识别敏感情形并升级。

---

## 目的

配置婚姻家事敏感度护栏，包括：

1. **家庭暴力识别等级**：当前事项中是否有家暴风险，以及处理策略
2. **未成年人利益保护**：是否涉及未成年子女，强制以最有利于未成年子女为原则
3. **当事人情绪危机**：识别情绪危机信号，暂停分析提示心理支持
4. **涉刑情形**：重婚/虐待/遗弃/拒不执行等涉刑风险识别
5. **重大财产隐匿阈值**：配置财产隐匿转移的强制升级阈值
6. **升级矩阵固化**：强制升级触发条件的完整配置

---

## 前置：访谈情绪护栏

> **本访谈子技能本身须特别注意情绪护栏**。询问家暴、情绪危机等敏感话题时：
> - **中立、克制的措辞**：「我需要了解一些敏感情形以配置安全护栏」，而非「你是否受过家暴」（后者可能触发情绪）
> - **识别情绪危机信号**：如当事人在回答中表现出强烈情绪化或危机信号，立即暂停，优先提示心理支持
> - **不替当事人判断**：记录当事人的描述，不作道德评判

---

## 访谈问卷

### 第一部分：家庭暴力识别配置

**问题 1：在当前事项中，是否存在家庭暴力、虐待或遗弃的相关情形或风险？**

> 目的：配置家暴识别等级，使后续技能在识别到相关信号时自动触发保护机制。

| 选项 | 说明 | 技能响应 |
|-----|------|---------|
| **A. 高风险（已有明确情形）** | 当事人已明确描述家庭暴力/虐待/遗弃情形，有证据或记录 | 所有技能强制开启家暴护栏；优先提示人身安全保护令路径 |
| **B. 中风险（有疑似情形）** | 当事人描述中有疑似家暴迹象（恐惧/受伤迹象/控制行为） | 开启疑似家暴护栏，评估安全状况 |
| **C. 低风险（无家暴情形）** | 无任何家暴相关描述或迹象 | 正常运行，保留识别机制 |
| **D. 待评估** | 不确定，需要进一步了解 | 默认开启识别，发现信号即触发 |

CLAUDE.md 目标字段：`## 敏感度校准 > ### 家庭暴力识别与处理 > 家暴识别等级`

**如选 A 或 B，立即输出家暴护栏**（不等待访谈完成）：

```
⚠️ [家暴敏感情形识别]

已识别到家庭暴力/虐待相关情形。

处理优先级：
1. 请先确认当前人身安全——如有危险请立即拨打 110 或联系妇联庇护热线
2. 家暴是诉讼离婚的法定准予条件（民法典第1079条，建议复核）
3. 家暴构成离婚损害赔偿的法定情形（第1091条，建议复核）
4. 可向法院申请人身安全保护令（反家庭暴力法，具体程序须核实）
5. 建议立即联系执业律师

本访谈可在确认安全后继续。
```

### 第二部分：未成年人利益保护配置

**问题 2：当前事项是否涉及未成年子女？**

| 选项 | 说明 | 技能响应 |
|-----|------|---------|
| **A. 是（涉及未成年子女）** | 有子女抚养/探望/财产影响未成年人利益等事项 | 所有涉子女事项强制以「最有利于未成年子女」为首要原则（第1084条，建议复核） |
| **B. 否** | 不涉及未成年子女 | 不适用 |
| **C. 不确定** | 可能涉及 | 默认开启保护，发现涉子女内容即激活 |

CLAUDE.md 目标字段：`## 敏感度校准 > ### 未成年人利益 > 涉未成年子女事项`

**如选 A，询问补充信息（选填）**：

| 字段 | 问题 |
|-----|------|
| 子女年龄 | 子女年龄？（影响抚养原则：不满2岁/2-8岁/8岁以上）`[建议核实民法典第1084条适用规则]` |
| 主要关切 | 主要抚养权争议 / 探望权安排 / 子女财产安排 / 其他 |

### 第三部分：当事人情绪危机配置

**问题 3：在当前婚姻家事事项中，当事人的情绪状态如何评估？**

> 说明：婚姻家事当事人常处于高度情绪化状态，技能输出须相应调整措辞。

| 选项 | 说明 |
|-----|------|
| **A. 高（识别到强烈情绪化表达/危机信号）** | 极度绝望、危机性语言（须立即暂停分析） |
| **B. 中（明显情绪波动）** | 当事人有较强情绪，但无危机信号 |
| **C. 低** | 相对平稳，能理性沟通 |
| **D. 待观察** | 不确定，随访谈进展评估 |

CLAUDE.md 目标字段：`## 敏感度校准 > ### 当事人情绪危机 > 情绪危机等级`

**所有情绪等级的通用原则**：技能输出**一律中立、克制、非煽动**；不激化与对方的对立情绪。

### 第四部分：涉刑情形配置

**问题 4：当前事项中是否存在涉刑情形的风险？**

| 选项 | 说明 | 技能响应 |
|-----|------|---------|
| **A. 是（已识别涉刑情形）** | 重婚/虐待家庭成员罪/遗弃家庭成员罪/拒不执行判决裁定罪 | 所有技能强制升级刑事律师 |
| **B. 待评估** | 有迹象但不确定 | 开启识别，发现涉刑信号即强制升级 |
| **C. 否** | 无涉刑情形 | 正常运行 |

CLAUDE.md 目标字段：`## 敏感度校准 > ### 涉刑情形识别 > 涉刑风险`

### 第五部分：重大财产隐匿阈值配置

**问题 5：触发「重大财产隐匿转移」强制升级的金额阈值是多少？**

> 说明：当识别到对方可能隐匿转移超过此金额的共同财产时，技能将强制升级给律师。

推荐参考：50万元（较保守），100万元（一般），300万元（高净值财产情况）

用户输入：[     ] 万元（如不确定，可填「不设阈值，由律师判断」）

CLAUDE.md 目标字段：`## 敏感度校准 > ### 敏感度等级与处理策略 > 重大财产隐匿转移（超过 [X] 万）`

### 第六部分：强制升级触发条件确认

**问题 6：以下强制升级触发条件，请确认是否采用默认配置（建议全部保留）：**

| 触发条件 | 默认配置 | 是否保留 |
|---------|---------|---------|
| 家庭暴力/虐待/遗弃（确认或高度疑似） | 强制升级，优先人身安全 + 保护令 | [ ] 保留（推荐）/ [ ] 自定义 |
| 涉刑事（重婚/虐待罪/遗弃罪/拒执罪） | 强制升级刑事律师 | [ ] 保留（推荐）/ [ ] 自定义 |
| 未成年人重大利益（主要抚养权/子女生活保障） | 强制升级，以最有利于未成年子女为首要原则 | [ ] 保留（推荐）/ [ ] 自定义 |
| 重大财产隐匿转移（超过上面配置的阈值） | 强制升级，提示申请财产保全 | [ ] 保留（推荐）/ [ ] 自定义 |
| 协议离婚生效前（非律师角色） | 须律师审核 | [ ] 保留（推荐）/ [ ] 自定义 |
| 遗嘱定稿前（非律师角色） | 须律师审核遗嘱形式要件 | [ ] 保留（推荐）/ [ ] 自定义 |
| 当事人情绪危机信号 | 暂停分析，提示心理援助 | [ ] 保留（推荐）/ [ ] 自定义 |

CLAUDE.md 目标字段：`## 升级矩阵 > ### 强制升级触发条件`

> **技术说明**：家暴/涉刑/未成年人三项强制升级条件是安全护栏，**不建议自定义降低**。如有特殊情况需自定义，须在 CLAUDE.md 中明确注明原因。

| 触发情形 | 检查操作 |

| 用户描述「家暴」后提及具体条号 | 核查条号是否正确（民法典第1079条/第1091条，建议复核）；旧法条号自动纠正 |

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 用户回避家暴问题 | 中立提示：「我了解这是敏感话题，配置家暴识别是为了确保系统在相关情形下自动提供保护。如暂时不确定，可设置为「待评估」」 |
| 用户主动描述家暴情形 | 立即输出家暴护栏提示，优先人身安全，暂停访谈进行 |
| 用户希望降低家暴护栏等级 | 告知风险，须在 CLAUDE.md 中明确注明；不建议降低 |
| 涉及未成年子女信息 | 不记录具体子女姓名或高度个人信息，只记录年龄范围和事项类型 |
