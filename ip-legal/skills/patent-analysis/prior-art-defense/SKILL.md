# 现有技术抗辩与不侵权抗辩（prior-art-defense）

> 前置读取：`ip-legal/skills/_shared/legal-basis-conventions.md`（引用规范）、`ip-legal/skills/_shared/ip-law-citations.md`（引用库）、`ip-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）、`ip-legal/CLAUDE.md`（实践配置）

---

## 目的

在字面侵权与等同侵权分析完成后，本子技能对被诉方面可提出的**抗辩理由**进行系统梳理与初步评估，并对涉案专利的稳定性进行初步预判，最终汇总专利侵权初步分析综合意见。

本技能覆盖四类核心抗辩与一项综合意见：
1. **现有技术抗辩**（被诉方案已属于现有技术，不构成侵权）
2. **先用权抗辩**（申请日前已在使用或作好必要准备，在原有范围内继续使用不侵权）
3. **不侵权抗辩**（基于全面覆盖与等同分析不成立的综合抗辩主张）
4. **专利权稳定性初判**（涉案专利是否存在被宣告无效的风险）
5. **专利侵权初步分析综合意见**（侵权可能性高/中/低）

**注意**：本技能为**初步分析**，非正式侵权鉴定。正式专利侵权鉴定须由具有专利侵权鉴定资质的专业机构或专利司法鉴定机构出具。

**本子技能可由律师/IP 经理/专利代理师单独触发：`/ip-legal:prior-art-defense`**

## 详细步骤

### 第一步：现有技术抗辩评估

**现有技术抗辩规则（来自专利法与专利侵权司法解释，描述规则，不写裸条号）**：

> 被诉侵权人有证据证明其实施的技术或者设计属于现有技术或者现有设计的，不构成侵犯专利权（现有技术抗辩）。

**现有技术抗辩构成要件：**

| 要件 | 说明 |
|-----|------|
| 「现有技术」的定义 | 专利申请日（或主张的优先权日）之前，在国内外为公众所知的技术（来自专利法，描述规则，条号 `[待验证]`） |
| 被诉方案实施的是现有技术 | 被诉方案整体上等同于某一现有技术方案（完整性要求：不是局部相同），而非仅个别特征类似 |
| 现有技术来源 | 申请日前已公开的专利文献、学术论文、产品、行业标准、公知常识等 |
| 不侵权逻辑 | 若被诉方案实施的是现有技术，则该方案本在专利保护范围之外（专利应不授权），即便通过比对看起来「落入保护范围」，亦不侵权（抗辩性防御） |

**现有技术抗辩评估表：**

| 评估项 | 结论 | 说明 |
|-------|------|------|
| 被诉方案核心技术特征是否在专利申请日前已有公开文献/资料 | 是/否/不确定 | [文献或资料来源] |
| 现有技术来源与被诉方案的对应程度 | 高度对应/部分对应/低对应 | [具体分析] |
| 现有技术抗辩成立可能性 | 高/中/低 | [综合判断] |
| 建议行动 | [检索现有技术文献/委托专业机构检索/不适用] | — |

### 第二步：先用权抗辩评估

**先用权抗辩规则（来自专利法，描述规则，条号 `[待验证]`）**：

> 在专利申请日**前**已经制造相同产品、使用相同方法，或者已经作好制造、使用的**必要准备**，并且**仅在原有范围内**继续制造、使用的，不构成侵犯专利权（先用权抗辩）。

**先用权抗辩构成要件：**

| 要件 | 说明 |
|-----|------|
| 时间要件 | 在涉案专利**申请日前**已经开始制造/使用，或已完成必要准备（设计图纸、设备采购、生产线建设等） |
| 独立开发 | 先用权人须独立研发，不得来源于对专利权人技术的窃取或不当获取 |
| 原有范围限制 | 先用权仅限于**申请日前已有的原有范围**内继续实施，不得扩大规模；若原有范围外扩大则构成侵权 |
| 举证责任 | 先用权由主张方举证，须提供申请日前的制造/使用证据（合同、发票、设备采购记录、技术文件等） |

**先用权抗辩评估表：**

| 评估项 | 结论 | 说明 |
|-------|------|------|
| 是否有证据证明专利申请日前已制造/使用或完成必要准备 | 有/无/不确定 | [证据清单或缺口] |
| 是否独立研发（非来源于专利权人） | 是/否/不确定 | [说明] |
| 当前实施规模是否在原有范围内 | 是/否/不确定 | [说明] |
| 先用权抗辩成立可能性 | 高/中/低 | [综合判断] |
| 建议行动 | [整理先用权证据/评估规模限制/不适用] | — |

### 第三步：不侵权抗辩梳理

**不侵权抗辩**是基于全面覆盖与等同分析的综合抗辩主张，汇总字面侵权和等同侵权的不成立理由：

**不侵权抗辩类型对照表：**

| 抗辩类型 | 来源分析 | 说明 |
|---------|---------|------|
| 缺少必要技术特征 | all-elements-rule 结论：有技术特征「缺少」 | 字面侵权不成立（缺一不可原则） |
| 技术特征字面不同且不等同 | doctrine-of-equivalents 结论：不构成等同 | 字面+等同均不成立 |
| 禁止反悔原则限制 | doctrine-of-equivalents 中禁止反悔适用 | 等同被排除，整体不侵权 |
| 现有技术抗辩（第一步） | 被诉方案属于现有技术 | 不构成侵权 |
| 先用权抗辩（第二步） | 申请日前已使用且在原有范围内 | 不构成侵权 |
| 合法来源抗辩 | 合法购买且不知情的销售者（注意限制） | 免赔但不免停止侵害 |
| 权利用尽 | 专利权人已授权销售，专利权已耗尽 `[待验证具体规则]` | 下游流通使用不侵权 |
| 科研例外 | 以科学研究为目的，非生产经营目的使用 `[待验证具体规则]` | 不构成侵权 |

**重要提示**：
- **合法来源抗辩**：仅能免除赔偿责任，不能免除**停止侵害**的责任（描述规则，不写裸条号）。即使合法购买，法院仍可判令停止销售侵权产品。
- **权利用尽原则**：专利权人或经其许可的人销售专利产品后，专利权在该特定产品上已耗尽，购买者和后续用户可自由使用、转卖（描述规则，条号 `[待验证]`）。

### 第四步：专利权稳定性初判

**专利权稳定性初判目的**：评估涉案专利被无效宣告的可能性。专利权不稳定时，被诉方可通过提出专利无效宣告请求（向国家知识产权局复审和无效审理部门）来应对侵权诉讼。

**新颖性与创造性瑕疵初步评估：**

| 评估维度 | 说明 | 风险提示 |
|---------|------|---------|
| 新颖性风险 | 专利申请日前是否存在与该专利相同技术方案的公开文献/产品 | 若现有技术抗辩中发现高度相似的申请日前文献，同时存在新颖性无效风险 |
| 创造性风险 | 申请日前是否存在主要技术特征的组合，本领域普通技术人员能轻易想到 | 若权利要求技术特征均为已知手段的简单组合，创造性存疑 |
| 公开充分性风险 | 说明书是否充分公开实施方式（若说明书过于简单无法支持权利要求范围，存在公开充分性问题） | — |
| 申请文件规范性 | 权利要求是否得到说明书支持、是否清楚简要等格式要件 | — |

**专利权稳定性风险初判表：**

| 风险类型 | 风险程度 | 初判理由 |
|---------|---------|---------|
| 新颖性被宣告无效风险 | 🔴高/🟠较高/🟡中等/🟢低 | [初判理由，标待验证] |
| 创造性被宣告无效风险 | 🔴高/🟠较高/🟡中等/🟢低 | [初判理由，标待验证] |
| 其他无效理由（公开充分等） | 🔴高/🟠较高/🟡中等/🟢低 | [初判理由] |
| **整体专利权稳定性** | **高稳定/中等稳定/不稳定** | [综合结论] |

**后续建议**：若稳定性不确定，建议委托专利代理师进行正式无效宣告预评估，通过专利检索系统检索相关现有技术文献（https://pss-system.cponline.cnipa.gov.cn）。

**重要提示**：专利无效宣告须通过专门程序向国家知识产权局复审和无效审理部门提出，不在侵权诉讼中直接处理（但侵权诉讼中可能援引稳定性作为抗辩）。

### 第五步：专利侵权初步分析综合意见

汇总 claim-construction、all-elements-rule、doctrine-of-equivalents 与本技能的全部分析，输出专利侵权初步分析综合意见：

**侵权可能性综合判断矩阵：**

| 侵权路径 | 分析结论 | 侵权可能性贡献 |
|---------|---------|------------|
| 字面侵权（全面覆盖原则） | 成立/不成立 | 成立→直接贡献高侵权可能 |
| 等同侵权（等同原则） | 成立/不成立/未分析 | 成立→贡献等同侵权可能 |
| 现有技术抗辩 | 强/弱/不适用 | 强→降低综合侵权可能 |
| 先用权抗辩 | 强/弱/不适用 | 强→降低综合侵权可能 |
| 专利权稳定性 | 高稳定/中等/不稳定 | 不稳定→降低侵权诉讼风险，但不直接影响现阶段侵权结论 |

**综合意见风险等级：**

| 等级 | 含义 | 典型情形 |
|-----|------|---------|
| 🔴 侵权可能性高 | 字面或等同侵权成立，无有效抗辩，专利权稳定性高 | 建议立即采取行动（维权方）/ 立即寻求法律救济（被诉方） |
| 🟠 侵权可能性较高 | 字面或等同侵权大概率成立，抗辩较弱，专利稳定性中等 | 须认真对待，建议律师深度参与 |
| 🟡 侵权可能性中等 | 侵权是否成立存在较大不确定性，或有有效抗辩但尚未确定 | 建议进一步深入分析，视商业重要性决策 |
| 🟢 侵权可能性低 | 字面与等同均不成立，或有明确强有力抗辩，或专利权稳定性低 | 综合侵权风险较低，可继续推进 |

**本技能为初步分析免责提示**：

> 本专利侵权分析为**初步分析**，非正式侵权鉴定，不能作为诉讼中的证据使用。正式专利侵权鉴定须由具有法定资质的专利司法鉴定机构出具。本分析的结论具有参考价值，建议由专利代理师或知识产权律师进一步核实。

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 本技能为「初步分析」非正式侵权鉴定 | 在所有输出中明确标注初步分析性质；正式侵权鉴定须由法定机构出具；本分析仅供参考，不作诉讼证据 |
| 合法来源抗辩的效力限制 | 合法来源抗辩仅免赔偿责任，不能免除停止侵害的责任；在分析中须明确说明此限制 |
| 专利无效宣告程序 | 无效宣告须向国家知识产权局复审和无效审理部门提出，属于专门程序；本技能仅作初步风险预判，提示后续里程碑 |
| 先用权的「原有范围」认定争议 | 「原有范围」的认定标准存在争议（是固定数量上限还是产能上限），建议咨询专利代理师具体分析 `[待验证具体规则]` |
| 方法专利的侵权证据难题 | 方法专利侵权举证困难（尤其是在被诉方工厂内部实施的方法），须考虑举证困难对侵权主张的影响（描述规则，不写裸条号） |
| FTO（自由实施）分析需求 | 若用户实际需要的是FTO分析（拟实施技术方案是否侵犯他人专利），建议调用 ip-clearance（06-03 专利 FTO）子技能，而非本侵权分析流程 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 用户要求本分析作为正式侵权鉴定报告 | 明确告知：本分析为初步分析，不具有正式侵权鉴定的法律效力；建议委托专利司法鉴定机构或专利代理机构出具正式意见 |
| 缺少前序子技能输出即触发 | 可基于用户直接描述案情进行分析，但须在输出中标注「基于口述信息，未经系统比对」 |
| 现有技术信息不足以评估抗辩强弱 | 标注「现有技术抗辩评估信息不足，建议委托专业检索机构进行现有技术检索」；不得凭猜测作出确定性结论 |
| 用户混淆「现有技术抗辩」与「专利无效宣告」 | 明确区分：现有技术抗辩是侵权诉讼中的抗辩方式（不需要另启程序）；专利无效宣告是独立的行政程序（须向国家知识产权局提出）；两者可同时使用 |
