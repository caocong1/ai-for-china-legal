## 访谈问题清单

> **访谈说明**：本节关于输出格式偏好，影响技能生成报告的结构与风格。你可随时说「使用默认」跳过某问题。

### 第一节：文书风格偏好

**问题 1.1 — 商标检索报告风格**

> 你们的商标检索报告通常是什么格式/风格？（多选或自由描述）

1. **结构化表格+综合结论**：检索结果清单（商标号/商标/类别/申请人/近似度/风险等级）+ 综合结论段落
2. **风险分级清单**：按🔴🟠🟡🟢分类列出障碍商标，每条含简短说明
3. **摘要+详细附件**：执行摘要（1-2页）+ 详细检索结果附件
4. **叙述式意见书**：律所意见书格式，叙述性段落为主，表格为辅
5. **自定义格式**（请描述）：

| 字段 | 对应位置 |
|-----|---------|
| 商标检索报告风格 | `## 文书风格` → 商标检索报告风格 |

**问题 1.2 — 侵权分析意见书风格**

> 专利侵权/商标侵权分析意见书的格式偏好是？

1. **法律意见书格式**：正式律所意见书，含案件标题、法律分析章节、意见结论、律师签署
2. **技术比对表格+法律结论**：权利要求比对表 + 侵权判断 + 法律结论
3. **内部研究笔记格式**：简洁的内部工作产物，不需正式格式
4. **保密级别说明**：（高度保密/律师工作产物/内部使用）
5. **其他（请描述）**：

| 字段 | 对应位置 |
|-----|---------|
| 侵权分析意见书风格 | `## 文书风格` → 侵权分析意见书风格 |

**问题 1.3 — 清权报告风格**

> 知识产权清权报告（覆盖商标/专利FTO/著作权字号域名多维度）的格式偏好是？

1. **多维度清权结论+障碍清单+建议方案**：按维度分章节，每维度含发现+风险+建议
2. **综合风险矩阵+结论**：汇总表（各维度风险等级矩阵）+ 清权结论 + 建议行动
3. **摘要结论+详细分维度报告**：执行摘要（三档结论）+ 各维度详细分析附件
4. **其他（请描述）**：

| 字段 | 对应位置 |
|-----|---------|
| 清权报告风格 | `## 文书风格` → 清权报告风格 |

**问题 1.4 — 维权函/律师函风格**

> 维权函或律师函的措辞风格是？（注意：发出前须律师确认）

1. **正式坚定但保留协商空间**：措辞正式，说明侵权事实和法律依据，但态度上保留协商/和解可能
2. **强硬不留余地**：措辞坚定，明确要求立即停止侵权并赔偿，不表达协商意向
3. **先礼后兵式**：先以礼貌通知方式告知对方，若不回应再升级措辞
4. **视情况（视侵权性质和对方身份决定）**：

| 字段 | 对应位置 |
|-----|---------|
| 维权函/律师函风格 | `## 文书风格` → 维权函/律师函风格 |

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 用户跳过文书风格问题 | 保留 `[PLACEHOLDER]`，技能将使用通用报告结构 |
| 用户不确定报告格式（如「客户不同格式不同」）| 填写「按项目定制」，技能在生成前询问偏好格式 |
| 检索数据库 identity-team 已填充无变化 | 本节确认后跳过，不重复修改 |
| 行业识别全部不适用 | 保留默认行业识别逻辑（所有行业均采用通用识别），不特别激活某行业 |
| 用户希望所有报告均为保密级别 | 在文书风格字段末添加「保密：律师工作产物/内部使用」，不额外修改 CLAUDE.md 结构 |
