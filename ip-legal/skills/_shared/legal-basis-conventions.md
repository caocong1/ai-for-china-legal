# 法律依据引用规范（知识产权版）

> 本文件定义 ip-legal 插件所有技能统一使用的法律依据引用规范。
> 后续计划（06-02、06-03）直接引用本规范，避免各技能重复维护。
> **知识产权法在 2019/2020/2021 年经过多次修正，条号重排风险极高，本规范特别强调保守标注原则。**

---

## 一、来源分类

所有法律依据必须归入以下类别之一，并在引用时明确标注类别标签。

### 1. 法律（全国人大及其常委会制定）

指全国人民代表大会及其常务委员会制定的法律，知识产权语境中主要包括：

- 《中华人民共和国商标法》（2019年修正）
- 《中华人民共和国专利法》（2020年修正，2021年6月1日施行）
- 《中华人民共和国著作权法》（2020年修正，2021年6月1日施行）
- 《中华人民共和国反不正当竞争法》（2019年修正）
- 《中华人民共和国民法典》（2021年1月1日施行，人格权编与知识产权相关）

**法律层级最高，是知识产权分析的核心基础。**

> **IP 法修正特别提示（必须遵守）**：商标法（2019修正）、专利法（2020修正/2021施行）、著作权法（2020修正/2021施行）、反不正当竞争法（2019修正）均经历修正，导致条号重排。**凡涉及上述法律的具体条号，除少数中度把握锚点外，一律标 `[待验证 — IP 法修正后条号重排，须核实现行版本]`，不臆造旧版条号。**

**格式要求（法律最小引用格式）：**

```
《[法律全名]》（[修正/施行年份]）第[X]条[第X款]  [来源标签]  [验证状态]
```

示例（中度把握锚点）：
```
《中华人民共和国商标法》（2019年修正）第57条  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
```

示例（一般条号）：
```
《中华人民共和国专利法》（2020年修正）第[X]条  `[模型知识 — 需验证]`  `[待验证 — IP 法修正后条号重排，须核实现行版本]`
```

### 2. 司法解释（最高人民法院发布）

指最高人民法院发布的解释性文件，知识产权语境中主要包括：

- 《最高人民法院关于审理商标授权确权行政案件若干问题的规定》（具体条号 `[待验证]`）
- 《最高人民法院关于审理专利侵权纠纷案件应用法律若干问题的解释》（一）（具体条号 `[待验证]`）
- 《最高人民法院关于审理专利侵权纠纷案件应用法律若干问题的解释（二）》（具体条号 `[待验证]`）
- 《最高人民法院关于审理著作权民事纠纷案件适用法律若干问题的解释》（具体条号 `[待验证]`）
- 《最高人民法院关于审理不正当竞争民事案件应用法律若干问题的解释》（具体条号 `[待验证]`）

**核心纪律（必须遵守）**：

> **司法解释的具体条号一律标 `[待验证]`，以「描述规则 + 待验证」呈现，不写裸条号。** 规则内容（如商标近似判断规则、等同原则三要素）可完整描述，但来源条号须待验证。

**格式要求：**

```
《[司法解释全名]》（[发布年份]，最高人民法院）规定：[描述规则]  [来源标签]  `[待验证]`
```

示例：
```
《最高人民法院关于审理商标授权确权行政案件若干问题的规定》（最高人民法院发布）规定：商标近似判断以相关公众一般注意力为标准，采用整体观察与隔离比对相结合的方法，对显著部分重点比对，以是否容易产生混淆为判断依据  `[模型知识 — 需验证]`  `[待验证 — 具体条号须核实]`
```

### 3. 部门规章与审查指南（国家知识产权局/国家版权局等制定）

指国务院各部门制定的规章、审查指南或规范性文件，知识产权语境中主要包括：

- 《商标审查审理指南》（国家知识产权局，`[待验证发布/修订年份]`）
- 《专利审查指南》（国家知识产权局，`[待验证发布/修订年份]`）
- 《作品自愿登记办法》（国家版权局/国家互联网信息办公室，`[待验证]`）
- 《计算机软件著作权登记办法》（国家版权局，`[待验证]`）
- 《商标审查标准》（国家知识产权局，`[待验证]`）

**核心纪律（必须遵守）**：

> **部门规章/审查指南的具体条号、具体审查标准数值、具体申报期限，一律标 `[待验证]`，以描述规则呈现，绝不硬写裸条号。**

**格式要求：**

```
《[规章/指南全名]》（[制定机关]，[年份]）[描述规则]  [来源标签]  `[待验证]`
```

### 4. 案例（指导性案例与知识产权法庭典型案例）

知识产权领域可参考的案例类型：

- **最高院指导性案例**：最高人民法院发布，对下级法院有参照作用，仅作参考，不具判例法约束力
- **最高院知识产权法庭典型案例**：知识产权法庭发布，实践参考价值较高，但**非判例法**
- **知识产权法院/中院典型案例**：北京/上海/广州知识产权法院及各地中院知识产权庭典型案例，参考价值有限

**核心纪律（必须遵守）**：

> **知识产权诉讼案例数量较多，但模型回忆的案号风险极高，所有案例引用默认标 `[待验证]`，未经检索核验不得移除该标注。** 中国采成文法体系，案例不具判例法约束力，须明确说明。

**格式要求：**

```
[案例名称或案号 `[待验证]`]  [裁判要点摘要]  [法院层级]  参考，不具判例法约束力  `[待验证]`
```

### 5. 学说（学术观点）

学说为辅助参考，存在争议性，引用时须标注来源和争议性：

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息]  "[观点摘要]"  [是否有争议：有/无]  `[待验证]`
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用五个标准标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站（全国人大/国家知识产权局/国家版权局） | 官网原始文件 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查；IP 法修正频繁，风险更高 |
| `[用户提供]` | 来自用户提供的文件 | 权利证书、合同、注册证等 |

**标签使用规则：**

1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 来源在依赖前**必须**对照原始来源检查——IP 法修正频繁，风险尤为突出
4. 来源标签不替代 `[待验证]` 标注——两者可同时出现在一行

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` 的条号 | 模型回忆的条文号风险高 | 国家法律法规数据库、国家知识产权局官网 |
| **IP 法 2019/2020/2021 修正后的任何具体条号**（中度把握锚点除外） | 商标法/专利法/著作权法/反不正当竞争法均经历修正，条号重排 | 国家法律法规数据库 flk.npc.gov.cn 核实现行版本 |
| 司法解释的任何具体条号 | 司法解释条号须核实 | 最高人民法院官网、法宝 |
| 部门规章/审查指南的具体条号或具体数值 | 规章修订频繁，条号与数值须核实 | 国家知识产权局官网、国家版权局官网 |
| 距最后更新超过 6 个月的法规 | 法规可能已修订或废止 | 查询官方数据库最新版本 |
| 用户口述的条款号或期限阈值 | 可能记忆偏差或引用旧版 | 核查官方法规全文 |
| 所有案例引用（默认） | IP 诉讼案例多，案号极易错误回忆 | 中国裁判文书网、国家知识产权局官网 |
| 施行状态不确定的法规 | 部分法规施行状态须核实 | 国家法律法规数据库 |

**待验证标记格式（示例）：**

```
《中华人民共和国专利法》（2020年修正）第[X]条  `[模型知识 — 需验证]`  `[待验证 — IP 法修正后条号重排，须核实现行版本]`
```

## 四、IP 法 2020/2021 修正版本提示 — 条号重排保守标注

> **这是知识产权语境最重要的纪律，必须严格遵守。**

### 修正情况说明

| 法律 | 修正年份 | 施行日期 | 条号重排风险 |
|------|---------|---------|------------|
| 《中华人民共和国商标法》 | 2019年修正 | 2019年11月1日 | 高 — 条号重排 |
| 《中华人民共和国专利法》 | 2020年修正 | 2021年6月1日施行 | 高 — 条号重排（外观设计保护期由10年改为15年） |
| 《中华人民共和国著作权法》 | 2020年修正 | 2021年6月1日施行 | 高 — 条号重排 |
| 《中华人民共和国反不正当竞争法》 | 2019年修正 | 2019年4月23日 | 中 — 局部调整 |

### 保守标注规则

**中度把握锚点（须标「已核实锚点（中度把握），建议复核」）：**

以下条号系中度把握，引用时须明确标注「已核实锚点（中度把握），建议复核」，且不得以此为由臆造其他条号：

| 锚点 | 法律依据 | 规则摘要 |
|------|---------|---------|
| 专利权期限 | 《专利法》（2020年修正）**第42条** | 发明专利 20 年；实用新型专利 10 年；外观设计专利 15 年（2021修正后，旧为10年）；均自申请日起算 |
| 商标侵权类型 | 《商标法》（2019年修正）**第57条** | 列举七类侵权行为，含未经许可在同一种商品使用相同/近似商标致混淆等 |

**规则稳定但条号建议复核（须描述规则，条号标「建议复核」）：**

| 规则 | 规则摘要 | 条号状态 |
|------|---------|---------|
| 商标注册有效期 | 10 年，期满可续展，每次续展 10 年 | 条号建议复核 `[待验证]` |
| 著作权保护期 | 自然人作品：作者终生及死亡后 50 年；法人作品/视听作品：首次发表后 50 年 | 条号建议复核 `[待验证]` |
| 著作权自愿登记 | 著作权自愿登记，非确权，但具有初步证据效力 | 条号建议复核 `[待验证]` |

**规则类（来自司法解释，描述规则即可，不写裸条号）：**

| 规则 | 规则摘要 |
|------|---------|
| 专利侵权判定 | 全面覆盖原则 + 等同原则（等同 = 手段-功能-效果基本相同 + 本领域普通技术人员无需创造性劳动可联想）；禁止反悔原则；捐献原则 |
| 商标近似判断 | 以相关公众一般注意力为标准；整体观察与隔离比对相结合；对显著部分重点比对；以混淆可能性为判断依据 |
| 类似商品判定 | 参照《类似商品和服务区分表》；结合功能用途/生产部门/销售渠道/消费对象综合判断；区分表为参照，不绝对 |
| 作品独创性与思想表达二分 | 受保护的是表达，不是思想；独创性要求最低创造性门槛 |

**保守原则（硬性要求）：**

> **不确定即标待验证，不臆造。** 除上述中度把握锚点外，商标法/专利法/著作权法/反不正当竞争法的其余具体条号，一律以「描述规则 + 待验证」呈现，绝不硬写无把握的条号。知识产权法律条号错误会直接误导权利范围认定、侵权判断、期限计算等关键法律结论。

## 五、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：知识产权分析的所有结论，必须首先援引现行成文法（商标法/专利法/著作权法/反不正当竞争法及相关法规）作为规范依据
2. **不得以案例替代法条**：即使存在明确的典型案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下司法解释/典型案例/学说」

### 案例的辅助定位

1. **最高院指导性案例**：仅作参考，标注「最高院指导，参照意义强，不具判例法约束力」
2. **最高院知识产权法庭典型案例**：实践参考价值较高，标注「知识产权法庭典型案例，参照意义强，不具判例法约束力」
3. **知识产权法院/中院典型案例**：较有参考价值，标注「知识产权法院/中院典型案例，参照」
4. **明确说明非判例法**：中国采成文法体系，案例不具判例法约束力

### 知识产权案例的特殊说明

- 知识产权诉讼案例远多于一般商事案例，但模型回忆案号的错误风险同样极高
- 典型案例（尤其最高院知识产权法庭发布的年度报告案例）对司法实践有重要参考价值
- **所有案例案号一律标 `[待验证]`，不依赖模型回忆**
- 建议通过中国裁判文书网（wenshu.court.gov.cn）、最高院官网检索核验

## 六、法院层级与机构定位

### 法院层级（知识产权案件）

| 层级 | 说明 | 知识产权特点 |
|------|------|------------|
