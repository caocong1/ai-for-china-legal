# 商标清权检索（trademark-clearance）

> 前置读取：`ip-legal/skills/_shared/legal-basis-conventions.md`（引用规范）、`ip-legal/skills/_shared/ip-law-citations.md`（引用库）、`ip-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）、`ip-legal/CLAUDE.md`（实践配置）

---

## 目的

在律师/IP 经理**决定使用或申请注册某商标/标识之前**，对拟用商标/标识进行系统性商标清权检索，识别在先注册商标、申请中商标、驰名商标及其他在先权利障碍，分别评估：

1. **可注册性**：是否存在绝对/相对注册障碍，拟用标识能否申请并通过注册
2. **可使用性**：即使无法注册或暂未注册，使用该标识是否会侵犯他人有效注册商标专用权

清权结论分三档：**✅ 可注册可使用 / ⚠ 有风险（区分可使用不可注册/有障碍需处理）/ ❌ 有侵权风险不建议用**

**本子技能可由律师/IP 经理单独触发：`/ip-legal:trademark-clearance`**

**本子技能也可由 ip-clearance 编排入口作为多维清权的商标维度调用。**

## 前置

### 读取实践配置

在开始商标清权检索前，读取 `ip-legal/CLAUDE.md`：

- **清权严格度**（`## 维权与申请立场` → `### 清权严格度`）：驱动近似风险标记阈值与清权结论门槛
- **商标策略**（`## 维权与申请立场` → `### 商标策略`）：近似容忍度、注册防御范围、异议倾向
- **整体维权立场**（`## 维权与申请立场`）：进取/平衡/防御 决定障碍严重性认定严格程度
- **IP 组合概况—商标**（`## 我们是谁` → IP 组合概况）：了解现有商标格局及拟用标识在整体 IP 战略中的定位
- **可用检索数据库**（`## 可用集成`）：确认中国商标网/商业商标数据库可用状态
- **执业角色**（`## 谁在使用`）：决定工作产物标题格式

**临时模式（配置缺失/含占位符时）：**

> 未检测到知识产权实践档案（或关键字段含占位符）。将使用临时模式默认值：清权严格度=保守（任何近似风险即标记）、整体立场=平衡型、近似容忍度=平衡、检索数据库=中国商标网。输出标记 `[临时模式]`。
> 运行 `/ip-legal:cold-start-interview` 可配置定制档案。

### 与其他子技能的分工

**清权（clearance）视角**与 trademark-search 的分工：

| 子技能/模块 | 视角 | 核心任务 |
|-----------|------|---------|
| trademark-search（搜索查询/近似判断/类似商品/在先权利） | 已有商标的查询与近似分析（评估他人对我的侵权，或我的商标能否注册）| 商标检索、近似比对、障碍识别，为维权/注册申请服务 |
| **本技能（trademark-clearance）** | **拟用/拟注册前的前瞻性清权视角**：「我能不能用这个标识？」 | 清权检索 + 可注册性初判 + **可使用性初判** + 清权结论三档，为拟用行为是否合规提供决策支持 |

本技能**复用** trademark-search 的近似判断方法（参见 `skills/trademark-search/similarity-assessment/SKILL.md`）和在先权利分析方法（参见 `skills/trademark-search/prior-rights-obstacles/SKILL.md`），**不重复实现**核心近似算法，聚焦清权决策视角。

## 详细步骤

### 第一步：厘清拟用标识与使用场景

| 信息项 | 说明 |
|-------|------|
| 拟用标识内容 | 文字（汉字/英文/拼音/数字）、图形、文字+图形组合 |
| 拟用商品/服务类别 | 尼斯分类（第 1-45 类），明确核定商品或服务的具体项目 |
| 拟用行为性质 | 仅使用（不申请注册）/ 申请注册后使用 / 先使用后申请 |
| 地域范围 | 中国大陆 / 港澳台 / 海外（本技能仅覆盖中国大陆注册商标法律框架） |
| 使用场景 | 商品包装、服务标识、广告宣传、企业名称、App 图标、域名 |
| 标识构成要素 | 识别标识的显著部分（文字读音/字形/含义 + 图形整体外观）|

**若信息不足**：不得自行假设拟用类别或场景，须向用户确认后继续。

### 第二步：确定清权检索范围

| 检索维度 | 说明 |
|---------|------|
| **核心类别（拟用类别）** | 与拟用商品/服务直接相同的尼斯分类 |
| **关联类别** | 与核心类别商品/服务构成类似群组或有关联的类别（按《类似商品和服务区分表》）|
| **驰名商标跨类保护** | 清权范围不限于相同/类似类别，需关注可能主张跨类保护的驰名商标 |
| **检索数据库** | 优先中国商标网（官方）；商业数据库作补充（可用性依 `ip-legal/CLAUDE.md` → `## 可用集成`）|

**商标清权检索范围表：**

| 类别层次 | 类别内容 | 检索必要性 |
|---------|---------|---------|
| 核心类别 | 拟用商品/服务直接所属的尼斯类别 | 必须检索 |
| 类似群组 | 与核心类别存在类似关系的其他类别（区分表类似群） | 应当检索 |
| 关联类别 | 与拟用商品/服务有业务关联的跨类别 | 建议检索 |
| 驰名商标全类 | 知名度较高的在先商标（可主张跨类保护） | 评估后检索 |

### 第三步：执行商标清权检索（复用 search-query 方法）

**检索方法参照 `skills/trademark-search/search-query/SKILL.md` 中的检索构造规则**，清权场景下的检索要点：

- **文字商标**：逐一检索汉字完整形式、常见简写、拼音、英文译名
- **图形商标**：按维也纳图形分类要素检索近似图形（需人工逐图比对）
- **组合商标**：分别检索文字部分与图形部分（显著部分为重点）
- **检索逻辑**：相同商标检索（精确匹配）→ 近似商标检索（读音/字形/含义近似）→ 主要视觉元素检索

**检索结果整理：**
列出全部在先商标结果，包括：商标号、商标图样、申请人、核定类别/商品项目、申请/注册日期、商标状态（注册有效/申请审查中/已撤销/已无效/已过期）。

### 第四步：近似判断（复用 similarity-assessment 方法）

**近似判断方法参照 `skills/trademark-search/similarity-assessment/SKILL.md`，清权场景聚焦「拟用行为是否导致混淆」视角：**

- **近似判断维度**：音（读音/发音）、形（字形/图形外观）、义（含义/联想）三维度
- **判断标准（描述规则，具体司法解释条号 `[待验证]`）**：以相关公众一般注意力为标准；整体观察与隔离比对相结合；对显著部分重点比对；以混淆可能性为核心判断依据
- **近似结论三级**：高度近似（混淆可能性高）/ 有一定近似（需结合商品类似性综合判断）/ 不近似

**近似判断表（按 `ip-legal/CLAUDE.md` → 清权严格度 调整阈值）：**

| 在先商标 | 商标号 | 类别 | 音近似 | 形近似 | 义近似 | 整体近似度 | 清权风险初判 |
|---------|-------|------|------|------|------|---------|-----------|
| [在先商标1] | [号] | [类别] | [高/中/低] | [高/中/低] | [高/中/低] | [高度/有一定/不近似] | [🔴/🟠/🟡/🟢] |

### 第五步：可注册性初判

评估拟用标识能否申请注册，分绝对障碍与相对障碍：

**绝对注册障碍（不因他人在先权利，而因标识自身性质）：**

| 障碍类型 | 判断要点 | 结论 |
|---------|---------|------|
| 缺乏显著性 | 描述性词汇/通用名称/行业通用符号，直接描述商品/服务特征 | 该商标可注册性低，须附加其他显著性要素 |
| 标志禁用（条号 `[待验证]`）| 国旗、国徽、国际组织标识等禁用标志 | 直接不可注册 |
| 欺骗公众（条号 `[待验证]`）| 标识内容使公众对商品/服务产生误认 | 不可注册 |
| 功能性（仅适用于外观设计/颜色商标）| 标识本身是实现技术功能所必须的 | 不可注册 |

**相对注册障碍（基于他人在先权利）：**

| 障碍类型 | 判断要点 | 参考法律依据 |
|---------|---------|-----------|
| 与在先注册商标相同或近似（同一/类似商品服务）| 步骤4近似判断结论 + 步骤3类别判断 | 商标法在先注册原则（条号 `[待验证]`） |
| 在先申请商标冲突（申请在先原则）| 他人申请日早于拟用标识申请日 | 商标法申请在先原则（条号 `[待验证]`） |
| 驰名商标跨类保护 | 检索到可能主张跨类保护的驰名商标 | 商标法驰名商标保护（条号 `[待验证]`） |
| 在先著作权/外观设计专利权冲突 | 拟用标识与他人美术作品/外观设计实质相似 | 商标法在先权利保护（条号 `[待验证]`） |
| 在先企业名称/字号冲突 | 拟用标识与他人有一定影响力的企业名称/字号相同或近似 | 商标法在先权利保护（条号 `[待验证]`）; 反不正当竞争法（条号 `[待验证]`）|

### 第六步：可使用性初判（清权关键步骤）

**可使用性与可注册性的区别**：即使拟用标识无法注册（如缺乏显著性），使用该标识不一定侵权；即使目前有注册障碍（如近似在先商标），若在先商标已被撤销/期满，使用也可能合法。**可使用性侵权判断的核心标准**：

> **是否构成商标侵权（描述规则，中度把握 — 建议复核）**：
> 《商标法》（2019年修正）第57条规定侵犯注册商标专用权的行为，主要包括：未经商标注册人许可，在**同一种商品**上使用与注册商标**相同**的商标；或在同一种商品上使用**近似**商标，或在**类似商品**上使用相同或近似商标，**容易导致混淆**。
>
> **已核实锚点（中度把握），建议复核**：《商标法》（2019年修正）第57条 `[模型知识 — 需验证]`

**可使用性判断要素：**

| 要素 | 判断问题 | 判断结论 |
|-----|---------|---------|
| 在先注册商标有效性 | 障碍商标是否仍为有效注册状态（非过期/撤三/无效）| [有效/已失效/待核实] |
| 商标相同性 | 拟用标识与在先注册商标是否相同 | [相同/近似/不相同] |
| 商品/服务类似性 | 拟用商品/服务是否与在先注册商标核定范围相同或类似 | [相同/类似/不类似] |
| 混淆可能性 | 综合近似度、商品类似度、相关公众注意力，是否容易导致混淆 | [高/中/低] |
| 注册商标权利状态 | 权利人是否仍在积极使用（撤三可能性）| [积极使用/使用存疑/不使用] |

**注意**：可使用性侵权判断结论具有法律不确定性，须提示升级律师确认（尤其是将发布产品/启动商标使用的场景）。

## 检查清单

在出具清权结论前，逐项核实：

- [ ] 拟用标识与使用场景已完整收集（文字/图形内容 + 类别 + 使用行为性质）
- [ ] 清权检索范围覆盖核心类别 + 关联类别 + 驰名商标跨类保护维度
- [ ] 检索数据库已确认可用（中国商标网官方为首选）
- [ ] 全部在先商标检索结果已整理（商标号/申请人/类别/状态）
- [ ] 对检索发现的在先商标完成音/形/义三维度近似判断
- [ ] 绝对注册障碍已评估（显著性/禁用标志等）
- [ ] 相对注册障碍已评估（在先注册商标/申请中商标/驰名商标/其他在先权利）
- [ ] **可使用性初判已完成**（是否构成对有效注册商标的侵权风险）
- [ ] 清权严格度已从 `ip-legal/CLAUDE.md` 读取（未读取须标 `[临时模式]`）
- [ ] 清权结论按三档给出（✅ / ⚠ / ❌）
- [ ] 清权报告组装提示（若为多维清权，链接 clearance-risk-report 子技能汇总）

## 边界条件

| 情形 | 处理方式 |
|-----|---------|
| 拟用标识为纯图形（无文字）| 重点说明图形商标清权须人工逐图比对，自动检索工具有局限，建议专业检索 |
| 未注册商标的在先使用抗辩 | 评估在先使用商标是否可主张在先使用抗辩（须积累使用证据），标注法律不确定性 |
| 拟用标识含他人字号/姓名 | 同时触发著作权清权与字号清权（标注：建议同步运行 copyright-tradename-domain-clearance 子技能）|
| 在先障碍商标疑似三年不使用 | 提示撤销注册（撤三）可能性，但撤三成功有不确定性，不得作为清权通过依据 |
| 检索数据库不可用 | 提示无法执行完整清权，须用户手动提供检索结果后继续 |
| 拟用标识将在海外使用 | 提示本技能仅覆盖中国大陆商标法律框架，海外清权须单独分析 |
| 驰名商标跨类风险不确定 | 标注驰名商标认定的不确定性，建议律师评估 |

## 错误处理

| 错误类型 | 错误信息 | 处理建议 |
|---------|---------|---------|
| 清权对象信息不足 | 未提供拟用类别或使用场景 | 向用户确认后再开始检索，不得假设 |
