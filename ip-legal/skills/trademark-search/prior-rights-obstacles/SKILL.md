# 在先权利与障碍分析（prior-rights-obstacles）

> 前置读取：`ip-legal/skills/_shared/legal-basis-conventions.md`（引用规范）、`ip-legal/skills/_shared/ip-law-citations.md`（引用库）、`ip-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）、`ip-legal/CLAUDE.md`（实践配置）

---

## 目的

综合商标检索结果（search-query）、近似判断（similarity-assessment）、类似商品判定（goods-services-classification）的结论，对拟申请注册或拟使用商标进行全面的在先权利障碍分析，识别绝对障碍与相对障碍，评估各障碍的严重程度，给出可注册性与可使用性初步结论，并提出障碍应对建议。

**本子技能是商标查询与近似分析流程的最终汇总步骤，也可由律师/IP 经理单独触发：`/ip-legal:prior-rights-obstacles`**

---

## 前置

### 读取实践配置

在开始障碍分析前，读取 `ip-legal/CLAUDE.md`：

- **整体维权立场与商标策略**（`## 维权与申请立场`）：决定障碍严重性分级与障碍应对建议的严格程度
- **近似容忍度**（`### 商标策略` → 近似容忍度）：影响障碍严重性阈值判断
- **商标异议/无效宣告倾向**（`### 商标策略`）：影响障碍处理建议（是否建议异议/清除在先障碍）
- **清权严格度**（`### 清权严格度`）：综合结论的严格程度（如进取维权型会将中等混淆可能性障碍升级为高风险）
- **执业角色**（`## 谁在使用`）：决定工作产物标题格式与是否需要升级提示

若配置含 `[PLACEHOLDER]` 或文件不存在，使用临时模式：按保守标准标记常见障碍，标注 `[临时模式]`。

### 与其他子技能的分工

| 子技能 | 分工说明 |
|-------|---------|
| search-query | 提供在先商标检索结果清单（在先注册/申请商标）|
| similarity-assessment | 提供拟用商标与各在先商标的音/形/义近似判断结论 |
| goods-services-classification | 提供商品/服务类似判定结论 |
| **本技能（prior-rights-obstacles）** | 综合上述三个子技能结论，识别所有类型在先权利，分级评估障碍严重性，给出综合结论与建议 |

**本技能也可独立触发（不依赖前三个子技能）**：此时本技能自行进行基础检索与初步分析，但结论会有限制，须在输出中注明。

## 详细步骤

### 第一步：系统性识别在先权利类型

在先权利涵盖多种类型，须逐一检查。**不遗漏任何类型是本步骤的核心要求。**

**在先权利类型清单：**

| 在先权利类型 | 说明 | 障碍性质 | 检索/识别路径 |
|-----------|------|---------|------------|
| **在先注册商标** | 他人已注册且有效的相同/近似商标（同一或类似商品/服务） | 相对障碍 | search-query 检索结果 |
| **在先申请商标** | 他人在本申请之前提交的商标申请（申请在先原则）| 相对障碍 | search-query 检索结果（申请中状态）|
| **驰名商标**（注册/未注册）| 他人已被认定的驰名商标，跨类保护的威胁 | 相对障碍（扩大保护范围）| 查询驰名商标认定记录 `[待验证路径]` |
| **在先字号（企业名称）** | 他人在先登记使用的具有一定知名度的字号/企业名称 | 相对障碍（可能主张在先权利）| 企业信用信息系统检索 |
| **在先著作权** | 他人的作品（文字/图形/美术等）在拟申请商标中被使用，可能侵犯著作权 | 相对障碍（在先权利主张）| 著作权登记查询（如有登记）；视觉比对 |
| **在先外观设计专利权** | 拟申请商标（图形/立体/颜色）与他人已注册外观设计专利相同/近似 | 相对障碍 | 专利检索系统检索 |
| **在先姓名权/肖像权** | 拟申请商标含有他人真实姓名/肖像，可能主张姓名权/肖像权 | 相对障碍（在先权利主张）| 视申请内容，须询问 |
| **绝对禁止注册事由** | 缺乏显著性/通用名称/欺骗性/有害社会公德等，不得注册 | 绝对障碍 | 商标显著性分析（见第二步）|

**在先权利的法律依据：**

> **不得损害他人在先权利规则（商标法相关条款，具体条号 `[待验证 — IP 法修正后条号重排]`）**：商标注册申请不得损害他人现有的在先权利，包括在先注册商标、在先申请商标（申请在先原则）、在先著作权、在先外观设计专利权、在先企业名称权（字号权）、在先姓名权、肖像权等。**描述规则，具体条号须核实**。

### 第二步：绝对障碍评估

绝对障碍是指商标本身因内在属性不得注册的情形，与他人权利无关。

**绝对禁止注册事由清单（基于商标法，具体条号 `[待验证]`）：**

| 绝对障碍类型 | 示例 | 风险等级 |
|-----------|------|---------|
| **缺乏显著性（通用名称）** | 拟注册商标是该类商品/服务的通用名称（如「苹果手机」中「手机」部分）| 🔴 绝对不可注册 |
| **高度描述性** | 商标直接描述商品/服务的主要特点（如「新鲜」用于食品、「快速」用于物流服务）| 🟠 难以注册（需证明通过使用获得显著性）|
| **欺骗性（虚假性质地理来源）** | 商标使消费者对商品/服务的性质、质量、产地等产生误解 | 🔴 绝对不可注册 |
| **社会公德/国家秩序** | 商标内容有害社会公德或有损社会主义精神文明 | 🔴 绝对不可注册 |
| **政治标志/地名禁用** | 含有国家名称/国旗/国徽/军徽、县级以上行政区划名称等 `[待验证具体规则]` | 🔴/🟠 视具体情形 |
| **通过使用获得显著性（第二含义）** | 原本描述性商标经长期使用已获得识别性（可抗辩绝对障碍）| 可主张第二含义争取注册 |

**绝对障碍分析流程：**

1. 识别商标中是否含有通用名称/高度描述性词汇
2. 评估商标是否涉及禁用标志
3. 评估商标是否可能产生欺骗性
4. 如存在描述性问题，评估是否可以通过第二含义克服

### 第三步：相对障碍分级评估

基于 search-query + similarity-assessment + goods-services-classification 的综合结论，对每一个发现的在先权利进行障碍分级：

**障碍严重性分级标准：**

| 障碍级别 | 判定标准 | 建议行动 |
|---------|---------|---------|
| 🔴 **严重障碍（不可克服）** | 在先注册商标相同/高度近似 AND 同一/高度类似商品；或涉及绝对禁止事由；或在先驰名商标（有效注册+活跃使用）且主张跨类保护 | 建议放弃该商标方案；或选择完全不同的商标 |
| 🟠 **重大障碍（可尝试克服）** | 在先注册商标较高程度近似 AND 类似商品；或驰名商标跨类保护风险（知名度较高但非最高级）；或在先字号/著作权主张有一定依据 | 须律师评估克服可能性：提无效宣告/申请撤三/谈判取得许可/修改商标方案 |
| 🟡 **一般障碍（有克服可能）** | 在先注册商标有一定程度近似 AND 商品存在一定关联；或在先权利主张较弱；或申请中/审查中状态（可等待结果）| 进一步评估，修改商标细节或等待在先申请结果 |
| 🟢 **轻微障碍/无障碍** | 在先商标近似程度低/商品不类似/在先权利已失效；无明显障碍 | 可继续推进 |

### 第四步：驰名商标跨类保护专项分析

驰名商标保护是在先权利分析中的重要专项，保护范围突破类别限制：

**驰名商标保护规则（来自商标法及相关规定，具体条号 `[待验证]`，描述规则）：**

- **已注册驰名商标**：不仅在相同/类似商品上受保护，还可跨类请求保护；申请注册与驰名商标相同/近似、可能误导公众、致使驰名商标注册人的利益受损的，不得注册
- **未注册驰名商标**：在相同或类似商品上，他人将未注册驰名商标复制/模仿/翻译并申请注册，权利人可提出异议；**未注册驰名商标不享有跨类保护**（仅已注册驰名商标有跨类保护）
- **驰名商标认定**：采「被动保护、个案认定」原则，不主动申请；认定结果有参考价值但非永久性

**驰名商标检索路径：**

- 国家知识产权局官网（商标局）公布的驰名商标认定记录 `[待验证检索路径]`
- 最高院/知识产权法院典型案例中的驰名商标认定 `[待验证]`

**驰名商标跨类风险评估：**

| 情形 | 风险级别 |
|------|---------|
| 拟用商标与知名全国性品牌（市场领先地位）相同/高度近似 | 🔴 严重跨类风险 |
| 拟用商标与省级/区域知名品牌相同/高度近似 | 🟠 有一定跨类风险 |
| 拟用商标与一般商标相同/近似 | 无跨类风险，按普通近似判断 |

### 第五步：综合障碍评估与可注册性结论

汇总所有障碍类型，给出综合评估：

**可注册性结论框架：**

| 结论 | 标准 | 说明 |
|------|------|------|
| **可注册/可使用（低风险）** | 无绝对障碍；无相同/高度近似在先商标；无驰名商标跨类风险；无其他实质性在先权利障碍 | 建议继续推进申请/使用 |
| **有障碍，可评估克服（中风险）** | 存在🟠重大障碍或多个🟡一般障碍；障碍有一定克服可能 | 须律师评估具体克服方案，告知申请被驳回/被异议的风险 |
| **有严重障碍，建议放弃/修改（高风险）** | 存在🔴严重障碍（与在先相同/高度近似商标冲突，无法克服）| 建议放弃该商标方案，选择不同商标；或通过全面修改降低风险 |

**与 ip-clearance 的衔接说明：**

本子技能完成商标申请注册/使用的障碍分析；若用户有全维度清权需求（商标+专利+著作权+字号+域名），须触发 ip-clearance（06-03）各子技能进行完整清权分析。

在完成障碍分析后，按以下清单验证：

- [ ] 在先注册商标障碍已评估（基于 search-query + similarity-assessment + goods-services-classification 结论）
- [ ] 在先申请商标（申请中状态）障碍已识别
- [ ] 驰名商标跨类风险已专项分析
- [ ] 在先字号权利障碍已识别（企业信用信息系统检索）
- [ ] 在先著作权障碍已识别（特别针对图形商标/组合商标）
- [ ] 在先外观设计专利障碍已识别（特别针对图形商标/立体商标）
- [ ] 在先姓名权/肖像权问题已识别（如商标含有人名/人像）
- [ ] 绝对障碍（通用名称/缺乏显著性/欺骗性/禁用标志）已评估
- [ ] 所有障碍已分级（🔴🟠🟡🟢）
- [ ] 综合可注册性结论已给出
- [ ] 已涉及清权需求时（全维度）已提示触发 ip-clearance

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 在先注册商标连续三年不使用（撤三可能） | 标注「权利人可能三年内未使用，可评估撤三申请清除障碍（须核实使用情况）」；不得直接推断为无障碍 |
| 在先权利已到期但未续展 | 标注「在先权利到期未续展，须确认已正式注销状态（非自动消灭）」`[待验证]` |
| 在先申请尚在审查中 | 标注「在先申请尚在审查中，若注册则形成障碍；若被驳回则障碍消除——可等待结果或提前制定应对预案」 |
| 拟用商标在申请前已有一定市场知名度（可争取驰名商标认定） | 说明权利人亦可寻求驰名商标认定以争取对在先权利的抗辩力，须律师评估 |
| 在先字号权利人/著作权人身份不明 | 标注「在先权利人身份须进一步核实」`[待验证]` |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 未获得 search-query/similarity-assessment/goods-services-classification 结论 | 在输出中注明「本分析未基于前三个子技能的完整结论，仅为初步障碍评估，建议进行完整检索分析后再作结论」|
| 驰名商标认定记录查询失败 | 标注「驰名商标认定记录须通过商标局官网核实 `[待验证]`」；提示律师确认 |
| 在先权利权利状态不明（如商标数据库显示异常状态） | 标注「状态不明的在先权利须通过商标局咨询或专业检索核实 `[待验证]`」，不得基于不明状态作出「无障碍」结论 |
