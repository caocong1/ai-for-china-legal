# 类似商品与服务判定（goods-services-classification）

> 前置读取：`ip-legal/skills/_shared/legal-basis-conventions.md`（引用规范）、`ip-legal/skills/_shared/ip-law-citations.md`（引用库）、`ip-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）、`ip-legal/CLAUDE.md`（实践配置）

---

## 目的

判定拟用商品/服务与在先商标核定使用的商品/服务是否构成类似商品或类似服务，结合近似判断结论（similarity-assessment）共同支撑混淆可能性评估，为在先权利障碍分析（prior-rights-obstacles）提供商品/服务类似性结论。

**本子技能可由律师/IP 经理单独触发：`/ip-legal:goods-services-classification`**

## 详细步骤

### 第一步：确认商品/服务信息

| 信息项 | 说明 |
|-------|------|
| 拟申请/使用的商品/服务 | 具体商品/服务名称（越具体越好，如「在线教育软件」而非「软件」） |
| 拟申请/使用的类别 | 尼斯分类类别号 |
| 在先商标核定商品/服务 | 来自 search-query 结果或用户提供 |
| 在先商标核定类别 | 尼斯分类类别号 |

**若商品/服务名称过于宽泛（如仅说「科技产品」「服务」），须请用户提供更具体的商品/服务描述，否则无法进行精确的类似判定。**

### 第二步：确定尼斯分类类别

《商标注册用商品和服务国际分类》（尼斯分类）当前版本将商品与服务分为 45 个类别：

**商品类（第 1-34 类）概览：**

| 类别 | 主要内容（示例） |
|------|--------------|
| 第 1-5 类 | 化工原料、涂料、洗涤剂/农药/药品 |
| 第 6-11 类 | 金属材料、机械/工具/器具、家用器具/照明/空调 |
| 第 12-14 类 | 交通工具/载具、烟草/火柴/打火机、贵金属/珠宝 |
| 第 15-21 类 | 乐器、纸张印刷品/文具/办公用品、橡胶塑料/非金属建材/皮革、家具/装饰品、炊具餐具/厨房用品 |
| 第 22-28 类 | 绳索/帐篷/帆布、纱线/织物/纺织品、地毯/墙纸/帷幕、服装/鞋帽/头饰、蕾丝/刺绣/丝带、游戏/玩具/运动器械 |
| 第 29-34 类 | 肉类/加工蔬菜/食用油脂、咖啡/茶/糖/调味品/面包糕点、啤酒/矿泉水/饮料、烟草/吸烟用品 |

**服务类（第 35-45 类）概览：**

| 类别 | 主要内容 |
|------|---------|
| 第 35 类 | 广告/商业经营/管理/办公服务 |
| 第 36 类 | 金融/货币/保险/不动产业务 |
| 第 37 类 | 建筑/修缮/安装维修服务 |
| 第 38 类 | 电信通信服务 |
| 第 39 类 | 运输/货物包装与储存/旅行安排 |
| 第 40 类 | 材料处理/废物处理/污染控制/印刷 |
| 第 41 类 | 教育/提供培训/娱乐/文体活动 |
| 第 42 类 | 科学技术/研究与设计/工业分析/计算机软件设计与开发/法律服务 |
| 第 43 类 | 提供食物和饮料/临时住所服务（餐厅/酒店等） |
| 第 44 类 | 医疗/兽医/保健/美容/农业园艺服务 |
| 第 45 类 | 法律服务/安全服务/个人社会服务（婚介/殡仪等） |

> **注意**：以上仅为概览；具体商品/服务所属类别须查阅最新版《类似商品和服务区分表》确认，不得凭印象判断。

### 第三步：查阅区分表与类似群组

**《类似商品和服务区分表》（以下简称「区分表」）的法律地位：**

> **区分表的地位（来自司法解释，描述规则，不写裸条号）**：区分表是商标审查中判断类似商品的重要参照，但在民事侵权诉讼中，区分表为参照而非绝对标准，法院可结合个案具体情形综合认定。

**区分表应用步骤：**

1. **查找商品/服务在区分表中的类似群组编号**
   - 区分表中每类商品/服务按类似群组编排（如第9类含多个类似群组：09-01计算机/数据处理设备、09-02摄影光学影院仪器等），具体编号须查阅最新版区分表 `[待验证 — 区分表版本]`
   - 同一类似群组内的商品/服务推定类似

2. **判断是否同一类似群组**
   - 同类同群：直接判定类似（同一类似群组内）
   - 同类不同群：需结合关联性因素进一步判断
   - 不同类别：需结合关联性因素判断是否构成跨类类似

3. **区分表参照的局限性**
   - 区分表版本须定期更新，新兴商品/服务可能尚未收录或分类不准确（如 NFT、区块链服务、AI 服务等新兴类目）
   - 区分表在行政审查中有较强参照效力，但在侵权诉讼中可被突破

**区分表突破情形（区分表不足以确定类似性）：**

| 情形 | 处理方式 |
|------|---------|
| 新兴商品/服务未被区分表收录 | 参照关联性因素综合判断，标注「区分表未明确收录，须结合关联性判断」|
| 同类但功能用途差异极大 | 可能认定不类似；须综合关联性因素 |
| 不同类但功能/消费者高度重叠 | 可能认定类似；须综合关联性因素 |
| 商品与服务之间的类似 | 须参照区分表交叉检索表 + 关联性因素综合判断（如零售商品服务与商品本身）|

### 第四步：关联性因素综合判断

当区分表无法直接给出明确结论时，综合以下因素判断：

**类似商品判断因素表（来自司法解释，描述规则）：**

| 判断因素 | 分析维度 | 高度类似信号 |
|---------|---------|-----------|
| **功能和用途** | 两种商品的使用功能/满足需求是否相同/相近 | 用于相同用途（如「智能手机」与「平板电脑」均用于移动通信与娱乐）|
| **主要原料/成分** | 两种商品的主要原材料是否相同/相近 | 相同原料制成的不同产品（部分情况适用）|
| **生产/制造部门** | 是否来自同一生产厂家或同一行业 | 通常由相同厂商生产（如饮料与果汁）|
| **销售渠道** | 是否通过相同渠道销售/展示 | 在同一卖场、同一货架或同一电商平台同类目销售 |
| **消费群体** | 是否面向相同类型的消费者 | 购买者高度重合（如儿童用品与儿童教育服务）|
| **行业通常做法** | 在行业实践中是否通常由同一经营者提供 | 行业中常见「商品+配套服务」一体提供 |

**商品与服务类似判断说明：**

> 商品与服务之间的类似判断较为特殊。通常认为，与某商品密切相关的配套服务（如「软件」商品与「软件开发服务」服务）可能被认定类似；但并非所有商品与其同名服务都类似，须结合具体使用场景判断。

### 第五步：类似判断结论分级

**类似商品/服务判断结论分级：**

| 级别 | 标准 | 说明 |
|------|------|------|
| **相同商品/服务** | 完全相同（拟申请商品与在先商标核定商品为同一种商品）| 注册障碍风险最高（无需混淆也可构成侵权）|
| **高度类似商品/服务** | 区分表同一类似群组；或关联性因素多项高度相符 | 与高度近似商标结合，混淆可能性极高 |
| **一定程度类似商品/服务** | 关联性因素部分相符；或区分表同类不同群 | 与较高近似程度商标结合，仍有混淆风险 |
| **不类似商品/服务** | 关联性因素差异大；区分表不同类别且无明显关联 | 单独不构成混淆前提（但须注意驰名商标跨类保护）|

## 检查清单

在完成类似商品判定后，按以下清单验证：

- [ ] 拟申请/使用的商品/服务已精确识别（不过于宽泛）
- [ ] 在先商标核定商品/服务已确认
- [ ] 双方商品/服务的尼斯分类类别已确认
- [ ] 区分表类似群组编号已查阅（标注查阅版本与日期）
- [ ] 关联性判断因素（功能用途/原料/生产部门/销售渠道/消费群体）已逐一分析
- [ ] 区分表突破情形已考虑（新兴类目/跨类情形）
- [ ] 商品与服务之间的类似情形已单独处理（如适用）
- [ ] 结论分级已明确（相同/高度类似/一定程度类似/不类似）
- [ ] 结论与近似判断（similarity-assessment）结论已联动（混淆可能性综合评估）

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 新兴商品/服务未被区分表收录（如 AI 服务/NFT/元宇宙相关服务） | 明确标注「区分表未明确收录」，依据关联性因素综合判断，建议律师确认 |
| 商品与服务之间的类似判断 | 单独说明判断逻辑；特别注意「零售服务」（第35类）与对应商品的关联性判断 |
| 区分表版本不确定（检索到旧版区分表） | 标注「须核实现行版本区分表」`[待验证 — 须核实最新版区分表]`，不得依赖旧版结论 |
| 驰名商标跨类保护情形 | 标注「涉及潜在驰名商标跨类风险，交由 prior-rights-obstacles 专项处理」，不在本技能判定 |
| 商品/服务高度重叠但类别不同 | 须突破区分表参照，结合关联性因素说明理由；建议律师确认 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 商品/服务描述过于宽泛（如仅说「电子产品」「服务」） | 停止判定，请用户提供具体描述；不得基于宽泛描述作出「类似/不类似」结论 |
| 区分表查阅结果存疑 | 标注 `[待验证 — 须核实区分表最新版]`；建议用户通过国家知识产权局官网查阅最新版区分表 |
| 跨语言商品/服务名称（外文商品/服务） | 翻译并对照中文区分表，翻译不确定时标注 `[待验证 — 商品/服务名称翻译须确认]` |
