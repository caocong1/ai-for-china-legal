# 商标近似判断（similarity-assessment）

> 前置读取：`ip-legal/skills/_shared/legal-basis-conventions.md`（引用规范）、`ip-legal/skills/_shared/ip-law-citations.md`（引用库）、`ip-legal/skills/_shared/practice-profile-schema.md`（配置读取契约）、`ip-legal/CLAUDE.md`（实践配置）

---

## 目的

对拟用商标与检索发现的在先商标进行系统性近似判断，从音（读音/发音）、形（字形/图形外观）、义（含义）三个维度进行比对，结合整体观察与隔离比对方法，判断是否容易产生混淆可能性，为在先权利障碍分析（prior-rights-obstacles）提供近似判断结论。

**本子技能可由律师/IP 经理单独触发：`/ip-legal:similarity-assessment`**

## 详细步骤

### 第一步：确认比对对象

| 信息项 | 说明 |
|-------|------|
| 拟用商标 | 完整商标内容（文字/图形/组合） |
| 在先商标（比对对象） | 来自 search-query 的检索结果，或用户指定的具体商标 |
| 比对目的 | 申请注册可行性评估 / 侵权判断 / 使用风险评估 |
| 商品/服务类别 | 双方各自对应的商品/服务类别（影响混淆可能性判断） |

若用户直接触发本技能（未经 search-query），须首先确认以上信息；若未提供完整商标信息，不得开始比对。

### 第二步：识别商标类型与显著部分

**商标类型识别：**

| 商标类型 | 比对重点 |
|---------|---------|
| 文字商标（中文） | 文字读音、字形、文字含义；注意文字的整体与独特部分 |
| 文字商标（英文/字母） | 字母组合的读音/字形/含义；大小写通常不影响近似判断 |
| 图形商标 | 图形整体外观、主要视觉要素、构图方式 |
| 组合商标（文字+图形） | 分别比较文字要素和图形要素；重点比对显著部分 |
| 立体商标 | 立体形状的整体外观与视觉印象（较少见）|
| 颜色商标 | 颜色的组合方式（颜色商标保护门槛极高，条号 `[待验证]`）|

**显著部分识别：**

> **核心判断方法（来自商标授权确权司法解释，描述规则，不写裸条号）**：
> 商标近似判断时，对文字商标的显著部分（识别性最强的部分）给予重点比对权重，但同时须进行整体观察。显著部分的识别基于相关公众对该商标的认知习惯。

| 情形 | 显著部分判断 |
|------|-----------|
| 文字商标含通用词汇 | 通用/描述性部分显著性较弱（如「苹果手机」中「手机」显著性弱）；独特词汇部分为显著部分 |
| 组合商标 | 文字部分通常比图形部分具有更强的显著性与识别性（但视个案而定） |
| 含企业名称简称 | 简称通常为显著部分；但须注意简称本身的独特性 |
| 弱显著性商标 | 通用名称/高度描述性商标显著性弱，保护范围较窄（见边界条件） |

### 第三步：音（读音/发音）维度比对

**文字商标音近判断标准：**

> 判断以**相关公众的听觉印象**为基础，考虑语音相同性或相似性是否足以使相关公众产生混淆。

| 比对维度 | 具体内容 | 示例 |
|---------|---------|------|
| 读音完全相同 | 汉字不同但读音完全相同（同音字） | 「云鹏」vs「运鹏」（云/运同音） |
| 声调差异 | 声调不同但读音字形相近 | 「天美」vs「添美」（tian mei 相同） |
| 声母/韵母近似 | 声母或韵母高度相近 | 「华为」vs「画为」（huawei 音近） |
| 英文读音近似 | 英文字母组合读音相近 | 「NIKE」vs「NYKE」（发音近似） |
| 中英文组合 | 中英文混合商标的读音近似 | 注意消费者的实际读法习惯 |
| 方言差异 | 部分行业/地域须考虑方言读音 | 食品/零售等面向广泛消费者的行业；通常以普通话为标准 |

**音近判断结论分级：**

| 级别 | 标准 | 说明 |
|------|------|------|
| 高度音近 | 读音完全相同或声调差异微小的同音字/近音字 | 在听觉上极难区分 |
| 一定程度音近 | 声母/韵母相近但存在明显差异 | 在听觉上有一定相似性，但可区分 |
| 不音近 | 读音差异明显 | 听觉上不易混淆 |

### 第四步：形（字形/图形外观）维度比对

**文字字形比对：**

| 比对维度 | 具体内容 |
|---------|---------|
| 字形整体外观 | 相同笔画数量、结构相近的汉字（如「己」「已」「巳」） |
| 繁简字形 | 繁体字与简体字对应（如「愛」vs「爱」）|
| 异体字/俗字 | 异体字形式（如「岳」vs「嶽」） |
| 英文字母形状 | 形状相近的字母组合（如「RN」vs「M」在某些字体中近似） |

**图形外观比对：**

| 比对维度 | 具体内容 |
|---------|---------|
| 整体视觉印象 | 相关公众以一般注意力观察图形时的整体印象 |
| 主要视觉要素 | 图形的主要构成要素（颜色、线条、形状、图案）是否相近 |
| 构图方式 | 图形的布局方式与视觉重心 |
| 颜色 | 注意：颜色通常不单独影响近似判断，彩色商标与黑白商标互相近似的，一般认定近似 `[待验证具体规则]` |

**整体观察方法：**

> **整体观察原则（来自司法解释，描述规则）**：以相关公众整体观察为基础，而非对各要素进行割裂分析。若两个商标整体视觉印象相似，即使某些细节有差异，仍可能认定近似。

**隔离比对方法：**

> **隔离比对原则（来自司法解释，描述规则）**：将两个商标分别呈现给相关公众（而非并排直接比较），考察相关公众在隔离状态下是否可能产生混淆。隔离比对比并排比对更接近实际交易中的认知场景。

**形近判断结论分级：**

| 级别 | 标准 |
|------|------|
| 高度形近 | 字形/图形整体外观高度相似，相关公众难以区分 |
| 一定程度形近 | 存在明显相似要素，但整体仍有一定差别 |
| 不形近 | 字形/图形差异明显，不易视觉混淆 |

### 第五步：义（含义）维度比对

| 比对维度 | 具体内容 | 示例 |
|---------|---------|------|
| 直接含义相同 | 两个商标的文字含义完全相同（不同文字，相同意思） | 「英格兰」vs「ENGLAND」（含义相同） |
| 引申含义相同 | 两个商标的引申意义或常见联想相同 | 「太阳」vs「SUN」（指向同一事物） |
| 词根/构词相同 | 含义来源相同（如英文词根相同） | 「赤心」vs「赤诚」（赤字含义相近） |
| 外文翻译对应 | 中文商标与其常见英文翻译 | 「长城」vs「GREAT WALL」 |
| 含义相反 | 含义相反的商标通常不近似（除非导致反讽混淆） | 一般不近似 |

**义近判断说明**：含义相同/相近通常需结合音形综合判断，单纯含义近似不一定构成整体近似，但会增加混淆可能性风险，特别是针对双语消费群体。

### 第六步：混淆可能性综合判断

> **混淆可能性判断规则（来自商标授权确权司法解释，描述规则，不写裸条号）**：
> 以是否容易导致相关公众对商品/服务的来源产生混淆，或误认为两商标存在特定联系（关联混淆/赞助混淆）为最终判断标准。近似判断与混淆可能性判断是递进关系：商标近似是产生混淆可能性的必要但非充分条件，还须结合商品/服务的类似程度综合判断。

**混淆可能性影响因素：**

| 因素 | 分析维度 | 对混淆可能性的影响 |
|------|---------|----------------|
| 商标近似程度 | 音/形/义三维度比对结论 | 高度近似时混淆可能性高；低度近似时低 |
| 商标显著性与知名度 | 是否为知名品牌/驰名商标 | 知名度越高，相关公众更熟悉，区分能力反而越强；弱显著性商标保护范围窄 |
| 商品/服务类似程度 | 见 goods-services-classification 子技能 | 商品越类似，消费者越可能混淆来源 |
| 相关公众的注意力程度 | 购买决策复杂度与商品价格 | 高价值/专业商品（如医疗设备、专业软件）消费者注意力高；普通消费品注意力低，混淆可能性更高 |
| 商标使用场景 | 实际市场中的使用方式（包装/广告/展示）| 视觉接触场景影响混淆概率 |

**混淆可能性结论分级（结合近似判断与商品类似程度）：**

| 级别 | 标准 | 建议行动 |
|------|------|---------|
| 🔴 高度混淆可能性 | 商标高度近似 AND 商品/服务相同或高度类似 | 极大障碍风险，须严肃处理（见 prior-rights-obstacles） |
| 🟠 较高混淆可能性 | 商标较高程度近似 AND 商品/服务类似；或商标高度近似 AND 商品一定程度类似 | 有较大障碍风险，建议修改商标或对拟注册类别进行策略调整 |
| 🟡 一定混淆可能性 | 商标有一定程度近似 AND 商品/服务存在一定关联 | 存在一定风险，建议进一步分析或咨询律师意见 |
| 🟢 混淆可能性低 | 商标近似程度低 OR 商品/服务差异大，不易产生混淆误认 | 风险较低，可继续推进申请/使用 |

## 检查清单

在完成近似判断后，按以下清单验证：

- [ ] 比对对象已明确（拟用商标 vs 在先商标，或指定比对对象）
- [ ] 音近维度已比对（发音/声调/声母韵母）
- [ ] 形近维度已比对（文字字形/图形外观，含整体观察与隔离比对）
- [ ] 义近维度已比对（文字含义/引申含义/外文对应）
- [ ] 显著部分已识别（并给予重点比对权重）
- [ ] 混淆可能性综合判断已完成（结合近似程度与其他影响因素）
- [ ] 判断结论与实践配置中的近似容忍度一致（或标注差异说明）
- [ ] 弱显著性情形已单独说明（如适用，见边界条件）
- [ ] 商品/服务类似程度判断已联动（或提示触发 goods-services-classification）

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 弱显著性商标（通用名称/高度描述性词汇）的近似判断 | 保护范围较窄，与相同通用词汇的近似认定更难成立；但若与强显著商标（驰名/知名商标）相关，仍须审慎；在判断中明确标注弱显著性 |
| 驰名商标的近似认定（跨类保护） | 驰名商标在类似商品范围外可主张跨类保护，近似判断标准可能不同；此类情形交由 prior-rights-obstacles 专门处理 |
| 商标整体近似但部分差异明显 | 以整体观察为主，不能因部分差异而否定整体近似；须完整描述整体印象 |
| 颜色在近似判断中的角色 | 颜色通常不单独决定近似与否，彩色与黑白商标互相近似须结合整体判断 `[待验证具体规则]` |
| 申请人同为一人（关联申请） | 若拟用商标与在先商标属同一申请人，通常不构成冲突；但须确认商标转让/许可情况 |
| 比对对象商标已被撤销/无效 | 已失效商标一般不构成注册障碍，但需确认失效的原因和时效 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 缺少完整商标信息（图形商标无图样） | 停止判断，请用户提供完整商标图样；不得基于文字描述替代图形近似判断 |
| 商标包含外文（不熟悉语言） | 对不熟悉语言的外文，标注「需专业译者核实读音/含义」`[待验证]`；不得臆造读音或含义 |
| 近似判断结论与相关公众实际认知差距大 | 说明判断局限性，建议进行消费者调研或咨询行业经验丰富的律师 |
| 多个在先商标需批量近似判断 | 对每一个在先商标独立比对，提供汇总表，不得笼统作出「均不近似」结论 |
