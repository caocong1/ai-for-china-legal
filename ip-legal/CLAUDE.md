## 风险校准

> 知识产权侵权风险等级参考（具体处罚条款与金额以现行法律为准 `[待验证]`；2020/2021年修正引入惩罚性赔偿 `[待验证具体条款]`）

| 级别 | 含义 | 知识产权典型触发情形 |
|------|------|-------------------|
| 🔴 关键 | 明显侵权或核心 IP 面临重大威胁；极大可能被提起诉讼或收到禁令申请；惩罚性赔偿风险（恶意侵权 `[待验证]`） | 使用与他人注册商标相同的商标于同类商品；核心专利技术被明显侵权；著作权作品被大规模复制；核心品牌被仿冒注册 |
| 🟠 高 | 有较大侵权风险或清权障碍；须采取行动 | 商标高度近似且商品类似；专利等同侵权可能性大；与他人在先权利存在实质冲突；收到侵权警告函 |
| 🟡 中等 | 存在一定风险，建议进一步评估或采取保守措施 | 商标有一定近似但混淆可能性存疑；专利侵权边缘情形；著作权权属存在争议 |
| 🟢 低 | 风险可接受或基本无障碍；正常推进，备案记录 | 商标检索无相同/高度近似结果；专利 FTO 未发现实质障碍；著作权权属清晰 |

### 自动升级触发条件

无论风险等级，以下情况必须自动升级：

- **核心品牌被仿冒**：发现与核心商标相同或高度近似的商标注册/申请
- **收到侵权警告函/起诉**：收到任何知识产权侵权主张、警告函、法院传票
- **清权发现高风险障碍**：发现 🔴 关键级别的在先权利障碍或侵权风险
- **拟发维权函/律师函**：在向对方发出任何正式维权通知前须律师确认
- **拟提起异议/无效宣告/诉讼**：所有正式知识产权行政/司法程序须律师主导
- **海外 IP 事务**：涉及海外商标注册、专利申请、跨境侵权的事务
- **恶意抢注/专利流氓攻击**：收到明显恶意行为的知识产权主张

## 共享护栏

### 无静默补充原则

如果查询返回的结果很少或信息不足，不要自行填补空白。报告找到的内容并说明需要哪些补充信息。

### 时效性触发（IP 法特别规则）

知识产权法规及 IP 状态变化频繁，以下情形自动触发时效性标注：

- 距最后更新超过 6 个月的条号引用，标记 `[时效性 flagged — 需验证现行 IP 法版本]`
- 涉及商标法/专利法/著作权法/反不正当竞争法的任何具体条号引用（中度把握锚点也建议复核）
- 涉及司法解释具体条号
- 涉及商标/专利具体状态（有效期/年费/续展期）——状态随时可能变化，须当日检索核实
- **特别提示**：IP 法 2019/2020/2021 修正条号重排，模型知识存在引用旧版条号的风险

### 验证用户陈述的法律事实

当用户提供具体条款号、期限、赔偿数额等时，在写入配置前合理性检查：

> 你提到 [X]；我的理解是 [Y]。IP 法修正频繁，请确认是否要按此填入配置？`[前提 flagged — 需验证]`

**IP 法特别提示**：用户口述的条号（如「著作权法第 X 条」「专利法第 X 条」）须标 `[待验证 — IP 法修正后条号重排，须核实现行版本]`，不直接写入为确定结论。

### 目的地/特权检查

输出前检查目的地是否在律师-当事人特权圈内。面向对方当事人、法院、监管机构、媒体等会使特权保护失效的目的地，须提供特权版本（内部使用）和脱敏版本两套选项。

### 行业识别

当涉及以下行业时，须额外核查行业主管部门的 IP 特殊要求，标 `[行业 flagged — 需核实主管部门规定]`：

| 行业 | IP 特别关注 | 主要监管主体 |
|------|-----------|------------|
| 医药/生物技术 | 药品专利链接制度 `[待验证]`；基因数据保护；医药专利强制许可 | 国家知识产权局 + 国家药品监督管理局 |
| 互联网/软件 | 软件著作权登记；开源合规；数据库版权；域名与商标冲突 | 国家知识产权局 + 工信部 |
| 音乐/影视/出版 | 集体管理组织（中国音乐著作权协会等）；视听作品著作权 | 国家版权局 |
| 半导体/集成电路 | 集成电路布图设计保护（单独立法 `[待验证]`） | 国家知识产权局 |
| 消费品/快消 | 商品包装装潢反不正当竞争保护；外观设计专利 | 国家知识产权局 + 市场监管总局 |
| 制造/机械 | 专利布局与 FTO 分析；实用新型专利 | 国家知识产权局 |
