# 法律依据引用规范（刑事合规版）

> 本文件定义 criminal-compliance 插件所有技能统一使用的法律依据引用规范。
> 后续计划（09-02、09-03）直接引用本规范，避免各技能重复维护引用规则。
> **刑事合规直接关涉企业与个人的刑事责任、罚金与人身自由。引用错误条号或数额会直接误导刑事风险判断与辩护策略。本规范特别强调具体罪名条号一律待验证纪律与涉案企业合规规范性文件快速演进标注纪律。**

---

## 一、来源分类

所有法律依据必须归入以下类别之一，并在引用时明确标注类别标签。

### 1. 法律（全国人大及其常委会制定）

指全国人民代表大会及其常务委员会制定的法律，刑事合规语境中主要包括：

- 《中华人民共和国刑法》（全国人大常委会历次修订，含刑法修正案；**刑法总则中度把握锚点见本规范第四章；各具体罪名条号一律标 `[条号待验证]`**）
- 《中华人民共和国刑事诉讼法》（全国人大常委会历次修订；**不起诉第177条、认罪认罚从宽第15条为中度把握锚点，标「建议复核」**）
- 《中华人民共和国公司法》、《证券法》、《税收征管法》、《环境保护法》、《安全生产法》等（作为特定罪名的前置违法行为规范参照，具体条号**一律标 `[待验证]`**）

**法律层级最高，是刑事合规分析的核心成文法基础。**

> **刑事合规法律引用特别提示（必须遵守）**：刑法总则有把握的锚点（单位犯罪第30/31条、自首第67条、立功第68条、缓刑第72条）与刑诉法（不起诉第177条、认罪认罚从宽第15条）可在标「建议复核」前提下引用；**但各具体罪名条号（分则各罪名条号）一律以构成要件描述呈现并标 `[条号待验证]`，绝不臆造具体罪名条号与法定刑档。**

**格式要求（法律最小引用格式）：**

```
《[法律全名]》第[X]条[第X款]  [来源标签]  [验证状态]
```

示例（中度把握锚点）：
```
《中华人民共和国刑法》第30条  `[模型知识 — 需验证]`  已核实锚点（中度把握），建议复核
```

示例（具体罪名 — 一律待验证）：
```
《中华人民共和国刑法》[条号待验证]（虚开增值税专用发票罪）  `[模型知识 — 需验证]`  `[条号待验证]`
```

### 2. 司法解释（最高人民法院 / 最高人民检察院发布）

指最高人民法院、最高人民检察院单独或联合发布的解释性文件，刑事合规语境中主要包括：

- 最高法、最高检关于各具体罪名定罪量刑标准的司法解释（虚开增值税专用发票罪、非法吸收公众存款罪、集资诈骗罪、职务侵占罪、挪用资金罪、受贿贿赂类罪名、污染环境罪、重大责任事故罪等的数额标准与入罪情节）
- 最高法、最高检关于办理具体类案的司法解释（如职务侵占案件、商业贿赂案件、食品药品类案件等）

**核心纪律（必须遵守）**：

> **司法解释的具体条号与定罪量刑数额标准一律标 `[待验证]`，以「描述规则 + 待验证」呈现，不写裸条号裸数额。** 规则内容（如虚开税额的入罪标准、非法吸收金额的数额门槛、贿赂数额的量刑区间等）可完整描述规则，但具体数额与条号须标待验证。

**格式要求：**

```
《[司法解释全名]》（最高人民法院/最高人民检察院）规定：[描述规则]  [来源标签]  `[待验证]`
```

示例：
```
最高法、最高检关于虚开增值税专用发票罪的司法解释规定：虚开税额达到一定数额构成犯罪，具体数额分档确定刑罚区间  `[模型知识 — 需验证]`  `[待验证 — 司法解释数额标准易变，须核实现行版本]`
```

### 3. 规范性文件（检察机关 / 多部门联合发布）

指最高检、公安部等机关单独或联合发布的规范性文件，刑事合规语境中主要包括涉案企业合规相关文件：

- 涉案企业合规改革试点工作相关文件（最高检主导，明确适用范围、启动程序、考察期等；**属快速演进规范性文件**）
- 企业合规第三方监督评估机制管理规定（2021年相关文件，规范管理委员会、评估组织、评估流程等；**属快速演进规范性文件**）
- 认罪认罚从宽制度相关指导意见（检察机关实施认罪认罚从宽的操作规程；**属快速演进规范性文件**）

**核心纪律（必须遵守）**：

> **涉案企业合规规范性文件一律描述规则并标 `[待验证 — 规范性文件快速演进，须核实现行版本]`，不硬写文号·发文日期·具体条款编号。** 涉案企业合规改革仍处试点演进中，文号、机制细节、有效合规认定标准快速变化，错误文号细节会直接误导合规不起诉策略。

**格式要求：**

```
[规范性文件名称描述]规定：[描述规则]  [来源标签]  `[待验证 — 规范性文件快速演进，须核实现行版本]`
```

示例：
```
涉案企业合规改革试点相关文件规定：企业涉嫌特定经济犯罪、认罪认罚、愿意合规整改的，可启动合规考察程序，考察期满评估后决定是否不起诉  `[模型知识 — 需验证]`  `[待验证 — 规范性文件快速演进，须核实现行版本]`
```

### 4. 案例（最高检 / 最高院发布的典型案例与指导性案例）

刑事合规领域可参考的案例类型：

- **最高检涉案企业合规典型案例**：最高人民检察院发布，展示合规不起诉适用实例，**仅作参考，不具判例法约束力**
- **最高院指导性案例**：最高人民法院发布，对下级法院有参照作用，**仅作参考，不具判例法约束力**
- **各地典型案例**：参考价值有限，地域差异大

**核心纪律（必须遵守）**：

> **所有案例案号一律标 `[待验证]`，未经检索核验不得移除该标注。** 中国采成文法体系，案例不具判例法约束力，须明确说明。

**格式要求：**

```
[案例名称或案号 `[待验证]`]  [要点摘要]  [发布机关]  参考，不具判例法约束力  `[待验证]`
```

### 5. 学说（学术观点）

学说为辅助参考，存在争议性，引用时须标注来源和争议性：

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息]  "[观点摘要]"  [是否有争议：有/无]  `[待验证]`
```

刑事合规常见学说争议点：有效合规的认定标准、合规计划的可验证性、单位犯罪责任人范围、合规不起诉适用边界等。

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用五个标准标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站（全国人大/最高检/最高法） | 官网原始文件 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查；具体罪名条号与数额标准风险尤为突出 |
| `[用户提供]` | 来自用户提供的文件 | 合规检查报告、案件材料、整改方案等 |

**标签使用规则：**

1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 来源在依赖前**必须**对照原始来源检查——具体罪名条号与司法解释数额标准风险突出
4. 来源标签不替代 `[待验证]` 标注——两者可同时出现在一行

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` 的具体条号 | 模型回忆的条文号风险高 | 国家法律法规数据库、最高检/最高法官网 |
| **各具体罪名条号（分则罪名）** | 条号繁多，含历次刑法修正案调整，风险极高 | 现行刑法全文核实 |
| **定罪量刑数额标准** | 多来自司法解释且经多次修订，极易变化 | 核实现行司法解释版本 |
| 司法解释具体条号 | 各类罪名司法解释修订频繁 | 最高人民法院/最高人民检察院官网 |
| **涉案企业合规规范性文件文号与机制细节** | 改革仍处试点演进中，文号细节快速变化 | 最高检官网 spp.gov.cn 核实现行文件 |
| 认罪认罚从宽指导意见具体条款 | 规范性文件快速演进 | 最高检官网核实现行版本 |
| 距最后更新超过 6 个月的法规 | 法规可能已修订或废止 | 查询官方数据库最新版本 |
| 所有案例引用（默认） | 刑事案例案号极易错误回忆 | 中国裁判文书网、最高检/最高法官网 |

## 四、具体罪名条号一律待验证纪律

> **这是刑事合规引用最重要的纪律，必须严格遵守。**

### 4.1 刑法总则中度把握锚点（可引用，须标「建议复核」）

以下刑法总则条号为中度把握锚点，可在标「已核实锚点（中度把握），建议复核」前提下引用：

| 锚点 | 条号 | 规则摘要 |
|------|------|---------|
| 单位犯罪定义 | **第30条** | 公司、企业、事业单位、机关、团体实施的危害社会的行为，法律规定为单位犯罪的，应当负刑事责任 |
| 单位犯罪双罚制 | **第31条** | 单位犯罪的，对单位判处罚金，并对其直接负责的主管人员和其他直接责任人员判处刑罚；分则另有规定的，依照规定 |
| 自首 | **第67条** | 犯罪以后自动投案，如实供述自己的罪行的，是自首；可以从轻或者减轻处罚；犯罪较轻的，可以免除处罚 |
| 立功 | **第68条** | 揭发他人犯罪行为经查证属实，或者提供重要线索得以侦破其他案件的，是立功；可以从轻或者减轻处罚 |
| 缓刑适用条件 | **第72条** | 被判处拘役、三年以下有期徒刑，犯罪情节较轻、有悔罪表现、没有再犯危险、宣告缓刑对所居住社区没有重大不良影响的，可以宣告缓刑 |

刑事诉讼法中度把握锚点：

| 锚点 | 条号 | 规则摘要 |
|------|------|---------|
| 不起诉 | **第177条** | 规定法定不起诉（情节显著轻微）、酌定相对不起诉（情节轻微不需要判处刑罚）及存疑不起诉等情形 |
| 认罪认罚从宽 | **第15条** | 犯罪嫌疑人、被告人自愿如实供述罪行，承认指控的犯罪事实，愿意接受处罚的，可以依法从宽处理 |

### 4.2 具体罪名一律构成要件描述 + 条号待验证

**关键纪律（硬性要求）：**

> **各具体罪名条号（刑法分则所有罪名条号）一律标 `[条号待验证]`，以构成要件描述呈现，绝不臆造具体罪名条号与法定刑档。定罪量刑数额标准一律描述规则 + `[待验证]`，不写裸数额裸条号。**

须谨慎处理的高频具体罪名（一律构成要件描述 + 条号待验证）：

| 罪名类型 | 代表罪名 | 处理原则 |
|---------|---------|---------|
| 税收类 | 虚开增值税专用发票罪、逃税罪、骗取出口退税罪 | 构成要件描述，条号 `[待验证]`，数额描述规则 `[待验证]` |
| 金融类 | 非法吸收公众存款罪、集资诈骗罪、操纵证券市场罪、内幕交易罪 | 同上 |
| 职务类 | 职务侵占罪、挪用资金罪、非国家工作人员受贿罪 | 同上 |
| 商业贿赂类 | 单位行贿罪、对非国家工作人员行贿罪、对单位行贿罪 | 同上 |
| 扰乱市场秩序类 | 非法经营罪、合同诈骗罪、假冒注册商标罪、串通投标罪 | 同上 |
| 环境与安全类 | 污染环境罪、重大责任事故罪、重大劳动安全事故罪 | 同上 |
| 走私类 | 走私普通货物物品罪等 | 同上 |

## 五、涉案企业合规规范性文件快速演进标注纪律

> **这是刑事合规引用的另一关键纪律，必须严格遵守。**

### 5.1 快速演进背景

涉案企业合规改革仍处试点演进阶段，相关规范性文件的文号、机制细节、有效合规认定标准快速变化。检察机关在各地推进试点过程中，具体适用口径也存在差异。

### 5.2 快速演进标注规则

**一律描述规则并标 `[待验证 — 规范性文件快速演进，须核实现行版本]`，不硬写文号、发文日期、具体条款编号。**

| 文件类型 | 快速演进程度 | 处理方式 |
|---------|------------|---------|
| 涉案企业合规改革试点文件 | 高（各地试点口径差异大） | 描述规则，`[待验证 — 规范性文件快速演进]`，标 `[地方 flagged]` |
| 第三方监督评估机制管理规定 | 高（管理委员会、评估流程细节演进） | 描述规则，`[待验证 — 规范性文件快速演进]` |
| 认罪认罚从宽制度指导意见 | 中（整体稳定但实施细则有调整） | 描述规则，`[待验证]` |

## 六、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：刑事合规法律分析的所有结论，必须首先援引现行成文法（刑法、刑事诉讼法及相关法律）作为规范依据
2. **不得以案例替代法条**：即使存在典型案例，也必须同时援引对应法条或规则描述
3. **法条缺失时**：若无直接成文法，须明确说明「无直接成文法规定，参照以下规范性文件/典型案例/学说」

### 案例的辅助定位

1. **最高检涉案企业合规典型案例**：最高人民检察院发布，标注「最高检典型案例，参照意义强，不具判例法约束力」
2. **最高院指导性案例**：标注「最高院指导，参照意义强，不具判例法约束力」
3. **地方典型案例**：参考价值有限，标注「地方案例，参照，不具判例法约束力，地区差异大」
4. **明确说明非判例法**：中国采成文法体系，案例不具判例法约束力

## 七、法院/检察机关/公安/监管机构/第三方监督评估机制层级与定位

### 法院层级（刑事案件）

| 层级 | 说明 | 刑事案件特点 |
|------|------|------------|
| 最高人民法院 | 全国最高审判机关 | 指导性案例与司法解释发布；刑事审判监督 |
| 高级人民法院 | 省级，刑事上诉二审；死刑复核（部分情形） | 重大刑事案件二审；省级死刑案件复核 |
| 中级人民法院 | 地市级，重大刑事案件一审；刑事上诉二审 | 可能判处无期徒刑以上或社会影响重大的刑事案件 |
| 基层人民法院 | 县区级，**一般刑事案件通常由基层法院一审** | 大多数经济犯罪、职务犯罪案件的一审 |

### 检察机关层级（不起诉决定权在检察机关）

| 层级 | 说明 |
|------|------|
| 最高人民检察院 | 全国最高检察机关；涉案企业合规改革的推动主体 |
| 省级人民检察院（含分院） | 省级检察机关；指导下级院涉案企业合规工作 |
| 市级人民检察院（分院/区分院） | 地市级；较大涉案企业合规案件的审查主体 |
| 基层人民检察院 | 县区级；**大多数合规不起诉决定由基层检察院作出** |

**重要说明**：不起诉决定权在检察机关，拟不起诉公开听证、第三方监督评估机制的启动均由检察机关主导。

### 公安机关

公安机关负责经济犯罪（侦查）、财物类刑事案件的侦查工作。企业合规整改期间，公安机关侦查活动仍可能继续，配合调查须在执业律师指导下进行。

### 行业监管机构（行刑衔接相关）

| 机构 | 主要职能 | 刑事合规特点 |
|------|---------|------------|
| 税务机关 | 税收征管、偷逃税行政处罚 | 行政处罚与刑事追诉衔接（移送标准须核实 `[待验证]`） |

## 八、合规预防与辩护定位

> **这是刑事合规插件的根本立场，任何立场不得覆盖本章节的护栏规则。**

### 8.1 本插件定位

本插件（criminal-compliance）**仅用于合规风险预防、合规体系建设、合规不起诉争取与刑事辩护研究**。所有输出立足于帮助企业事前预防刑事风险、涉案后依法合规整改争取从宽，不替企业作有罪无罪的最终结论（那是司法机关与辩护律师的职责）。

### 8.2 绝对禁止提供的内容

**绝不提供以下任何内容：**

- 规避侦查、对抗调查的方法
- 伪造或毁灭证据的建议
- 串供、统一口径以逃避责任的方案
- 转移资产、隐匿财产以逃避刑事追缴的操作
- 帮助逃避法律追究的任何方法

**识别到上述需求时，一律拒绝，并提示**：合规的唯一正道是依法整改、如实供述、认罪认罚争取从宽，输出标 `[合规边界 — 拒绝]`。

### 8.3 涉刑强制升级

识别到企业或个人**已被立案、被采取强制措施（拘留、逮捕、取保候审、监视居住）、面临侦查、可能构成具体罪名、重大涉案**等情形时，**强制升级给执业刑事辩护律师**，输出标 `[强制升级 — 涉刑情形]`。本插件在上述情形下仅提供合规研究与初步分析，不替代辩护律师的专业判断与辩护工作。
