# 不起诉申请与听证（application-hearing）

> **可单独触发**：本子技能可由律师/合规官/法务单独触发（`/criminal-compliance:application-hearing`），也可由 non-prosecution 编排入口调用。
>
> **与其他子技能分工**：本子技能专注于不起诉申请提交与听证配合；适用条件评估见 applicability-assessment；合规整改计划制定见 compliance-plan；第三方监督评估配合见 third-party-monitoring。
>
> **合规边界**：本子技能指导企业在法律允许范围内完整论证与有效呈现，不提供任何夸大整改成效、虚构事实、规避侦查的方法。申请与听证中的陈述须如实、客观、真实。识别到此类需求一律拒绝并标 `[合规边界 — 拒绝]`。
>
> **涉刑强制升级（硬性要求）**：不起诉申请提交与拟不起诉公开听证是重大法律程序步骤，**一律须执业刑事辩护律师主导**。本子技能提供准备框架供律师参考与复核，非律师角色在任何申请提交与听证环节前必须升级给执业律师。

---

## 目的

在完成合规整改（compliance-plan）并经第三方监督评估机构评估（third-party-monitoring）后，指导企业与律师配合向检察机关提交合规整改报告与不起诉申请，有效参与拟不起诉公开听证（如启动），以及处理不起诉决定后的后续事项（包括合规承诺履行与行刑反向衔接提示）。本子技能是 non-prosecution 四子技能的最后一步，直接决定合规不起诉路径的最终结果。

---

## 前置

### 读取实践配置

技能启动时读取 `criminal-compliance/CLAUDE.md`：

1. **服务立场**（`## 合规与服务立场`）：
   - **涉案企业合规不起诉立场**：聚焦完整论证符合不起诉条件、有效呈现整改成效，目标是获得检察机关作出不起诉决定
   - **单位犯罪辩护研究立场**：在推动不起诉申请的同时，须同步评估万一不起诉申请不被接受的替代路径

2. **是否已涉案**（`## 合规与服务立场` > `### 是否已涉案`）：
   - **已立案或被采取强制措施**：立即触发 `[强制升级 — 涉刑情形]`；所有申请与听证步骤**强制须执业刑事辩护律师主导**

3. **可用集成**（`## 可用集成`）：读取当地检察院联系信息（检察机关是申请与听证的受理机关）；第三方监督评估组织（评估报告提交方）

若配置缺失，进入临时模式：所有步骤须律师主导（保守处理）；标 `[临时模式]`。

### 前置条件确认

本子技能须在以下条件满足后启动：

| 前置条件 | 确认方式 |
|---------|---------|
| applicability-assessment 已确认适用条件 | 确认路径评估结论 |
| compliance-plan 已完成专项整改计划并实施 | 整改里程碑已基本完成 |
| third-party-monitoring 评估已完成或进行中 | 评估报告已出具或即将出具 |
| 认罪认罚从宽已完成（具结书签署） | 律师确认，具结书已签署 |

若前置条件未满足，提示先完成前置子技能。

## 详细步骤

### 步骤一：准备合规整改报告

合规整改报告是不起诉申请的核心材料，须如实反映整改情况：

**合规整改报告核心要素**：

| 要素 | 内容要求 | 注意事项 |
|------|---------|---------|
| 涉案问题描述 | 客观描述涉案行为的性质与范围（基于认罪认罚的事实认定） | 须与认罪认罚的事实保持一致，不得自相矛盾 |
| 合规漏洞诊断 | 诚实呈现涉案行为暴露的制度、流程、人员等合规漏洞 | 诚实面对漏洞是有效合规的体现，隐瞒漏洞会影响整改认定 |
| 整改措施清单 | 针对各漏洞的具体整改措施（对应 compliance-plan 输出） | 措施须已实际落实，不得写未落实的措施 |
| 整改进度与成果 | 逐一列示各整改措施的完成情况与可验证成果 | 须有实物证据支撑（制度文本、培训记录、系统截图等） |
| 整改效果评估 | 合规整改后的制度有效性自评 | 客观评估，不夸大成效；第三方监督评估报告是权威评估 |
| 合规承诺 | 高层的书面整改承诺；今后持续合规的承诺 | 须高管级别签署，体现持续合规决心 |
| 赔偿/弥补情况 | 对被害方的赔偿、退赔及谅解情况（如有） | 被害方谅解是酌定不起诉的积极因素 |

**合规整改报告撰写合规边界**：

> `[合规边界]` 合规整改报告须如实反映整改情况，不得夸大整改成效、虚构整改措施、掩盖未完成的整改项目。整改报告将作为第三方监督评估的参考材料，也将由检察机关审查。任何不实内容不仅危及不起诉可能性，还可能加重刑事责任。

---

### 步骤二：准备不起诉申请书

不起诉申请书是向检察机关提出申请的正式法律文书，须由执业律师主导起草：

**不起诉申请书核心结构**：

| 章节 | 内容 | 写作要点 |
|------|------|---------|
| 基本情况 | 企业名称、案件基本信息、申请不起诉的类型（酌定相对不起诉） | 准确呈现，与认罪认罚事实一致 |
| 法律依据 | 援引刑事诉讼法第177条（建议复核）、涉案企业合规相关规定（规范性文件描述，`[待验证]`） | 依据须与不起诉类型匹配 |
| 论证符合不起诉条件 | 逐条论证企业满足适用条件（认罪认罚+合规整改有效+社会效果） | 核心论据：整改的针对性与有效性；不需要刑事处罚的理由 |
| 整改成果摘要 | 合规整改的核心成果（引用整改报告的主要证据） | 须有证据支撑，不可空洞 |
| 第三方评估结论引用 | 引述第三方监督评估机构的积极评估结论 | 须引用评估报告原文，不得断章取义 |
| 被害方处理情况 | 赔偿、退赔、谅解情况（如有） | 谅解书原件须附备 |
| 申请事项 | 明确请求检察机关作出不起诉决定 | 表述须明确、礼貌、合规 |

**不起诉申请材料清单**：

| 材料 | 说明 | 必要性 |
|------|------|-------|
| 不起诉申请书（正文） | 律师起草、法定代表人签署 | 必须 |
| 合规整改报告 | 完整版，附可验证成果证据 | 必须 |
| 第三方监督评估报告 | 评估组织出具的评估结论 | 如已完成评估 |
| 高管整改承诺书 | 持续合规承诺 | 必须 |
| 认罪认罚具结书（副本） | 已签署并有律师/值班律师见证 | 必须 |
| 被害方谅解书（如有） | 被害方签署的谅解声明 | 如有被害方 |
| 赔偿凭证（如有） | 退赔、赔偿的银行记录或收据 | 如有退赔 |
| 其他补充材料 | 律师认为必要的其他证明材料 | 按需 |

**律师审查硬性要求**：

> `[律师审查 — 硬性要求]` 不起诉申请书及所有附件材料**须由执业刑事辩护律师审核确认**后方可提交检察机关。非律师角色（合规官/法务/当事人）不得独立提交申请材料。

### 步骤三：配合拟不起诉公开听证

**听证制度说明**（描述规则，`[待验证 — 规范性文件快速演进，须核实现行版本]`）：

相关规定：检察机关在决定对涉案企业作不起诉处理前，可组织公开听证，充分听取各方意见。听证并非必经程序，是否启动由检察机关决定（`[待验证]`）。

**听证参加方**（描述规则，`[待验证]`）：

| 参加方 | 角色 | 参加方式 |
|-------|------|---------|
| 检察机关 | 主持听证，说明案情与申请情况 | 主持 |
| 企业（申请方） | 陈述整改情况，论证符合不起诉条件 | 发言陈述 |
| 律师 | 代理企业，补充陈述，回答听证员询问 | 代理陈述 |
| 侦查机关（公安） | 发表意见（是否同意不起诉） | 意见陈述 |
| 被害方（如有） | 发表意见（谅解情况/对不起诉的态度） | 意见陈述 |
| 人民监督员/听证员 | 听取各方意见，发表意见，供检察机关参考 | 意见陈述 |
| 第三方监督评估机构（可能） | 介绍评估情况与结论 | 情况介绍 |

**听证准备要点**：

| 准备事项 | 内容 | 合规边界 |
|---------|------|---------|
| 陈述提纲准备 | 企业陈述：案件认罪认罚情况→整改措施与成果→社会效果→未来合规承诺 | 陈述须真实；不得虚构或夸大整改效果 |
| 证据材料准备 | 整改报告核心亮点提炼；评估报告关键结论准备；被害方谅解材料 | 材料须真实原件或经核实副本 |
| 答询准备 | 预判听证员可能提问（整改真实性/是否已经弥补损失/今后合规承诺）；律师与企业代表对答询进行模拟演练 | 答询须如实回答；对确实不知道的问题说明不知道，不猜测 |
| 被害方关系处理 | 尽量在听证前完成赔偿与谅解；听证时被害方谅解是积极因素 | 谅解须真实自愿，不得威胁或诱导 |

**听证应对话术框架（合规版）**：

企业代表在听证中的核心表达框架：

```
（一）认罪认罚的真诚性：
「我司（企业）对涉嫌的[罪名类别，不写具体条号]行为，
真诚认罪认罚，对相关人员造成的损害深感歉意。」

（二）整改的针对性与真实性：
「针对涉案行为暴露的[具体合规漏洞]，我司已采取
以下具体措施：[简述核心整改措施]。相关措施已落实并经
第三方监督评估机构核实。」

（三）社会效果与今后合规承诺：
「我司是[行业]企业，对[就业/税收/地区经济]有[X]
的贡献。我司承诺今后依法合规经营，已建立[关键合规制度]
长效机制。」
```

**合规边界**：陈述须如实；不得陈述虚假整改成果；不得否认已认罪认罚的犯罪事实；不得对侦查机关或被害方作出不实指控。

**听证非必经情形**（描述规则，`[待验证]`）：并非所有合规不起诉案件均须经过公开听证。检察机关决定是否组织听证，企业须配合而非主动要求或规避听证。

### 步骤四：不起诉决定后的后续处理

**不起诉决定后的事项**（描述规则，`[待验证 — 规范性文件快速演进]`）：

| 事项 | 内容 | 处理要点 |
|------|------|---------|
| **合规承诺履行** | 不起诉决定通常附随继续履行合规承诺的要求 | 持续按整改计划推进后续合规措施；不得因获得不起诉决定而停止整改 |
| **行刑反向衔接** | 检察机关作出不起诉决定后，可向有关行政机关提出行政处罚建议（规范性文件，`[待验证]`） | 企业须准备配合行政处罚程序（如税务处罚、市场监管处罚等）；行刑反向衔接意味着不起诉不等于无任何处罚 |
| **合规文件保存** | 整改报告、评估报告、整改证据材料长期保存 | 作为持续合规的历史记录；如后续有类似风险，历史合规记录有参考价值 |
| **被害方补偿完成** | 若听证前未完成全额赔偿，不起诉决定后继续履行赔偿承诺 | 诚信履行承诺，维护企业信用 |
| **不起诉被撤回的风险** | 如合规承诺履行不力，检察机关是否有权撤回不起诉决定（描述规则，`[待验证]`） | 持续合规是保证不起诉决定稳定性的关键 |

**不起诉申请被否决时的救济**（描述规则，`[待验证]`）：

- 检察机关否决不起诉申请后，案件将依法提起公诉，进入审判程序
- 律师须立即启动审判阶段辩护准备（本插件不替代辩护律师工作）
- 企业认罪认罚的从宽情节在审判阶段仍可主张（刑事诉讼法第15条，建议复核）
- 合规整改作为悔罪表现也可作为量刑从宽情节（须律师主导）

## 边界条件

| 情形 | 处置方式 |
|------|---------|
| 检察机关不启动听证（非必经程序） | 无需准备听证；关注申请书与整改报告的质量 |
| 被害方明确反对不起诉 | 继续推进赔偿谈判；律师评估被害方反对的影响程度；不得威胁或诱导被害方撤回意见 |
| 申请被否决，案件提起公诉 | 立即升级律师启动审判阶段辩护准备；认罪认罚从宽情节仍可主张；整改作为量刑情节 |
| 听证员提出合规整改范围外的问题 | 律师及时介入，说明问题超出本次申请范围，如实回答或说明无法答复的原因 |
| 不起诉决定后检察机关要求继续合规报告 | 按要求履行持续报告义务，是合规承诺的组成部分 |
| 不起诉被撤回的风险 | 律师评估撤回条件（`[待验证]`），持续合规以维持不起诉稳定性 |
