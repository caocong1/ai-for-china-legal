# 常见单位犯罪类型（common-unit-crime-types）

> **可单独触发**：本子技能可由律师/合规官/法务单独触发（`/criminal-compliance:common-unit-crime-types`），也可由 corporate-crime 编排入口调用。
>
> **与其他子技能分工**：本子技能专注于按类别梳理单位常见罪名的构成要件；单位犯罪构成要件的通用判断见 unit-crime-constitution；内控防范见 internal-control-prevention；涉案应对见 incident-response。
>
> **合规边界**：本子技能用于刑事风险识别与预防研究，不作「是否构成具体罪名」的最终结论（那是司法机关与辩护律师的职责）。不提供规避侦查或逃避责任的方法。识别到此类需求一律拒绝并标 `[合规边界 — 拒绝]`。
>
> **关键纪律**：所有具体罪名条号一律 `[条号待验证]`，以构成要件描述呈现；定罪量刑数额标准一律描述规则 + `[待验证]`，绝不臆造具体罪名条号与法定刑档。
>
> **涉刑强制升级**：识别到企业具体行为可能匹配某罪名的构成要件，已达🔴关键风险等级或已立案等情形时，一律强制升级执业刑事辩护律师。

---

## 目的

按行业高发类别，系统梳理单位（公司、企业等）可能构成的常见刑事罪名，描述各罪名的构成要件特征（不写具体条号，条号须核实），帮助律师/合规官/法务识别本企业可能面临的刑事风险类型，用于风险评估、内控设计与辩护研究。

## 前置

### 读取实践配置

技能启动时读取 `criminal-compliance/CLAUDE.md`：

1. **企业画像**（`## 我们是谁` > 行业/业务模式）：根据企业行业聚焦最相关的罪名类别（如制造企业重点看税收/走私/环境类；金融机构重点看金融类；医药企业重点看商业贿赂类）

2. **服务立场**（`## 合规与服务立场`）：
   - **单位犯罪预防立场**：重点识别本企业业务中可能触及的罪名类型，输出风险点清单用于内控设计
   - **单位犯罪辩护研究立场**：重点分析构成要件是否完整满足，为辩护策略提供构成要件比对研究
   - **事前刑事风险评估立场**：全面梳理，输出行业高发罪名风险地图

3. **是否已涉案**（`## 合规与服务立场` > `### 是否已涉案`）：
   - **已立案或被采取强制措施**：立即触发 `[强制升级 — 涉刑情形]`；罪名分析仅作辩护律师参考背景

若配置缺失，进入临时模式：默认全类别梳理；按用户描述的企业行业聚焦相关类别；标 `[临时模式]`。

## 详细步骤

### 步骤一：确认聚焦类别

根据企业行业与用户需求，确认重点梳理的罪名类别：

| 类别 | 高发行业/场景 |
|------|-------------|
| 走私类 | 进出口企业、贸易企业、跨境电商 |
| 金融类 | 金融机构、投资理财、融资租赁、互联网金融 |
| 税收类 | 几乎所有企业（尤其是增值税发票链条密集的制造/贸易企业） |
| 商业贿赂类 | 医药/医疗器械、政府采购、工程建设、金融销售 |
| 扰乱市场秩序类 | 贸易企业、特许经营领域、招投标行业、互联网企业 |
| 环境类 | 制造企业、化工、矿产、养殖、危险废物处置 |
| 安全生产类 | 建筑施工、矿山、化工、危险化学品、特种设备 |

### 步骤二：按类别梳理构成要件

以下所有具体罪名条号均 `[条号待验证]`，定罪量刑数额标准均描述规则 + `[待验证]`。

#### C1. 走私类

**走私普通货物、物品罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 国家对外贸易管理秩序、税收利益 |
| 客观行为 | 逃避海关监管，运输、携带、邮寄国家禁止/限制进出口以外的普通货物、物品进出国（边）境，偷逃应缴税额达一定标准 |
| 主体 | 单位（✓）与自然人均可构成 |
| 主观要件 | 故意 |
| 定罪量刑数额 | 偷逃税额达到一定标准，分档量刑，依相关司法解释  `[待验证 — 须核实现行司法解释版本]` |
| 单位犯罪 | 刑法规定单位可构成本罪，双罚制适用  `[条号待验证]`  `[模型知识 — 需验证]` |
| 内控要点 | 贸易企业须严格控制报关单据的真实性，确保申报价格与实际成交价一致；禁止阴阳合同 |

**走私其他罪名类型**（文物/贵重金属/珍贵动植物/国家禁止进出口商品等）：保护法益各有侧重，构成要件描述类似，具体条号 `[条号待验证]`，高风险行业应核实具体罪名类型。

---

#### C2. 金融类

**非法吸收公众存款罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 国家金融管理秩序、社会公众资金安全 |
| 客观行为 | 未经有关部门依法批准，或借用合法经营形式，向社会公众（含单位和个人）吸收资金，承诺还本付息或给付回报 |
| 主体 | 单位（✓）与自然人 |
| 主观要件 | 故意（明知未获批准仍吸收） |
| 定罪量刑数额 | 依司法解释，数额或人数达一定标准构成犯罪，分档量刑  `[待验证]` |
| 合规红线 | 金融产品销售须持牌；募集说明书须真实不误导；任何「保本保收益」的承诺须事先合规审查 |

**集资诈骗罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 与非法吸收公众存款罪的关键区别 | 集资诈骗罪要求行为人**以非法占有为目的**（主观要件不同）；否则即使吸收大额资金，也定非法吸收公众存款罪而非集资诈骗罪 |
| 客观行为 | 使用诈骗方法（虚构资金用途、虚假宣传等）非法集资 |
| 主体 | 单位（✓）与自然人 |
| 定罪量刑数额 | 数额标准高于非法吸收公众存款罪，分档量刑  `[待验证]` |

**操纵证券、期货市场罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 证券、期货市场秩序与信息公平 |
| 客观行为 | 连续买卖、对倒交易、虚报撤单、散布谣言操纵市场等多种方式 |
| 主体 | 单位（✓）（机构投资者需特别关注）与自然人 |
| 主观要件 | 故意 |
| 合规红线 | 上市公司大股东/机构应建立合规交易制度，禁止操纵价格的交易行为 |

**内幕交易罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 内幕信息知情人（含单位）利用未公开内幕信息进行证券期货交易，或泄露内幕信息 |
| 主体 | 内幕信息知情人（含单位）及非法获取内幕信息者 |
| 合规红线 | 建立内幕信息知情人登记制度；重大事项窗口期禁止交易制度 |

#### C3. 税收类

**虚开增值税专用发票罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 国家税收征管制度、增值税专用发票管理秩序 |
| 客观行为 | 为他人虚开、为自己虚开、让他人为自己虚开、介绍他人虚开增值税专用发票；「虚开」包括无真实交易而开具、有真实交易但超开金额、以假冒主体名义开具等情形 |
| 主体 | 单位（✓）与自然人 |
| 主观要件 | 故意 |
| 定罪量刑数额 | 虚开税额达到一定数额构成犯罪，数额分档对应不同法定刑区间  `[待验证 — 须核实现行司法解释版本]` |
| 合规红线 | 增值税专用发票须与真实业务一一对应；禁止开具或接受与实际业务不符的发票；付款前核实交易对手的真实性 |

**逃税罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 采取欺骗、隐瞒手段进行虚假纳税申报或不申报，逃避缴纳税款 |
| 主体 | 纳税人（单位（✓）和自然人） |
| 特别规定 | 经税务机关下达追缴通知后，补缴税款、缴纳滞纳金、已受行政处罚的，可不予追诉（具体规则须核实  `[待验证]`） |
| 定罪量刑 | 逃税数额及比例达到一定标准，依司法解释  `[待验证]` |

**骗取出口退税罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 以假报出口或其他欺骗手段，骗取国家出口退税款 |
| 主体 | 单位（✓）与自然人 |
| 合规红线 | 出口贸易须真实，配套的报关文件、合同须与实际业务一致；禁止「假出口」 |

---

#### C4. 商业贿赂类

**单位行贿罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 国家工作人员职务廉洁性 |
| 客观行为 | 单位为谋取不正当利益，给予国家工作人员财物（以单位名义行贿） |
| 主体 | 单位（✓）（双罚制：单位处罚金，对直接负责主管人员和其他直接责任人员处刑） |
| 主观要件 | 故意 + 为谋取**不正当**利益的目的（「不正当利益」是构成要件，须注意） |
| 定罪量刑数额 | 依司法解释  `[待验证]` |
| 合规红线 | 对国家工作人员（含国有企业人员）的任何财物给付须事先合规审查；禁止以「业务招待」「礼品」名义变相行贿 |

**对非国家工作人员行贿罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 公司、企业等单位工作人员廉洁性与公平竞争秩序 |
| 客观行为 | 为谋取不正当商业利益，给予公司、企业或其他单位的工作人员财物 |
| 主体 | 单位（✓）与自然人 |
| 与单位行贿罪的区别 | 受贿方不同：本罪受贿方为非国家工作人员（商业对象）；单位行贿罪受贿方为国家工作人员 |
| 合规红线 | 对合作单位采购员、销售人员等的任何财物给付须审查；高风险行业（医药、工程）须建立严格的商业招待记录制度 |

**对单位行贿罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 为谋取不正当利益，给予国家机关、国有公司企业等单位财物（以及违规给予回扣、手续费） |
| 主体 | 单位（✓）与自然人 |
| 合规红线 | 合法的回扣/折扣须在合同中明示并如实入账；禁止账外返利 |

---

#### C5. 扰乱市场秩序类

**非法经营罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 市场经营秩序 |
| 客观行为 | 违反国家规定，无证经营须国家许可的行业、买卖专营专卖物品、非法经营证券期货保险业务、非法资金支付结算等 |
| 主体 | 单位（✓）与自然人 |
| 注意事项 | 兜底条款适用争议较大，须谨慎  `[模型知识 — 需验证]` |
| 合规红线 | 需取得许可证方可经营的业务须确保持牌；互联网企业的特许经营业务须特别注意许可证要求 |

**合同诈骗罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 以虚构主体资格、虚假担保、无履行能力骗取预付款等方式，在合同签订履行中骗取财物 |
| 主体 | 单位（✓）与自然人 |
| 与民事欺诈的区别 | 须有非法占有目的；合理的商业判断失误不构成合同诈骗 |

**串通投标罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 投标人相互串通投标报价，损害招标人或其他投标人利益；或投标人与招标人串通，排斥其他投标人 |
| 主体 | 投标人（单位（✓）与自然人） |
| 合规红线 | 招投标全程记录保存；禁止与竞争者的任何价格协调；独立投标 |

**假冒注册商标罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 未经许可，在同一种商品上使用与注册商标相同的商标 |
| 主体 | 单位（✓）与自然人 |
| 合规红线 | 产品包装须核实商标授权；采购渠道须核实知识产权状态 |

---

#### C6. 环境类

**污染环境罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 环境质量与生态安全 |
| 客观行为 | 违反国家规定，排放、倾倒或处置有放射性废物、含传染病病原体废物、有毒物质或其他有害物质，严重污染环境 |
| 主体 | 单位（✓）与自然人 |
| 主观要件 | 故意（司法解释中争议较大，须核实  `[待验证]`） |
| 定罪标准 | 「严重污染环境」依相关司法解释（死亡人数/重伤人数/环境损害面积/污染物种类等认定标准）  `[待验证]` |
| 合规红线 | 须持有排污许可证且在批准范围内排放；危险废物须交有资质机构处置，禁止私自填埋、非法转运 |

---

#### C7. 安全生产类

**重大责任事故罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 保护法益 | 生产安全与从业人员人身安全 |
| 客观行为 | 在生产、作业中违反有关安全管理的规定，造成重大伤亡事故或其他严重后果 |
| 主体 | 生产、作业直接责任人员（自然人，含法定代表人、高管等负有安全管理职责者）；**注意：此罪主体为自然人，单位不直接作为主体，但单位负责人可能被追诉**  `[模型知识 — 需验证]` |
| 主观要件 | 过失 |
| 合规红线 | 须建立安全生产管理制度；定期开展安全生产检查；落实安全生产培训 |

**重大劳动安全事故罪** `[条号待验证]`

| 要素 | 内容 |
|------|------|
| 客观行为 | 安全生产设施或条件不符合国家规定，经有关部门或单位职工提出后，对事故隐患仍不采取措施，造成严重后果 |
| 主体 | 厂矿企业、事业单位中直接负责的主管人员（自然人） |
| 注意事项 | 须有「经提出后仍不采取措施」的主观怠慢要件 |

### 步骤三：基于行业的重点罪名推荐

根据企业行业，推荐重点关注的罪名类别：

**行业风险地图速查**：

| 行业 | 重点关注罪名类别 | 高发场景 |
|------|--------------|---------|
| 制造业 | 税收类（虚开发票）、环境类（污染排放）、商业贿赂类 | 供应链发票管理、危废处置、采购回扣 |
| 金融投资 | 金融类（非法集资/操纵市场/内幕交易）、商业贿赂类 | 产品销售、股票交易、理财募集 |
| 医药/医疗器械 | 商业贿赂类（最高发）、税收类 | 医药代表推广费、学术会议 |
| 建筑工程 | 商业贿赂类（串通投标）、安全生产类 | 招投标、分包、现场施工 |
| 贸易/进出口 | 走私类、税收类（虚开发票骗退税）、扰乱市场 | 关税申报、出口退税、产品资质 |
| 互联网/科技 | 金融类（无证经营支付/证券）、扰乱市场类 | 支付结算、网络借贷、数据服务 |
| 能源/化工 | 环境类、安全生产类、税收类 | 危险化学品管理、废物处置、发票管理 |

## 边界条件

| 情形 | 处置方式 |
|------|---------|
| 企业行为可能触发多个罪名（罪名竞合） | 逐一说明各罪名构成要件，标注竞合风险，须律师研究具体适用罪名 |
| 某罪名单位是否可构成存在争议 | 标 `[待验证]`，建议律师核实该具体罪名的单位主体规定 |
| 新型业务模式罪名定性不明确（如某新型互联网业务） | 标 `[待验证]`，描述可能适用的罪名类型，建议律师专项研究；触发新型业务合规审查 |
| 用户要求评估具体行为是否已构成罪名 | 以构成要件比对方式描述，明确「最终定性须律师判断，本子技能不替代司法机关结论」 |
