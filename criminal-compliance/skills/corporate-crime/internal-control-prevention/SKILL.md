# 内控与防范机制（internal-control-prevention）

> **可单独触发**：本子技能可由律师/合规官/法务单独触发（`/criminal-compliance:internal-control-prevention`），也可由 corporate-crime 编排入口调用。
>
> **与其他子技能分工**：本子技能专注于单位犯罪风险的事前防范内控要点；单位犯罪构成要件分析见 unit-crime-constitution；常见罪名类型识别见 common-unit-crime-types；涉案后应对见 incident-response。
> **与 compliance-system-building 的关系**：本子技能聚焦单位犯罪防范的核心内控要点（实践操作层面），刑事合规体系的全面建设（风险识别体系/制度体系/组织架构/有效性评估）见 compliance-system-building。
>
> **合规边界**：本子技能用于事前防范，输出内控建议与防范机制设计，不提供规避侦查或逃避责任的方法。防范机制的目的是使企业行为保持合法合规，而非规避法律追究。识别到此类需求一律拒绝并标 `[合规边界 — 拒绝]`。
>
> **涉刑强制升级**：若发现企业已发生风险事件（已有违规行为苗头或已立案），应立即强制升级执业刑事辩护律师，本子技能转为提供应对建议框架（见 incident-response）。

---

## 目的

针对单位犯罪常见风险领域（税收、商业贿赂、金融、环境、安全生产等），设计实操性强的内控与防范机制，帮助企业从制度建设、流程管控、岗位制衡与举报机制四个维度形成完整的单位犯罪事前防范体系，将刑事风险控制在合法合规的范围内。

### 读取实践配置

技能启动时读取 `criminal-compliance/CLAUDE.md`：

1. **企业画像**（`## 我们是谁`）：读取行业、规模、业务模式，据此聚焦最相关的重点风险环节
2. **服务立场**（`## 合规与服务立场`）：
   - **单位犯罪预防立场**：本子技能的主要使用场景；侧重输出可落地的内控建议
   - **事前合规体系建设立场**：与 compliance-system-building 衔接，本子技能提供单位犯罪防范的专项要点
3. **是否已涉案**（`## 合规与服务立场` > `### 是否已涉案`）：
   - **已立案或被采取强制措施**：立即触发 `[强制升级 — 涉刑情形]`；内控完善须在律师指导下推进，避免被认为是事后毁证（合规边界）

## 详细步骤

### 步骤一：事前防范基础制度建设

有效的内控防范须以书面制度为基础：

**合规制度体系要素**：

| 制度类型 | 核心内容 | 覆盖的单位犯罪风险 | 优先级 |
|---------|---------|----------------|-------|
| **合规政策与行为准则** | 禁止事项（零容忍清单）、合规承诺、举报渠道声明、违规后果 | 所有罪名类别 | 最高 |
| **反商业贿赂制度** | 礼品/招待/赞助/回扣的审批标准与上限；「不正当利益」的识别；对国家工作人员/商业对象的区分要求 | 单位行贿罪、对非国家工作人员行贿罪 | 高 |
| **发票管理制度** | 增值税专用发票开具与接受的审核流程；「三流一致」要求（资金流/货物流/发票流）；付款前核实业务真实性 | 虚开增值税专用发票罪、逃税罪 | 高 |
| **税务合规制度** | 纳税申报审核流程；税务筹划合规边界（禁止逃税行为）；与税务机关的沟通记录 | 逃税罪、骗取出口退税罪 | 高 |
| **招投标合规制度** | 独立投标要求；禁止与竞争者协调价格；禁止围标串标；招投标文件留存 | 串通投标罪 | 中高 |
| **环境合规制度** | 排污许可证管理；危险废物处置合规要求（禁止私自填埋/非法转运）；环保监测记录 | 污染环境罪 | 中高（制造/化工行业高） |
| **安全生产制度** | 安全生产检查制度；事故隐患报告与整改制度；安全生产培训记录 | 重大责任事故罪、重大劳动安全事故罪 | 中高（建筑/化工/矿山行业高） |
| **关联交易制度** | 关联方识别；关联交易的独立定价与审批；关联交易信息披露 | 职务侵占罪（利用关联交易）、税收类 | 中 |

### 步骤二：禁止性行为清单

禁止性行为清单（红线清单）是预防单位犯罪最直接的制度工具，须具体可识别：

**全行业通用禁止性行为清单模板**：

| 禁止行为类别 | 具体描述 | 对应风险罪名 | 违规后果 |
|------------|---------|-----------|---------|
| **商业贿赂红线** | 向任何国家工作人员（含国有企业人员）提供任何形式的财物（含现金/购物卡/礼品/旅游/股权等） | 单位行贿罪 `[条号待验证]` | 立即解除劳动合同+刑事追诉风险 |
| **商业贿赂红线** | 以「咨询费」「顾问费」「会议赞助」「服务费」等名义向对方采购人员支付任何额外报酬 | 对非国家工作人员行贿罪 `[条号待验证]` | 立即解除+刑事追诉风险 |
| **发票红线** | 开具与实际业务不符的增值税专用发票（无论数额大小） | 虚开增值税专用发票罪 `[条号待验证]` | 立即解除+刑事追诉风险 |
| **发票红线** | 接受明知与实际业务不符的增值税专用发票 | 同上 | 立即解除+刑事追诉风险 |
| **资金安全红线** | 未经审批，个人借用或挪用公司账户资金用于非公司业务 | 挪用资金罪 `[条号待验证]` | 立即解除+刑事追诉风险 |
| **资产安全红线** | 利用职务便利将公司财产非法占为己有 | 职务侵占罪 `[条号待验证]` | 立即解除+刑事追诉风险 |
| **环保红线** | 未经审批超标排放或非法处置危险废物 | 污染环境罪 `[条号待验证]` | 立即停产整改+刑事追诉风险 |
| **招投标红线** | 与竞争者在投标前协调价格或分配项目 | 串通投标罪 `[条号待验证]` | 立即终止合同+刑事追诉风险 |

**行业定制化禁止行为**（根据 `## 我们是谁` 中的行业配置）：

- **金融行业**：禁止利用内幕信息进行证券交易；禁止向公众募集资金（无牌照）；禁止虚假陈述诱导投资
- **医药行业**：禁止以任何形式向医生/医院管理人员支付回扣；禁止以「学术活动」名义行贿
- **制造/贸易**：禁止低报申报价格（走私）；禁止假出口骗取退税
- **互联网**：禁止无证开展支付结算/网络借贷；禁止利用算法操纵竞争

### 步骤三：重点环节风险控制

根据企业业务流程，对刑事风险集中的重点环节实施专项控制：

**重点环节风险控制表**：

| 业务环节 | 刑事风险 | 控制措施 | 责任部门 |
|---------|---------|---------|---------|
| **采购管理** | 商业贿赂（对供应商采购员行贿）、虚开发票（采购发票与货物不符） | 供应商评审分离采购决策与执行；采购员轮岗；发票与货物/服务验收单三方核对 | 采购部+合规部 |
| **销售管理** | 商业贿赂（向客户采购员/关键决策人行贿）、虚开发票（销售收入确认与发票一致性） | 客户招待费用审批上限；年度反腐败承诺书；销售发票与合同、收款核对 | 销售部+合规部+财务部 |
| **财务税务** | 虚开发票（税款抵扣）、逃税（申报不实）、挪用资金（资金挪用） | 发票审核专职岗位；申报前合规复核；账外资金禁止（银行账户统一管控） | 财务部+合规部 |
| **招投标** | 串通投标（与竞争者协调）、商业贿赂（行贿评标人员） | 投标决策须合规审批；投标方案独立制定不与竞争者沟通；评标人员接触须记录 | 业务部+合规部 |
| **关联交易** | 职务侵占（利用关联交易抽取利益）、税收（不公允关联定价） | 关联方独立识别与定期更新；关联交易须按独立交易原则定价；关联交易须董事会/合规委员会审批 | 财务部+合规部+董事会 |
| **对外捐赠赞助** | 商业贿赂（以捐赠名义行贿）、资产流失（未经审批捐赠） | 捐赠须合规审批（审核是否有利益关联）；受捐方须为合法机构；捐赠记录留存 | 合规部+财务部+高管审批 |
| **外包/分包** | 商业贿赂（外包单位代为行贿）、责任转移 | 外包商合规尽职调查；合同含反腐败承诺条款；外包活动不得作为规避内控的工具 | 采购部+合规部 |
| **危险废物处置** | 污染环境（非法处置危废） | 危险废物须交持证单位处置；转移联单全程留存；禁止私自填埋或委托无资质单位 | 生产部+环保合规 |

### 步骤四：岗位制衡与举报机制

**岗位制衡设计原则**：

「不相容职务分离」是防范职务类犯罪的核心机制：

| 不相容职务组合 | 风险 | 制衡要求 |
|------------|------|---------|
| 采购决策 + 采购执行 | 采购人员倒卖信息、吃回扣 | 采购决策与执行分离；大额采购需独立评审 |
| 资金支付审批 + 会计核算 | 挪用/侵占资金不易被发现 | 出纳、会计、审批三方分离 |
| 发票开具 + 销售确认 | 虚开发票难以发现 | 发票开具须经独立核验 |
| 业务签约 + 合同履行验收 | 合同造假、虚假验收 | 合同签约与验收由不同部门负责 |
| 资产保管 + 资产核实 | 资产侵占 | 实物保管与账目核对由不同人员负责 |

**举报机制设计要素**：

| 要素 | 内容 | 实施要点 |
|------|------|---------|
| **举报渠道** | 多渠道举报（邮箱/热线/线上系统/书面）；可匿名 | 渠道须公开易得；管理层不得截获举报信息 |
| **举报人保护** | 举报人信息严格保密；禁止对举报人打击报复；举报人保护须有制度保障 | 违反举报人保护的管理者须受到处分 |
| **调查程序** | 举报须有专职人员记录、受理、调查；重大问题升级合规委员会/律师审查 | 调查不得由被举报部门自行调查 |
| **反馈机制** | 向举报人（匿名举报可向全员）反馈处理结果 | 定期公告合规举报处理情况 |
| **记录留存** | 所有举报与调查记录须留存（期限须符合规定） | 避免调查记录被删除 |

### 步骤五：内控有效性自查清单

**年度/定期内控自查**：

| 自查维度 | 检查项 | 参考结论 |
|---------|-------|---------|
| 制度覆盖 | 反商业贿赂/发票管理/招投标/关联交易/环保/安全生产制度是否均已制定并发布 | 未覆盖的类别标 `❌` 需补充 |
| 制度落地 | 制度是否有配套培训记录、员工签收记录 | 无记录的标 `⚠️` 需补充 |
| 高危环节控制 | 重点环节的控制措施是否有执行记录（审批/核对/三方验收）（采购/销售/财务/招投标） | 执行记录缺失标 `⚠️` 需整改 |
| 岗位制衡 | 不相容职务是否已分离（出纳会计/采购决策执行/合同签约验收） | 未分离的标 `❌` 需整改 |
| 举报机制 | 举报渠道是否畅通；近期是否有举报记录；举报处理是否留存 | 无记录的标 `⚠️` 需核实 |
| 培训记录 | 年度合规培训是否完成；高风险岗位是否有专项培训 | 未完成的标 `⚠️` 需补充 |
| 外部评估 | 是否有外部律师/审计对内控进行独立评估 | 未评估标 `⚠️` 建议安排 |

## 边界条件

| 情形 | 处置方式 |
|------|---------|
| 中小企业内控资源有限（无法配置合规部门） | 提供简化版内控方案（关键红线+高管审批+兼职合规负责人）；标注简化的局限性 |
| 内控形式化风险（企业只出制度不落地） | 明确提示：制度须有配套培训与执行记录，否则有效合规认定可能失败 |
| 内控成本高（如系统改造需要大量投资） | 按风险等级分阶段实施；优先高风险环节；标注分阶段计划的风险提示 |
| 发现已发生的风险事件（举报或审计发现异常） | 立即强制升级执业刑事辩护律师；启动内部调查须律师指导（证据保全与法律特权保护） |
| 上市公司/国有企业特殊要求 | 说明上市公司/国有企业监管合规要求（证监会/国资委要求），与普通私营企业有所不同 |

## 错误处理

| 错误情形 | 处置方式 | 输出标记 |
|---------|---------|---------|
| 发现已发生风险事件（非纯预防场景） | 立即强制升级律师，转为 incident-response 子技能框架 | `[强制升级 — 发现风险事件]` |
