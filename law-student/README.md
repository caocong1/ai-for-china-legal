# law-student

> 法考培训技能套件 — 辅助法律职业资格考试备考

本插件提供系统化的法考备考训练工具，覆盖从备考规划到查漏补缺的全链路。

## 功能模块

### 冷启动访谈（cold-start-interview）

通过结构化访谈建立学习者画像，个性化所有后续训练。

- 身份与学习画像（identity-enterprise-profile）
- 学习立场与目标（study-stance-goals）
- 薄弱环节与升级（weakness-escalation）
- 学习风格与资源（style-resources）

### 案例研习（case-study）

训练法律案例分析方法论，适应法考主观题要求。

- 案件事实识别与争议焦点归纳（case-fact-identification）
- 法律问题定性与请求权基础分析（legal-issue-analysis）
- 多方观点论证与利益衡量（multi-perspective-argumentation）
- 结论形成与结构化表达（conclusion-structured-output）

### 法条记忆（law-memorization）

系统训练高效记忆法考核心法条。

- 法律体系与部门法知识图谱（legal-system-mapping）
- 重点法条识别与高频考点（key-provision-identification）
- 记忆技巧应用（memory-technique-application）
- 间隔复习与自测（spaced-repetition-review）

### 模拟题生成（practice-questions）

按法考标准生成高质量模拟题。

- 题型设计与知识点覆盖（question-type-design）
- 案例情景构建（scenario-construction）
- 答案与解析撰写（answer-key-explanation）
- 难度校准与区分度（difficulty-calibration）

### 错题分析（wrong-answer-analysis）

系统化分析错题，找到根本原因并制定补强方案。

- 错误模式识别（error-pattern-recognition）
- 知识薄弱点诊断（knowledge-gap-diagnosis）
- 针对性补强方案（targeted-remediation）
- 进步追踪与自测（progress-tracking）

## 快速开始

1. 运行冷启动访谈配置学习档案：`/law-student:cold-start-interview`
2. 根据需要选择对应的训练模块

## 教育声明

本插件所有输出仅供法考学习参考，不构成法律建议。法条内容请以当年考试大纲和指定教材为准。

学习者配置文件由冷启动访谈生成，存放于 `law-student/CLAUDE.md`。
