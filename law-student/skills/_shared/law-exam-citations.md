## 四、引用库使用说明

### 使用方式

技能引用本库时，在「前置」中声明：

```
前置：读取 law-student/skills/_shared/law-exam-citations.md（法考高频法条引用库）
```

然后在「法律依据」章节按法条/案例/学说三类组织，从本库提取相关条文，保留来源标签和待验证标注。

### 更新规则

1. **法条更新**：法规修订时，更新对应版本年份
2. **案例填充**：经检索核验的案例，填入对应占位位置
3. **学说填充**：引用时须有完整出处，注明通说/少数说
4. **大纲调整**：法考大纲调整后，更新对应学科的考查要点

### 与引用规范的关系

本库遵循 `law-student/skills/_shared/legal-basis-conventions.md` 定义的引用规范。
本库是**学习辅助工具**，不构成法律建议。完整法条正文请通过：

- 国家法律法规数据库：https://flk.npc.gov.cn
- 最高人民法院官网：https://www.court.gov.cn
- 法考指定教材（以当年大纲为准）
