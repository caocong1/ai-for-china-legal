# 法考培训法律依据引用规范

> 本文件定义 law-student 插件所有技能统一使用的法律依据引用规范。
> 法考培训定位为辅助法律学习与备考，不替代法律执业，所有输出仅供学习参考。

---

## 一、来源分类

所有法律依据必须归入以下三类之一，并在引用时明确标注类别标签。

### 1. 法条（成文法）

指现行有效的成文法规范，法考涉及的主要渊源包括：

- **法律**：全国人民代表大会及其常委会制定（民法典、刑法、行政处罚法、公司法等）
- **行政法规**：国务院制定（如招标投标法实施条例等）
- **部门规章**：国务院各部委制定（如市场监管总局规章等）
- **司法解释**：最高人民法院、最高人民检察院发布的解释性文件
- **地方性法规**：省级人大及其常委会制定（法考中较少直接考查，但涉及管辖等知识点时会出现）

**成文法是法考分析基础。** 所有案例分析、模拟题解析的结论，必须首先对照现行成文法条文。

**格式要求（法条最小引用格式）：**

```
《[法律/法规全名]》（[版本/修订年份]）第[X]条[第X款]
```

示例：
```
《中华人民共和国民法典》（2020年通过）第143条
```

**版本标注要求：**
- 引用民法典须标注「2020年通过」（2021年1月1日施行，现行版本）
- 引用刑法须标注「1997年修订」并注明修正案编号（如「刑法修正案（十二）」）
- 引用已修订的版本，须注明修订情况

### 2. 案例（指导性案例与参考案例）

指最高人民法院发布的指导性案例，以及法考真题中引用的典型案例。

**定位说明：指导性案例仅作参考，不具有判例法约束力。** 中国采用成文法体系，案例不能替代法条。法考培训中使用案例的目的是帮助理解法条适用方式，而非创设法律规则。

**格式要求（案例最小引用格式）：**

```
[最高人民法院指导案例第X号] / [案号：（年份）法院级别字号数字号] — [裁判要点摘要] — [法院层级]
```

**模型回忆案例风险说明：** 案号、指导案例编号极易被模型错误回忆，**所有案例引用默认标注 `[待验证]`**，必须经检索核验后方可去除。

### 3. 学说（学术观点）

指法考培训中涉及的法理学说、学术争议观点。

**学说在法考中的定位：** 法考以通说为准，涉及学说争议时须标注「通说」与「少数说」的区别。引用学说时必须标注：
- 作者/出处（教材/著作名称）
- 是否为通说
- 法考中的考查倾向

**格式要求：**

```
[作者姓名]，《[著作/教材名]》，[出版信息] — "[观点摘要]" [通说/少数说]
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[教材]` | 来自法考培训教材或辅导书 | 教材引用 |
| `[法规网站]` | 来自政府官方网站 | 全国人大、最高院等官网 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查 |
| `[用户提供]` | 来自用户提供的文件 | 课堂笔记、错题本等 |
| `[真题]` | 来自历年法考真题 | 真题引用 |

**标签使用规则：**
1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 和 `[网络搜索 — 需验证]` 的引用在依赖前**必须**对照原始来源检查

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` | 模型回忆的条文号/案号风险高 | 通过国家法律法规数据库核查 |
| 引用条文距最后更新超过 6 个月 | 法规可能已修订或废止 | 查询官方法规数据库最新版本 |
| 所有案例引用（默认） | 案号易被错误回忆 | 通过中国裁判文书网、法宝等核查 |
| 法规版本未确认 | 无法确认是否为现行有效版本 | 查询国家法律法规数据库 |

**待验证标记格式：**

```
《民法典》第XXX条 `[模型知识 — 需验证]` `[待验证 — 验证途径：国家法律法规数据库]`
```

**验证资源：**
- 国家法律法规数据库：https://flk.npc.gov.cn
- 最高人民法院官网：https://www.court.gov.cn
- 中国裁判文书网：https://wenshu.court.gov.cn
- 北大法宝（如已接入）

## 四、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：案例分析、模拟题解析的每个结论，必须首先援引现行成文法作为规范依据
2. **不得以案例替代法条**：即使存在清晰的指导性案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下案例/学说」

### 案例的辅助定位

1. **指导性案例**：用于说明法条适用方式，标注「最高院指导，参照意义强」
2. **法考真题**：用于说明考查方式和答题思路，标注「真题参考」
3. **明确说明非判例法**：引用案例时，须附注「中国采成文法体系，本案例为参考，不具判例法约束力」

## 五、法考特殊引用规范

### 法条号记忆规则

法考培训中引用法条号时：

1. **描述规则 + `[待验证]`**：对所有具体条文号，先描述该条文规定的规则内容，再标注 `[待验证]`，绝不编造条文号
2. **高频考点条文**：对法考高频考查的核心条文（如民法典第143条民事法律行为有效要件），可在确认教材一致性后使用
3. **条号不确定时**：使用「《XX法》关于[规则描述]的规定」替代具体条号

### 学说争议处理

| 情形 | 处理方式 |
|------|---------|
| 通说观点 | 直接引用，标注「通说」 |
| 少数说观点 | 标注「少数说」，并说明法考中通常不以此为准 |
| 法考大纲明确采纳的观点 | 按大纲要求处理，标注「大纲采纳」 |
| 存在多种学说但法考未明确 | 列出主要观点，标注「法考中依大纲为准 `[待验证]`」 |

### 学科分类引用

法考涉及的学科分类及各学科主要法律渊源：

| 学科 | 主要法律渊源 |
|------|------------|
| 民法 | 民法典、相关司法解释 |
| 刑法 | 刑法及修正案、相关司法解释 |
| 行政法与行政诉讼法 | 行政处罚法、行政许可法、行政强制法、行政诉讼法、行政复议法 |
| 民事诉讼法 | 民事诉讼法及相关司法解释 |
| 刑事诉讼法 | 刑事诉讼法及相关司法解释 |
| 商法 | 公司法、合伙企业法、破产法、票据法、保险法、证券法 |
| 经济法 | 劳动法、劳动合同法、消费者权益保护法、反垄断法、反不正当竞争法、税法等 |
| 国际法 | 国际公法、国际私法、国际经济法相关条约和国内法 |
| 理论法学 | 法理学、宪法学、中国法制史 |
| 法律职业道德 | 律师法、法官法、检察官法及相关职业规范 |

## 七、教育护栏

### 法考培训定位声明

所有输出须遵循以下护栏：

1. **学习辅助**：本插件定位为辅助法律学习与备考，不替代法律执业
2. **不构成法律建议**：所有案例分析、法条解读仅供学习参考，不构成针对具体案件的法律意见
3. **教材为准**：法考答案以当年考试大纲和指定教材为准，本插件辅助理解但不替代教材
4. **条文号审慎**：所有具体条文号描述规则后标注 `[待验证]`，建议学员对照教材核实
5. **学说中立**：涉及学说争议时客观呈现，注明法考中的考查倾向

### 输出标记

在非学习场景下误用时，须标记：

```
[教育声明：本内容仅供法考学习参考，不构成法律建议。具体法律问题请咨询执业律师。]
```
