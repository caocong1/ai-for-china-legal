# 法律问题定性与请求权基础分析（legal-issue-analysis）

> **所属编排**：`law-student/skills/case-study/SKILL.md`（案例研习）
> **遵循规范**：`law-student/skills/_shared/legal-basis-conventions.md`（法考培训引用规范）
> **引用库**：`law-student/skills/_shared/law-exam-citations.md`（法考高频法条引用库）
> **分工说明**：本子技能负责**法律定性与请求权基础分析**。事实识别由 `case-fact-identification` 处理；
> 多方论证由 `multi-perspective-argumentation` 处理；结论输出由 `conclusion-structured-output` 处理。
> **可单独触发**：`/law-student:legal-issue-analysis`

---

## 目的

训练学习者掌握法律案例分析的核心步骤——将事实问题转化为法律问题，检索请求权基础，
拆解法律构成要件，建立从事实到规范的分析链条。这是法考主观题考查的核心能力。

## 详细步骤

### 步骤 1：法律关系类型识别

#### 1.1 法律关系分类

| 法律关系类型 | 典型特征 | 适用编/法 |
|------------|---------|---------|
| 合同关系 | 双方合意、权利义务约定 | 民法典·合同编 |
| 物权关系 | 物的归属与利用 | 民法典·物权编 |
| 侵权关系 | 侵权行为→损害结果→因果关系 | 民法典·侵权责任编 |
| 婚姻家庭关系 | 身份关系、财产关系 | 民法典·婚姻家庭编 |
| 继承关系 | 遗产分配 | 民法典·继承编 |
| 劳动关系 | 用人单位与劳动者 | 劳动法、劳动合同法 |
| 行政法律关系 | 行政主体与行政相对人 | 行政法与行政诉讼法 |
| 刑事法律关系 | 犯罪行为与刑罚 | 刑法 |

#### 1.2 法律关系识别方法

1. **看主体**：主体类型决定法律关系性质（如用人单位与劳动者→劳动关系）
2. **看行为**：行为类型决定法律关系性质（如签订合同→合同关系；侵害权益→侵权关系）
3. **看标的**：标的物类型影响法律关系定性（如不动产→物权关系可能性大）

#### 1.3 法律关系竞合与聚合

| 情形 | 定义 | 处理方式 |
|------|------|---------|
| **请求权竞合** | 同一事实产生多个请求权，目的相同 | 择一行使（如违约与侵权竞合） |
| **请求权聚合** | 同一事实产生多个请求权，目的不同 | 可同时行使（如违约损害赔偿+解除合同） |
| **请求权规范竞合** | 同一事实可适用多个法律规范 | 选择最有利于权利人的规范 |

### 步骤 2：请求权基础检索

#### 2.1 请求权基础分析法概述

请求权基础分析法是民法案例分析的核心方法，由王泽鉴教授倡导，在法考中广泛应用。

**分析路径**：

```
谁得向谁，依据何种法律规范，主张何种权利？
```

#### 2.2 请求权基础检索顺序

按以下顺序检索请求权基础（避免遗漏）：

| 顺序 | 请求权类型 | 检索范围 |
|------|----------|---------|
| 1 | **合同上请求权** | 合同约定的权利+法定的合同权利（违约责任等） |
| 2 | **类合同请求权** | 缔约过失责任、无因管理等 |
| 3 | **物权上请求权** | 返还原物、排除妨害、消除危险 |
| 4 | **不当得利请求权** | 无法律根据取得利益致他人受损 |
| 5 | **侵权损害赔偿请求权** | 一般侵权+特殊侵权 |

| 请求权人 | 相对人 | 请求权内容 | 请求权基础（法条） | 构成要件 | 是否成立 |

### 步骤 3：法律要件拆解

#### 3.1 要件拆解方法

对每个请求权基础的法条进行构成要件拆解：

```
法条结构：如果 A + B + C → 法律效果 R

要件 A：[具体要件内容]
要件 B：[具体要件内容]
要件 C：[具体要件内容]

对应事实：
事实是否满足要件 A：[分析]
事实是否满足要件 B：[分析]
事实是否满足要件 C：[分析]

结论：R 是否成立
```

#### 3.2 要件拆解示例（一般侵权责任）

```
请求权基础：《民法典》第1165条（一般侵权责任）`[待验证]`

构成要件：
A. 侵权行为（加害行为）
B. 损害结果
C. 因果关系
D. 过错（故意或过失）

对应事实分析：
A. 侵权行为：[事实是否满足]
B. 损害结果：[事实是否满足]
C. 因果关系：[事实是否满足]
D. 过错：[事实是否满足]

结论：一般侵权责任是否成立
```

### 步骤 4：抗辩权检索

#### 4.1 抗辩权类型

在请求权分析后，须检索相对人可能的抗辩权：

| 抗辩权类型 | 定义 | 示例 |
|----------|------|------|
| **权利未发生抗辩** | 请求权自始未产生 | 合同无效、行为能力欠缺 |
| **权利消灭抗辩** | 请求权已产生但已消灭 | 清偿、抵销、免除、提存 |
| **权利阻止抗辩** | 请求权存在但暂时不能行使 | 同时履行抗辩权、先诉抗辩权、时效抗辩 |

#### 4.2 抗辩权检索表

|---------|---------|---------|---------|---------|

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 案例涉及多个法律关系 | 逐一识别，分别分析请求权基础 |
| 请求权竞合 | 列出竞合的请求权，说明择一行使的规则 |
| 法条检索困难（不确定适用哪条） | 使用「《XX法》关于[规则描述]的规定」替代具体条号，标注 `[待验证]` |
| 学习者对请求权基础分析法不熟悉 | 先简要介绍方法论，再开始分析 |

---

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 学习者遗漏了关键请求权 | 温和提示：「除了你分析的X请求权外，是否还考虑过Y请求权？」 |
| 请求权基础引用错误 | 指出错误，解释正确条文的适用范围 |
| 要件拆解不完整 | 补充遗漏的要件，解释为何该要件重要 |
