# 答案与解析撰写（answer-key-explanation）

> **所属编排**：`law-student/skills/practice-questions/SKILL.md`（模拟题生成）
> **遵循规范**：`law-student/skills/_shared/legal-basis-conventions.md`（法考培训引用规范）
> **引用库**：`law-student/skills/_shared/law-exam-citations.md`（法考高频法条引用库）
> **分工说明**：本子技能负责**答案与解析撰写**。题型设计由 `question-type-design` 处理；
> 情景构建由 `scenario-construction` 处理；难度校准由 `difficulty-calibration` 处理。
> **可单独触发**：`/law-student:answer-key-explanation`

---

## 目的

为法考模拟题生成准确的标准答案和详细的解析说明，帮助学习者理解每道题的
解题思路、法律依据和易错点，达到「做一道题，掌握一个知识点」的效果。

---

## 前置

1. **读取 `law-student/CLAUDE.md`** — 检查学习者偏好（解析详细度）
2. **读取 `law-student/skills/_shared/legal-basis-conventions.md`** — 法考培训引用规范
3. **读取 `law-student/skills/_shared/law-exam-citations.md`** — 法考高频法条引用库
4. 本子技能产出是答案与解析，**不构成法律建议**

## 详细步骤

### 步骤 1：客观题解析结构

#### 1.1 单选题解析模板

```markdown
**第X题答案：** [正确选项]

**解析：**

本题考查[知识点名称]。

根据《XX法》第X条的规定，[法条内容概述]。`[待验证]`

A 选项：[分析为何对/错]
B 选项：[分析为何对/错]
C 选项：[分析为何对/错]
D 选项：[分析为何对/错]

**易错点：** [学习者常见的错误选择及原因]

**关联知识点：** [需要一并复习的相关知识点]
```

#### 1.2 多选题/不定项解析模板

```markdown
**第X题答案：** [正确选项组合]

**解析：**

本题考查[知识点名称]。

根据《XX法》第X条的规定，[法条内容概述]。`[待验证]`

逐项分析：
- A 选项：[分析] — [正确/错误]
- B 选项：[分析] — [正确/错误]
- C 选项：[分析] — [正确/错误]
- D 选项：[分析] — [正确/错误]

**正确答案组合的理由：** [为何是这些选项]

**易错点：** [常见漏选/多选的情况]
```

### 步骤 2：主观题解析结构

#### 2.1 案例分析题解析模板

```markdown
**第X题参考答案：**

**问题一：**
根据《XX法》第X条 `[待验证]`，[法律规定]。
本案中，[事实与法律的对应分析]。
因此，[结论]。

**问题二：**
[同上结构]

**问题三：**
[同上结构]

**答题要点总结：**

| 问题 | 得分要点 | 分值 |
|------|---------|------|
| 问题一 | [要点列表] | X分 |
| 问题二 | [要点列表] | X分 |
| 问题三 | [要点列表] | X分 |

**常见失分点：**
1. [常见失分情况]
2. [常见失分情况]
```

### 步骤 3：解析质量标准

#### 3.1 好解析的标准

| 标准 | 说明 | 评分 |
|------|------|------|
| **准确性** | 答案正确，法律依据准确 | 必须满足 |
| **完整性** | 每个选项/问题都有分析 | 必须满足 |
| **逻辑性** | 分析有条理，推理严密 | 必须满足 |
| **教育性** | 能帮助学习者理解解题方法 | 重要 |
| **延展性** | 关联相关知识点，拓展学习 | 加分 |

#### 3.2 常见解析问题

| 问题 | 表现 | 改进方式 |
|------|------|---------|
| 解析过简 | 只给答案不给理由 | 增加法律分析 |
| 只分析正确项 | 不分析为何其他选项错 | 逐项分析 |
| 无法律依据 | 纯理论分析不引法条 | 补充法条引用 |
| 无易错点提示 | 不指出常见错误 | 增加易错点分析 |

### 步骤 4：解析分层

#### 4.1 根据学习者需求调整解析详细度

| 详细度 | 内容 | 适用场景 |
|-------|------|---------|
| **完整解析** | 法条引用+逐项分析+易错点+关联知识点 | 基础阶段学习 |
| **简要解析** | 法条引用+关键分析+答案 | 强化阶段练习 |
| **答案速查** | 仅答案+核心法条 | 冲刺阶段自测 |

## 检查清单

- [ ] 答案正确
- [ ] 每个选项/问题都有分析
- [ ] 法律依据已引用（含 `[待验证]`）
- [ ] 易错点已说明
- [ ] 关联知识点已提示
- [ ] 解析详细度符合学习者偏好
- [ ] 主观题答题要点已列出

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 题目存在争议 | 标注「本题存在争议」，列出不同观点 |
| 法条近期有修订 | 标注新旧法条差异 |
| 学习者要求只看答案 | 提供答案速查版，但建议查看完整解析 |
