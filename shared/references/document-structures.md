# 诉讼文书结构模板

## 诉讼方案

1. **案件基本情况**（按时间顺序的事实叙述）
2. **原告**（主体资格、诉讼地位、请求权基础）
3. **被告 / 第三人**（主体资格、责任范围、追加理由）
4. **案由**
   - 首选案由（必须唯一明确，不得以"X或Y"未决方式收尾）
   - 备选案由（如首选不成立时的退路）
   - 对易混案由的主动排除论证
5. **管辖法院**
   - 地域管辖（被告住所地 / 合同履行地 / 专属管辖）
   - 级别管辖（基层 / 中院 / 高院 / 最高院）
   - 管辖权异议风险预判
6. **起诉策略对比**（**必含**：单独起诉担保人 / 单独起诉主债务人 / 同时起诉两被告的管辖、举证、保全、执行复杂度对比）
7. **诉讼请求**
   - 主请求（精确到金额、起算日、止算日、计息标准）
   - 附带请求（诉讼费、保全费、律师费等）
8. **事实和理由**
   - 时间线
   - 法律关系分析
   - 法律依据
   - 类案支撑
9. **证据目录和举证要点**（按证据类型分组，见下方"证据目录模板"）
10. **法律依据和类案依据**（精确到条文号、案号）
11. **风险及应对**（必须覆盖下方"风险段强制要求"）
12. **结论和诉前准备清单**
13. **文末落款**：律所名称、经办律师署名位、日期

## 起诉状

1. 标题
2. 当事人信息
3. 诉讼请求（可执行、可计算）
4. 事实和理由（按时间顺序、证据关联）
5. 证据和证据来源
6. 致送法院
7. 具状人和日期

## 答辩状 / 代理意见

1. 核心答辩意见
2. 事实回应
3. 法律分析
4. 证据质证要点
5. 程序抗辩
6. 结论

## 证据目录模板

**必须按以下 3 大类分组列示**（每类至少 1 项；缺类视为质量闸门不通过）：

### 第一类：基础交易关系类
证明"基础违约事实"。包括：
- 招标公告/招标文件
- 投标文件
- 资格审查承诺
- 审计报告
- 监管机关认定函
- 取消中标资格/解除合同/终止交易等动作的官方文书及送达凭证

### 第二类：保函/合同法律关系及索赔事实类
证明法律关系有效成立。包括：
- 保函原件/合同文本
- 索赔函/催告函原件
- 索赔函/催告函送达凭证（EMS/邮政/当面签收）

### 第三类：被告违约事实类
证明被告未履行义务。包括：
- 催告记录（邮件/微信/通话录音）
- 对方拒付或拖延的会议纪要/函件
- 律师催告函及回执

## 风险段强制要求

**保函/担保案件风险段必须覆盖以下 5 大类**（缺一即视为不通过）：

1. **损失认定风险**：成渝地区判例中"提交投标保函即视为已交保证金、需自证因投标人失信遭受损失"的隐藏路径
2. **违约事实主观故意认定风险**：被告可能用"非主动故意"抗辩切断索赔基础
3. **保函有效期/索赔时效风险**：含扫描件 OCR 误读日期的 sanity check 提醒
4. **保证方式被改判风险**：被改判为一般保证后的先诉抗辩权后果
5. **执行风险**：被告是否已为被执行人/失信/大量被诉

## 保函/担保案件强化要求

凡涉及 `保函`、`融资担保公司`、`担保公司`、`连带责任保证` 等任一关键词，下列要求 **MUST** 满足：

### 诉讼请求措辞必须显式包含保证责任类型

- 单独起诉担保人：`判令被告 X 就 Y 元（基础义务项下款项）承担连带责任保证项下的支付责任 / 承担连带清偿责任`
- **不允许**只写 `判令被告支付 Y 元`
- 利息/损失：必须写出起算日（精确到日）、止算日（"至实际清偿之日"）、计息标准（LPR/约定利率）
- 必含诉讼费、保全费等程序费用

### 法律依据章节必须含的最小法条集合

| 论点 | 必引法条 |
|------|----------|
| 排除独立保函 | 法释〔2016〕24号 第1条 |
| 主体资格定位 | 金融监管总局令〔2023〕第3号 + 国务院令第683号 |
| 保证方式认定 | 民法典第686条 + 第688条第一款 + 第二款 + 担保制度解释第25条第二款 |
| 管辖 | 民事诉讼法第24条 + 担保制度解释第21条 |

### 起诉策略对比章节

对于保函/担保案件，**必须**在"起诉策略对比"章节明确论证三选一：

| 策略 | 管辖法院 | 举证复杂度 | 保全复杂度 | 执行风险 |
|------|----------|------------|------------|----------|
| 单独起诉担保人 | 担保合同管辖 | 中 | 中 | 担保人偿付能力 |
| 单独起诉主债务人 | 主合同管辖 | 低 | 低 | 主债务人偿付能力 |
| 同时起诉两被告 | 主合同管辖 | 高 | 高 | 两被告中任一可执行 |

不允许直接跳到结论而无对比依据。
