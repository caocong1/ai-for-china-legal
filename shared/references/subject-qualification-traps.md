# 主体资格陷阱清单

**凡涉以下主体类型，必须先查主体资格，否则可能案由错误、立案被驳或诉讼请求无法支持。**

## 融资担保公司/担保公司/地方金融组织

| 项目 | 内容 |
|------|------|
| **风险** | 无权开立独立保函（不属于"银行或非银行金融机构"） |
| **必查依据** | 法释〔2016〕24号 第1条 + 金融监管总局令〔2023〕第3号 + 国务院令第683号 |
| **正确案由** | 保证合同纠纷（非独立保函纠纷） |
| **保证方式** | 按保函措辞判断（见索即付/无条件支付 → 连带责任保证） |
| **管辖** | 按主合同确定（担保制度解释第21条） |

## 银行分支机构（无总行授权）

| 项目 | 内容 |
|------|------|
| **风险** | 可能无对外担保权限 |
| **必查依据** | 民法典第504/506条 + 银保监会授权规则 |
| **处理** | 需核验总行授权文件或追认 |

## 保险公司（出具保证保险）

| 项目 | 内容 |
|------|------|
| **风险** | 性质争议（保证保险 vs 担保） |
| **必查依据** | 保险法 + 相关司法解释 |
| **处理** | 区分保险责任和保证责任，案由不同 |

## 上市公司（无董事会/股东会决议）

| 项目 | 内容 |
|------|------|
| **风险** | 担保可能无效 |
| **必查依据** | 民法典第504条 + 上市公司监管规则 |
| **处理** | 需核验董事会/股东会决议，无决议可能构成越权担保 |

## 国企/事业单位（无国资委审批）

| 项目 | 内容 |
|------|------|
| **风险** | 程序瑕疵，担保效力待定 |
| **必查依据** | 国资监管规则 |
| **处理** | 需核验国资委审批文件或内部决策程序 |

## 地方政府融资平台

| 项目 | 内容 |
|------|------|
| **风险** | 担保可能违反预算法或地方政府债务管理规则 |
| **必查依据** | 预算法 + 地方政府债务管理相关规定 |
| **处理** | 需区分政府债务与企业债务 |

## 外商投资企业

| 项目 | 内容 |
|------|------|
| **风险** | 对外担保需外汇局备案/登记 |
| **必查依据** | 外汇管理条例 + 跨境担保外汇管理规定 |
| **处理** | 需核验外汇登记或备案文件 |

## 使用规则

1. 在案由确定前，**先核验主体资格**
2. 如发现主体资格问题，**调整案由和请求权基础**
3. 在研究底稿中记录主体资格核验过程和结论
4. 对不确定的主体，标注 `待律师复核主体资格`
