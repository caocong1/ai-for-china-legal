# 研究质量闸门

所有法律研究在进入文书起草前，必须通过以下三层质量闸门。任一闸门不通过，必须重新研究。

## Gate 1: 法律依据完整性

- [ ] 每个重大结论都有法条、司法解释、案例或明确标注的实务来源支撑
- [ ] 所有法律法规是否为现行有效版本？如已修订，是否标注版本差异？
- [ ] 条文号、案号是否精确可查？
- [ ] L1 来源（法律/司法解释全文）是否已获取？不得仅依赖 L5 转述

**不通过信号**：结论无来源、条文号错误、法规已废止未标注。

## Gate 2: 反向依据检索

- [ ] 是否至少检索了一种不利表述？
- [ ] 是否识别了不利案例、少数观点、程序异议或主体资格陷阱？
- [ ] 如未发现不利依据，是否列明了已执行的搜索关键词？

**不通过信号**：只找支持性依据、未尝试反向搜索、未说明搜索范围。

## Gate 3: 文书就绪性

- [ ] 事实、缺失事实、假设和证据缺口是否分离？
- [ ] 来源层级和置信度是否明确？
- [ ] 未决事项是否标注 `待律师复核`？
- [ ] 是否所有论证都可以追溯到具体来源？

**不通过信号**：事实与假设混同、来源层级不清、未标注未验证结论。

## 专项闸门（按案件类型附加）

### 保函/担保案件附加闸门

- [ ] 融资担保公司主体资格已核验（是否属于"银行或非银行金融机构"）
- [ ] 独立保函 vs 保证合同路径已论证（不得以"X或Y"未决方式收尾）
- [ ] 保证方式（一般 vs 连带）已按保函措辞分析
- [ ] 管辖策略已分别就"单告"和"同时起诉"两种方案列出

### 合同审查附加闸门

- [ ] 管辖法律已确认（不同法律下审查立场可执行性不同）
- [ ] 违约金/责任限制的四个维度已分析（直接vs间接、基数定义、例外、审查立场）
- [ ] 主体资格已核验（上市公司决议、国资审批、分支机构授权等）
