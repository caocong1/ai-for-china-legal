# 中国法检索 Playbooks

配合 `authoritative-sources.md` 使用 — 本文件规定**搜索什么**，authoritative-sources 规定**到哪里搜**。

## 通用五维度搜索模式

对每个法律问题，**必须**从五个维度构建搜索词：

| 维度 | 示例 | 用途 |
|------|------|------|
| 文书标签词 | `投标保函`、`见索即付`、`独立保函` | 匹配文书中的专业表述 |
| 主体资格词 | `主体资格`、`非银行金融机构`、`地方金融组织`、`融资担保公司` | 识别主体陷阱 |
| 替代案由词 | `保证合同纠纷`、`担保合同纠纷`、`合同纠纷` | 防止案由锁定错误 |
| 程序词 | `管辖权异议`、`追加第三人`、`一般保证`、`连带责任保证` | 发现程序风险 |
| 不利结果词 | `驳回诉讼请求`、`不具备开立独立保函主体资格`、`不予支持` | 主动找反例 |

## 保函/担保案件 Playbook

**触发条件**：文书中出现 `保函`、`见索即付`、`独立`、`无条件支付`、`投标担保`、`履约担保`、`融资担保公司`、`担保公司`。

### 必须固定调起的检索

```
融资担保公司 非银行金融机构 独立保函 site:court.gov.cn OR site:gov.cn
融资担保公司 不具备 开立独立保函 主体资格
见索即付 投标保函 连带责任保证
投标保函 融资担保公司 保证合同纠纷
独立保函 主体资格 融资担保公司 管辖权异议
"2024-10-2-358-001"          ← 入库案例编号
成渝金融法院 融资担保 独立保函
重庆 融资担保公司 保函 保证合同
```

### 必须核验的法规

| 文件 | 条文 | 预期结论 |
|------|------|----------|
| 法释〔2016〕24号（独立保函司法解释） | 第1条 | 开立人限**银行或非银行金融机构** |
| 金融监管总局令〔2023〕第3号 | 全文 | **不含**融资担保公司 |
| 融资担保公司监督管理条例（国务院令第683号） | 全文 | 业务范围**不含**开立独立保函 |
| 民法典 | 第686、688条 | 一般保证 vs 连带责任保证 |
| 担保制度解释（法释〔2020〕28号） | 第25条 | "无条件支付"等 → **连带责任保证** |
| 担保制度解释（法释〔2020〕28号） | 第21条 | 合并起诉时按**主合同**确定管辖 |
| 民事诉讼法 | 第24条 | 被告住所地/合同履行地 |

### 2024-2025 裁判共识（回 L1/L2 核验）

各级法院已形成共识：

1. **担保公司（含融资担保公司）不属于"银行或非银行金融机构"**，无权开立独立保函
2. 保函即使载"见索即付"/"无条件支付"，**不构成独立保函**
3. 通常按**保证合同**处理；保证方式按《担保制度解释》第25条判断
4. **基础合同无效**（如违法转包）会导致保函一并无效 — 应核验基础合同效力

## 管辖 Playbook

**触发条件**：需要选择法院，或涉及合同、担保、建设工程、招投标、分支机构。

```
投标保函 管辖权异议 融资担保公司
独立保函纠纷 管辖权异议 担保公司
保证合同纠纷 管辖 被告住所地 合同履行地
建设工程 招投标 保函 管辖权异议 专属管辖
```

重点关注**程序性裁定**（管辖权异议裁定），不仅是实体判决。

## 主体资格陷阱 Playbook

**触发条件**：涉及以下任一主体类型：

| 主体类型 | 风险 | 必查依据 |
|----------|------|----------|
| 融资担保公司/担保公司 | 无开立独立保函主体资格 | 法释〔2016〕24号第1条 + 金融监管总局令〔2023〕第3号 |
| 银行分支机构（无总行授权） | 可能无对外担保权限 | 民法典第504/506条 |
| 保险公司（保证保险） | 性质争议（保险 vs 担保） | 保险法 + 司法解释 |
| 上市公司（无董事会/股东会决议） | 担保可能无效 | 民法典第504条 + 上市公司监管规则 |
| 国企/事业单位（无国资委审批） | 程序瑕疵 | 国资监管规则 |

## 地方司法实务反例预警

**凡涉投标保函索赔，必查以下反例路径**：

| 反例路径 | 检索词 | 应对 |
|----------|--------|------|
| "提交保函即视为已交保证金，需自证损失" | `投标保函 视为 履行 交保证金 损失` `重庆 投标保函 缔约过失 损失` | 风险段列入"损失认定风险" |
| "基础合同无效 → 保函一并无效" | `基础合同 无效 保函 无效 投标` | 论证本案基础关系是招投标关系 |
| "弄虚作假需主观故意" | `弄虚作假 主观 故意 投标 抗辩` | 准备权威机关客观认定函 |
| "管辖约定不明" | `管辖约定 不明 投标保函` | 论证按民诉法第24条确定 |

未检索到反例 ≠ 不存在，必须在研究底稿中明示"已检索 X、Y 关键词未发现反例"。

## 公众号/律所文章使用规则

- 公众号文章和律所文章**仅作为发现案例、检索词、争议点的线索**
- 找到 L5 提到的案例后，**必须回到 L1/L2 校验全文**
- 校验失败时，标注 `实务观点，未核验原裁判全文`
- 法律公众号、律所文章**不能当作绑定性依据**

## 评论性文章使用

使用公众号和律所文章发现隐藏案例和搜索词，然后用法律、官方案例、判决文本或法院发布材料验证。如验证失败，标注评论为 `实务观点，未核验原裁判全文`。
