# 申请材料准备

> 本子技能可单独运行 `/legal-aid:material-preparation`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件，含本地资源信息）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

协助当事人齐备法律援助申请所需的全部材料。

---

## 详细步骤

### 第一步：材料清单生成

根据案件类型和当事人情况，生成个性化材料清单：

#### 必备材料

| 序号 | 材料名称 | 说明 | 获取方式 |
|------|---------|------|---------|
| 1 | 法律援助申请表 | 按法援中心统一格式 | 法援中心领取/下载 |
| 2 | 身份证或其他有效身份证明 | 核对原件，提交复印件 | 当事人持有 |
| 3 | 经济困难证明 | 低保证/特困证/街道证明 | 见下方详细说明 |
| 4 | 与案件相关的证据材料 | 支持诉讼请求的证据 | 当事人收集和协助调取 |

#### 经济困难证明获取指南

| 证明类型 | 适用情形 | 获取部门 | 获取方式 |
|---------|---------|---------|---------|
| 最低生活保障证 | 已享受低保 | 民政局/街道办 | 当事人持有 |
| 特困人员救助供养证 | 特困人员 | 民政局 | 当事人持有 |
| 经济困难证明 | 无低保证但经济困难 | 户籍所在地街道办事处/乡镇政府 | 申请开具 |
| 收入证明 | 有固定收入 | 工作单位 | 申请开具 |
| 无收入声明 | 无固定收入 | 社区/村委会 | 申请开具 |

从实践配置文件读取本地民政部门信息：
- 民政局名称：`[从 legal-aid/CLAUDE.md 读取]`
- 联系方式：`[从 legal-aid/CLAUDE.md 读取]`
- 开具流程：`[从 legal-aid/CLAUDE.md 读取]`

#### 案件相关材料

根据案件类型列出需要的证据材料：

| 案件类型 | 常见证据材料 |
|---------|------------|
| 劳动争议 | 劳动合同、工资条、考勤记录、解除通知、社保缴纳记录 |
| 赡养/抚养费 | 亲属关系证明、对方收入证明、已支付/未支付记录 |
| 交通事故 | 事故认定书、医疗费票据、伤残鉴定、保险单 |
| 工伤赔偿 | 工伤认定决定书、劳动能力鉴定、医疗费票据 |
| 刑事辩护 | 拘留通知书、逮捕通知书、起诉书（如有） |

### 第二步：材料完整性检查

| 序号 | 材料 | 状态 | 备注 |
|------|------|------|------|
| 1 | 法律援助申请表 | [已备/待准备] | — |
| 2 | 身份证明 | [已备/待准备] | — |
| 3 | 经济困难证明 | [已备/待准备/免予] | — |
| 4 | [案件材料] | [已备/待准备] | — |

### 第三步：材料瑕疵处理

| 瑕疵类型 | 处理方式 |
|---------|---------|
| 材料缺失 | 告知获取方式，必要时协助联系 |
| 材料过期 | 告知需更新（如低保证年审） |
| 信息不一致 | 请当事人说明原因，必要时更正 |
| 材料不完整 | 指出缺失部分，限期补充 |
| 无法取得某材料 | 评估是否可以其他材料替代，或说明情况 |

### 第四步：材料整理与装订

按法援中心要求整理材料：
1. 按目录顺序排列
2. 编制材料目录
3. 统一纸张大小
4. 装订成册

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 当事人行动不便 | 提供上门收集服务或委托他人代办 |
| 材料在异地 | 协助联系异地部门或邮寄获取 |
| 无法取得经济困难证明 | 说明情况，由法援机构核实 |
| 免予核查经济困难 | 跳过经济证明，准备免予核查情形证明 |
| 当事人为文盲或半文盲 | 协助填写表格，口头陈述由工作人员记录并确认 |
| 当事人为外国人 | 增加护照复印件、出入境记录、翻译件 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 经济困难证明被拒绝开具 | 协助分析原因，建议向民政部门上级机关反映 |
| 关键证据确实无法取得 | 标注「关键证据缺失」，评估对案件的影响 |
| 材料之间信息矛盾 | 请当事人说明原因，必要时更正 |
| 材料伪造嫌疑 | 标注「材料真实性存疑」，报告负责人 |

## 材料整理与装订规范

### 装订顺序

1. 材料目录
2. 法律援助申请表
3. 身份证明材料
4. 经济困难证明材料
5. 案件证据材料（按逻辑顺序排列）
6. 其他材料

### 装订要求

| 要求 | 说明 |
|------|------|
| 纸张统一 | A4纸，单面打印 |
| 页码编排 | 从第一页开始连续编页码 |
| 材料目录 | 首页为材料目录，标明各材料起止页码 |
| 装订方式 | 左侧装订或线装 |
| 复印件标注 | 复印件注明「与原件核对一致」并签名 |
| 外文材料 | 附中文翻译件 |
