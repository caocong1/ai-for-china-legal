# 申请提交与审查跟踪

> 本子技能可单独运行 `/legal-aid:application-submission-tracking`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

1. 协助提交法律援助申请
2. 跟踪审查进度
3. 处理审查期间的补正要求

---

## 详细步骤

### 第一步：申请提交

#### 1.1 提交方式

| 提交方式 | 说明 |
|---------|------|
| 现场提交 | 到法援中心窗口提交（推荐） |
| 邮寄提交 | 邮寄申请材料（注意保留邮寄凭证） |
| 网上申请 | 通过法援机构网站/APP提交（如有） |
| 转交申请 | 通过法院/看守所/社区等转交 `[待验证]` |

#### 1.2 提交确认

提交后获取以下确认信息：

| 确认项 | 说明 |
|-------|------|
| 收件回执 | 法援机构出具的收件凭证 |
| 申请编号 | 法援机构的受理编号 |
| 收件日期 | 正式收件日期（影响审查期限起算） |
| 联系人 | 法援机构的审查联系人 |
| 预计审查期限 | 法援机构告知的预计审查时间 |

#### 1.3 提交记录

| 项目 | 内容 |
|------|------|
| 提交日期 | [日期] |
| 提交方式 | [现场/邮寄/网上/转交] |
| 收件机构 | [法援中心名称] |
| 申请编号 | [编号] |
| 收件回执编号 | [编号] |
| 提交材料清单 | [附材料目录] |

### 第二步：审查期限跟踪

#### 2.1 法定审查期限

| 期限 | 说明 | 法律依据 |
|------|------|---------|
| 审查决定期限 | 收到申请后 [期限待验证] 内作出决定 | 《法律援助法》相关规定 `[待验证]` |
| 补正期限 | 收到补正通知后 [期限待验证] 内补充 | 相关规定 `[待验证]` |

#### 2.2 进度跟踪

| 时间节点 | 状态 | 备注 |
|---------|------|------|
| 提交日期 | [日期] | — |
| 审查期限届满日 | [日期] | 自动计算 |
| 当前状态 | [审查中/已补正/已决定] | — |
| 距期限届满 | [N] 天 | — |

#### 2.3 超期预警

| 预警级别 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 🟢 正常 | 距期限 > 7日 | 等待 |
| 🟡 关注 | 距期限 3-7日 | 主动询问进度 |
| 🟠 警告 | 距期限 1-3日 | 联系法援机构催办 |
| 🔴 超期 | 超过审查期限 | 投诉或申诉 |

### 第三步：补正处理

#### 3.1 收到补正通知

法援机构认为材料不齐全时，可一次性告知需补充的材料 `[待验证]`。

**补正响应流程**：

1. 仔细阅读补正通知
2. 逐项准备补充材料
3. 在补正期限内提交
4. 保留提交凭证

#### 3.2 补正记录

| 项目 | 内容 |
|------|------|
| 补正通知日期 | [日期] |
| 补正期限 | [日期] |
| 需补充材料 | [列举] |
| 补正提交日期 | [日期] |
| 补正材料清单 | [附清单] |

### 第四步：紧急情况处理

若案件紧急（时效即将届满等），在申请时一并申请加急处理：

| 紧急情形 | 处理方式 |
|---------|---------|
| 时效即将届满 | 申请加急审查，说明紧急情况 |
| 羁押期限将满 | 通过看守所/法院转交并申请加急 |
| 人身安全受威胁 | 说明紧急情况，申请优先处理 |

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 法援机构不予受理 | 了解原因，判断是否合理，必要时申诉 |
| 审查超期 | 主动催办，必要时向司法行政机关反映 |
| 补正期限不足 | 申请延期或说明情况先行提交已有材料 |
| 联系不上法援机构 | 通过上级司法行政机关联系 |
