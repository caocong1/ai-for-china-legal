# 法律依据引用规范

> 本文件定义 legal-aid 插件所有技能统一使用的法律依据引用规范。
> 全插件各子技能直接引用本规范，避免各技能重复维护。

---

## 一、来源分类

所有法律依据必须归入以下三类之一，并在引用时明确标注类别标签。

### 1. 法条（成文法）

指现行有效的成文法规范，包括：
- **法律**：全国人民代表大会及其常委会制定（如法律援助法、律师法、民事诉讼法、刑事诉讼法等）
- **行政法规**：国务院制定（如法律援助条例等）
- **部门规章**：司法部、民政部等部门制定（如办理法律援助案件程序规定、法律援助质量标准和评估办法等）
- **地方性法规与规章**：省级人大及其常委会、省级政府制定（如各省法律援助条例或实施办法）
- **司法解释**：最高人民法院、最高人民检察院发布的解释性文件（具有规范效力）

**成文法是分析基础。** 法律援助工作的所有结论，必须首先对照现行成文法条文，而非惯例或经验。

**格式要求（法条最小引用格式）：**

```
《[法律/法规全名]》（[版本/修订年份]）第[X]条[第X款]
```

示例：
```
《中华人民共和国法律援助法》（2021年）第[X]条 `[待验证]`
```

**版本标注要求：**
- 引用的法条必须标注现行有效版本（修订年份或施行年份）
- 法律援助法（2021年8月20日通过，2022年1月1日施行）为现行法律援助领域基本法律
- 如引用已修订的版本，须注明"（已修订，适用旧法情形）"

### 2. 案例（指导性案例与参考案例）

指最高人民法院发布的指导性案例或参考案例。

**定位说明：指导性案例仅作参考，不具有判例法约束力。** 中国采用成文法体系，案例不能替代法条。但最高人民法院指导性案例对下级法院有参照作用，可用于说明裁判思路。

**格式要求（案例最小引用格式）：**

```
[最高人民法院指导案例第X号] / [案号：（年份）法院级别字号数字号] — [裁判要点摘要] — [法院层级]
```

**法院层级标注：**

| 层级 | 简称 | 说明 |
|------|------|------|
| 最高人民法院 | 最高院 | 参照价值最高 |
| 高级人民法院 | 高院 | 省级，参照本省 |
| 中级人民法院 | 中院 | 地市级 |
| 基层人民法院 | 基层法院 | 县区级 |

**模型回忆案例风险说明：** 案号、指导案例编号极易被模型错误回忆，**所有案例引用默认标注 `[待验证]`**，必须经检索核验后方可去除。

### 3. 学说（学术观点）

指学术著作、法学论文中的学术观点，或法律援助实务经验总结。

**学说为辅助参考，存在争议性。** 引用学说时必须标注：
- 作者/出处（书名/期刊名+年份）
- 是否存在对立观点

**格式要求：**

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息] — "[观点摘要]" [是否有争议：有/无]
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站 | 全国人大、国务院、司法部等官网 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查 |
| `[用户提供]` | 来自用户提供的文件 | 当事人材料、法援中心文件等 |

**标签使用规则：**
1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 和 `[网络搜索 — 需验证]` 的引用在依赖前**必须**对照原始来源检查

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` | 模型回忆的条文号/案号风险高 | 通过法宝、国家法律法规数据库核查 |
| 引用条文距最后更新超过 6 个月 | 法规可能已修订或废止 | 查询官方法规数据库最新版本 |
| 用户口述的条款号 | 可能记忆偏差 | 核查法规原文 |
| 所有案例引用（默认） | 案号易被错误回忆 | 通过中国裁判文书网、法宝等核查 |
| 法规版本未确认 | 无法确认是否为现行有效版本 | 查询国家法律法规数据库 |

**待验证标记格式：**

```
《法律援助法》（2021年）第[X]条 `[模型知识 — 需验证]` `[待验证 — 验证途径：国家法律法规数据库]`
```

**验证资源：**
- 国家法律法规数据库：https://flk.npc.gov.cn
- 最高人民法院官网：https://www.court.gov.cn
- 司法部官网：https://www.moj.gov.cn
- 北大法宝（如已接入）
- 中国裁判文书网：https://wenshu.court.gov.cn

## 四、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：法律援助条件评估、案件管理的每个结论，必须首先援引现行成文法（法律、行政法规、司法解释）作为规范依据
2. **不得以案例替代法条**：即使存在清晰的指导性案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明"无直接成文法规定，参照以下案例/学说"

### 案例的辅助定位

1. **指导性案例**：用于说明法条适用方式，标注「最高院指导，参照意义强」
2. **一般生效判决**：用于说明司法实践倾向，标注「参考，非指导案例」
3. **明确说明非判例法**：引用案例时，须附注「中国采成文法体系，本案例为参考，不具判例法约束力」

## 五、引用最小格式汇总

| 类别 | 最小格式 | 示例 |
|------|---------|------|
| 法条 | 《法律名称》（版本年份）第X条[第X款] [来源标签] | 《法律援助法》（2021年）第[X]条 `[待验证]` |
| 行政法规 | 《行政法规名称》（发布年份）第X条 [来源标签] | 《法律援助条例》（2003年）第X条 `[待验证]` |
| 部门规章 | 《部门规章名称》（文号）第X条 [来源标签] | 《办理法律援助案件程序规定》第X条 `[待验证]` |
| 司法解释 | 《司法解释名称》（文号）第X条 [来源标签] | 最高院相关司法解释第X条 `[待验证]` |
| 指导案例 | 最高人民法院指导案例第X号 — [裁判要点] — [法院层级] `[待验证]` | 最高人民法院指导案例第X号 — [要点] — 最高院 `[待验证]` |
| 学说 | [作者]，《[著作]》，[出处] — "[观点]" [争议性] | [作者]，《[著作]》，[出版年份] — "..." [争议性] |
