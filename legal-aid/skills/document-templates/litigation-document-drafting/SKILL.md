# 诉讼文书起草

> 本子技能可单独运行 `/legal-aid:litigation-document-drafting`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件，含文书风格和本地法院信息）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）
> 前置：读取 `legal-aid/skills/_shared/legal-aid-citations.md`（法律援助引用库）

---

## 目的

辅助起草法援案件常用诉讼文书：

1. 民事/行政起诉状
2. 民事/行政答辩状
3. 代理词
4. 刑事辩护词

---

## 详细步骤

### 第一步：确定文书类型和基本信息

| 确定项 | 选项 |
|-------|------|
| 文书类型 | 起诉状 / 答辩状 / 代理词 / 辩护词 |
| 案件类型 | 民事 / 行政 / 刑事 |
| 我方立场 | 原告/申请人 / 被告/被申请人 / 辩护人 |
| 管辖法院 | 从实践配置文件读取本地法院信息 |

### 第二步：民事起诉状模板

```markdown
# 民事起诉状

**原告：** [姓名]，[性别]，[民族]，[出生日期] 出生，住 [地址]，
身份证号码：[号码]。联系电话：[电话]。

**被告：** [姓名/名称]，[性别/类型]，[民族]，[出生日期] 出生，
住 [地址]，身份证号码/统一社会信用代码：[号码]。
法定代表人（如适用）：[姓名]。
联系电话：[电话]。

**诉讼请求：**

1. [具体请求一]；
2. [具体请求二]；
3. 本案诉讼费由被告承担。

**事实与理由：**

[案件事实经过，按时间顺序叙述]

[法律依据和分析]

根据《中华人民共和国民法典》/相关法律 `[待验证]` 的规定，
原告特向贵院提起诉讼，请求依法支持原告的诉讼请求。

此致

[法院名称]

具状人（签名）：
日期：    年  月  日

**附：**
1. 本起诉状副本 [N] 份
2. 证据清单及证据材料
```

**起诉状起草要点**：
- 诉讼请求明确具体，可执行
- 事实叙述客观完整
- 法律依据引用准确，标注 `[待验证]`
- 管辖法院正确（从实践配置文件读取）

### 第三步：民事答辩状模板

```markdown
# 民事答辩状

**答辩人（被告）：** [姓名]，[性别]，[民族]，[出生日期] 出生，
住 [地址]，身份证号码：[号码]。联系电话：[电话]。

**被答辩人（原告）：** [姓名/名称]，[基本信息]。

答辩人就 [被答辩人姓名] 诉答辩人 [案由] 一案（案号：[—]），
提出答辩如下：

**一、关于原告诉讼请求**

[逐项回应原告的诉讼请求，说明同意或不同意的理由]

**二、关于事实与理由**

[针对原告陈述的事实进行回应，纠正不实之处，补充遗漏事实]

**三、法律分析**

[法律依据和论证]

综上所述，原告的诉讼请求 [缺乏事实/法律依据]，
请求贵院依法 [驳回原告诉讼请求/支持答辩人主张]。

此致

[法院名称]

答辩人（签名）：
日期：    年  月  日
```

### 第四步：代理词模板

```markdown
# 代理词

**尊敬的审判长、审判员：**

[律师事务所/法律援助中心] 接受 [当事人姓名] 的委托，
指派 [律师姓名] 律师担任其与 [对方当事人] [案由] 一案的
[原告/被告] 诉讼代理人。代理人经过阅卷、调查和参加庭审，
现发表如下代理意见：

**一、案件事实**

[简要概述案件事实]

**二、法律分析**

[逐项论证己方主张的法律依据]

**三、结论**

综上所述，[总结己方立场和请求]。

恳请合议庭依法支持 [原告/被告] 的诉讼请求。

此致

[法院名称]

代理人（签名）：
[律师事务所/法律援助中心]
日期：    年  月  日
```

### 第五步：刑事辩护词模板

```markdown
# 辩护词

**尊敬的审判长、审判员：**

[法律援助中心] 指派 [律师姓名] 律师担任被告人 [姓名]
涉嫌 [罪名] 一案（案号：[—]）的辩护人。
辩护人经过会见被告人、阅卷和参加庭审，
现发表如下辩护意见：

**一、案件事实方面**

[对案件事实的分析，指出控方证据的不足或事实认定的偏差]

**二、法律适用方面**

[法律适用的分析，包括罪名认定、量刑情节等]

**三、量刑建议**

[如有法定或酌定从轻/减轻情节，提出量刑建议]

辩护人认为，[总结辩护观点]。
恳请合议庭依法作出公正判决。

此致

[法院名称]

辩护人（签名）：
[法律援助中心]
日期：    年  月  日
```

### 第六步：文书质量自查

| 检查项 | 标准 |
|-------|------|
| 当事人信息准确 | 与身份证件/案卷一致 |
| 诉讼请求明确 | 具体、可执行 |
| 事实叙述完整 | 关键事实不遗漏 |
| 法律引用规范 | 遵循引用规范，标注 `[待验证]` |
| 管辖法院正确 | 与配置文件一致 |
| 格式规范 | 符合法院格式要求 |
| 敏感信息处理 | 对外版本脱敏 |

---

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 案件事实复杂 | 建议分段叙述，必要时附事实清单 |
| 法律适用有争议 | 客观呈现不同观点，突出己方论证 |
| 多被告/多原告 | 分别列明各方当事人信息 |
| 反诉 | 使用反诉状格式，注明反诉请求 |
| 管辖权异议 | 另行起草管辖权异议申请书 |

---

## 输出

根据选择的文书类型，生成对应的诉讼文书草稿。
