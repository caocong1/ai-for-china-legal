# 法律援助申请书

> 本子技能可单独运行 `/legal-aid:legal-aid-application-form`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-aid-citations.md`（法律援助引用库）

---

## 目的

生成符合规范的法律援助申请书，包含法定必备要素。

---

## 详细步骤

### 第一步：收集申请书要素

| 要素 | 说明 | 来源 |
|------|------|------|
| 申请人基本信息 | 姓名、性别、出生日期、身份证号、住址、联系方式 | party-identity-verification |
| 申请事项 | 请求法律援助的具体事项 | case-fact-collection |
| 事实与理由 | 案件基本事实和申请法律援助的理由 | case-fact-collection |
| 经济困难说明 | 家庭经济状况说明 | eligibility-preliminary-assessment |
| 附件清单 | 随申请书提交的材料清单 | — |

### 第二步：生成申请书

#### 标准模板

```markdown
# 法律援助申请书

**申请人：** [姓名]，[性别]，[民族]，[出生日期] 出生，住 [地址]，身份证号码：[号码]。
联系电话：[电话]。

**申请事项：**

请求 [法援中心名称] 给予法律援助，指派律师为申请人提供 [代理/辩护] 服务，
帮助申请人 [具体诉求]。

**事实与理由：**

[案件事实经过，简明扼要]

[申请法律援助的理由，包括为什么需要法律援助]

**经济状况说明：**

申请人家庭共 [人数] 口人，家庭月总收入约 ¥[金额] 元。
[持有低保证/特困证/其他经济困难证明]。
家庭经济困难，无力承担律师服务费用。

根据《中华人民共和国法律援助法》相关规定 `[待验证]`，
申请人符合法律援助条件，恳请批准。

**附件：**

1. 身份证复印件
2. 经济困难证明（低保证/特困证/街道办事处证明）
3. [案件相关证据材料]
4. [其他材料]

此致

[法律援助中心名称]

申请人（签名）：
日期：    年  月  日
```

### 第三步：刑事案件专用模板

刑事案件的法律援助申请书需增加以下内容：

```markdown
# 法律援助申请书（刑事辩护）

**申请人/被羁押人：** [姓名]，[性别]，[出生日期] 出生，
身份证号码：[号码]。

**现羁押于：** [看守所名称]

**涉嫌罪名：** [罪名]

**申请人（近亲属）：** [姓名]，与犯罪嫌疑人/被告人的关系：[关系]。
联系电话：[电话]。

**申请事项：**

请求 [法援中心名称] 为 [犯罪嫌疑人/被告人姓名] 指派辩护律师，
提供刑事辩护法律援助服务。

**事实与理由：**

[犯罪嫌疑人/被告人姓名] 因涉嫌 [罪名] 于 [日期] 被 [拘留/逮捕]，
现羁押于 [看守所名称]。

[犯罪嫌疑人/被告人] 因 [经济困难没有委托辩护人 / 属于应当通知法援的情形]，
根据《中华人民共和国刑事诉讼法》相关规定 `[待验证]`，
申请指派辩护律师为其提供法律帮助和辩护。

**经济状况说明（如适用）：**

[家庭经济困难说明]

**附件：**

1. 申请人身份证复印件
2. 与犯罪嫌疑人/被告人的亲属关系证明
3. 拘留通知书/逮捕通知书（复印件）
4. 经济困难证明（如适用）

此致

[法律援助中心名称]

申请人（签名）：
日期：    年  月  日
```

### 第四步：申请书写作要点

#### 4.1 事实与理由写作要求

| 要求 | 说明 |
|------|------|
| 简明扼要 | 控制在 500 字以内，突出关键事实 |
| 时间清晰 | 关键事件的时间节点必须明确 |
| 因果关系 | 说明对方行为与损害后果的因果关系 |
| 法律依据 | 适当引用法律条文，标注 `[待验证]` |
| 诉求合理 | 申请的援助内容应具体明确 |

#### 4.2 经济状况说明写作要点

| 要求 | 说明 |
|------|------|
| 数据准确 | 家庭人口、收入等数据须与证明材料一致 |
| 困难突出 | 说明无力承担法律服务费用的具体原因 |
| 证明关联 | 提及随附的经济困难证明材料 |

#### 4.3 常见案件类型的申请书要点

| 案件类型 | 特别注意事项 |
|---------|------------|
| 劳动争议 | 注明入职时间、工资标准、争议焦点 |
| 赡养/抚养费 | 注明亲属关系、对方经济能力、拖欠金额 |
| 交通事故 | 注明事故时间、责任认定、伤情鉴定 |
| 工伤赔偿 | 注明工伤认定情况、劳动能力鉴定等级 |
| 刑事辩护 | 注明涉嫌罪名、羁押时间、辩护需求 |

### 第五步：申请书自查

| 检查项 | 标准 |
|-------|------|
| 申请人信息完整 | 与身份证件一致 |
| 申请事项明确 | 具体说明请求援助的内容 |
| 事实与理由充分 | 包含基本事实和法律理由 |
| 经济状况说明 | 与实际相符 |
| 附件清单完整 | 必备材料均已列明 |
| 法律引用标注 | 所有法条引用标注 `[待验证]` |
| 格式规范 | 符合法援中心统一格式要求 |
| 签名位置预留 | 申请人签名和日期位置已预留 |

---

## 检查清单

- [ ] 已确认案件类型（民事/刑事/行政）
- [ ] 已收集申请人基本信息
- [ ] 事实与理由已简明扼要撰写
- [ ] 经济状况说明已撰写（或免予核查已标注）
- [ ] 法律条文引用已标注 `[待验证]`
- [ ] 附件清单已列明
- [ ] 格式符合法援中心要求
- [ ] 签名位置已预留

---

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 免予核查经济困难 | 在申请书中注明免予核查情形，省略经济困难说明 |
| 代为申请 | 注明代理人信息（姓名、身份证号、与申请人关系、联系方式） |
| 刑事案件申请 | 使用刑事案件专用模板，增加涉嫌罪名、羁押场所等信息 |
| 多名申请人（群体性案件） | 每人在申请书中列明，推选代表人 |
| 当事人为未成年人 | 法定代理人作为申请人，注明代理关系 |
| 当事人为外国人 | 增加国籍、护照号码、出入境证件信息 |
| 申请材料语言 | 如当事人不通晓中文，需附翻译件 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 当事人信息不完整 | 标注「待补充」，生成申请书草稿，限期补充 |
| 案件事实不清楚 | 标注「事实待核实」，建议先完善 case-fact-collection |
| 经济困难证明缺失 | 在申请书中如实说明，标注「经济困难证明待补充」 |

---

## 输出

生成完整的法律援助申请书文本，供当事人签名提交。

输出格式继承实践配置文件 `## 输出` → `### 工作产物标题` 中的标题规范。
