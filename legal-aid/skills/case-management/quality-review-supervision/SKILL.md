# 质量审查与监督

> 本子技能可被 `/legal-aid:case-management` 编排入口调用，
> 也可单独运行 `/legal-aid:quality-review-supervision`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件，含质量标准）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）
> 前置：读取 `legal-aid/skills/_shared/legal-aid-citations.md`（法律援助引用库）

---

## 目的

1. 对法援案件进行质量审查
2. 评估承办律师的服务质量
3. 监督案件办理规范性
4. 处理质量投诉和问题

---

## 详细步骤

### 第一步：质量标准加载

从实践配置文件加载质量标准：
- 质量标准依据：`[从 legal-aid/CLAUDE.md 读取]`
- 案卷归档要求：`[从 legal-aid/CLAUDE.md 读取]`
- 受援人满意度评估：`[从 legal-aid/CLAUDE.md 读取]`

### 第二步：案件质量评估

#### 2.1 程序规范性评估

| 评估项目 | 评估标准 | 评分 |
|---------|---------|------|
| 及时受理 | 在规定期限内完成受理 | [合格/不合格] |
| 及时指派 | 在规定期限内完成指派 | [合格/不合格] |
| 及时会见/联系 | 刑事案件在 [期限] 内会见 `[待验证]` | [合格/不合格/不适用] |
| 及时阅卷 | 合理时间内完成阅卷 | [合格/不合格/不适用] |
| 按时出庭 | 不缺席、不迟到 | [合格/不合格/不适用] |
| 按时提交材料 | 在法定期限内提交 | [合格/不合格/不适用] |

#### 2.2 实体质量评估

| 评估项目 | 评估标准 | 评分 |
|---------|---------|------|
| 法律分析准确性 | 法律关系判断和适用法条是否正确 | [优秀/良好/一般/差] |
| 证据组织充分性 | 证据收集是否充分、组织是否合理 | [优秀/良好/一般/差] |
| 文书质量 | 起诉状/答辩状/代理词/辩护词质量 | [优秀/良好/一般/差] |
| 出庭表现 | 庭审中是否充分发表意见 | [优秀/良好/一般/差/不适用] |
| 受援人沟通 | 是否及时向受援人通报进展 | [优秀/良好/一般/差] |

#### 2.3 综合质量评分

| 等级 | 评分标准 | 处理方式 |
|------|---------|---------|
| A 优秀 | 全部项目合格，实体质量评分均在良好以上 | 表扬，作为示范案件 |
| B 良好 | 全部项目合格，实体质量评分一般在中等以上 | 正常结案 |
| C 一般 | 存在部分问题但不影响案件结果 | 指出问题，要求整改 |
| D 不合格 | 存在重大问题或程序违规 | 报告负责人，视情况处理 |

### 第四步：受援人满意度评估

#### 4.1 满意度调查

向受援人了解以下情况（可采用回访表或电话回访）：

| 调查项目 | 问题 |
|---------|------|
| 服务态度 | 承办律师态度是否好？ |
| 专业能力 | 承办律师是否专业？ |
| 沟通情况 | 承办律师是否及时向你通报案件进展？ |
| 办案效果 | 你对案件的办理结果是否满意？ |
| 总体评价 | 你对法律援助服务总体是否满意？ |

#### 4.2 满意度评分

| 评分 | 含义 |
|------|------|
| 非常满意 | 所有项目满意 |
| 满意 | 大部分项目满意 |
| 一般 | 部分项目有改进空间 |
| 不满意 | 存在明显问题 |
| 非常不满意 | 存在严重问题 |

### 第五步：问题处理

#### 5.1 质量问题处理

| 问题级别 | 处理方式 |
|---------|---------|
| 轻微问题 | 口头提醒承办律师改进 |
| 一般问题 | 书面通知，限期整改 |
| 严重问题 | 报告负责人，视情况更换承办人 |
| 违规问题 | 报告司法行政机关处理 |

#### 5.2 投诉处理

收到受援人投诉时：
1. 记录投诉内容和时间
2. 调查核实
3. 提出处理意见
4. 向投诉人反馈处理结果

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 案件未结案 | 可进行过程质量检查，不做最终评分 |
| 承办律师不配合检查 | 报告主任，作为质量问题记录 |
| 受援人拒绝配合满意度调查 | 标注「未获得受援人评价」，其他项目正常评估 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 发现承办律师违规行为 | 立即报告负责人和司法行政机关 |
| 案卷材料严重缺失 | 要求承办律师限期补正 |
| 受援人提出严重投诉 | 启动投诉调查程序 |
