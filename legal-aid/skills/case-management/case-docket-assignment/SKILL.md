# 案件登记与指派

> 本子技能可被 `/legal-aid:case-management` 编排入口调用，
> 也可单独运行 `/legal-aid:case-docket-assignment`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

完成法援案件的正式登记和承办律师指派：

1. 生成案件编号，录入案件信息
2. 审查案件类型和紧急程度
3. 根据案件特点指派承办律师
4. 建立案件档案

---

## 详细步骤

### 第一步：案件登记

#### 1.1 基本信息录入

| 登记项 | 说明 |
|-------|------|
| 案件编号 | 按本机构编号规则自动生成 |
| 受理日期 | 法援机构收到完整申请之日 |
| 案件类型 | 民事/刑事/行政 |
| 案由 | 具体案由（劳动争议/赡养纠纷/故意伤害辩护等） |
| 受援人信息 | 从 intake 结果导入 |
| 对方当事人信息 | 姓名/名称、联系方式 |
| 受理机构 | 本法律援助中心 |
| 承办律师 | 待指派 |

#### 1.2 紧急程度标记

| 紧急级别 | 适用情形 | 处理要求 |
|---------|---------|---------|
| 🔴 紧急 | 时效即将届满（7日内）/ 羁押期限将满 / 人身安全受威胁 | 24小时内指派 |
| 🟠 加急 | 时效较紧（30日内）/ 受援人被羁押 | 3个工作日内指派 |
| 🟡 一般 | 常规案件 | 正常排队指派 |
| 🟢 不急 | 无时效压力 | 按序处理 |

#### 1.3 重大案件标记

检查是否属于重大/群体性案件：
- 群体性案件（10人以上）→ 标记并升级
- 社会影响较大 → 标记并报告
- 涉访涉诉 → 标记并报告
- 跨地区案件 → 标记并协调

### 第二步：承办律师指派

#### 2.1 指派原则

| 原则 | 说明 |
|------|------|
| 专业对口 | 根据律师专业特长分配对应类型案件 |
| 工作量均衡 | 考虑律师当前承办案件数量 |
| 回避审查 | 确认律师与案件无利益冲突 |
| 能力匹配 | 复杂/重大案件指派经验丰富的律师 |
| 地域便利 | 考虑案件管辖地与律师办公地点的距离 |

#### 2.2 指派流程

1. **筛选候选律师**：按专业领域和工作量筛选
2. **回避审查**：确认候选律师无利益冲突
3. **沟通确认**：与候选律师沟通，确认可以承办
4. **正式指派**：出具指派通知书
5. **通知受援人**：告知受援人承办律师信息

#### 2.3 特殊案件指派

| 特殊情形 | 指派规则 |
|---------|---------|
| 刑事辩护 | 须指派有刑事辩护经验的律师 |
| 未成年人案件 | 须指派熟悉未成年人保护的律师 |
| 重大复杂案件 | 经集体讨论后指派，可安排两名律师 |
| 群体性案件 | 经主任审批后指派，可组建承办小组 |

### 第三步：建立案件档案

#### 3.1 初始档案目录

| 序号 | 材料名称 | 状态 |
|------|---------|------|
| 1 | 法律援助申请表 | [已收/待补] |
| 2 | 受援人身份证明 | [已收/待补] |
| 3 | 经济困难证明 | [已收/待补/免予核查] |
| 4 | 案件相关材料 | [已收/待补] |
| 5 | 指派通知书 | [待出具] |
| 6 | 委托代理协议/指定辩护函 | [待签署] |

#### 3.2 档案编号规则

按本机构档案编号规则建立：
- 年度编号
- 案件类型编号
- 顺序号

### 第四步：通知与确认

1. **通知承办律师**：发送指派通知，附案件材料
2. **通知受援人**：告知承办律师姓名、联系方式
3. **系统更新**：在管理系统中更新案件状态为「已指派」

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 无法找到合适承办律师 | 报告主任协调；必要时转介其他法援机构 |
| 受援人要求更换律师 | 记录原因，评估合理性，按程序决定 |
| 案件信息不完整 | 标注缺失项，限期补充，暂不正式指派 |
| 利益冲突无法回避 | 转介至其他有管辖权的法援机构 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 案件编号重复 | 重新生成，标注原编号作废 |
| 指派律师无法联系 | 顺延指派下一候选律师 |
| 受援人拒收通知 | 记录拒收情况，报告负责人 |
