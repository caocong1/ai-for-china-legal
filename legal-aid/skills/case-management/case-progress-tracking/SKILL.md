# 案件进展跟踪

> 本子技能可被 `/legal-aid:case-management` 编排入口调用，
> 也可单独运行 `/legal-aid:case-progress-tracking`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）

---

## 目的

1. 跟踪法援案件各阶段进展
2. 管理关键时间节点和超期预警
3. 记录案件变更情况
4. 定期向受援人通报进展

---

## 详细步骤

### 第一步：案件阶段识别

根据案件类型识别当前所处阶段：

#### 民事/行政案件阶段

| 阶段 | 关键节点 | 预计时间 |
|------|---------|---------|
| 受理阶段 | 立案/提交申请 | 即时 |
| 审理阶段 | 举证期限、开庭日期、调解 | 按法院排期 |
| 裁判阶段 | 收到判决/裁定 | 按审理期限 |
| 执行阶段 | 申请执行、执行进展 | 6个月内（一般）`[待验证]` |
| 上诉阶段 | 上诉期限（判决15日/裁定10日） | `[待验证]` |

#### 刑事案件阶段

| 阶段 | 关键节点 | 预计时间 |
|------|---------|---------|
| 侦查阶段 | 拘留期限、提请批捕、批捕 | 按刑诉法规定 `[待验证]` |
| 审查起诉阶段 | 移送审查起诉、起诉期限 | 一般1个月+15日 `[待验证]` |
| 审判阶段 | 开庭、判决 | 按审限规定 `[待验证]` |
| 执行阶段 | 交付执行 | — |

### 第二步：进展记录

每次案件推进时记录：

| 记录项 | 说明 |
|-------|------|
| 日期 | 事件发生日期 |
| 事件类型 | 会见/阅卷/开庭/调解/提交材料/收到文书等 |
| 事件描述 | 具体经过 |
| 参与人 | 承办律师/受援人/对方当事人/法官等 |
| 结果 | 本次进展的结果 |
| 下一步 | 后续工作安排 |
| 下次节点 | 下一个关键时间节点 |

### 第三步：超期预警

#### 3.1 预警级别

| 预警级别 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 🟢 正常 | 距期限届满 >30日 | 正常推进 |
| 🟡 关注 | 距期限届满 15-30日 | 提醒承办律师加快进度 |
| 🟠 警告 | 距期限届满 7-15日 | 报告负责人，督促办理 |
| 🔴 紧急 | 距期限届满 ≤7日 | 立即升级，采取紧急措施 |

#### 3.2 预警处理流程

1. 系统自动检查所有在办案件的关键节点
2. 触发预警的案件生成预警通知
3. 通知承办律师和负责人
4. 记录预警处理结果

### 第四步：案件变更管理

#### 4.1 承办律师变更

| 变更原因 | 处理流程 |
|---------|---------|
| 律师辞职/调离 | 重新指派，通知受援人 |
| 受援人申请更换 | 评估合理性，决定后重新指派 |
| 律师回避 | 回避审查后重新指派 |
| 案件移交 | 办理移交手续，通知受援人 |

#### 4.2 受援人变更

| 变更类型 | 处理方式 |
|---------|---------|
| 受援人撤回申请 | 记录原因，办理撤回手续，终止援助 |
| 受援人经济状况改善 | 评估是否继续符合法援条件 |
| 受援人死亡/失踪 | 报告负责人，决定是否终止或继续 |

#### 4.3 案件终止

| 终止原因 | 法律依据 | 处理方式 |
|---------|---------|---------|
| 受援人不再符合法援条件 | 经济状况变化等 `[待验证]` | 评估后决定是否终止 |
| 案件已结案 | 判决生效/执行完毕 | 正常结案归档 |
| 受援人拒绝继续援助 | 受援人自愿放弃 `[待验证]` | 书面确认，办理终止 |

### 第五步：定期通报

按规定向受援人通报案件进展：

- **常规通报**：每次重要进展后通报
- **定期通报**：长期未结案件，每月至少通报一次
- **通报方式**：电话/面谈/短信（根据受援人偏好）

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 案件长期停滞 | 标注「案件停滞」，报告负责人，分析原因 |
| 法院延期开庭 | 记录延期原因，更新下次开庭日期 |
| 受援人联系不上 | 记录联系尝试，标注「待联系」，报告负责人 |
| 案件涉及多程序 | 分别跟踪各程序进展，标注程序间关联 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 案件已超期但未预警 | 立即标记为「超期案件」，升级处理 |
| 承办律师不配合 | 报告主任，评估是否更换承办人 |
| 关键日期信息错误 | 更正记录，评估是否影响案件 |
