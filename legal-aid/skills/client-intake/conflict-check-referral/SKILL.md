# 利益冲突审查与案件分流

> 本子技能可被 `/legal-aid:client-intake` 编排入口按顺序调用，
> 也可单独运行 `/legal-aid:conflict-check-referral`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

1. 审查是否存在利益冲突
2. 对案件进行分流处理
3. 汇总完整 intake 结果
4. 引导当事人进入下一步流程

---

## 详细步骤

### 第一步：利益冲突审查

#### 1.1 利益冲突识别

检查以下利益冲突情形：

| 冲突类型 | 说明 | 处理方式 |
|---------|------|---------|
| 同一案件代理双方 | 本机构已为对方当事人提供法援 | 🔴 不可受理 |
| 律师个人利益冲突 | 承办律师与对方当事人有利害关系 | 🔴 该律师回避 |
| 前当事人冲突 | 新案件与已承办的旧案件存在利益冲突 | 🟠 需评估后决定 |
| 关联案件冲突 | 系列案件中存在利益冲突 | 🟠 需评估后决定 |
| 内部人员关联 | 法援中心工作人员与案件有关联 | 🟠 该人员回避 |

#### 1.2 冲突审查流程

**询问当事人**：

> 为了保障你的合法权益，我需要了解几个情况：
> 1. 你是否在我们法援中心或其他律所咨询过这个案件？
> 2. 对方当事人是否有律师？你认识对方的律师吗？
> 3. 你是否认识我们法援中心的工作人员？

**内部核查**：
- 查询本机构已受理案件数据库
- 核查是否有承办律师与对方当事人存在关联
- 核查系列案件是否已有代理安排

#### 1.3 冲突处理决定

| 审查结果 | 处理 |
|---------|------|
| 未发现冲突 | ✅ 继续分流程序 |
| 发现可回避的冲突 | 🟠 指定其他承办人/回避处理 |
| 发现不可回避的冲突 | 🔴 不予本机构受理，转介其他法援机构 |

### 第二步：案件分流

#### 2.1 分流决策树

```
是否符合法援条件？
├─ 是 → 是否属于本机构管辖范围？
│      ├─ 是 → 是否属于本机构受理偏好？
│      │      ├─ 是 → ✅ 受理，进入申请流程
│      │      └─ 否 → 转介其他机构/告知其他途径
│      └─ 否 → 转介有管辖权的法援机构
├─ 不确定 → 建议正式申请，由法援机构审查
└─ 否 → 告知其他解决途径
```

#### 2.2 分流类型

| 分流类型 | 适用情形 | 后续处理 |
|---------|---------|---------|
| **受理** | 符合法援条件 + 本机构管辖 + 无冲突 | 引导进入 legal-aid-application 流程 |
| **转介本系统** | 符合法援条件但不属于本机构管辖 | 转介至有管辖权的法援机构 |
| **转介外部** | 不完全符合法援条件但有其他援助渠道 | 推荐其他资源 |
| **不予受理** | 明确不符合法援条件且无其他援助渠道 | 书面告知并说明理由 |
| **需进一步审查** | 信息不完整或存在疑点 | 补充材料后再决定 |

#### 2.3 转介资源清单

根据实践配置文件中的本地资源信息，为当事人提供转介建议：

| 转介渠道 | 适用情形 | 联系方式来源 |
|---------|---------|------------|
| 其他法律援助中心 | 管辖权不在本机构 | `legal-aid/CLAUDE.md` → 本地资源 |
| 律师协会公益服务 | 不符合法援条件但有困难 | 当地律协联系方式 |
| 高校法律诊所 | 一般性法律咨询 | 当地高校法学院 |
| 人民调解委员会 | 适合调解的纠纷 | 当地司法局 |
| 妇联/残联/工会 | 特定群体维权 | 对应组织联系方式 |
| 12348 法律服务热线 | 一般法律咨询 | 全国统一热线 |
| 社会救助机构 | 需要社会救助 | 当地民政部门 |

根据分流结论，引导当事人：

> 经初步评估，你的案件基本符合法律援助条件。接下来我们将协助你正式申请法律援助。

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 利益冲突难以判断 | 标注「利益冲突需进一步审查」，报告负责人决定 |
| 当事人对分流决定不满 | 耐心解释理由，提供书面说明，告知申诉渠道 |
| 多个当事人涉及同一纠纷 | 分别评估冲突，注意系列案件的冲突处理 |
| 群体性案件线索 | **立即升级**，报告法援中心主任，按群体性案件处理规则办理 |
| 紧急案件（时效即将届满） | 优先处理申请流程，简化 intake 步骤 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 冲突审查数据库不完整 | 标注「冲突审查结果可能有遗漏」，人工核查 |
| 转介机构信息已过期 | 标注 `[时效性 flagged — 需验证联系方式]`，建议查询最新信息 |
| 当事人拒绝接受分流决定 | 记录当事人意见，书面告知决定和理由，告知申诉渠道 |
