# 当事人身份与基本信息核实

> 本子技能可被 `/legal-aid:client-intake` 编排入口按顺序调用，
> 也可单独运行 `/legal-aid:party-identity-verification`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

收集并核实法援案件当事人的身份信息和基本背景，建立当事人档案：

1. 确认当事人主体资格
2. 获取有效联系方式
3. 收集经济状况基本信息（为法援资格评估做准备）
4. 识别特殊群体需求

---

## 详细步骤

### 第一步：基本身份信息

收集以下信息：

| 信息项 | 说明 | 必填 |
|-------|------|------|
| 姓名 | 与身份证件一致 | 是 |
| 性别 | — | 是 |
| 出生日期 | 用于判断是否未成年/老年 | 是 |
| 身份证号码 | 核实身份 | 是 |
| 民族 | — | 是 |
| 户籍所在地 | 影响管辖和法援机构选择 | 是 |
| 经常居住地 | 若与户籍不同 | 否 |
| 联系电话 | 主要联系方式 | 是 |
| 备选联系方式 | 备用联系人 | 否 |

**核实方式**：
- 要求出示身份证原件，核对信息
- 若未携带身份证，记录情况，后续补充
- 核对当事人自述与证件信息是否一致

### 第二步：经济状况基本信息

以下信息用于初步判断是否可能符合经济困难标准（详细评估在 eligibility-preliminary-assessment 子技能中完成）：

| 信息项 | 说明 | 必填 |
|-------|------|------|
| 家庭人口数 | 共同生活的家庭成员人数 | 是 |
| 主要收入来源 | 工资/低保/养老金/其他 | 是 |
| 月收入（大约） | 家庭月总收入 | 是 |
| 是否有低保证 | 最低生活保障证 | 是 |
| 是否有特困证明 | 特困人员救助供养证 | 否 |
| 是否有其他救助 | 残疾补贴/优抚金等 | 否 |

**特殊情形处理**：
- 流浪乞讨人员：记录救助站信息
- 无法提供收入证明：记录原因，后续协助开具
- 当事人不愿透露经济信息：解释法援需审查经济状况，鼓励配合

### 第三步：特殊群体识别

检查以下特殊身份，标记适用的特殊保护规则：

| 特殊群体 | 识别标志 | 特殊规则 |
|---------|---------|---------|
| 未成年人 | 未满18周岁 | 须法定代理人参与；刑事法援强制辩护 |
| 残疾人 | 残疾证 | 提供无障碍沟通；部分情形免予经济审查 |
| 老年人 | 65周岁以上（各地标准不同） | 适当简化程序；部分地区有绿色通道 |
| 农民工 | 务工身份证明 | 请求支付劳动报酬/工伤赔偿可免予经济审查 `[待验证]` |
| 军人/军属 | 军官证/军属证明 | 部分地区有军人法援绿色通道 `[待验证]` |
| 外国人/无国籍人 | 护照/出入境证件 | 按对等原则或国际条约处理 `[待验证]` |

### 第四步：联系方式确认

确认以下沟通安排：

- **首选沟通方式**：电话 / 面谈 / 短信 / 微信
- **最佳联系时间**：工作日白天 / 晚上 / 周末
- **是否需要翻译**：不通晓普通话 / 听障 / 语障
- **紧急联系人**：姓名、关系、联系方式

---

## 检查清单

- [ ] 姓名、身份证号等基本信息已核实（与身份证件核对）
- [ ] 联系电话已确认有效
- [ ] 经济状况基本信息已收集（收入、人口、低保证等）
- [ ] 特殊群体已识别并标记
- [ ] 沟通方式和紧急联系人已确认
- [ ] 若未携带证件，已告知需后续补充

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 当事人未携带身份证件 | 记录基本信息，标注「待补充身份证件」，不阻断后续流程 |
| 当事人无法清楚表达 | 放慢节奏；必要时请陪同人员协助；标记「沟通困难」 |
| 当事人拒绝提供经济信息 | 解释法律援助需审查经济状况；若仍拒绝，标注「经济信息缺失，无法进行法援资格评估」 |
| 身份信息与证件不符 | 标注「身份信息存疑」，请当事人解释；必要时暂停 intake |
| 未成年人单独来访 | 联系法定代理人到场；若涉及家暴等紧急情形，先保障安全再补办手续 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 当事人提供虚假身份信息 | 标注「身份信息疑似虚假」，向负责人报告，暂停 intake |
| 信息不完整 | 列出缺失项，告知当事人在 [期限] 内补充，暂存档案 |
| 当事人中途离开 | 保存已收集信息，标注「intake 中断」，后续跟进 |
