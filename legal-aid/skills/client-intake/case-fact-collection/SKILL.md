# 案件事实初步收集

> 本子技能可被 `/legal-aid:client-intake` 编排入口按顺序调用，
> 也可单独运行 `/legal-aid:case-fact-collection`。
>
> 前置：读取 `legal-aid/CLAUDE.md`（实践配置文件）
> 前置：读取 `legal-aid/skills/_shared/legal-basis-conventions.md`（引用规范）

---

## 目的

通过结构化访谈，全面了解案件事实经过，收集证据线索，初步判断法律关系性质，
为法援资格评估和后续案件承办提供事实基础。

---

## 详细步骤

### 第一步：案件类型定位

根据当事人陈述，初步判断案件属于以下哪类：

| 案件大类 | 常见子类 |
|---------|---------|
| 民事案件 | 劳动争议、赡养/抚养/扶养、交通事故人身损害、工伤赔偿、医疗事故、国家赔偿 |
| 刑事案件 | 犯罪嫌疑人/被告人辩护、被害人代理、自诉案件代理 |
| 行政案件 | 行政复议、行政诉讼、行政赔偿 |

**追问逻辑**：
- 案件类型不明显时，先让当事人自由陈述，再归纳
- 当事人可能混淆法律概念，需要用通俗语言确认
- 涉及多类法律关系的，分别记录，后续确认以哪个为主

### 第二步：事实经过收集

**开放式提问**（让当事人自由陈述）：

> 请你从头到尾把事情经过讲一遍，什么时候发生的，发生了什么事情，现在怎么样了。

**记录要点**：
- 时间线（按时间顺序记录关键事件）
- 涉及主体（各方当事人、关系人）
- 关键行为（合同签订、侵权行为、行政决定等）
- 损害后果（经济损失、人身损害、精神损害等）
- 已采取的行动（协商、投诉、报警、起诉等）
- 目前状态（协商中、已起诉、执行中等）

**结构化追问**：

针对不同类型的案件，追问特定要素：

| 案件类型 | 关键追问要素 |
|---------|------------|
| 劳动争议 | 入职时间、合同情况、工资标准、争议起因、解除时间/原因 |
| 赡养/抚养费 | 亲属关系、对方经济能力、已支付的金额、拖欠期间 |
| 交通事故 | 事故时间/地点、责任认定、伤情鉴定、保险情况、医疗费金额 |
| 工伤赔偿 | 工作时间/地点、事故经过、工伤认定情况、劳动能力鉴定 |
| 刑事辩护 | 涉嫌罪名、被拘留/逮捕时间、是否收到起诉书、有无同案犯 |

### 第三步：证据线索收集

了解当事人目前掌握的证据和可能的证据来源：

| 证据类型 | 追问方式 |
|---------|---------|
| 书证 | 「你手头有什么文件、合同、收据、通知？」 |
| 物证 | 「有没有相关的实物证据？」 |
| 证人 | 「还有谁知道这件事？能联系到他们吗？」 |
| 视听资料 | 「有没有录音、录像、照片、微信聊天记录？」 |
| 电子数据 | 「手机里有没有相关记录？」 |
| 鉴定意见 | 「做过伤残鉴定/工伤鉴定/医疗事故鉴定吗？」 |
| 官方文件 | 「有没有收到法院/仲裁委/行政机关的文件？」 |

**证据现状评估**：
- 证据充分性初步判断
- 缺失的关键证据标注
- 需要调取的证据线索（如：需要去哪个部门调取档案）

### 第四步：法律关系初步判断

基于收集的事实，初步判断：

1. **法律关系性质**：劳动/合同/侵权/婚姻家事/行政等
2. **对方当事人信息**：姓名/名称、与当事人的关系
3. **诉讼请求方向**：当事人希望通过法律途径达到什么目的
4. **时效评估**：
   - 是否可能超过诉讼时效（一般民事3年 `[待验证]`）
   - 劳动仲裁时效（1年 `[待验证]`）
   - 上诉期限（判决15日/裁定10日 `[待验证]`）

**时效紧急标记**：
若发现可能超过时效，立即标记并优先处理：
> ⚠️ **时效警告**：该案件可能存在超过 [诉讼时效/仲裁时效/上诉期限] 的风险。
> 建议立即采取 [起诉/申请仲裁/上诉] 行动，或确认是否存在时效中断/中止事由。

### 第五步：事实摘要确认

将收集的事实整理为摘要，向当事人确认：

> 我整理一下你刚才说的情况：[事实摘要]。你看是否准确？有没有需要补充或纠正的？

**确认记录**：
- 当事人确认事实摘要准确
- 当事人补充或纠正的内容
- 当事人签字确认（若条件允许）

## 边界条件

| 场景 | 处理方式 |
|-----|---------|
| 当事人叙述混乱 | 用时间线引导：「我们先从最早发生的事情开始说起？」 |
| 当事人情绪激动 | 给予充分时间；必要时暂停；不催促 |
| 涉及多个法律关系 | 分别记录；帮助当事人理清主次；后续确认以哪个为主申请法援 |
| 当事人表述与法律判断不一致 | 不纠正当事人的事实描述，但在法律关系判断中独立分析 |
| 关键事实当事人不清楚 | 标注「待核实」，告知后续可能需要补充调查 |

---

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 当事人前后陈述矛盾 | 指出矛盾点，请当事人澄清，记录最终版本 |
| 发现可能虚假陈述 | 标注「陈述一致性存疑」，不拒绝但报告负责人 |
| 事实涉及犯罪行为 | 如当事人是被害人，正常收集；如当事人可能涉及犯罪，注意保密义务边界 |
