**第585条/第588条审查要点（违约金/定金四维分析）：**

> **重要条款结构说明（`[待验证]`）：**
> - **第585条第1款** = 可约定违约金，也可约定损失赔偿额的计算方法
> - **第585条第2款** = 违约金调整规则（低于损失可请求增加；过分高于损失可请求适当减少）
> - **第585条第3款** = 迟延履行约定违约金的，支付违约金后还应继续履行债务
> - **第588条** = 违约金与定金竞合规则（择一适用、不得并用）
>
> 违约金与定金的「不可并用/选择适用」规则出自**第588条**，而非第585条第2款。

- 直接损失 vs 间接损失适用范围
- 基数定义（合同总价/已支付费用/月费等）
- 例外事项（数据泄露/IP/保密是否在限制之外）
- 与审查立场的对应
- 若同时约定定金，适用第588条择一选择规则

## 四、引用库使用说明

### 使用方式

技能引用本库时，在「前置」中声明：
```
前置：读取 commercial-legal/skills/_shared/civil-code-contract-citations.md（民法典合同编引用库）
```

然后在「法律依据」章节按法条/案例/学说三类组织，从本库提取相关条文，并保留来源标签和待验证标注。

### 更新规则

1. **法条更新**：法规修订时，更新对应版本年份，并在旧版本标注「已修订，见[新条文号]」
2. **案例填充**：经检索核验的案例，填入对应占位位置，去除 `[待验证]` 标注（须注明核验工具和日期）
3. **学说填充**：引用时须有完整出处，去除 `[待验证]` 须经律师确认

### 与法条的关系

本库是**引用工具**，不存放完整法条正文。完整法条正文请通过：
- 国家法律法规数据库：https://flk.npc.gov.cn
- 北大法宝（如已接入）
