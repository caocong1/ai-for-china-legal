## 步骤 0：定位违约责任条款

在进入子技能前，快速阅读合同，定位所有涉及违约责任的条款：

| 条款类型 | 常见位置 | 定位状态 |
|---------|---------|---------|
| 责任限制条款（cap） | 通常在合同后半部分「责任限制」章节 | ☐ 已定位 / ☐ 未找到 |
| 违约金条款 | 通常在「违约责任」章节 | ☐ 已定位 / ☐ 未找到 |
| 赔偿条款 | 通常在「赔偿/indemnification」章节 | ☐ 已定位 / ☐ 未找到 |
| 免责条款 | 通常在「不可抗力/责任限制」章节 | ☐ 已定位 / ☐ 未找到 |
| 例外清单（carveouts） | 通常在责任限制条款末尾「但以下情形除外」 | ☐ 已定位 / ☐ 未找到 |

## 子技能编排顺序

下列四个子技能按顺序执行，构成完整违约责任分析工作流。
**每个子技能均可单独触发**（通过 `/commercial-legal:<子技能名>` 或直接引用路径）。

| 步骤 | 子技能 | 路径 | 可单独触发 | 说明 |
|-----|-------|-----|---------|-----|
| 1 | `liability-cap-analysis` | `skills/liability-analysis/liability-cap-analysis/SKILL.md` | ✓ | 四维深度分析（直接/间接损失、基数定义、例外交互、审查立场对照）；基数不明确时给出标准错误处理 |
| 2 | `penalty-assessment` | `skills/liability-analysis/penalty-assessment/SKILL.md` | ✓ | 违约金合法性（民法典585条）、过高/过低调整风险、定金并用规则、商业影响评估 |
| 3 | `damages-scope` | `skills/liability-analysis/damages-scope/SKILL.md` | ✓ | 赔偿范围界定（民法典584条可预见性规则）、律师费约定、减损义务、赔偿程序 |
| 4 | `carveout-review` | `skills/liability-analysis/carveout-review/SKILL.md` | ✓ | 例外清单全面审查（完整性/过宽过窄/争议定义）；法律强制例外核查（第506条） |

## 编排调用方式

### 完整流程（从头到尾）

触发条件：用户提供合同/条款请求全面违约责任分析。

执行顺序：
```
定位违约责任条款 → liability-cap-analysis → penalty-assessment → damages-scope → carveout-review → 风险自查输出
```

### 单一子技能触发

用户可随时直接触发某一子技能，例如：
- `/commercial-legal:liability-cap-analysis` — 仅分析责任限制上限
- `/commercial-legal:penalty-assessment` — 仅评估违约金合法性
- `/commercial-legal:damages-scope` — 仅界定赔偿范围
- `/commercial-legal:carveout-review` — 仅审查例外清单

### 风险自查清单

- [ ] 是否分析了所有违约责任条款？（责任限制/违约金/赔偿范围/例外清单）

- [ ] 是否评估了违约金的合法性（第585条）和商业影响？
- [ ] 是否界定了赔偿范围（第584条可预见性规则）？
- [ ] 是否审查了例外清单完整性与定义争议？
