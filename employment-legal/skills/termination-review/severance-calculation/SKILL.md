# 经济补偿与赔偿金计算（severance-calculation）

> **前置**：
> 1. 读取 `employment-legal/CLAUDE.md`（实践配置文件），加载 `## 审查立场 > 解雇审查` —
>    `#### 经济补偿`（计算标准倾向 N/N+1；高于法定标准倾向）。
>    若含占位符，进入**临时模式**（见下方）。
> 2. 读取 `employment-legal/skills/_shared/legal-basis-conventions.md`（引用规范）。
> 3. 读取 `employment-legal/skills/_shared/labor-law-citations.md`（引用库，
>    重点：§A.10 经济补偿N第47条、§A.11 违法解除2N第87条、§A.8 解除事由、§A.9 经济补偿情形）。
> 4. 分工说明：本子技能专注「给多少」（经济补偿/代通知金/赔偿金计算）；
>    `grounds-legality` 负责「能不能解」（事由合法性）；
>    `termination-procedure` 负责「怎么解」（程序合规）；
>    `unlawful-termination-remedy` 负责「解错了如何救济」（救济路径与预防）。
>    本子技能不评估解除合法性，仅在合法/违法两条路径下分别计算金额。
> 5. 本子技能可被 `employment-legal/skills/termination-review/SKILL.md` 编排入口调用，
>    也可通过 `/employment-legal:severance-calculation` 单独触发。

---

**临时模式**：若实践配置文件经济补偿立场含占位符，使用以下默认值并标注 `[临时模式]`：

- 按法定标准 N 计算经济补偿，不高于也不低于法定标准 `[临时模式]`
- 非过失性辞退未提前30天通知时，额外支付一个月代通知金（N+1情形）`[临时模式]`
- 违法解除按第87条支付2N，不与N并用 `[临时模式]`

---

## 目的

精确计算劳动合同解除时劳动者应获得的经济补偿、代通知金或违法解除赔偿金，
清晰区分 N / N+1 / 2N 三种情形，提供计算工作表和金额汇总。

---

## 详细步骤

### 步骤 1：确定适用情形（N / N+1 / 2N 路径）

首先确认解除方式和合法性，进入对应计算路径：

| 情形 | 触发条件 | 计算公式 |
|------|---------|---------|
| **N（经济补偿）** | 合法解除且须支付经济补偿（协商由用人单位提出/非过失/裁员/合同到期维持条件不续等，约第46条 `[待验证]`） | N = 工龄年限 × 月工资基数（封顶规则见步骤3） |
| **N+1（经济补偿+代通知金）** | 非过失性辞退但**未提前30天书面通知**，以额外一个月工资代替提前通知 | N+1 = 经济补偿N + 1个月代通知金（两者性质不同，代通知金不计入N基数） |
| **2N（违法解除赔偿金）** | 用人单位违法解除或违法终止劳动合同（第87条**已核实锚点**） | 2N = N × 2（按第47条经济补偿标准二倍支付，与N互斥不并用） |
| **零（无需补偿）** | 过失性辞退（约第39条 `[待验证]`）且劳动者无过错申索 | 0 |
| **劳动者主动辞职** | 劳动者无故辞职（非因用人单位过错） | 0（但劳动者依约第38条因用人单位过错解除，须支付N `[待验证]`） |

> **核心区分**：
> - **N** 是合法解除的经济补偿，来源：第47条（**已核实锚点**）
> - **+1（代通知金）** 是替代提前30天通知的对价，性质为工资，不是经济补偿
> - **2N** 是违法解除的赔偿金，来源：第87条（**已核实锚点**），与N **互斥不并用**

### 步骤 2：确定月工资基数

月工资基数是计算经济补偿/赔偿金的关键变量：

**法定定义**（基于第47条，**已核实锚点**）：

| 定义项 | 标准 |
|-------|------|
| 计算区间 | 劳动合同解除或终止前**12个月**的平均工资 |
| 包含项目 | **应得**工资（含基本工资、绩效奖金、津贴、补贴等，解除前12个月实际发放的各项工资性收入） |
| 扣减后税前 | 以税前应得工资计算（非税后实得），含个人所得税部分 |
| 最低保障 | 劳动者月工资低于用人单位所在地最低工资标准的，按最低工资标准计算 |

**高收入封顶规则**（第47条**已核实锚点**）：

若劳动者月工资**高于用人单位所在地上年度职工月平均工资三倍**：
- **月工资基数封顶**：按「三倍」平均工资计算，超出部分不计
- **工龄年限封顶**：最高支付12年经济补偿（不论实际工龄多长）

> 示例：上年度当地职工月平均工资为 ¥8,000，三倍为 ¥24,000。
> 若劳动者月工资为 ¥30,000，基数封顶为 ¥24,000；若工龄15年，年限封顶12年。
> 经济补偿 = ¥24,000 × 12年 = ¥288,000

**争议性项目（须确认是否纳入基数）**：

| 项目 | 一般处理 | 风险提示 |
|------|---------|---------|
| 年终奖 | 月均摊入12个月 `[待验证]` | 各地仲裁口径不一，争议多 |
| 股权/期权收益 | 通常不纳入工资基数 `[待验证]` | 须核查合同约定 |
| 报销款项/差旅补贴 | 通常不纳入（非劳动报酬性质） | — |
| 住房补贴 | 纳入（属工资性质） `[待验证]` | — |

### 步骤 3：计算经济补偿 N（第47条，已核实锚点）

**工龄年限认定规则**（第47条**已核实锚点**）：

| 工龄段 | 计算方式 |
|--------|---------|
| 满1年 | 计1个月工资 |
| 满6个月不满1年 | 按1年计算，即1个月工资 |
| 不满6个月 | 支付半个月工资（0.5个月） |

> **工龄起算**：从劳动者入职本单位当日起算至劳动合同解除或终止日。
> 多次签订合同（固定期限续签）的，连续工龄累计计算。
> **历史问题（须提示）**：2008年1月1日（《劳动合同法》施行日）前后的工龄是否分段计算，
> 各地实践存在差异 `[待验证]`，建议核查当地最新仲裁口径。

**计算工作表**：

```
步骤 A：确认月工资基数
  解除前12个月应得工资总额：¥[  ]
  ÷ 12 = 月均工资基数：¥[  ]
  ----------------------------------------
  上年度当地职工月平均工资三倍：¥[  ]（当地标准 ¥[  ] × 3）
  月工资基数是否超三倍？[是/否]
  → 若是：月工资基数封顶 = ¥[  ]（三倍值）

步骤 B：确认工龄年限
  入职日期：[    ]
  解除日期：[    ]
  工龄：[X]年[Y]个月
  ----------------------------------------
  年限换算（按第47条）：
    满整年数：[  ]年 → [  ]个月工资
    余月处理：
      ≥6个月 → 按1年算，+1个月工资
      <6个月 → +0.5个月工资

  → 若是：年限封顶 = 12年

步骤 C：计算经济补偿 N
  N = 月工资基数（封顶后）× 工龄年限（封顶后）
  N = ¥[  ] × [  ]年 = ¥[  ]
```

### 步骤 4：代通知金计算（N+1 中的「+1」）

代通知金适用于：**非过失性辞退且未提前30天书面通知**。

| 项目 | 说明 |
|------|------|
| 金额 | 一个月工资（按劳动者月工资计算，非按三倍封顶基数） |
| 性质 | 替代提前30天通知义务的对价，属工资性质，**非经济补偿** |
| 与N的关系 | 二者并列，N+1 = 经济补偿N + 一个月代通知金 |
| 是否计入N基数 | **不计入**：代通知金不影响N的月工资基数计算 |

> **N+1 总计**：
> - N = 经济补偿（月工资基数 × 工龄年限）
> - +1 = 一个月工资（代通知金）
> - **合计 = N + 1个月工资**

### 步骤 5：违法解除赔偿金 2N（第87条，已核实锚点）

**触发条件**：用人单位违法解除或违法终止劳动合同（第87条**已核实锚点**）。

**计算规则**（第87条**已核实锚点**）：

> 按照第47条规定的经济补偿标准的**二倍**向劳动者支付赔偿金。

**选择权归属**（劳动者方主张时）：

| 选项 | 条件 | 结果 |
|------|------|------|
| 恢复劳动关系 | 劳动者要求继续履行且合同能够继续履行 | 继续履行 + 支付解除至恢复期间工资（不支付2N） |
| 支付2N赔偿金 | 劳动者不要求恢复，或合同已不能继续履行 | 支付2N，不另支付N |

> **2N 与 N 互斥**：违法解除支付2N，不另支付N；合法解除支付N，无需支付2N。

**2N 计算工作表**：

```
步骤 A：先按第47条算出经济补偿 N（见步骤3）
  N = ¥[  ]

步骤 B：2N = N × 2
  2N = ¥[  ] × 2 = ¥[  ]

注意：2N计算基数与N相同（月工资基数 × 工龄年限），封顶规则同样适用。
```

### 步骤 6：综合金额汇总

| 支付项目 | 金额 | 备注 |
|---------|------|------|
| 经济补偿 N（合法解除情形） | ¥[  ] | 第47条，已核实锚点 |
| 代通知金（+1，如适用） | ¥[  ] | 一个月工资，性质为工资 |
| 违法解除赔偿金 2N（违法情形） | ¥[  ] | 第87条，已核实锚点；与N互斥 |
| 未休年假补偿（如有未休年休假）| ¥[  ] | 300%（含正常工资100%+另付200%）`[待验证]` |
| 未支付工资/绩效 | ¥[  ] | — |
| **合计应付款项** | **¥[  ]** | — |

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 高收入员工（月薪超三倍当地平均工资） | 月工资基数封顶 + 工龄年限封顶12年（第47条），明确说明封顶后的计算结果 |
| 工龄不满6个月 | 支付0.5个月工资，已含于第47条规则（已核实锚点），直接适用 |
| 工龄跨越2008年1月1日 | 2008年前工龄是否分段计算存在争议，建议核查当地仲裁最新口径 `[待验证]`，报告中明确标注 |
| 月工资低于最低工资 | 按当地最低工资标准计算基数（第47条规定的下限保障） |
| 非过失性辞退已提前30天通知 | N（经济补偿），无代通知金（+1），合计仅N |
| 非过失性辞退未提前30天通知 | N+1（经济补偿+代通知金），代通知金按月工资计（非三倍封顶基数） |
| 违法解除且劳动者要求恢复劳动关系 | 继续履行 + 支付中间期工资，不支付2N；不恢复的支付2N，两者不并用 |
| 合同约定解除补偿高于法定标准 | 按合同约定执行（法定标准为最低标准），提示「高于法定约定有效」 |
| 实践配置文件有「高于法定标准倾向（如N+2）」 | 按配置文件立场呈现，并同时列出法定标准对照，标注「超法定标准」 |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 无法获取解除前12个月工资记录 | 说明：「无法计算精确基数，以下使用可获取信息估算，结论仅供参考；建议补充工资记录后重新计算」 |
| 绩效奖金是否纳入基数存在争议 | 列出两种计算方案（含/不含奖金），说明各自适用的情形和风险 |
| 违法解除但劳动者尚未表态（恢复vs2N） | 说明劳动者选择权，提供两种金额：（a）恢复劳动关系方案+中间期工资；（b）2N赔偿金方案，供劳动者选择 |
| 工龄记录不完整（入职日期不明） | 建议补充劳动合同/社保记录核实，提示「工龄确认影响补偿金额，须在最终支付前核实」 |
