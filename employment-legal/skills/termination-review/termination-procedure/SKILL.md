# 解除程序合规（termination-procedure）

> **前置**：
> 1. 读取 `employment-legal/CLAUDE.md`（实践配置文件），加载 `## 审查立场 > 解雇审查` —
>    了解内部流程要求（通知工会 / 解除证明风格）。
>    若含占位符，进入**临时模式**（见下方）。
> 2. 读取 `employment-legal/skills/_shared/legal-basis-conventions.md`（引用规范）。
> 3. 读取 `employment-legal/skills/_shared/labor-law-citations.md`（引用库，
>    重点：§A.8 解除事由（各类解除适用程序）、§A.9 经济补偿情形）。
> 4. 分工说明：本子技能专注「怎么解才合规」（法定程序履行）；
>    `grounds-legality` 负责「能不能解」（事由合法性）；
>    `severance-calculation` 负责「给多少」（补偿计算）；
>    `unlawful-termination-remedy` 负责「解错了如何救济」（救济与预防）。
>    本子技能不评估解除事由合法性，仅审查程序是否依法履行。
> 5. 本子技能可被 `employment-legal/skills/termination-review/SKILL.md` 编排入口调用，
>    也可通过 `/employment-legal:termination-procedure` 单独触发。

---

**临时模式**：若实践配置文件含占位符，使用以下默认值并标注 `[临时模式]`：

- 非过失性辞退须提前30天书面通知，否则支付代通知金 `[临时模式]`
- 全部解除类型均须出具书面解除证明 `[临时模式]`
- 均须在解除后15日内办理档案和社保转移 `[临时模式]`

---

## 目的

核查拟议解除或已发生解除的法定程序是否依法履行，
识别程序瑕疵（可能导致解除无效或赔偿责任），
提供程序合规清单和送达留痕要点。

---

## 详细步骤

### 步骤 1：确认解除类型与对应程序差异

不同解除类型对应不同法定程序要求，须首先确认类型：

| 解除类型 | 提前30天通知/代通知金 | 通知工会 | 向劳动行政部门报告 | 出具解除证明 | 档案社保转移 |
|---------|-------------------|---------|-----------------|-----------|-----------|
| 协商解除（约第36条 `[待验证]`） | ☐ 不要求（双方协商） | ☐ 建议通知 `[待验证]` | ☐ 不要求 | **☐ 必须** | **☐ 必须** |
| 过失性辞退（约第39条 `[待验证]`） | ☐ **无需提前通知** | **☐ 须事先通知工会** | ☐ 不要求 | **☐ 必须** | **☐ 必须** |
| 非过失性辞退（约第40条 `[待验证]`） | **☐ 提前30天书面通知或代通知金** | **☐ 须事先通知工会** | ☐ 不要求 | **☐ 必须** | **☐ 必须** |
| 经济性裁员（约第41条 `[待验证]`） | **☐ 提前30天向工会/职工说明** | **☐ 须提前说明并听取意见** | **☐ 须向劳动行政部门报告** | **☐ 必须** | **☐ 必须** |

### 步骤 2：提前通知/代通知金（非过失性辞退专项）

**适用范围**：仅非过失性辞退须履行此程序。
经济性裁员须提前30天向工会/职工**说明**，性质不同。

| 审查项 | 合规要求 | 状态 |
|-------|---------|------|
| 是否已提前30天书面通知 | 书面形式，须留存送达证明 | ☐ 是 / ☐ 否→支付代通知金 |
| 书面通知是否载明解除事由 | 须说明解除依据（约第40条情形） | ☐ 是 / ☐ 否🟠 |
| 30天期间劳动者是否正常工作 | 通知期内劳动者照常工作、领取工资 | ☐ 已履行 / ☐ 未履行🟠 |
| 如未提前通知：代通知金是否支付 | 额外支付一个月工资 | ☐ 已支付 / ☐ 未支付🔴 |

**送达合规**（书面通知的举证关键）：

| 送达方式 | 举证要求 | 建议 |
|---------|---------|------|
| 当面送达 | 劳动者签字确认、注明日期 | 首选，留存签收记录 |
| EMS 快递 | 快递回执 + 签收记录 | 务必保存回执，记录发件地址 |
| 电子邮件/企业微信/钉钉 | 发送记录 + 已读回执或回复 | 须有被送达方读取证明 |
| 留置送达 | 劳动者拒签时，见证人在场 + 照片/视频 | 最后手段，须有见证人 |
| 公告送达 | 用于无法送达的情形，须在劳动合同约定/企业公告栏公示 | 效力最弱，争议多 `[待验证]` |

### 步骤 3：工会通知（过失性辞退 / 非过失性辞退专项）

**适用范围**：过失性辞退和非过失性辞退均须**事先**通知工会，并说明解除理由。
若企业无工会，该程序豁免（但须确认无工会的事实）。

| 审查项 | 合规要求 | 状态 |
|-------|---------|------|
| 企业是否设有工会 | 确认工会存在情况 | ☐ 有工会 / ☐ 无工会（记录在案） |
| 是否**事先**通知工会（非事后） | 必须在解除**前**通知 | ☐ 已事先通知 / ☐ 未通知🔴 / ☐ 无工会 |
| 是否说明解除理由 | 通知应载明理由和依据 | ☐ 是 / ☐ 否🟠 |
| 工会是否提出异议 | 工会认为不合法可提出意见 | ☐ 无异议 / ☐ 有异议（处理方式：[  ]） |
| 工会异议是否处理 | 用人单位须考虑工会意见，再作决定 `[待验证]` | ☐ 已处理 / ☐ 未处理🟠 |

**重要边界**：

- 工会通知程序瑕疵（如解除后才通知工会）：
  - 一般认为程序违法，但部分地区认定可在诉讼前补正效力，标 `[待验证]`（各地实践不一）
  - 建议不依赖补正，严格在解除前完成通知

**经济性裁员中的工会程序**（区别于过失/非过失）：

| 程序要求 | 说明 |
|---------|------|
| 提前30天向工会或全体职工说明 | 主动说明，非仅「通知」（约第41条 `[待验证]`） |
| 听取工会或职工意见 | 须有书面记录，意见须被考虑（非仅形式性听取） |
| 裁员方案须经过意见程序 | 实践中意见程序结束后方可正式裁员 |

### 步骤 4：向劳动行政部门报告（经济性裁员专项）

**适用范围**：仅经济性裁员须履行此程序（约第41条 `[待验证]`）。

| 审查项 | 合规要求 | 状态 |
|-------|---------|------|
| 是否已向劳动行政部门报告 | 须提交裁员方案及说明 | ☐ 已报告 / ☐ 未报告🔴 |
| 报告时间 | 在正式裁员前完成 | ☐ 是 / ☐ 否🔴 |
| 报告材料 | 裁员原因说明 / 裁员名单 / 经济补偿方案 | ☐ 完备 / ☐ 不完备🟠 |

### 步骤 5：出具解除证明

**全部解除类型**均须出具书面解除证明（约劳动合同法相关条款 `[待验证]`）。

**解除证明的法定内容**（须核查是否齐全）：

| 内容项 | 是否必须 | 状态 |
|-------|---------|------|
| 劳动合同期限 | 必须 | ☐ 含 / ☐ 缺 |
| 解除或终止劳动合同的日期 | 必须 | ☐ 含 / ☐ 缺 |
| 工作岗位 | 必须 | ☐ 含 / ☐ 缺 |
| 在本单位的工作年限 | 必须 | ☐ 含 / ☐ 缺 |
| 解除或终止劳动合同的原因（如用人单位提出） | 劳动者要求时应提供 `[待验证]` | ☐ 按要求处理 |
| 是否注明「协商解除」/「非过失辞退」等具体方式 | 按约定和惯例 | ☐ 注明 / ☐ 未注明 |

**未出具解除证明的法律后果**：

- 用人单位未及时出具解除/终止劳动合同的书面证明，给劳动者造成损害的，须承担赔偿责任
  （约劳动合同法相关条款 `[待验证]`）
- 实践中，未出具解除证明可能导致劳动者无法在新单位入职，用人单位面临赔偿责任🟠

### 步骤 6：档案和社保转移

**时限要求**（约劳动合同法相关条款 `[待验证]`）：

| 事项 | 时限 | 操作主体 | 状态 |
|------|------|---------|------|
| 档案转移手续办理 | 解除/终止后**15日**内 | 用人单位协助办理 | ☐ 已完成 / ☐ 超期🟠 |
| 社保关系转移 | 解除后当月或次月社保中断 | 用人单位在系统中及时操作 | ☐ 已完成 / ☐ 超期🟠 |
| 住房公积金封存/转移 | 按公积金管理中心规定 `[待验证]` | 用人单位办理封存手续 | ☐ 已完成 / ☐ 待处理 |
| 解除/终止劳动合同证明归入档案 | 须归入劳动者档案 | 用人单位 | ☐ 已归档 / ☐ 未归档🟠 |

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 工会通知瑕疵（解除后才通知工会） | 标🔴：程序违法；部分地区认为诉讼前补正可有效 `[待验证]`，但依赖补正风险大，建议不以此为策略 |
| 电子送达有效性 | 若劳动合同约定电子送达方式有效，或劳动者实际已读回复，一般认可；无约定且劳动者不认可的风险较高 `[待验证]` |
| 劳动者拒绝签收解除通知 | 采取留置送达（见证人在场、照片/视频记录）或EMS送达，保留回执；劳动者拒签不影响通知的法律效力 |
| 未出具解除证明导致劳动者无法入职 | 用人单位承担赔偿责任：赔偿范围以劳动者实际损失为准 `[待验证]` |
| 档案转移因劳动者原因受阻 | 做好自身履行义务的书面证明，请求行政部门协助；若劳动者故意拖延，记录用人单位已主动联系的证据 |
| 经济性裁员6个月内重新招用 | 须优先通知被裁减员工（约第41条 `[待验证]`），未通知存在纠纷风险🟠 |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 解除通知已发出但未送达劳动者 | 分析送达方式的有效性；建议补充送达或重新送达；在送达确认前，解除效力存疑🔴 |
| 工会通知在解除后才补做 | 实践中争议大 `[待验证]`；建议咨询当地劳动行政部门，标注🔴风险，不应以「补正」为常规操作 |
| 经济性裁员未向劳动行政部门报告 | 程序违法，标🔴；但劳动行政部门是否有权撤销解除，各地实践不一 `[待验证]`；须在当地确认 |
| 解除证明内容缺失（如无工作年限） | 须补充出具完整解除证明，并就已造成的损失与劳动者协商处理 |
| 未在15日内办理档案社保转移 | 承担迟延赔偿责任；立即办理，同时书面告知劳动者进度，留存记录 |
