# 解除事由合法性（grounds-legality）

> **前置**：
> 1. 读取 `employment-legal/CLAUDE.md`（实践配置文件），加载 `## 审查立场 > 解雇审查` —
>    `#### 法定解除情形`（用人单位内部认定标准 / 提前通知倾向）。
>    若配置文件含占位符，进入**临时模式**（见下方临时模式段落）。
> 2. 读取 `employment-legal/skills/_shared/legal-basis-conventions.md`（劳动人事引用规范）。
> 3. 读取 `employment-legal/skills/_shared/labor-law-citations.md`（劳动人事引用库，
>    重点：§A.8 解除事由、§F.2 三期保护）。
> 4. 分工说明：本子技能负责「能不能解」（事由合法性 + 禁止解除情形核查）；
>    `severance-calculation` 负责「合法解除给多少」；
>    `termination-procedure` 负责「怎么解才合规」；
>    `unlawful-termination-remedy` 负责「解错了如何救济」。
>    四个子技能互补，分工不重叠。
> 5. 本子技能可被 `employment-legal/skills/termination-review/SKILL.md` 编排入口调用，
>    也可通过 `/employment-legal:grounds-legality` 单独触发。

---

**临时模式**：若 `employment-legal/CLAUDE.md` 的 `## 审查立场 > 解雇审查` 含占位符，
使用以下默认值并标注 `[临时模式]`：

- 过失性辞退须有明确规章制度依据，规章制度经民主程序制定并公示 `[临时模式]`
- 末位淘汰不等于「不能胜任工作」，须经培训/调岗验证 `[临时模式]`
- 经济性裁员须满足法定数量门槛（20人或10%） `[临时模式]`

如需更新为实际立场，运行 `/employment-legal:cold-start-interview` 或直接编辑配置文件。

## 目的

核查拟议解除方案的法律事由是否属于《劳动合同法》（2012年修正）所规定的合法解除情形，
并核查是否触发禁止解除情形，输出「能不能解」的合法性结论及风险等级评估。

---

## 详细步骤

### 步骤 1：识别解除类型

首先确认拟议解除属于哪一类：

| 解除类型 | 特征 | 是否须支付经济补偿 |
|---------|------|-----------------|
| 协商解除（约第36条 `[待验证]`） | 双方协商一致 | 用人单位提出时须支付 |
| 过失性辞退（约第39条 `[待验证]`） | 劳动者有过错情形 | **不须支付** |
| 非过失性辞退（约第40条 `[待验证]`） | 劳动者无过错，但有特定情形 | **须支付 N** |
| 经济性裁员（约第41条 `[待验证]`） | 企业层面经济原因，批量裁减 | **须支付 N** |

> **引用纪律**：第36/39/40/41条已经相当确定地对应上述内容，但使用时仍标注 `[待验证]`，
> 引用前通过 https://flk.npc.gov.cn 核验。

### 步骤 2：核查各类解除的法定构成要件

#### 2.1 协商解除（《劳动合同法》约第36条 `[待验证]`）

**核心要件**：

- [ ] 用人单位与劳动者协商一致（须有双方意思表示）
- [ ] 签署书面协议（强烈建议，防止争议）
- [ ] 协议内容明确（解除日期 / 经济补偿金额 / 社保转移安排 / 工资结清）

**注意**：

- 若由**用人单位**提出协商解除，须支付经济补偿（约第46条 `[待验证]`）
- 若由**劳动者**提出协商解除，用人单位无须支付经济补偿
- 协商解除协议签署后，劳动者原则上不可反悔（须确认签署时意思表示自愿真实）

| 审查项 | 状态 |
|-------|------|
| 解除系哪方提出？ | ☐ 用人单位 / ☐ 劳动者 |
| 双方达成一致的证据 | ☐ 已有书面协议 / ☐ 仅口头 |
| 协议是否载明补偿金额 | ☐ 是 / ☐ 否 |
| 劳动者签字时是否有「被迫」迹象 | ☐ 是 🔴 / ☐ 否 |

#### 2.2 过失性辞退（《劳动合同法》约第39条 `[待验证]`）

过失性辞退无需提前通知，**无需支付经济补偿**。但须满足以下任一情形：

| 情形 | 法定描述 | 审查要点 |
|------|---------|---------|
| 试用期不符合录用条件 | 在试用期内被证明不符合录用条件 | 录用条件是否事先明确告知并有书面记录？ |
| 严重违反规章制度 | 严重违反用人单位的规章制度 | 规章制度是否经民主程序制定并公示（或书面告知）？「严重」的标准是否在制度中明确？ |
| 严重失职、营私舞弊 | 严重失职、营私舞弊，给用人单位造成重大损害 | 「重大损害」的认定依据？有无量化或明确标准？ |
| 双重劳动关系 | 同时与其他用人单位建立劳动关系，影响本职工作，经提出拒不改正 | 是否已提出（有书面记录）但劳动者拒不改正？ |
| 欺诈胁迫致合同无效 | 以欺诈、胁迫或乘人之危方式订立劳动合同，导致合同无效 | 欺诈/胁迫的证据？是否经仲裁或司法确认合同无效？ |
| 被追究刑事责任 | 被依法追究刑事责任 | 是否已有生效刑事判决或检察院起诉？拘留/羁押≠追究刑责 |

**重要边界条件（规章制度适用）**：

| 规章制度缺陷 | 法律后果 |
|------------|---------|
| 规章制度未经民主程序制定 | 不得作为解除依据，标🔴 |
| 规章制度经民主程序但未向劳动者公示/告知 | 不得作为解除依据，标🔴 |
| 「严重」标准缺失或模糊 | 在仲裁/诉讼中可能被认定不合规，标🟠 |
| 末位淘汰制度 | **末位淘汰≠「不能胜任工作」**，不适用非过失性辞退规定，绝不可直接依据此辞退劳动者，标🔴 |

#### 2.3 非过失性辞退（《劳动合同法》约第40条 `[待验证]`）

非过失性辞退须**提前30天书面通知**或支付**代通知金**（一个月工资），并**须支付经济补偿 N**。
须满足以下任一情形：

| 情形 | 法定描述 | 前置程序 | 审查要点 |
|------|---------|---------|---------|
| 医疗期满不能从事工作 | 劳动者患病或非因工负伤，医疗期满后不能从事原工作，也不能从事用人单位另行安排的工作 | 须先「另行安排工作」 | 医疗期是否已满？是否真的安排了其他工作？劳动者是否确实不能从事？ |
| 不能胜任工作 | 劳动者不能胜任工作，经过培训或者调整工作岗位，仍不能胜任工作 | 须先「培训或调岗」 | 是否进行了真实的培训或调岗？有无记录？调岗后仍不胜任的证据？ |
| 客观情况重大变化 | 劳动合同订立时所依据的客观情况发生重大变化，致使劳动合同无法履行，经与劳动者协商未能就变更劳动合同内容达成协议 | 须先「协商变更」 | 什么是「重大变化」？有无协商记录？协商是否诚实进行？ |

**重要说明**：

- 「不能胜任工作」须经**培训或调岗**验证，不能仅凭绩效考核末位即认定
- 「客观情况重大变化」须是外部重大变化（如政策法规变化、市场环境），不包括内部重组
  （注：内部重组是否构成本情形存在争议，标 `[待验证]`）
- 每种情形均须**先履行前置程序**（安排工作/培训调岗/协商变更），否则解除无效

#### 2.4 经济性裁员（《劳动合同法》约第41条 `[待验证]`）

经济性裁员有严格的数量门槛和法定情形：

**数量门槛**：

- 需要裁减人员**20人以上**；或
- 裁减人数不足20人但**占企业职工总数10%以上**

（同时满足其一即可）

**法定情形（须满足其一）**：

| 法定情形 | 说明 |
|---------|------|
| 企业破产重整 | 依照企业破产法规定进行重整 |
| 生产经营严重困难 | 实践中认定标准有争议 `[待验证]` |
| 企业转产、技术革新、经营方式调整 | 须有客观背景，非主观压缩人员 |
| 其他客观经济情况重大变化 | 兜底条款，须有实质支撑 |

**法定程序（必须依序履行）**：

1. [ ] **提前30天向工会或全体职工说明情况**（注意：须说明，非仅通知）
2. [ ] **听取工会或职工的意见**（须有记录）
3. [ ] **向劳动行政部门报告裁减人员方案**
4. [ ] **优先留用**以下人员：
   - 与本单位订立较长期限固定期限劳动合同的
   - 与本单位订立无固定期限劳动合同的
   - 家庭无其他就业人员、有需要赡养的老人或者未成年人（约第41条 `[待验证]`）
5. [ ] **6个月内重新招用人员的，须优先通知被裁减人员**（约第41条 `[待验证]`）

### 步骤 3：禁止解除情形前置核查（必须优先执行）

> **警告**：以下情形存在时，**非过失性辞退**和**经济性裁员**不得适用。
> 若存在禁止解除情形，须立即标记🔴，在本步骤停止，**不得继续推进解除方案**。

| 保护情形 | 禁止范围 | 审查状态 |
|---------|---------|---------|
| 从事接触职业病危害作业的劳动者未进行离岗前职业健康检查，或疑似职业病病人在诊断或医学观察期间 | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |
| 在本单位患职业病或因工负伤并被确认丧失或者部分丧失劳动能力 | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |
| 患病或非因工负伤，在规定的医疗期内 | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |
| 女职工在孕期、产期、哺乳期 | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |
| 在本单位连续工作满十五年，且距法定退休年龄不足五年（`[待验证]`） | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |
| 法律、行政法规规定的其他情形 | 非过失性辞退 + 经济性裁员 | ☐ 排除 / ☐ 触发🔴 |

> **注意**：上述禁止情形条号（约第42条 `[待验证]`）。但规则本身已高度确信——
> 医疗期内 / 孕产哺乳期 / 工伤后部分丧失劳动能力不得以非过失性辞退或经济性裁员解除。
> **过失性辞退（约第39条）不受上述禁止限制。**

### 步骤 4：合法性综合评估

完成以上三步后，出具合法性综合结论：

|---------|------|---------|
| 解除类型归属是否明确 | [结论] | [🔴/🟠/🟡/🟢] |

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 劳动者处于医疗期/孕产哺乳期，拟用过失性辞退绕开禁止规定 | 虽然过失性辞退不受禁止情形约束，但须确认有真实过失证据；若仅为规避禁止规定，存在被认定违法解除风险🔴（仲裁委/法院可能穿透审查）`[待验证]` |
| 规章制度民主程序有瑕疵（如未正式公示但劳动者事实上知晓） | 不能以「劳动者事实知晓」替代正式公示程序，仍存在解除无效风险🟠 |
| 末位淘汰制度希望归入「不能胜任工作」 | 绝对不可直接适用，须转化：基于末位表现设计真实的「不能胜任」认定标准，经培训/调岗验证后方可适用非过失性辞退🔴 |
| 协商解除时劳动者后悔反悔，主张协议无效 | 若劳动者能证明签署时存在欺诈/胁迫/重大误解，协议可能被撤销；须确保协议签署程序规范（建议留有劳动者充分考虑时间的记录） |
| 实践配置文件过失辞退认定标准为占位符 | 使用法定标准进行审查，标`[临时模式]`，并建议运行冷启动访谈 |
| 客观情况重大变化与内部重组区分不清 | 内部重组（如部门合并、业务调整）是否构成「客观情况重大变化」存在争议，建议升级`[待验证]` |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 用人单位无法提供规章制度民主程序证明 | 标🔴：「规章制度民主程序无证据，无法依据该制度解除，建议暂缓解除方案，先补充完善内部程序」 |
| 解除事由含「不能胜任工作」但无培训/调岗记录 | 标🔴：「未经培训/调岗直接解除，违反法定前置程序，解除方案无效」 |
| 经济性裁员数量未达20人/10%门槛 | 标🟠：「数量未达经济性裁员门槛，建议改用其他合法解除方式或等待条件满足」 |
| 用人单位以「试用期不合格」解除已过试用期员工 | 标🔴：「试用期已满，不可适用试用期不符合录用条件解除，需另找合法事由」 |
| 无法获取劳动者孕期/医疗期证明信息 | 标🟠：「无法排除禁止解除情形，建议在确认无相关情形前暂缓解除」 |
| 劳动者刑拘但未被起诉 | 标🟡：「羁押/刑拘≠被依法追究刑事责任，须等待检察院起诉或法院生效判决再适用该解除事由」 |
