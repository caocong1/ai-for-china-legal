# 雇佣审查

## 目的

审查雇佣相关文件，对照《中华人民共和国劳动合同法》及相关法规，识别法律风险点，提供具体的修改建议，确保雇佣文件符合中国劳动法要求。

## 前置条件：加载审查立场

**在审查之前，读取实践配置文件。** 如果缺失或有占位符，提示用户运行 `/employment-legal:cold-start-interview` 或使用临时模式。

## 工作流

### 第一步：定位

快速阅读文件，回答：

| 问题 | 回答 |
|---|---|
| 什么类型的文件？ | 劳动合同 / 录用通知书 / 竞业限制协议 / 保密协议 / 员工手册 / 其他 |
| 用人单位性质？ | 企业 / 个体经济组织 / 民办非企业 / 国家机关 / 事业单位 / 社会团体 |
| 劳动者类型？ | 全日制 / 非全日制 / 劳务派遣 / 实习生 / 退休返聘 / 其他 |
| 工作地点？ | 具体城市（影响地方性规定，如社保基数、最低工资） |
| 合同期限？ | 固定期限 / 无固定期限 / 以完成一定工作任务为期限 |
| 是否涉及竞业限制？ | 是 / 否 |

### 第二步：法定强制性条款检查

检查是否包含《劳动合同法》第十七条规定的必备条款：

1. ✅ 用人单位名称、住所和法定代表人
2. ✅ 劳动者姓名、住址和身份证号码
3. ✅ 劳动合同期限
4. ✅ 工作内容和工作地点
5. ✅ 工作时间和休息休假
6. ✅ 劳动报酬
7. ✅ 社会保险
8. ✅ 劳动保护、劳动条件和职业危害防护
9. ✅ 法律、法规规定应当纳入的其他事项

**缺失任何必备条款：** 标记为 🟡 中等风险，建议补充。

### 第三步：关键风险点审查

#### 3.1 试用期

**法定标准（《劳动合同法》第十九条）：**

| 合同期限 | 试用期上限 |
|----------|-----------|
| 3个月 ≤ 期限 < 1年 | 不超过 1 个月 |
| 1年 ≤ 期限 < 3年 | 不超过 2 个月 |
| 3年以上或无固定期限 | 不超过 6 个月 |
| 以完成一定工作任务为期限 | 不得约定试用期 |
| 期限 < 3个月 | 不得约定试用期 |

**检查要点：**
- [ ] 试用期期限是否符合法定上限
- [ ] 试用期工资是否不低于转正后工资的 80% 且不低于当地最低工资
- [ ] 是否同一用人单位与同一劳动者只能约定一次试用期
- [ ] 试用期是否包含在劳动合同期限内

**如果发现偏离：**

```markdown
### 试用期不符合法定要求

**法律规定：** 《劳动合同法》第19条 — [具体规定]

**文件写的是：**
> "[原文引用]"

**风险：** 🟠 高 — 超过法定上限的试用期无效，劳动者可以要求按转正后工资补足差额

**建议修改：**
> "[具体替换语言]"
```

#### 3.2 工时制度

**法定标准：**
- 标准工时制：每日不超过 8 小时，每周不超过 40 小时
- 综合计算工时制：需劳动行政部门批准
- 不定时工时制：需劳动行政部门批准，适用于高管、外勤等

**加班费标准：**
- 工作日延长：不低于工资的 150%
- 休息日加班且不能补休：不低于工资的 200%
- 法定节假日：不低于工资的 300%

**检查要点：**
- [ ] 工时制度是否明确
- [ ] 加班费计算基数是否合法
- [ ] 是否有"自愿加班"规避加班费的条款（🔴 关键风险）

#### 3.3 社会保险和住房公积金

**法定要求：** 用人单位必须为劳动者缴纳五险（养老、医疗、失业、工伤、生育）和住房公积金。

**检查要点：**
- [ ] 是否有"自愿放弃社保"条款（🔴 关键风险，该条款无效）
- [ ] 是否有"社保补贴代替缴纳"条款（🟠 高风险，不免除法定缴纳义务）
- [ ] 缴纳基数是否合规

#### 3.4 竞业限制

**法定标准（《劳动合同法》第23-24条）：**

| 项目 | 法定要求 |
|------|---------|
| 适用人员 | 高管、高技和其他负有保密义务的人员 |
| 期限 | 解除/终止后不超过 2 年 |
| 经济补偿 | 必须支付，标准由双方约定（地方有不同指引） |
| 范围 | 与本单位生产或经营同类产品、从事同类业务的有竞争关系的用人单位 |

**检查要点：**
- [ ] 竞业限制人员是否属于法定范围
- [ ] 竞业限制期限是否超过 2 年
- [ ] 是否约定了经济补偿（没有补偿的竞业限制条款可能无效）
- [ ] 竞业限制范围是否合理

#### 3.5 违约金

**法定限制（《劳动合同法》第25条）：** 除以下两种情形外，用人单位不得与劳动者约定由劳动者承担违约金：

1. 用人单位为劳动者提供专项培训费用，对其进行专业技术培训的，可以约定服务期和违约金（违约金不超过培训费用）
2. 竞业限制条款中的违约金

**检查要点：**
- [ ] 是否有提前离职违约金条款（🔴 关键风险，除上述两种情形外无效）
- [ ] 是否有"未提前30天通知的违约金"（🔴 关键风险，无效）

#### 3.6 劳动合同解除

**法定情形（《劳动合同法》第36-48条）：**

检查是否有违法解除条款：
- [ ] 是否有"用人单位可以随时解除"条款（🔴 关键风险）
- [ ] 是否有"劳动者提前30天书面通知即可解除"的确认（法定权利）
- [ ] 经济补偿/赔偿金的约定是否符合法定标准

### 第四步：地方性规定检查

根据工作地点，检查适用的地方性规定：

| 地区 | 特别规定 |
|------|---------|
| 北京 | 社保基数上下限、最低工资标准 |
| 上海 | 竞业限制补偿指引标准 |
| 广东 | 高温津贴、特殊工时 |
| 深圳 | 经济特区劳动合同条例 |
| 其他 | 根据具体城市查询 |
