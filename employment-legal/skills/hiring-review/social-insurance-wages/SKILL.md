> 4. 分工说明：本子技能负责**社保公积金合规**（五险一金法定义务/无效放弃条款）与**劳动报酬**（工资标准/支付周期/最低工资合规）及**未订书面合同二倍工资风险**（第82条）。试用期工资见 `probation-working-hours`；违约金见 `service-period-penalty`；竞业限制经济补偿见 `non-compete-review`（04-02）。

## 目的

检查劳动合同在社会保险、住房公积金和劳动报酬方面的合规性，重点识别：
1. 「自愿放弃社保」等无效条款及其法律风险（🔴）
2. 工资约定低于最低工资标准、工资支付不合规的风险
3. 未订书面劳动合同二倍工资风险（《劳动合同法》第82条，已核实锚点）
4. 社保缴纳基数合规性（地方差异显著，须核查工作地点标准）

---

## 详细步骤

### 步骤 1：社会保险合规检查

#### 1.1 五险法定缴纳义务（已核实规则）

**法律框架**：《社会保险法》（2011年施行，条号 `[待验证]`）规定，用人单位须依法为劳动者缴纳以下五险：

| 险种 | 缴纳方式 | 强制性 | 放弃是否有效 |
|------|---------|-------|------------|
| 养老保险 | 用人单位 + 劳动者共同缴纳 | **强制** | **无效** |
| 医疗保险 | 用人单位 + 劳动者共同缴纳 | **强制** | **无效** |
| 失业保险 | 用人单位 + 劳动者共同缴纳 | **强制** | **无效** |
| 工伤保险 | 用人单位**单独**缴纳 | **强制** | **无效** |
| 生育保险 | 用人单位**单独**缴纳 | **强制** | **无效** |

> **「自愿放弃社保」规则（已核实）**：无论劳动者是否「自愿」书面声明放弃社保，用人单位的法定缴纳义务不因此免除。「自愿放弃社保」条款依法无效；仍需补缴，并可能面临滞纳金和罚款。

#### 1.2 社保条款扫描

| 检查项 | 识别指标 | 风险等级 |
|-------|---------|---------|
| 「自愿放弃社保」条款 | 「员工自愿选择不参加社会保险」「员工已知晓并同意不缴纳社保」等表述 | 🔴 关键风险 |
| 「社保补贴代替缴纳」 | 「按月向员工支付X元社保补贴，员工自行购买社保/商业保险」等表述 | 🟠 高风险 — 不免除法定义务 |
| 社保条款缺失（仅作形式声明） | 「依法缴纳社保」但未明确险种与缴纳标准 | 🟡 中等风险 |
| 社保缴纳基数约定偏低 | 以「基本工资」为社保基数，而非实际工资（如排除绩效、津贴等） `[待验证]` | 🟠 高风险 |

**「自愿放弃社保」偏离输出模板**：

```markdown
### 「自愿放弃社保」条款——关键风险

**法律规定：** 《社会保险法》（2011年施行）规定，用人单位须依法为劳动者缴纳五险（条号 `[待验证]`）

**文件写的是：**
> "[原文引用]"

**风险：** 🔴 关键风险 — 该条款依法无效；用人单位仍须补缴所有欠缴的社保费用，
另需支付滞纳金（通常按日计算，`[待验证]`），并可能面临劳动行政部门罚款

**建议修改：**
> 删除「自愿放弃社保」条款；替换为：「用人单位依法为劳动者缴纳养老、医疗、失业、
> 工伤、生育保险，缴纳基数和比例按当地现行规定执行」
```

#### 1.3 社保缴纳基数合规性

社保缴纳基数有上下限，各地规定不同：

| 基数规则 | 说明 |
|---------|-----|
| 基数下限 | 一般为当地上年度职工月平均工资的 60%（各地不同，须核查 `[管辖 flagged — 需验证地方性规定]`） |
| 基数上限 | 一般为当地上年度职工月平均工资的 300%（各地不同 `[管辖 flagged — 需验证地方性规定]`） |
| 实际工资低于下限 | 按下限缴纳 |
| 实际工资高于上限 | 按上限缴纳 |

**审查要点**：合同中明确约定「以基本工资 [X] 元作为社保缴纳基数」时，若基本工资明显低于实际工资且低于下限，标记 🟠。

### 步骤 4：未订书面劳动合同二倍工资风险（第82条专项，已核实）

> **本步骤是雇佣审查中最重要的二倍工资风险专项检查。**

**法律依据**：《劳动合同法》（2012年修正）**第82条**（**已核实锚点**）

**规则摘要**：
- 用人单位自用工之日起**超过一个月不满一年**未与劳动者订立书面劳动合同的，应向劳动者**每月支付二倍工资**
- 用工之日起满**一年**仍未订立书面劳动合同的，**视为已订立无固定期限劳动合同**（第14条触发），二倍工资义务消灭，转为无固定期限合同纠纷
- 二倍工资的「二倍」含义：第一倍是正常工资（已实际支付），第二倍是额外赔偿，共计支付两倍月工资

#### 4.1 二倍工资风险触发检查

| 检查项 | 状态 | 风险等级 |
|-------|------|---------|
| 劳动合同是否已签署（书面形式）| [已签/待签/未签] | [等级] |
| 若未签：用工开始日期与当前日期的间隔 | [X 天/月] | [等级] |
| 若已签：签署日期与用工开始日期的间隔 | [X 天/月] | [等级] |

**风险等级矩阵**：

| 间隔 | 风险 |
|------|------|
| 用工 ≤ 1 个月 | 在法定期限内，暂无二倍工资风险 🟢 |
| 用工 > 1 个月且 < 12 个月，合同仍未签 | 二倍工资风险，每月持续累积 🔴 |
| 用工已满 12 个月，仍未签书面合同 | 视为已订立无固定期限劳动合同，另有违法责任 🔴 |
| 合同已签但签署日期晚于用工日期超过 1 个月 | 未签期间二倍工资风险，须评估是否已超时效 `[待验证]` |

#### 4.2 二倍工资计算

**计算公式**（`[待验证]`）：

```
应支付的额外工资 = 劳动者每月工资 × 未签书面合同的月数（超1个月不满12个月部分）
```

**计算说明**：
- 「每月工资」以实际工资为准（含绩效津贴，不仅基本工资）`[待验证]`
- 起算点：用工之日起第二个月（超过一个月起）
- 终止点：书面合同签署当日（或用工满12个月时）
- 仲裁时效1年（《劳动争议调解仲裁法》第27条，**已核实**）

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 劳动者本人书面要求不缴社保，并提供免责声明 | 明确告知：「声明/协议不免除法定缴纳义务，声明本身无效；用人单位仍须依法补缴」🔴 |
| 外籍员工同时在本国缴纳社保（双边协议国家） | 须查询中国是否与该国签订社会保险双边协议，如有协议可豁免 `[待验证]` `[管辖 flagged]` |
| 用人单位通过商业保险替代工伤保险 | 商业保险不能代替法定工伤保险；工伤保险属强制性社保，必须依法参保 `[待验证]` 🔴 |
| 「工资中包含社保份额，社保由用人单位一并缴纳」 | 需区分：若用人单位实际已缴且金额符合法定标准，合规；若仅名义上「包含」而未实际缴纳，🔴 |
| 配置文件缺失或含占位符 | 按 practice-profile-schema.md 临时模式处理，工作地点社保基数标 `[临时模式，按工作地点当地标准确认]` |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 工资约定仅为范围（「X万至X万」）而非固定值 | 分析「以最低值作为社保基数是否合规」；标注「具体金额须在入职时确认」 |
| 无法确认用工开始日期（合同未填写/字迹不清） | 标记 🟡：「须确认用工开始日期，方可评估二倍工资风险；建议核查入职记录、社保缴纳记录」 |
| 文件中同时存在多份工资约定（如劳动合同+补充协议） | 梳理各文件优先适用顺序，以最终确认的工资标准为准 |
