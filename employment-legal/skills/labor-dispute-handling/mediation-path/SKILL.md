# 调解路径（mediation-path）

> **所属编排**：`employment-legal/skills/labor-dispute-handling/SKILL.md`（劳动争议处理）
> **前置依赖**：在进入本子技能前，须先经 `dispute-classification-limitation` 确认争议属于劳动争议范围且时效未届满。
> **分工说明**：本子技能专责调解路径。仲裁路径由 `arbitration-path` 处理；诉讼路径由 `litigation-path` 处理。
> **可单独触发**：`/employment-legal:mediation-path`

---

## 目的

评估劳动争议是否适合通过调解解决，指导当事人选择合适的调解机构，核实调解协议效力，以及协助决策是否将调解与仲裁并行（防止时效流失）。

---

## 前置

在执行本子技能前：

1. **读取 `employment-legal/CLAUDE.md`**：
   - `## 谁在使用` → 服务对象（用人单位方/劳动者方，决定调解策略）
   - `## 风险校准` → 风险等级（评估调解失败后的仲裁/诉讼风险）
   - `## 升级矩阵` → 升级条件（调解是否须升级）
2. **确认 `dispute-classification-limitation` 已运行**（争议定性确认、时效核查完成）
3. **读取 `employment-legal/skills/_shared/legal-basis-conventions.md`**（引用规范）
4. **读取 `employment-legal/skills/_shared/labor-law-citations.md` 章节 C**（调解仲裁法相关条文）

## 第一步：调解适用性评估

### 调解的适用场景

| 情形 | 调解适合性 | 原因 |
|------|-----------|------|
| 劳动关系仍存续，双方希望保持关系 | 高度适合 | 避免对抗，维护用工关系 |
| 欠薪金额较小，双方均愿和解 | 适合 | 快速解决，成本低 |
| 争议事实清晰，仅金额有争议 | 适合 | 可直接协商最终数字 |
| 解雇/违法解除，劳动者坚持复职 | 不太适合 | 复职诉求通常无法通过调解实现 |
| 用人单位拒绝参与任何调解 | 不适合 | 直接进入仲裁，调解不是强制程序 |
| 时效临近届满 | 谨慎使用 | 调解不自动中断时效（须书面主张权利或申请仲裁中断） |

**关键提醒**：**调解期间时效继续计算**——若时效即将届满，须同步书面主张权利以中断时效，或直接申请仲裁（仲裁申请会中断时效）。

## 第二步：调解组织选择

### 三类调解组织（均为自愿参与，非强制）`[待验证]`

| 调解组织 | 主要特点 | 适用场景 |
|---------|---------|---------|
| **企业劳动争议调解委员会** | 设于用人单位内部，由职工代表和用人单位代表组成；快速便捷，但中立性存疑（劳动者方慎用）| 双方关系尚可、用人单位愿意内部解决 |
| **基层人民调解组织** | 依托基层司法所/调解委员会，第三方中立调解；覆盖社区/村镇 | 欠薪等简单劳动报酬纠纷，第三方中立重要时 |
| **乡镇街道劳动争议调解组织** | 由基层政府设立，专职或兼职调解员；有一定官方背书 | 需要政府背景背书时；劳动者不信任企业内部调解时 |

### 调解申请要素 `[待验证]`

| 要素 | 内容 |
|------|------|
| 申请方式 | 书面申请（口头申请可被记录）`[待验证]` |
| 受理期限 | 调解组织收到申请后 5 个工作日内决定是否受理 `[待验证]` |
| 调解期限 | 自受理调解申请之日起 15 个工作日内结束，延长须双方同意 `[待验证]` |
| 调解失败处理 | 调解不成，当事人可申请仲裁 |

## 第三步：调解协议效力与支付令

### 调解协议的法律效力

调解协议书经双方签字（或盖章）后生效，但**不具有强制执行力**（不同于法院判决或仲裁裁决）。

若用人单位事后拒绝履行调解协议，劳动者须另行申请仲裁或起诉。

### 支付令（重要救济路径）`[待验证]`

**适用条件**：因以下事项达成调解协议，用人单位不履行的，劳动者可持**调解协议书**依法向**人民法院申请支付令** `[待验证]`：

| 适用事项 |
|---------|
| 拖欠劳动报酬 |
| 工伤医疗费 |
| 经济补偿 |
| 赔偿金 |

> **支付令的意义**：支付令是督促程序，无需经过完整诉讼程序；法院发出支付令后，用人单位须在法定期限内履行或提出异议。如无异议，支付令具有强制执行力，可申请强制执行。
>
> **关键纪律**：支付令申请的条件、程序与法院受理范围条号标 `[待验证]`，引用前通过官方渠道核验。

## 第四步：调解与仲裁/诉讼的衔接

### 调解不成后的路径衔接

```
调解申请
    ↓
调解成功 → 签署调解协议书
    ↓                   ↓
用人单位履行        用人单位不履行
（结案）          → 申请支付令（符合条件时）[待验证]
                  → 申请仲裁（补充或替代路径）

调解不成（或拒绝）
    ↓
→ /employment-legal:arbitration-path（仲裁路径）
→ 注意时效：调解期间时效未中断，须确认时效是否仍在1年内（第27条）
```

### 时效保全提示

| 场景 | 建议操作 |
|------|---------|
| 时效距届满 > 3 个月 | 可先尝试调解（3-6周），调解失败后及时申请仲裁 |
| 时效距届满 1-3 个月 | 建议调解与书面主张权利并行，或直接申请仲裁（仲裁申请中断时效） |
| 时效距届满 < 1 个月 | 立即申请仲裁；调解可在仲裁过程中由仲裁庭主持（仲裁庭在裁决前须先行调解 `[待验证]`） |

## 边界条件

| 场景 | 处理方式 | 风险提示 |
|------|---------|---------|
| 用人单位拒绝参与调解 | 调解是自愿程序，对方拒绝调解不影响后续申请仲裁 | 🟢 直接进入仲裁路径 |
| 调解协议签署后用人单位反悔 | 劳动者须申请仲裁或起诉，主张调解协议的约束力；或申请支付令（符合条件时） | 🔴 调解协议无直接执行力，须另行救济 |
| 仲裁过程中申请调解 | 仲裁庭在裁决前须先行调解（调解成功出具调解书，具有法律效力）`[待验证]` | 🟢 仲裁调解书效力高于自行签署的调解协议 |
| 调解期间用人单位转移资产 | 提示申请仲裁/诉讼前做好财产保全准备 | 🔴 高风险，建议直接进入仲裁路径并考虑财产保全 |

---

## 错误处理

| 错误场景 | 识别方式 | 处理方案 |
|---------|---------|---------|
| 时效届满后才申请调解 | 时效已超过1年，且非存续期间拖欠报酬 | 🔴 告知当事人时效风险，评估对方是否会提出时效抗辩，同步评估和解可能性 |
| 将调解协议视为与裁决/判决同等执行力 | 误认为签署调解协议后可直接申请强制执行 | 🟠 纠正认知：须通过支付令（符合条件）或另行仲裁/诉讼实现强制执行 |
| 仲裁时效届满后希望以调解代替仲裁 | 时效已过，不想申请仲裁（认为调解更快） | 🔴 调解不能替代仲裁；时效届满须评估其他补救路径 |
