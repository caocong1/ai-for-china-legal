# 仲裁路径（arbitration-path）

> **所属编排**：`employment-legal/skills/labor-dispute-handling/SKILL.md`（劳动争议处理）
> **前置依赖**：在进入本子技能前，须先经 `dispute-classification-limitation` 确认争议属于劳动争议范围且时效未届满。
> **分工说明**：本子技能专责仲裁路径（含一裁终局识别）。调解路径由 `mediation-path` 处理；诉讼路径由 `litigation-path` 处理。
> **可单独触发**：`/employment-legal:arbitration-path`

---

## 目的

指导律师/HR 完成劳动争议仲裁程序，核心任务：

1. **仲裁前置确认**：劳动争议须先经仲裁，不服裁决方可起诉（一裁终局除外）；
2. **一裁终局识别（第47条，已核实）**：判断争议是否属于终局裁决范围，影响双方的后续救济权利；
3. 仲裁申请书要素、证据准备、管辖选择；
4. 举证责任分配（特别是用人单位掌握的证据）。

---

## 第一步：仲裁前置原则确认

### 仲裁前置规则 `[具体条号待验证]`

**劳动争议须先经仲裁，对仲裁裁决不服方可向法院起诉**（仲裁前置原则，《劳动争议调解仲裁法》相关条款 `[待验证]`）。

**例外情形 `[待验证]`：**

部分情形可不经仲裁直接起诉（具体情形须核实 `[待验证]`）——实务中较为罕见，大多数劳动争议须经仲裁前置。

**流程：**

```
提起劳动争议
    ↓
（可先行调解，自愿）
    ↓
申请劳动人事争议仲裁委员会仲裁（前置必须）
    ↓
    ├─→ 一裁终局（第47条情形）→ 仲裁裁决终局，双方救济受限（见第三步）
    └─→ 非终局裁决 → 对裁决不服，15日内向法院起诉 `[待验证]`
```

## 第二步：一裁终局识别（《劳动争议调解仲裁法》第47条，已核实）

### 一裁终局范围（已核实锚点）

以下两类劳动争议，仲裁裁决为**终局裁决**，自作出之日起发生法律效力：

| 类型 | 判断标准 | 示例 |
|------|---------|------|
| **（一）小额追索** | 追索**劳动报酬、工伤医疗费、经济补偿或者赔偿金**，争议金额**不超过当地月最低工资标准12个月金额**的 | 追索欠薪 ¥5,000（当地月最低工资×12=¥48,000），属一裁终局 |
| **（二）执行国家劳动标准** | 因执行国家劳动标准在**工作时间、休息休假、社会保险**等方面发生的争议 | 加班时间超标争议、年休假未休补偿争议（若金额不超小额标准也适用第一类） |

**判断步骤：**

1. 争议请求是否属于「劳动报酬/工伤医疗费/经济补偿/赔偿金」或「工时/休假/社保」范围？
2. 如属追索金额类，争议金额是否 ≤ 当地月最低工资 × 12？
3. 两个条件满足其一，即为一裁终局。

**核实当地月最低工资标准** `[管辖 flagged — 需验证地方性规定]`：

各地最低工资标准每1-2年调整，须核查仲裁申请时所在地（劳动合同履行地或用人单位所在地）的最新最低工资标准。

---

## 第三步：一裁终局对双方的不同救济

### 劳动者方的救济

**劳动者对终局裁决不服：可在收到裁决书之日起 15 日内向人民法院起诉** `[待验证]`

| 救济方式 | 条件 | 期限 |
|---------|------|------|
| 向法院起诉 | 劳动者不服终局裁决 | 收到裁决书之日起 15 日 `[待验证]` |

### 用人单位方的救济（有限）

**用人单位对终局裁决不服：不能向法院起诉，仅可在法定情形下申请中级法院撤销裁决** `[待验证]`

| 情形 | 说明 |
|------|------|
| 适用法律确有错误 | `[待验证]` |
| 仲裁庭无管辖权 | `[待验证]` |
| 违反法定程序 | `[待验证]` |
| 裁决依据的证据系伪造 | `[待验证]` |
| 对方当事人隐瞒了足以影响公正裁决的证据 | `[待验证]` |
| 仲裁员索贿受贿、徇私舞弊、枉法裁决 | `[待验证]` |

**申请撤销期限**：裁决书送达之日起 **30 日内**向用人单位住所地中级法院申请 `[待验证]`

> **关键纪律**：15日起诉期限与30日撤销期限、撤销情形条号均标 `[待验证]`，引用前通过官方渠道核验。

### 非终局裁决的救济

**双方均可**在收到裁决书之日起 15 日内向有管辖权的法院起诉 `[待验证]`。起诉不影响裁决的先行执行性（用人单位先行执行部分 `[待验证]`）。

## 第四步：仲裁申请书要素

### 仲裁申请书标准要素 `[待验证]`

| 要素 | 内容说明 |
|------|---------|
| 申请人信息 | 姓名/名称、住所、联系方式（劳动者/用人单位） |
| 被申请人信息 | 用人单位名称（须与营业执照一致）、住所、法定代表人 |
| 仲裁请求 | 明确列示每项请求及金额（请求要具体，如：「支付2023年1月至12月欠发工资共计¥XX」） |
| 事实与理由 | 简明描述争议事实，引用法律依据 |
| 证据清单 | 列出提交的证据目录 |
| 申请日期 | 影响时效截止计算 |

**实务建议**：
- 仲裁请求须具体明确，金额须事先核算
- 一裁终局情形须识别并在申请中说明（如申请人为劳动者时）
- 请求中若含非一裁终局事项（如要求恢复劳动关系），整个申请可能不适用一裁终局 `[待验证]`

## 第五步：管辖选择

### 管辖规则（仲裁委选择）`[待验证]`

| 管辖依据 | 说明 |
|---------|------|
| 劳动合同**履行地**的仲裁委 | 劳动者实际工作地点所在市/区劳动人事争议仲裁委员会 |
| 用人单位**所在地**的仲裁委 | 用人单位注册地所在市/区仲裁委员会 |

**两处均有管辖权，劳动者可选择其一申请**（一般优选劳动合同履行地，证据收集便利）。

**多地管辖争议处理** `[待验证]`：双方均申请仲裁时，由先受理的仲裁委管辖；先后关系不明时，由争议各方协商确定。

---

## 第六步：举证责任分配（劳动争议特殊规则）

### 用人单位掌握证据的举证责任倒置 `[待验证]`

| 情形 | 举证责任方 |
|------|---------|
| 用人单位掌管的书面劳动合同、考勤记录、工资发放记录 | **用人单位举证**（不提交承担不利后果） |
| 解除/终止劳动合同的理由和程序 | **用人单位举证** |
| 社保缴纳记录 | 双方均可提供；劳动者可向社保局调取 |
| 劳动者绩效考核记录（以不胜任工作为由解除） | **用人单位举证** |

**劳动者主动举证的常见证据：**

| 证据类型 | 目的 |
|---------|------|
| 劳动合同原件（如有） | 证明劳动关系、工资约定 |
| 工资条/银行流水 | 证明实发工资及欠薪情况 |
| 工作证/名片/邮件签名 | 证明劳动关系（无书面合同时） |
| 解除通知书 | 证明解除事实及日期 |
| 出勤记录/打卡记录 | 证明加班事实 |
| 通讯记录（微信/钉钉/邮件） | 证明工作内容、指令关系、工资约定等 |
| 医院诊断证明 | 证明病假/工伤事实 |

## 边界条件

| 场景 | 处理方式 | 风险提示 |
|------|---------|---------|
| 请求中含终局与非终局混合事项 | 一般按最高价值的请求定性；含非终局请求（如恢复劳动关系）时整体可能不适用一裁终局 `[待验证]` | 🟠 须确认混合请求的处理方式 |
| 用人单位在被申请前转移资产 | 在仲裁过程中申请财产保全（须向法院申请，仲裁委无财产保全权） `[待验证]` | 🔴 高风险，尽早评估财产保全必要性 |
| 劳动者在境外（无法亲自出席）| 可委托代理人出席 `[待验证]` | 🟡 提前确认委托手续 |
| 仲裁裁决出来前和解 | 可在仲裁庭主持下达成和解，仲裁委出具调解书（具有执行力） `[待验证]` | 🟢 推荐在仲裁程序中和解，调解书效力高 |

## 错误处理

| 错误场景 | 识别方式 | 处理方案 |
|---------|---------|---------|
| 将一裁终局误认为不可再起诉 | 劳动者对终局裁决仍坚持起诉 | 🟢 劳动者有权在15日内起诉 `[待验证]`；用人单位不能起诉，只能申请撤销 |
| 仲裁申请超过1年时效 | 受理通知书未收到，或被仲裁委不予受理 | 🔴 评估对方是否提出时效抗辩，无抗辩时仍可尝试；有抗辩时评估补救 |
| 管辖错误（向无管辖权仲裁委申请） | 仲裁委裁定不予受理或移送管辖 | 🟠 及时向正确管辖地仲裁委重新申请（注意时效） |
