# 劳动争议定性与时效（dispute-classification-limitation）

> **所属编排**：`employment-legal/skills/labor-dispute-handling/SKILL.md`（劳动争议处理）
> **分工说明**：本子技能专责争议定性与仲裁时效核查。调解路径由 `mediation-path` 处理；仲裁路径由 `arbitration-path` 处理；诉讼路径由 `litigation-path` 处理。
> **时效优先原则**：进入调解/仲裁/诉讼路径前，必须先经本子技能确认时效是否届满，防止因时效问题导致权利丧失。
> **可单独触发**：`/employment-legal:dispute-classification-limitation`

---

## 目的

在进入任何争议解决路径之前：

1. **争议定性**：确认纠纷是否属于劳动争议范围（须适用仲裁前置程序），区分劳动争议与劳务/侵权/合同纠纷；
2. **时效核查**：核实仲裁申请时效是否届满（一般 1 年，《劳动争议调解仲裁法》第27条，已核实锚点）；
3. **时效起算**：确认起算点（知道或应当知道权利受侵害之日）及存续期间拖欠报酬特例；
4. **时效中止/中断**：识别时效中止/中断情形，防止时效被错误消灭。

## 第一步：劳动争议范围认定

### 属于劳动争议范围（仲裁前置适用）`[待验证 — 具体条号待核实]`

以下事项属于劳动争议，须先经劳动人事争议仲裁委员会仲裁，对裁决不服方可起诉：

| 争议类型 | 说明 | 典型情形 |
|---------|------|---------|
| 确认劳动关系 | 双方对是否存在劳动关系有争议 | 劳务协议被主张劳动关系、无书面合同 |
| 订立/履行/变更/解除/终止劳动合同 | 合同内容与履行争议 | 条款效力、变更未经协商、解除程序违法 |
| 除名、辞退、辞职、离职 | 各类终止雇佣关系的合法性争议 | 违法辞退、自动离职认定 |
| 工作时间、休息休假、社会保险、福利 | 劳动条件争议 | 加班费计算、年休假未休补偿、社保欠缴 |
| 劳动报酬 | 工资、奖金、提成、经济补偿等 | 欠薪、绩效克扣、经济补偿不足 |
| 工伤医疗费 | 工伤赔偿争议 | 医疗费报销、停工留薪期工资 |
| 培训费用/竞业限制补偿/赔偿金 | 违约金与补偿争议 | 竞业违约金、经济补偿、2N赔偿金 |

### 不属于劳动争议范围（须走普通民事途径）`[待验证]`

| 情形 | 分析 |
|------|------|
| 劳务关系争议（非劳动关系） | 独立承包人、个人劳务合同中不存在劳动关系的，不适用仲裁前置 |
| 社保行政争议（不服社保行政决定） | 属行政法律关系，应走行政复议/行政诉讼 |
| 用人单位内部规定不满（无具体权益侵害） | 不构成劳动争议，须先穷尽内部申诉 |

**审查要点：**

- [ ] 纠纷是否发生在劳动关系存续期间或终止后
- [ ] 是否存在书面劳动合同（无书面合同时，须先确认劳动关系）
- [ ] 争议标的是否在劳动争议范围内

## 第二步：仲裁时效核查（已核实锚点）

### 一般仲裁时效（《劳动争议调解仲裁法》第27条，已核实）

| 要素 | 内容 |
|------|------|
| **时效期间** | **1 年**（《劳动争议调解仲裁法》第27条，已核实锚点） |
| **起算点** | 当事人**知道或者应当知道**其权利被侵害之日起计算 |
| **法律后果** | 超过仲裁时效，仲裁委可能不予受理；即使受理，对方提出时效抗辩，胜诉可能性降低 |

### 时效起算点认定

| 争议类型 | 起算点认定方式 | 注意事项 |
|---------|-------------|---------|
| 欠薪争议（每月发薪日届期） | 每月发薪日后次日（分期欠薪按每次发薪日分别起算） | 🔴 若多月欠薪，须分月核算时效 |
| 违法解除争议 | 收到解除通知书或知道解除事实之日 | |
| 经济补偿不足 | 离职之日（协议约定金额未足额支付时，自约定支付日起算）`[待验证]` | |
| 工伤赔偿 | 工伤认定决定书下达后 `[待验证]` | |
| 竞业限制补偿 | 用人单位停止支付之日 `[待验证]` | |
| 未休年休假工资 | 存在争议：按年逐年计算 vs 离职时统一计算 `[学说争议，待验证]` | 🟡 建议尽早主张，不等到离职 |

**时效起算点实务技巧（劳动者方）：**

> 若劳动者「不知道」权利被侵害（如不知道社保应由用人单位全额缴纳），起算点可能延后——但举证「应当知道」难度大，不建议依赖该主张，应尽早申请仲裁。

## 第三步：时效中止与中断

### 时效中止（阻碍消除后继续计算）`[待验证]`

| 中止事由 | 说明 |
|---------|------|
| 不可抗力 | 地震、战争等无法申请仲裁的情形 |
| 其他正当理由 | 具体情形较模糊，须结合案情判断 `[待验证]` |

时效中止期间，时效暂停计算；阻碍消除后，从中止期间届满时继续计算剩余时效。

### 时效中断（重新起算1年）`[待验证]`

| 中断事由 | 说明 | 实务操作 |
|---------|------|---------|
| 申请调解 | 向企业调解委员会或基层调解组织申请调解 | 🟢 操作简便，可有效中断时效 |
| 申请劳动行政部门救济 | 向劳动局投诉、举报 | 🟢 有记录的书面投诉 |
| 对方同意履行 | 用人单位书面承诺补偿/补缴等 | 🟢 须保留书面证据 |
| 主张权利 | 劳动者向用人单位以书面方式主张权利 `[待验证]` | 🟢 书面邮件/微信/钉钉留证即可 |

**实务建议**：时效届满前，至少发送一封书面（或可留存证据的电子通信）主张权利，以中断时效并重新计算。

---

## 第四步：拖欠劳动报酬特例（重要）

### 存续期间拖欠报酬不受1年限制（《劳动争议调解仲裁法》第27条，已核实）

> **特别规则**：劳动关系**存续期间**因**拖欠劳动报酬**发生争议的，劳动者申请仲裁**不受一年仲裁时效期间的限制**。
>
> **但是**：劳动关系**终止**后，应当在劳动关系终止之日起**一年内**提出申请。

| 情形 | 时效规则 |
|------|---------|
| 劳动关系存续期间欠薪（仍在职） | 不受1年限制，随时可申请仲裁 |
| 劳动关系已终止（已离职）后追欠薪 | 须在终止之日起1年内申请，超过则受时效限制 |

**审查要点：**

- [ ] 劳动关系是否仍存续（影响是否适用特例）
- [ ] 如已终止，终止日期是否已超过1年
- [ ] 追索的欠薪是否全部属于劳动报酬（非劳动报酬的其他金额不适用此特例）

## 第五步：时效届满的处理

### 超过仲裁时效但对方未抗辩

仲裁时效是可以被放弃的抗辩权——如果对方（通常是用人单位）在仲裁中未提出时效抗辩，仲裁委不应主动援引时效驳回。实务中，仲裁委的主动援引做法存在争议 `[待验证]`。

| 对方行为 | 处理方式 |
|---------|---------|
| 未提出时效抗辩 | 程序上可能仍可获支持，但须谨慎，不要依赖对方失误 |
| 提出时效抗辩 | 举证时效起算点（证明起算点晚于一年前）或时效中止/中断事实 |

## 边界条件

| 场景 | 处理方式 | 风险提示 |
|------|---------|---------|
| 时效起算点争议（用人单位与劳动者认定不一致） | 综合证据分析：知晓权利侵害的时间节点 | 🟠 举证责任在主张时效未届满的一方 |
| 多项主张时效不同 | 逐项核算（如欠薪1年、经济补偿2年前离职，两项时效分别计算） | 🟠 分项处理，不混同 |
| 劳动关系不明确（无书面合同） | 须先确认劳动关系再核算时效 | 🔴 确认劳动关系本身也是争议标的，须一并提请仲裁 |
| 对方在时效快届满时拖延谈判 | 建议一边谈判、一边申请仲裁（或至少书面主张权利中断时效） | 🔴 谈判不中断时效，仅书面申请调解/主张权利才中断 |

---

## 错误处理

| 错误场景 | 识别方式 | 处理方案 |
|---------|---------|---------|
| 将民法典3年诉讼时效误用于劳动争议 | 时效计算用3年而非1年 | 🔴 劳动争议仲裁时效为1年（第27条），须修正 |
| 遗漏存续期间拖欠报酬特例 | 劳动者仍在职但说「时效已过」 | 🟠 存续期间拖欠报酬不受1年限制，重新核查 |
| 时效起算点错误（从事件发生日而非「知道」之日起算） | 起算点早于实际得知时间 | 🟠 须举证「知道或应当知道」的时间节点 |
| 以谈判代替书面主张时效中断 | 仅有口头谈判，无书面记录 | 🔴 口头谈判不中断时效；须有书面证据 |
