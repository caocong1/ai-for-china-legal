# 其他法定假期与考勤制度（statutory-leave-attendance）

> **所属编排**：`employment-legal/skills/leave-management/SKILL.md`（假期管理）
> **分工说明**：本子技能专责婚丧假/陪产假/工伤停工留薪/法定节假日及考勤制度合规审查。年休假由 `annual-leave` 处理；病假/医疗期由 `sick-leave-medical-period` 处理；产假/三期保护由 `maternity-female-protection` 处理。
> **可单独触发**：`/employment-legal:statutory-leave-attendance`

---

## 目的

审查「年休假/病假/产假之外」的法定假期制度及考勤规章制度，核实：

1. 婚假、丧假天数是否符合法定标准（地方性规定差异显著）；
2. 陪产假/护理假是否按当地最新规定落实；
3. 工伤停工留薪期与病假医疗期是否区分处理；
4. 法定节假日安排与调休是否合规（节假日工作须支付300%加班费）；
5. 考勤制度（旷工认定标准、请假程序、未批准休假的处理）是否合法，是否经民主程序制定并公示。

## 第一步：婚假与丧假核实

### 全国统一标准 `[待验证]`

| 假期类型 | 法定天数（全国统一） | 工资待遇 |
|---------|----------------|---------|
| 婚假 | 3 天 `[待验证]` | 正常工资 |
| 丧假（直系亲属） | 1-3 天 `[待验证]` | 正常工资 |

> **关键说明**：全国统一婚假天数（3天）与各地曾有的「晚婚奖励假」（原10-30天不等）已因计划生育政策调整发生变化，**现行天数须核查当地最新规定** `[管辖 flagged — 需验证地方性规定]`。

### 地方差异（须按工作地点核实）`[管辖 flagged — 需验证地方性规定]`

| 地区 | 婚假（示例，须核最新） | 丧假（示例，须核最新） | 备注 |
|------|---------------------|---------------------|------|
| 北京市 | `[待验证，须核当地最新规定]` | `[待验证]` | 各省市地方性规定频繁变动 |
| 上海市 | `[待验证，须核当地最新规定]` | `[待验证]` | 须查询当地人社局最新通知 |
| 广东省 | `[待验证，须核当地最新规定]` | `[待验证]` | |
| 深圳市 | `[待验证，须核当地最新规定]` | `[待验证]` | |

**审查要点：**

- [ ] 规章制度中婚假/丧假天数是否符合当地最新规定（不低于地方规定）
- [ ] 是否将已废止的「晚婚奖励假」误作有效规定继续适用
- [ ] 直系亲属范围认定（父母/配偶/子女是否含继父母/岳父母 `[待验证]`）
- [ ] 婚假工资是否正常发放（不扣发）

### 陪产假（男职工）`[管辖 flagged — 需验证地方性规定]`

陪产假无全国统一强制性标准（部分法规有规定 `[待验证]`），由各省市地方性规定确定：

| 地区 | 陪产假天数（示例，须核最新） | 备注 |
|------|---------------------------|------|
| 北京市 | `[待验证，须核当地最新规定]` | 地方性差异大 |
| 上海市 | `[待验证，须核当地最新规定]` | |
| 广东省 | `[待验证，须核当地最新规定]` | |
| 其他地区 | `[管辖 flagged — 必须核验当地最新规定]` | |

**审查要点：**

- [ ] 是否已核实工作地点所在省市的陪产假规定

## 第三步：工伤停工留薪期核实

### 停工留薪期与医疗期的关键区别

| 特征 | 停工留薪期（工伤） | 医疗期（因病/非因工负伤） |
|------|----------------|---------------------|
| 适用场景 | 工伤/职业病 | 普通疾病/非因工受伤 |
| 工资待遇 | **原工资福利待遇不变（全额发放）** | 病假工资（不低于最低工资80%） |
| 时长 | 一般不超过 12 个月；伤情严重可延长，总期限不超过 24 个月 `[待验证]` | 按实际/本单位年限分档（3-30个月） |
| 支付主体 | 用人单位（在工伤保险基金支付前垫付）`[待验证]` | 用人单位（须达病假工资底线） |
| 解除保护 | 停工留薪期内不得解除 `[待验证]` | 医疗期内不得非过失解除 |

**审查要点：**

- [ ] 是否将工伤停工留薪期误用病假处理，导致工资发放不足（全额 vs 80%最低工资）
- [ ] 停工留薪期是否超出法定期限仍在发放
- [ ] 工伤认定是否已完成（未认定工伤情况下不应适用停工留薪期规则）

## 第四步：法定节假日与调休合规

### 法定节假日（全国统一）`[待验证]`

| 节假日 | 法定天数 | 加班工资标准 |
|-------|---------|------------|
| 元旦 | 1天 | 300% `[待验证]` |
| 春节 | 3天 | 300% |
| 清明节 | 1天 | 300% |
| 劳动节 | 1天 | 300% |
| 端午节 | 1天 | 300% |
| 中秋节 | 1天 | 300% |
| 国庆节 | 3天 | 300% |

> 上述法定节假日期间安排工作，不可以调休代替加班费支付，须支付不低于工资300%的加班费（含已付正常工资，另付200%）`[待验证]`。

**调休日（非法定节假日）：**
调休安排在休息日（周末）上班，如不能安排补休，须支付不低于工资200%加班费 `[待验证]`（含已付100%，另付100%）。注意与法定节假日300%的区别。

**审查要点：**

- [ ] 规章制度是否允许以调休代替法定节假日加班费（违法，不可替代）
- [ ] 法定节假日加班费计算基数是否合规
- [ ] 调休日安排工作是否按200%（或安排补休）处理

## 第五步：考勤与请假制度合规审查

### 考勤制度合法性核查要点

#### 规章制度效力（民主程序与公示）`[待验证]`

用人单位考勤规章制度须经民主程序制定（经职工代表大会或全体职工讨论，提出方案和意见，与工会或职工代表平等协商确定 `[待验证]`）并公示才对劳动者具有约束力。

| 程序要求 | 合规性审查 |
|---------|---------|
| 经民主程序制定 | [ ] 是 / [ ] 否（否→🟠 规章制度效力存疑） |
| 向劳动者公示或告知 | [ ] 是 / [ ] 否（否→🟠 不得对抗劳动者） |
| 劳动者签字确认 | [ ] 是 / [ ] 否（否→🟡 无公示记录，举证风险） |

#### 旷工认定标准

| 场景 | 合规处理 | 风险 |
|------|---------|------|
| 未批准申请假期即缺勤 | 可按旷工处理，但须请假程序在规章制度中明确且合理 | 🟠 程序不明确时认定困难 |
| 已提交病假证明但未走正式请假流程 | 不可直接认定为旷工，须综合判断 | 🟠 一律按旷工处理面临争议风险 |
| 超出批准假期仍缺勤 | 可认定为旷工，但须核实延假申请是否被拒绝或未申请 | 🟡 须有明确记录 |

#### 考勤罚款的合法性

> **关键规则**：用人单位**不得**在工资中自行扣罚款（无法定授权，自设罚款条款无效）`[待验证]`。

允许的措施：
- 扣减迟到/早退的对应工资（按实际工作时间计算）
- 依规章制度认定旷工并扣发对应工资
- 按规章制度给予警告、记过等纪律处分，情节严重者可依法解除劳动合同

不允许的措施：
- 直接从工资中扣除「罚款」金额
- 以「罚款」名义设立高额扣减

**审查要点：**

- [ ] 规章制度是否含「迟到X分钟罚款Y元」等罚款条款（无效，须删除）
- [ ] 「旷工认定」标准是否清晰、程序是否合规
- [ ] 扣发工资部分是否以「对应工作时间」为限（不得超过应付工资的法定扣减范围）

## 边界条件

| 场景 | 处理方式 | 风险提示 |
|------|---------|---------|
| 法定节假日前后调休 | 调休日（非法定节假日）工作按200%；法定节假日本身仍须300%，不可混同 | 🔴 混淆节假日与调休日加班费标准 |
| 规章制度经「告知」而非「民主程序」 | 效力存疑，可能被认定为无效 | 🟠 建议重新经民主程序 |
| 丧假「直系亲属」范围争议 | 须在规章制度中明确界定（父母/配偶/子女，是否含岳父母/继父母） | 🟡 界定不清时争议风险 |
| 工伤认定期间是否享受停工留薪期 | 工伤认定期间原则上应按停工留薪期处理，但实务存在争议 `[待验证]` | 🟠 |

---

## 错误处理

| 错误场景 | 识别方式 | 处理方案 |
|---------|---------|---------|
| 将节假日调休（休息日）工作误按法定节假日300%计算 | 加班费计算基础混淆 | 🟠 核实日期类型，调休休息日按200%，法定节假日当日按300% |
| 以「弹性工作制」为由不支付加班费 | 规章制度含「弹性工作无加班」 | 🔴 须经劳动行政部门批准不定时工时制方可；否则法定节假日仍须支付300% |
| 规章制度中「旷工两次即解除」但未明确旷工认定 | 解除通知以旷工为由但无明确旷工标准 | 🟠 解除可能被认定为违法 |
| 未区分工伤停工留薪期与普通病假，按80%最低工资发放 | 工伤职工工资发放低于原工资水平 | 🔴 须按原工资全额补发差额 |
