# 劳动人事法律依据引用规范

> 本文件定义 employment-legal 插件所有技能统一使用的法律依据引用规范。
> 后续计划（04-02、04-03）及全插件各子技能直接引用本规范，避免各技能重复维护。

---

## 一、来源分类

所有法律依据必须归入以下三类之一，并在引用时明确标注类别标签。

### 1. 法条（成文法）

指现行有效的成文法规范，包括：
- **法律**：全国人民代表大会及其常委会制定（如劳动合同法、劳动法、劳动争议调解仲裁法、社会保险法等）
- **行政法规**：国务院制定（如职工带薪年休假条例、女职工劳动保护特别规定等）
- **部门规章**：人力资源和社会保障部、财政部等部门制定（如企业职工带薪年休假实施办法等）
- **地方性法规与规章**：省级人大及其常委会、省级政府制定（如地方最低工资规定、竞业限制补偿指引等）
- **司法解释**：最高人民法院发布的解释性文件（具有规范效力，如劳动争议相关司法解释）

**成文法是分析基础。** 劳动合同合规、争议应对的所有结论，必须首先对照现行成文法条文，而非市场惯例或案例。

**格式要求（法条最小引用格式）：**

```
《[法律/法规全名]》（[版本/修订年份]）第[X]条[第X款]
```

示例：
```
《中华人民共和国劳动合同法》（2012年修正）第47条
```

**版本标注要求：**
- 引用劳动合同法须标注「2012年修正」（2007年通过，2012年修正，为现行版本）
- 劳动法须标注「1994年通过，1995年1月1日施行」（基础法律，后续有局部修改，须确认现行版本）
- 引用已修订的版本，须注明「（已修订，适用旧法情形）」

### 2. 案例（指导性案例与参考案例）

指最高人民法院发布的指导性案例或参考案例，以及劳动人事争议仲裁委员会的典型案例（作辅助参考）。

**定位说明：指导性案例仅作参考，不具有判例法约束力。** 中国采用成文法体系，案例不能替代法条。但最高人民法院指导性案例对下级法院有参照作用，劳动仲裁委员会典型案例对仲裁实践有参照作用，可用于说明裁判/裁决思路。

**格式要求（案例最小引用格式）：**

```
[最高人民法院指导案例第X号] / [案号：（年份）法院级别字号数字号] — [裁判要点摘要] — [法院层级]
```

示例：
```
最高人民法院指导案例第XX号 — [裁判要点] — 最高人民法院 `[待验证]`
```

**法院层级与争议机构标注：**

| 机构 | 简称 | 说明 |
|------|------|------|
| 劳动人事争议仲裁委员会 | 劳动仲裁委 | 劳动争议前置仲裁机构，仲裁前置原则（《劳动争议调解仲裁法》） |
| 基层人民法院 | 基层法院 | 县区级，大多数劳动争议诉讼一审 |
| 中级人民法院 | 中院 | 地市级，重大劳动争议案件一审或上诉 |
| 高级人民法院 | 高院 | 省级，上诉审及在本省有重大影响的案件 |
| 最高人民法院 | 最高院 | 全国最高，指导案例参照价值最高 |

**劳动争议程序说明：** 劳动争议须经仲裁前置——先向劳动人事争议仲裁委员会申请仲裁，不服仲裁裁决方可向法院起诉（一裁终局案件除外，见《劳动争议调解仲裁法》第47条 `[已核实]`）。

**模型回忆案例风险说明：** 案号、指导案例编号极易被模型错误回忆，**所有案例引用默认标注 `[待验证]`**，必须经检索核验后方可去除。

### 3. 学说（学术观点）

指劳动法领域学术著作、法学论文中的学术观点，或律所/HR 法务内部实务经验总结。

**学说为辅助参考，存在争议性。** 引用学说时必须标注：
- 作者/出处（书名/期刊名+年份）
- 是否存在对立观点

**格式要求：**

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息] — "[观点摘要]" [是否有争议：有/无]
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用 employment-legal/CLAUDE.md 中已定义的五个标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站 | 全国人大、人社部等官网 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查 |
| `[用户提供]` | 来自用户提供的文件 | 合同原文、内部规定等 |

**标签使用规则：**
1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 和 `[网络搜索 — 需验证]` 的引用在依赖前**必须**对照原始来源检查

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` | 模型回忆的条文号/案号风险高 | 通过法宝、国家法律法规数据库核查 |
| 引用条文距最后更新超过 6 个月 | 法规可能已修订或废止 | 查询官方法规数据库最新版本 |
| 用户口述的条款号或金额/期限/比例阈值 | 可能记忆偏差 | 核查合同原文或官方法规全文 |
| 所有案例引用（默认） | 案号易被错误回忆 | 通过中国裁判文书网、法宝等核查 |
| 法规版本未确认 | 无法确认是否为现行有效版本 | 查询国家法律法规数据库 |

**待验证标记格式：**

```
《劳动合同法》（2012年修正）第XX条 `[模型知识 — 需验证]` `[待验证 — 验证途径：国家法律法规数据库]`
```

**验证资源：**
- 国家法律法规数据库：https://flk.npc.gov.cn
- 人力资源和社会保障部官网：http://www.mohrss.gov.cn
- 最高人民法院官网：https://www.court.gov.cn
- 中国裁判文书网：https://wenshu.court.gov.cn
- 北大法宝（如已接入）

## 四、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：劳动合同审查、解雇合规、竞业限制评估的每个结论，必须首先援引现行成文法（法律、行政法规、司法解释）作为规范依据
2. **不得以案例替代法条**：即使存在清晰的指导性案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下案例/学说」

### 案例的辅助定位

1. **指导性案例**：用于说明法条适用方式，标注「最高院指导，参照意义强」
2. **仲裁委典型案例**：用于说明仲裁实践倾向，标注「仲裁参考，非司法判决」
3. **明确说明非判例法**：引用案例时，须附注「中国采成文法体系，本案例为参考，不具判例法约束力」

## 五、地方性规定与争议处理机构标注

### 地方性规定识别

劳动用工涉及大量地方性差异，以下常见地区差异须在引用时标注 `[管辖 flagged — 需验证地方性规定]`：

| 地区 | 常见地方性差异 |
|------|--------------|
| 北京 | 社保基数上下限、最低工资标准、竞业限制补偿指引 |
| 上海 | 最低工资标准、综合计算工时制审批、竞业补偿月工资认定 |
| 广东省 | 高温津贴标准、特殊工时制审批、佛山/东莞等地特别规定 |
| 深圳 | 经济特区劳动条例、竞业限制补偿比例指引 |
| 浙江/江苏 | 地方最低工资标准、高温津贴 |
| 其他地区 | 根据工作地点查询当地人力资源和社会保障局 |

**地方性规定引用规则：**
- 明确引用地方性规定时，须标注「[地区名称]地方性规定 + `[管辖 flagged — 需验证地方性规定]`」
- 不得以一个地区的地方性规定代替全国适用规则
- 地方性规定原则上仅适用于工作地点在该地区的劳动合同

### 争议处理机构定位

劳动人事争议的解决路径（须区别于一般民事纠纷）：

```
调解（自愿） → 劳动人事争议仲裁委员会（前置，强制） → 人民法院（不服裁决方可起诉）
```

- **调解**：企业劳动争议调解委员会或第三方调解机构（自愿，非强制）
- **仲裁（前置）**：劳动人事争议仲裁委员会——一般时效 1 年（《劳动争议调解仲裁法》第27条，**已核实**）；部分案件一裁终局（第47条，**已核实**）
- **诉讼**：不服仲裁裁决（除一裁终局外）可向基层法院起诉，适用普通民事诉讼程序
