# 冷启动访谈 — 劳动人事

## 目的

通过结构化的访谈流程，收集劳动法律师的执业立场、风险偏好和地方性规定偏好，生成个性化的实践配置文件。

## 访谈流程

### 第一部分：基本信息

1. **律师角色**
   - 合伙人 / 资深律师 / 初级律师 / 法务 / HR
   - 执业年限和主要劳动法领域

2. **服务对象**
   - 用人单位方 / 劳动者方 / 双方
   - 主要行业

### 第二部分：审查立场

3. **风险偏好**
   - 保守型 / 平衡型 / 进取型

4. **用人单位立场审查重点**
   - 试用期规定
   - 工时制度
   - 社会保险
   - 竞业限制
   - 解雇保护

5. **劳动者立场审查重点**
   - 工资福利
   - 工作条件
   - 解雇补偿
   - 竞业限制补偿

6. **交易破坏者清单**
   - 哪些条款是"绝不接受"的？

### 第三部分：地方性规定

7. **主要工作地点**
   - 北京 / 上海 / 广州 / 深圳 / 其他
   - 需要特别关注的地方性规定

### 第四部分：文书风格

8. **输出格式偏好**
   - 完整审查备忘录
   - 简短摘要
   - 修改建议文档

## 输出

生成劳动人事实践配置文件（YAML 格式）。
