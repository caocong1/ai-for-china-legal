## 目的

核查竞业限制约定的人员资格，判断劳动者是否属于法定三类可约定竞业限制的人员，
区分保密义务与竞业限制的法律性质，分析商业秘密的认定要件，
输出「谁能被约束」的资格结论与风险评级。

---

## 详细步骤

### 步骤 1：人员范围法定框架核查

**法定依据**（**第23条已核实锚点**）：

《劳动合同法》（2012年修正）第23条规定，用人单位可以对「负有保密义务的劳动者」
约定竞业限制；第24条进一步明确，竞业限制人员限于：

| 法定类别 | 说明 | 认定要点 |
|---------|------|---------|
| **高级管理人员** | 公司董事、监事、高级管理人员（《公司法》意义上，具体范围 `[待验证]`） | 依公司章程、公司法界定职位 |
| **高级技术人员** | 核心技术人员、研发人员等掌握关键技术/研发信息的人员 | 依实际掌握的技术信息类型判断，非仅依职称 |
| **其他负有保密义务的人员** | 接触用人单位商业秘密的其他岗位人员（如销售、市场、财务等接触核心经营信息者） | 依**实际接触**的商业秘密范围判断 |

**关键原则**：竞业限制的适用范围以**实际负有保密义务**为核心判断依据，
对于不实际接触商业秘密的普通员工，即便签署竞业限制协议，
在仲裁或诉讼中可能被认定不属于法定三类人员而影响效力（争议较大，标 `[待验证]`）。

### 步骤 2：具体人员资格核查

针对拟约定或已约定竞业限制的劳动者，逐项核查：

#### 2.1 职位核查（职位列表法）

| 职位类别 | 竞业限制适用资格 | 风险等级 |
|---------|--------------|---------|
| 公司董事/监事 | 适用（高级管理人员） | 🟢 |
| CEO/总经理/副总经理/财务总监/技术总监等高管 | 适用（高级管理人员，具体范围依公司章程 `[待验证]`） | 🟢 |
| 核心研发工程师/技术骨干（掌握核心技术） | 适用（高级技术人员） | 🟢 |
| 销售总监/大客户经理（掌握核心客户信息/定价策略） | 适用（其他负有保密义务人员，须核实接触商业秘密情况） | 🟡 |
| 一般行政/客服/前台等基础岗位 | 通常**不适用**（不属三类人员，竞业限制约定效力存争议🔴） | 🔴 |

#### 2.2 实际接触商业秘密核查

即使职位不在高管/高技列表，如果实际接触商业秘密，仍可能属于「其他负有保密义务的人员」：

| 信息类型 | 是否属于商业秘密（须满足三要件）| 劳动者是否实际接触 |
|---------|--------------------------|----------------|
| 核心技术配方/算法/源代码 | 须核查（秘密性/价值性/保密措施） | ☐ 是 / ☐ 否 |
| 核心客户名单（含联系方式/需求偏好）| 须核查（仅有公开信息的客户名单不构成） | ☐ 是 / ☐ 否 |
| 定价策略/成本结构 | 须核查 | ☐ 是 / ☐ 否 |
| 市场战略/产品路线图（未公开） | 须核查 | ☐ 是 / ☐ 否 |
| 内部管理流程/财务数据 | 视敏感程度判断 | ☐ 是 / ☐ 否 |

### 步骤 3：商业秘密三要件认定

竞业限制以保护商业秘密为前提。商业秘密须同时满足**三个要件**
（通常引用《反不正当竞争法》定义，具体条号 `[待验证]`）：

| 要件 | 说明 | 核查要点 |
|------|------|---------|
| **秘密性** | 不为公众所知晓（即「非公知性」）；公开出版/公开可获取的信息不构成商业秘密 | 竞争对手能否通过正当途径获取？是否有公开记录？ |
| **价值性** | 因其秘密性而具有商业价值，能为权利人带来竞争优势 | 该信息对业务的实际价值大小？泄露会造成什么损害？ |
| **保密措施** | 权利人采取了合理的保密措施（如加密/物理隔离/签署保密协议/限制知悉范围等） | 是否有书面保密制度？信息的实际保密状态如何？ |

**三要件任一欠缺，则该信息不构成商业秘密，不能作为竞业限制的正当理由**。

**常见认定争议**：

| 情形 | 风险说明 |
|------|---------|
| 客户名单：仅记录公开可查的公司名称 | 通常不构成商业秘密（无秘密性）`[待验证]` |
| 客户名单：含联系方式/特殊需求/合作历史 | 可能构成商业秘密（须核查保密措施） |
| 技术信息已申请专利（公开） | 专利公开后不再具有秘密性，不构成商业秘密 |
| 员工从公开渠道自主积累的技能/经验 | 个人技能和经验不属于商业秘密，不能被竞业限制约束 |

### 步骤 4：保密义务与竞业限制的区别

**这是法律实践中最常见的混淆点，须明确区分：**

| 维度 | 保密义务 | 竞业限制 |
|------|---------|---------|
| **适用人员范围** | **全体劳动者**均可约定 | **仅限三类人员**（高管/高技/其他负有保密义务者） |
| **是否须支付补偿** | **无须支付**额外补偿（属劳动者附随义务） | **必须按月支付**经济补偿（强制性） |
| **法律依据** | 诚实信用原则 + 约定（可约定于合同/保密协议） | 《劳动合同法》第23-24条（**已核实锚点**） |
| **约束内容** | 不泄露/不使用商业秘密 | 不到竞品单位工作/不自行开展竞争业务 |
| **约束期限** | 在职期间 + 离职后（商业秘密存续期间有效） | 离职后最长2年（第24条，**已核实锚点**） |
| **违约金** | 可约定，无法定限制 | 须区分：用人单位只能在竞业限制范围内约定违约金（约第25条 `[待验证]`） |

**常见错误做法**（须在审查中发现并纠正）：

- 将保密条款冠名为「竞业限制条款」：若实质上仅约定不泄露信息，而非禁止就业，须区分性质
- 对不属于三类人员的普通员工约定竞业限制：效力存争议，建议改为保密义务约定
- 保密义务约定无补偿（合法）与竞业限制无补偿（违法或效力存疑）混同：须分别审查

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 对普通员工（基础岗位）约定竞业限制 | 标🔴：效力存争议，即便支付补偿也可能因不属三类人员而存在纠纷；建议改为保密义务约定（无需补偿） `[待验证]` |
| 劳动者职位模糊（如「高级专员」是否属高级技术人员） | 以实际工作内容和接触信息判断，不依赖职位名称；建议用人单位提供实际职责说明书 |
| 保密协议与竞业限制协议合并为一份文件 | 须识别各条款的法律性质，分别按保密义务和竞业限制的规则审查（补偿/人员/期限分别核查） |
| 商业秘密已泄露（签协议前已泄露）| 竞业限制本意是事前防范，商业秘密已泄露后约定竞业限制的目的性存疑；建议同时追究违反保密义务的责任 |
| 知悉商业秘密的范围极广（几乎全公司员工）| 须审查保密措施是否真实有效，过宽的保密范围可能导致商业秘密三要件（保密措施）认定存疑 |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 用人单位无法说明何为「商业秘密」 | 建议用人单位先梳理商业秘密清单（按三要件列举），再评估哪些岗位实际接触；未完成前，竞业限制的正当性无法确认 |
| 劳动者主张自己不接触商业秘密 | 按三要件核查实际接触情况；若确无接触，建议用人单位撤销竞业限制约定，或与劳动者协商修改为保密义务 |
| 实践配置文件人员范围为占位符 | 使用法定三类人员标准进行审查，标`[临时模式]`；建议运行冷启动访谈完善配置 |
