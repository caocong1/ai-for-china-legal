## 目的

从地域范围、行业竞品范围、岗位业务范围三个维度，
评估竞业限制约定范围的合理性与可执行性，
识别范围过宽导致被认定不可执行的风险，
以及与劳动者生存权/择业权的合理利益平衡，
输出「限制是否合理」的可执行性结论。

---

## 详细步骤

### 步骤 1：法定竞业范围框架核查

**法定范围基准**（**第24条已核实锚点**）：

《劳动合同法》（2012年修正）第24条规定，竞业限制范围为：

- **不得到**与本单位生产或经营**同类产品**、从事**同类业务**的有竞争关系的用人单位就职
- **不得自己开业**生产或经营同类产品、从事同类业务

**要点解析**：

| 要素 | 说明 |
|------|------|
| 「同类产品/同类业务」 | 以用人单位**实际**生产/经营的产品/服务为基准，而非可能涉及的所有业务 |
| 「有竞争关系」 | 须存在实际竞争关系，与本单位无竞争关系的单位不在限制范围内 |
| 「本单位」 | 指用人单位本身，关联公司通常需另行约定（若扩展至集团公司须明确列举 `[待验证]`） |

### 步骤 2：地域范围合理性分析

地域范围是竞业限制可执行性争议的高频领域：

#### 2.1 地域范围核查表

| 约定地域范围 | 合理性评估 | 风险等级 |
|-----------|----------|---------|
| 仅限用人单位实际开展业务的城市/省份 | 通常合理 | 🟢 |
| 全国范围，用人单位在全国均有实质业务 | 通常可支持（须有证据证明全国业务存在）| 🟡 |
| 全国范围，但用人单位实际仅在少数城市运营 | 范围明显过宽，不可执行风险🔴 | 🔴 |
| 全球/境外范围 | 范围过宽，中国法律框架下执行可能性极低🔴 | 🔴 |
| 特定省份/区域，与用人单位业务匹配 | 合理 | 🟢 |

#### 2.2 过宽地域范围的法律后果

- 地域范围明显超出用人单位实际业务范围的，仲裁委/法院可能：
  - 缩减至合理地域范围（司法适当缩减，各地实践不一 `[待验证]`）
  - 认定该条款整体无效（较少见 `[待验证]`）
- 劳动者可以「范围不合理」为由主张竞业限制条款不可执行

**建议**：

地域范围须与用人单位的实际商业利益（商业秘密的地理影响范围）相匹配，
用人单位须能举证说明该地域范围对保护其商业利益的必要性。

### 步骤 3：行业/竞品范围合理性分析

#### 3.1 竞品/竞争单位范围核查

| 约定方式 | 合理性评估 | 风险 |
|---------|----------|------|
| 明确列举竞争对手公司名称 | 清晰具体，通常合理 | 🟢（但名单可能不完整） |
| 定义为「从事同类产品/服务」的单位 | 符合法定标准，通常合理 | 🟡（「同类」的认定存在弹性） |
| 「包括但不限于以下竞争对手」+开放列举 | 开放列举存在扩大解释风险 | 🟠 |
| 所有「相关行业」或「类似行业」 | 定义模糊，明显过宽 | 🔴 |
| 劳动者未来任何工作单位 | 极端过宽，等同剥夺劳动权，通常不可执行🔴 | 🔴 |

#### 3.2 「包括但不限于」开放式列举的风险

协议常见表述：「不得就职于以下公司，包括但不限于：[竞品列表]...」

**风险分析**：

- 「包括但不限于」措辞使竞业范围无法明确，劳动者无法确知哪些单位属禁止范围
- 在仲裁/诉讼中，此类模糊约定可能被认定为范围不明确而影响执行力
- **建议**：改为明确列举或明确定义「同类产品」的判断标准

#### 3.3 自行创业的限制

**法定范围包含**自己开业生产或经营同类产品/从事同类业务（第24条，**已核实锚点**）。

但「自行创业」的限制须合理：
- 若劳动者仅以个人名义提供与竞争单位无关的服务，通常不构成违反竞业限制
- 若劳动者实际控制的公司从事同类业务，即便名义上非劳动者直接就业，也可能构成违反

### 步骤 4：岗位/业务范围合理性分析

竞业限制应与劳动者**原岗位接触的商业秘密**相关，而非泛化至所有工作：

| 审查项 | 说明 | 建议 |
|-------|------|------|
| 限制岗位是否与原岗位相关 | 须与原岗位接触的商业秘密/客户/技术相关 | 界定具体岗位类型 |
| 是否限制劳动者在竞争单位的**所有**岗位（含无关岗位）| 过宽：若劳动者去竞争单位做行政/财务，与原商业秘密无关，过宽限制影响择业权 | 建议限制与原岗位相关的具体职能 |
| 是否限制劳动者在非竞争单位的相关岗位 | 不合理：若用人单位不竞争该领域，此限制无正当利益 | 应删除 |

### 步骤 5：择业权与商业利益的平衡

**劳动者的基本权利保护**：

中国劳动法律体系保护劳动者的**劳动权利和生存权**，竞业限制作为对劳动者择业自由的限制，
在设计上须有合理性约束：

| 平衡维度 | 分析框架 |
|---------|---------|
| 是否有正当保护利益 | 用人单位须有真实的商业秘密/正当利益须保护，而非仅排挤竞争对手 |
| 限制范围是否超必要 | 限制范围不得超过保护正当利益的必要程度 |
| 劳动者是否获得充分对价 | 经济补偿须与限制程度相称（见`term-and-compensation`） |
| 是否严重妨碍劳动者维持生计 | 若劳动者技能专一且范围过宽导致无法就业，极端情形下可能被认为违反公共政策 `[待验证]` |

**不合理限制的认定后果**（各地实践 `[待验证]`）：

| 情形 | 可能后果 |
|------|---------|
| 地域/范围过宽 | 仲裁/法院缩减范围或认定部分无效 |
| 范围明显不合理损害劳动者基本生存权 | 条款可能被整体认定不可执行 `[待验证]` |
| 补偿明显过低 + 范围过宽 | 综合因素下条款效力存风险🔴 |

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 竞业限制范围约定极宽（全行业/全球/所有单位）| 标🔴：明显超出必要范围，可执行性极低；建议重新协商设定合理范围 |
| 用人单位经营范围不断扩张，原来约定的竞品已不能覆盖所有竞争对手 | 仅能执行协议约定时的范围，未来新竞争对手不能直接援引；须重新签署协议 `[待验证]` |
| 「关联公司」是否纳入禁止就职范围 | 须协议明确约定；未约定的情况下，关联公司是否被认为是「有竞争关系」存在争议 `[待验证]` |
| 劳动者的技能高度专一（如特殊领域工程师），范围合理但劳动者几乎无法就业 | 须评估补偿与限制的相称性；极端情形下可能影响效力 `[待验证]`；建议提高补偿以支持可执行性 |
| 约定禁止在特定行业内的「任何职位」（含非竞争岗位）| 标🟠：过宽；建议限制为与原岗位同类或相关的职能岗位 |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 竞业范围约定使用中文模糊词汇（「类似业务」「相关行业」） | 分析：「定义不清晰，在执行时存在较大争议空间；建议明确列举或定义判断标准」 |
| 用人单位无法提供实际经营范围证明（无营业执照/年报）| 建议补充：竞业范围的合理性须与用人单位实际经营范围对应，若无法证明，范围约定的正当性存疑 |
| 劳动者主张「我的工作与竞争对手无关」| 按岗位职责和实际接触信息分析，而非仅依据劳动者主张；若确实无关，支持劳动者的主张 |
