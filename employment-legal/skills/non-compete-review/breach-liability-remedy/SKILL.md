# 违约责任与救济（breach-liability-remedy）

> **前置**：
> 1. 读取 `employment-legal/CLAUDE.md`（实践配置文件），加载 `## 审查立场 > 竞业限制`
>    和 `## 谁在使用`（服务对象立场）。
>    若含占位符，进入**临时模式**（见下方）。
> 2. 读取 `employment-legal/skills/_shared/legal-basis-conventions.md`（引用规范）。
> 3. 读取 `employment-legal/skills/_shared/labor-law-citations.md`（引用库，
>    重点：§A.6 竞业限制第23-24条、§A.7 违约金限制第25条）。
> 4. 分工说明：本子技能专注「违约怎么办」（双向违约责任与救济）；
>    `personnel-scope-confidentiality` 负责「谁能被约束」（人员资格）；
>    `term-and-compensation` 负责「多久、给多少」（期限与补偿），含三个月未付补偿的解除权；
>    `scope-enforceability` 负责「限制范围是否合理」（可执行性）。
>    **本子技能不重复分析三个月未付补偿的解除权（在term-and-compensation中已覆盖），
>    而是聚焦违约发生后的双向责任与救济路径。**
>    **与 hiring-review/service-period-penalty 子技能的分工**：服务期违约金由
>    `hiring-review/service-period-penalty` 子技能深度分析；本子技能专注**竞业限制违约金**，
>    二者不重叠（均属第25条允许的两种违约金情形）。
> 5. 本子技能可被 `employment-legal/skills/non-compete-review/SKILL.md` 编排入口调用，
>    也可通过 `/employment-legal:breach-liability-remedy` 单独触发。

## 目的

分析竞业限制违约发生时的双向责任与救济路径：
劳动者违约时（违约金 + 继续履行竞业限制）；
用人单位违约时（劳动者解除权 + 追偿欠付补偿）；
违约金合理性调整机制；
以及与服务期违约金的制度分工边界，
输出「违约怎么办」的双向救济方案。

---

## 详细步骤

### 步骤 1：立场确认与违约性质判断

首先确认立场和违约情形：

| 违约类型 | 违约方 | 适用路径 |
|---------|-------|---------|
| **劳动者违约** | 劳动者违反竞业限制（去竞争单位就职/自行开展竞争业务）| 步骤2（劳动者违约责任与救济） |
| **用人单位违约** | 用人单位连续三个月未支付竞业限制补偿 | 步骤3（用人单位违约救济）+ 见`term-and-compensation` |
| **双向违约** | 劳动者违约且用人单位也存在补偿欠付 | 步骤2 + 步骤3 + 步骤4 |
| **尚未违约，事前评估** | 协议已签，评估潜在违约风险 | 步骤2、3 的风险分析框架 |

### 步骤 2：劳动者违约的救济路径（用人单位方视角）

**劳动者违约时，用人单位的救济手段**：

#### 2.1 违约金请求

**法律基础**（约第25条 `[待验证]`）：

《劳动合同法》（2012年修正）第25条规定，仅以下两种情形可约定由劳动者承担违约金：
1. 服务期约定（第22条，见`hiring-review/service-period-penalty`）
2. 竞业限制约定（**第23条**，本子技能分析范围）

**审查违约金约定**：

| 审查项 | 说明 | 状态 |
|-------|------|------|
| 是否约定了违约金 | 协议中是否有明确违约金条款 | ☐ 是 / ☐ 否（须主张实际损失） |
| 违约金金额/计算方式 | 固定金额 / 按月工资倍数 / 按竞业补偿总额计算 | [约定方式：  ] |
| 违约金是否明确适用竞业限制违约 | 须区分：若协议有多个违约金条款须分别评估 | ☐ 明确 / ☐ 不明确 |
| 违约金是否过高 | 须与实际损失相称（见2.2 违约金合理性分析） | [评估结论] |

#### 2.2 违约金合理性分析（可请求调整）

**调整机制**（`[待验证]` — 须核查劳动合同法相关司法解释）：

劳动者可向仲裁委/法院申请调整过高的竞业限制违约金。参考因素：

| 考量因素 | 说明 |
|---------|------|
| 用人单位的实际损失 | 劳动者违约造成的损失（商业秘密泄露的实际损害/客户流失等）|
| 劳动者的违约情节 | 违约程度（是否主动泄露机密/是否拉拢原客户）/ 违约持续时间 |
| 补偿与违约金的比例关系 | 违约金是否明显高于竞业限制补偿总额，是否有威慑过度之嫌 |
| 劳动者的支付能力 | `[待验证]`（是否纳入考量存在争议） |

**违约金调整原则**（`[待验证]`）：

> 「约定的违约金过分高于造成的损失，当事人可以请求适当减少」
>（类比民法典第585条第2款 `[待验证]`，劳动合同领域的具体适用须核查劳动法司法解释）

**重要提示**：劳动合同法语境下的违约金调整标准，与一般民事合同的调整标准
在适用依据上有差异，须核查最高院专项劳动争议司法解释 `[待验证]`。

#### 2.3 要求继续履行竞业限制义务

除违约金外，用人单位**还可以**要求劳动者继续履行竞业限制义务（即离开竞争单位/停止竞争业务），
**二者可并存**（违约金 + 继续履行，非二选一）：

| 救济措施 | 说明 | 是否可并存 |
|---------|------|---------|
| 违约金 | 对过去违约行为的损失赔偿 | ✓ |
| 继续履行竞业限制义务 | 对未来期间的约束延续（停止违约行为） | ✓（可与违约金并存） |
| 实际损失赔偿 | 违约金不足以覆盖实际损失时，主张差额（`[待验证]`） | ✓（须核查是否可补充主张） |

**继续履行竞业限制的可行性**：

- 若劳动者离职后已在竞争单位就职：要求继续履行意味着须退出该竞争单位
- 实际执行难度较高（无法强制劳动者辞职），通常通过申请禁令/调解实现 `[待验证]`
- 竞业限制期限已过的，继续履行不再有意义，仅剩违约金请求

#### 2.4 劳动者违约责任汇总

| 救济措施 | 适用条件 | 金额估算 |
|---------|---------|---------|
| 违约金（约定额或调整后额） | 协议有约定 | ¥[约定额] / 估算调整范围 |
| 继续履行竞业限制 | 期限未满 | 须离开竞争单位 |
| 损失赔偿（超出违约金部分，`[待验证]`） | 实际损失超出违约金 | ¥[估算损失 - 违约金] |

### 步骤 3：用人单位违约的劳动者救济（劳动者方视角）

**用人单位违约的两类情形**：

#### 3.1 连续三个月未支付补偿（解除权）

（与`term-and-compensation`子技能联动，此处简述救济路径）

| 劳动者救济手段 | 说明 |
|-------------|------|
| 解除竞业限制义务 | 可请求解除竞业限制约定，恢复自由就业 `[待验证]` |
| 追讨欠付补偿 | 已履行期间的应付补偿须追缴（属工资性质债权，仲裁时效1年） |
| 不承担违约金 | 用人单位违约导致解除，劳动者为守约方，不承担违约金 |

#### 3.2 用人单位单方终止竞业限制约定

若用人单位主动提出解除竞业限制约定（不再要求劳动者遵守），须：
- 书面通知劳动者解除竞业限制
- 支付劳动者已履行竞业限制期间的全部补偿（即便单方解除，已履行期间须付）
- 解除后劳动者恢复择业自由，不受原竞业限制约束

**若用人单位在解除通知发出前已停止支付补偿**：
按三个月未付规则，劳动者已可主张解除；
用人单位单方解除通知系确认该解除，通常不影响劳动者追讨欠付补偿的权利。

### 步骤 4：双向违约场景分析

当双方均存在违约（劳动者违反竞业限制 + 用人单位欠付补偿）时，责任分配较复杂：

| 场景 | 分析框架 |
|------|---------|
| **用人单位先欠付（超3个月），劳动者后违约** | 劳动者已有权解除竞业限制义务，其「违约」可能被认定为合法解除后的行为，违约金风险较低 `[待验证]` |
| **劳动者先违约（已就职竞争单位），用人单位后停付补偿** | 劳动者的先期违约通常不能免除用人单位的补偿义务（二者各自承担违约责任）`[待验证]` |
| **双方同步违约** | 须分析哪一方违约在先，以及各自违约对对方义务的影响 `[待验证]`；建议升级分析 |

**双向违约处理建议**：

1. 首先厘清违约时间线（谁先谁后）
2. 分别评估各自违约责任
3. 考虑双方是否可通过谈判和解，减少双方损失

### 步骤 5：与服务期违约金的制度分工

**本子技能与`hiring-review/service-period-penalty`的边界**：

| 维度 | 竞业限制违约金（本子技能）| 服务期违约金（service-period-penalty）|
|------|----------------------|--------------------------------------|
| **法律依据** | 《劳动合同法》第23条 + 第25条（约 `[待验证]`） | 《劳动合同法》第22条 + 第25条（约 `[待验证]`） |
| **触发情形** | 劳动者违反竞业限制义务（离职后就职竞争单位） | 劳动者接受专项培训后未满服务期离职 |
| **约定限制** | 无明确上限，但过高可申请调整 `[待验证]` | 不超过培训费用，且按剩余服务期分摊 |
| **补偿关系** | 竞业限制违约金与经济补偿相互独立 | 服务期违约金与经济补偿不冲突（服务期本身合法） |
| **深度审查位置** | **本子技能** | `hiring-review/service-period-penalty` |

**识别混同约定**：

若协议将竞业限制违约金和服务期违约金合并为同一条款，须分拆评估：
- 分别识别哪些金额对应竞业限制违约，哪些对应服务期违约
- 各自按对应的法律规则评估合理性

## 边界条件

| 场景 | 处理方式 |
|------|---------|
| 约定违约金畸高（如超过劳动者年薪10倍）| 标🔴：极大概率被调整；建议在争议前与对方谈判压低，或准备举证实际损失较低的证据 |
| 协议未约定违约金但劳动者确已违约 | 用人单位须自行举证实际损失，按实际损失主张赔偿（举证难度较高）；建议在签署时补充违约金条款 |
| 竞业限制期限已届满后劳动者违约（如末月才入职竞争单位）| 仍属违约（期限内有效），须按协议处理 |
| 用人单位欲主动解除竞业限制（无需劳动者遵守）| 可书面通知解除，并支付已履行期间补偿；无需劳动者同意 `[待验证]` |
| 劳动者违约后已离开竞争单位并回头履行竞业限制 | 已发生的违约期间的违约金仍可主张，但继续履行不再有实际意义 |

## 错误处理

| 错误场景 | 处理方式 |
|---------|---------|
| 协议违约金条款语言模糊（「视情况赔偿损失」）| 分析：「此表述实质为约定按实际损失赔偿，无固定违约金，须举证损失金额」 |
| 违约金与服务期违约金合并约定为同一条款 | 须分拆评估：识别各自对应部分，分别按竞业限制和服务期规则审查 |
| 用人单位主张违约金且同时主张停止违约行为，劳动者认为二者不能并存 | 分析：「中国法律框架下，竞业限制违约金与继续履行请求可并存（均可主张），须向仲裁委/法院主张两项请求」 `[待验证]` |
| 实践配置文件无竞业限制违约金立场 | 使用一般原则：以合理补偿实际损失为基准，过高须申请调整，标`[临时模式]` |
