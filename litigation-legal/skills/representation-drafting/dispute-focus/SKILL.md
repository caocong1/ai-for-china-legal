# 争议焦点梳理

> **研究闸门**：除非用户明确豁免联网检索，必须先确认 `shared/research-gate/SKILL.md` 已运行，研究底稿已完成，再进入本子技能。
>
> **单独触发说明**：本子技能可通过 `/litigation-legal:dispute-focus` 单独触发，只运行争议焦点梳理，无需重跑整个代理词起草流程。

## 目的

系统梳理案件争议焦点，明确我方与对方的核心分歧，为后续事实论证（fact-argumentation）和法律论证（legal-argumentation）奠定逻辑基础。

争议焦点梳理的作用：
1. 防止代理词失焦——只对准法院需要裁决的事项展开论证
2. 识别我方强势焦点（有利）与弱势焦点（不利）
3. 为庭审辩论提供结构化应对框架

## 步骤一：识别信息来源

### 1.1 争议焦点信息来源优先级

| 来源 | 优先级 | 说明 |
|------|--------|------|
| 法院在庭前会议/开庭时归纳的争议焦点 | 最高 | 以法院归纳为准，不得自行替换 |
| 双方提交的答辩状/起诉状中显现的分歧 | 次高 | 庭前阶段使用 |
| 律师根据案件事实自行归纳 | 参考 | 庭前草案，须与法院归纳核对 |

> **边界条件**：庭审已由法官归纳争议焦点时，本步骤直接采纳法院归纳，不作调整，仅补充我方应对框架。

### 1.2 收集基础信息

向律师确认以下信息（如事项工作区已有，自动读取）：

| 字段 | 说明 |
|------|------|
| 案件类型 | 合同纠纷 / 侵权 / 劳动 / 知识产权 / 其他 |
| 诉讼阶段 | 庭前 / 一审庭审中 / 二审 |
| 我方角色 | 原告 / 被告（读自配置文件，向律师确认）|
| 对方诉讼请求 | 列出全部诉讼请求（如我方为被告）|
| 我方诉讼请求 | 列出全部诉讼请求（如我方为原告）|
| 法院归纳的争议焦点 | 如有，直接引用庭审笔录原文 |

## 步骤二：梳理争议焦点

### 2.1 争议焦点分类

将所有争议焦点归入以下分类：

| 类型 | 说明 | 示例 |
|------|------|------|
| **事实争议** | 双方对某项事实是否存在/如何认定存在分歧 | 合同是否已成立、货物是否已交付 |
| **法律适用争议** | 双方对适用何种法律规则或如何解释存在分歧 | 违约金是否过高、损害赔偿范围如何计算 |
| **程序性争议** | 管辖权、诉讼资格、举证责任分配等 | 仲裁协议是否有效、管辖法院是否正确 |
| **混合争议** | 同时涉及事实认定和法律适用 | 是否构成根本违约（需认定事实 + 法律判断）|

### 2.2 逐条梳理争议焦点

对每个争议焦点，完成以下分析：

**焦点 [N]：[焦点名称/描述]**

- **焦点类型**：[事实争议 / 法律适用争议 / 程序性争议 / 混合争议]
- **我方立场**：[用 1-2 句话概括我方核心主张]
- **关键事实支持**：[支持我方立场的关键事实 1-3 条，注明对应证据序号]
- **对方立场**：[用 1-2 句话概括对方核心主张]
- **对方关键主张**：[对方可能援引的事实或法律依据]
- **我方优势/劣势评估**：[我方在此焦点的胜算评估：强 / 中 / 弱 / 不确定]
- **需补充的证据或论证**：[如有缺口，标记需补充内容]

### 2.3 交叉质证要点

对对方可能援引的事实，提前准备反驳要点：

| 对方主张 | 反驳方向 | 所需证据 |
|---------|---------|---------|
| [对方主张 1] | [反驳思路] | [所需证据或法律依据] |
| [对方主张 2] | [反驳思路] | [所需证据或法律依据] |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 法院已归纳争议焦点 | 以法院归纳为权威来源，不作实质性替换；在我方立场栏填充我方应对立场 |
| 存在尚未质证的证据 | 在事实支持列标注「[未质证 — 仅参考，不作为确认事实]」 |
| 对方提出新证据/主张（尚未质证）| 标注「[对方新证据/主张 — 待质证，立场存不确定性]」 |
| 争议焦点超过 5 个 | 按对代理词最终结果影响排序，优先处理前 5 个核心焦点；其余列为「次要焦点」附录 |
| 仲裁案件 | 争议焦点梳理逻辑相同；注意仲裁庭可能不公开宣告争议焦点，需从双方文件中自行梳理 |
| 二审案件 | 二审争议焦点通常限于对一审判决有异议的事项，梳理时以上诉状/答辩状为起点 |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 律师无法提供对方主张 | 根据起诉状/仲裁申请书推断，标注「[对方主张推断 — 基于起诉状，需与律师确认]」 |
| 案件材料不完整 | 列出已知焦点，对信息缺口标注「[待补充：需提供[具体材料]]」，不得自行补全 |
| 焦点高度法律专业，无法独立判断 | 描述分歧点，标注「[焦点定性需律师确认 — 涉及专业法律判断]」 |
