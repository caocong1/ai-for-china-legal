# 代理词起草

> **研究闸门**：除非用户明确豁免联网检索，必须先运行 `shared/research-gate/SKILL.md`，完成研究底稿后再进入本 skill。

## 目的

根据案件事实、证据和法律依据，起草逻辑严密、论证有力的代理词或法庭陈述材料。

## 前置：加载共享资源

在进入工作流前，读取以下共享资源：
- `shared/references/document-structures.md` — 诉讼文书结构模板
- `shared/research-gate/references/quality-gates.md` — 研究质量闸门
- `shared/research-gate/references/search-playbooks.md` — 搜索策略（如涉保函/担保/主体资格）

## 工作流

### 第一步：案件回顾

- 争议焦点
- 我方立场
- 对方立场
- 已提交证据
- 庭审情况（如适用）

### 第二步：代理词结构

1. **开头**
   - 代理人身份
   - 代理权限
   - 简要陈述代理意见

2. **事实部分**
   - 案件事实概述
   - 证据支持
   - 对方事实的反驳

3. **法律部分**
   - 适用法律
   - 法律分析
   - 案例支持（如适用）

4. **结论部分**
   - 代理请求
   - 理由总结

### 第三步：法律依据检索

- 相关法律条文
- 司法解释
- 指导性案例
- 学术观点（如适用）

### 第四步：起草代理词

```markdown
# 代理词

**审判长、审判员：**

根据《中华人民共和国民事诉讼法》的相关规定，我接受 [当事人] 的委托，担任本案的诉讼代理人。经过庭前准备和今天的庭审，现发表如下代理意见：

## 一、案件事实

[事实陈述]

## 二、法律分析

[法律分析]

## 三、代理请求

综上所述，请求法院：

1. [请求 1]
2. [请求 2]

**代理人：** [签名]
**日期：** [日期]
```

## 输出

**代理词：** 完整的代理词。

**辩论提纲：** 庭审辩论要点。

**法庭陈述：** 开庭陈述或结案陈词。
