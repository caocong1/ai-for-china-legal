# 冷启动访谈 — 诉讼仲裁

## 目的

通过结构化的访谈流程，收集诉讼律师的执业立场、案件管理偏好、风险校准和文书风格，生成个性化的实践配置文件。

## 访谈流程

### 第一部分：基本信息

1. **律师角色和团队**
   - 合伙人 / 资深律师 / 初级律师 / 法务
   - 执业年限和主要诉讼领域
   - 团队结构（独立/小团队/大团队）

2. **案件类型偏好**
   - 合同纠纷 / 劳动争议 / 知识产权 / 侵权 / 公司治理 / 房地产
   - 主要角色倾向（原告/被告/两者）

### 第二部分：风险校准

3. **严重程度定义**
   - 高：什么构成高严重程度？
   - 中：什么构成中等严重程度？
   - 低：什么构成低严重程度？

4. **可能性定义**
   - 高：什么构成高可能性？
   - 中：什么构成中等可能性？
   - 低：什么构成低可能性？

5. **重大性阈值**
   - 准备金阈值（金额）
   - 披露阈值（金额）
   - 监控阈值（金额）

### 第三部分：案件管理

6. **利益冲突检查方法**
   - 律所系统 / 外部律师 / 人工检查 / 其他

7. **证据保全偏好**
   - 自动发出保全通知 / 逐案决定
   - 保全范围偏好

8. **外部律师管理**
   - 是否有偏好的外部律师团队？
   - 聘用协议标准条款

### 第四部分：文书风格

9. **输出格式偏好**
   - 完整案件备忘录
   - 案件状态摘要
   - 法院文件草稿

10. **文书风格**
    - 正式/半正式/简洁
    - 是否需要案例引用？

## 输出

生成诉讼仲裁实践配置文件（YAML 格式）。
