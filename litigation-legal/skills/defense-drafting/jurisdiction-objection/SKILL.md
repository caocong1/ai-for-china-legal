# 管辖权异议

> 引用规范：`litigation-legal/skills/_shared/legal-basis-conventions.md`
> 引用库：`litigation-legal/skills/_shared/civil-procedure-citations.md`（重点：A.2节级别管辖 + A.3节地域/专属管辖 + A.4节管辖权异议程序）

> 也可通过 `/litigation-legal:jurisdiction-objection` 单独触发。

---

## 目的

审查起诉法院是否具有级别管辖权和地域管辖权，判断是否存在管辖错误，并在管辖权异议有胜算时起草管辖权异议申请书。

**仲裁案件特殊说明**：本子技能的「提管辖权异议」路径适用于法院诉讼；仲裁案件管辖问题走「仲裁协议效力异议」路径，见步骤五。

## 步骤三：地域管辖审查

### 3.1 管辖判断链

按以下顺序逐项检查：

```
1. 是否有专属管辖？→ 有则按专属管辖
2. 是否有约定管辖（书面协议）？→ 有则审查约定管辖是否有效
3. 无以上两项 → 按一般/特殊地域管辖规则
```

## 步骤五：仲裁场景 — 仲裁协议效力异议

**适用情形**：本案为仲裁程序，且认为仲裁协议无效或仲裁机构无管辖权。

| 仲裁协议效力异议路径 | 说明 |
|-------------------|------|
| 向仲裁委员会申请 | 请求仲裁委员会就仲裁协议效力或管辖权作出决定 `[待验证]` |
| 向人民法院申请裁定 | 同时向人民法院申请确认仲裁协议无效；但同时提出时，以仲裁委员会决定为准 `[待验证]` |
| 无效情形 | 仲裁事项超出仲裁范围（如婚姻、收养等人身关系不得仲裁）；无民事行为能力人签订；胁迫签订 `[待验证]` |
| 缺陷情形 | 仲裁协议未明确约定仲裁委员会名称（约定不明时的补救方法）`[待验证]` |

## 边界条件

| 情形 | 处理方式 |
|------|---------|
| 约定管辖与专属管辖冲突 | 专属管辖优先，约定管辖无效，说明约定管辖条款无效的理由 |
| 无法判断当地级别管辖标准 | 标记 `[需查当地高院级别管辖规定 — 不自行假设金额标准]`，提示查询途径 |
| 已超过答辩期限 | 标记 `[管辖权异议时机已过 — 答辩期届满后不得再提]`，不再起草异议申请书 |
| 专门法院管辖（海事/知产/金融）| 检查是否应由专门法院管辖，说明专门法院管辖的适用情形 |
| 仲裁协议约定的仲裁机构不明确 | 说明约定不明的补救规则（须双方协议或无法确定时仲裁条款无效的风险）`[待验证]` |

## 错误处理

| 错误情形 | 处理方式 |
|---------|---------|
| 起诉法院地域不明（网上立案无信息）| 标记 `[管辖法院待确认]`，提示从立案通知书确认具体法院名称和级别 |
| 律师已超过答辩期再咨询此问题 | 明确告知时机已过，管辖权异议不再可行，但可在二审阶段主张管辖错误（如判决存在程序瑕疵）`[待验证]` |
| 研究闸门运行失败 | 标记 `[研究闸门未完成 — 以下管辖分析基于已知信息，建议核验当地级别管辖标准]` |
