> 本文件定义 litigation-legal 插件所有技能统一使用的法律依据引用规范。
> 后续计划（03-02、03-03）直接引用本规范，避免各技能重复维护。
> 本规范以 `commercial-legal/skills/_shared/legal-basis-conventions.md` 为骨架，针对诉讼仲裁语境改编。

## 一、来源分类

所有法律依据必须归入以下三类之一，并在引用时明确标注类别标签。

### 1. 法条（成文法）

指现行有效的成文法规范，包括：

- **法律**：全国人民代表大会及其常委会制定（如民事诉讼法、仲裁法、民法典等）
- **行政法规**：国务院制定（如《中华人民共和国民事诉讼法实施条例》等）
- **部门规章**：国务院各部门制定（如司法部规章、法院审判程序规章等）
- **地方性法规**：省级人大及其常委会制定（适用于管辖识别）
- **司法解释**：最高人民法院、最高人民检察院发布的解释性文件（具有规范效力）
  - 重点包括：《关于民事诉讼证据的若干规定》（2019年修正，2020年施行）等

**成文法是分析基础。** 诉讼仲裁风险评估和文书起草的所有结论，必须首先对照现行成文法条文，而非单纯依赖案例倾向或司法惯例。

**格式要求（法条最小引用格式）：**

```
《[法律/法规全名]》（[版本/修订年份]）第[X]条[第X款]
```

示例：
```
《中华人民共和国民法典》（2020年）第188条
《中华人民共和国民事诉讼法》（2023年修正）第[X]条 `[待验证]`
```

**版本标注要求：**

- 引用的法条必须标注现行有效版本（修订年份或施行年份）
- 民事诉讼法最新版本：2023年修正版（2023年9月1日起施行）
- 仲裁法：现行版本为1994年通过、1995年9月1日施行，后续是否有修订须核查 `[待验证]`
- 如引用已修订版本，须注明「（已修订，适用旧法情形）」

### 2. 案例（指导性案例与参考案例）

指最高人民法院发布的指导性案例或参考案例。

**定位说明：指导性案例仅作参考，不具有判例法约束力。** 中国采用成文法体系，案例不能替代法条。但最高人民法院指导性案例对下级法院有参照作用，可用于说明裁判思路和司法实践倾向。

**格式要求（案例最小引用格式）：**

```
[最高人民法院指导案例第X号] / [案号：（年份）法院级别字号数字号] — [裁判要点摘要] — [法院层级] `[待验证]`
```

示例：
```
最高人民法院指导案例第XX号 — [裁判要点] — 最高人民法院 `[待验证]`
（YYYY）最高法民再XXXX号 — [争议焦点裁判要旨] — 最高院 `[待验证]`
```

**法院层级标注：**

| 层级 | 简称 | 字号特征 | 说明 |
|------|------|---------|------|
| 最高人民法院 | 最高院 | 最高法 | 参照价值最高 |
| 高级人民法院 | 高院 | XX高法 | 省级，参照本省 |
| 中级人民法院 | 中院 | XX中法 | 地市级 |
| 基层人民法院 | 基层法院 | XX法 | 县区级 |

**专门法院标注：**

| 专门法院类型 | 说明 |
|------------|------|
| 海事法院 | 涉海纠纷专属管辖 |
| 知识产权法院 | 重大知识产权案件 |
| 金融法院（上海） | 特定金融纠纷 |
| 互联网法院 | 互联网相关纠纷 |

**仲裁机构标注（区别于法院）：**

| 仲裁机构 | 简称 | 说明 |
|---------|------|------|
| 中国国际经济贸易仲裁委员会 | CIETAC | 国际商事仲裁为主 |
| 北京仲裁委员会 | BAC | 国内商事仲裁 |
| 上海国际仲裁中心 | SHIAC | 涉外仲裁 |
| 深圳国际仲裁院 | SCIA | 大湾区仲裁 |
| 地方仲裁委员会 | 各地仲委 | 地方商事仲裁 |

引用仲裁裁决须注明：仲裁机构名称 + 案号（如有）+ 裁决年份 + 核心裁判要旨 `[待验证]`

**模型回忆案例风险说明：** 案号、指导案例编号极易被模型错误回忆，**所有案例引用默认标注 `[待验证]`**，必须经检索核验后方可去除。

### 3. 学说（学术观点）

指学术著作、法学论文中的学术观点，或律所内部实务经验总结。

**学说为辅助参考，存在争议性。** 引用学说时必须标注：

- 作者/出处（书名/期刊名 + 年份）
- 是否存在对立观点

**格式要求：**

```
[作者姓名]，《[著作/论文名]》，[出版/期刊信息] — "[观点摘要]" [是否有争议：有/无]
```

## 二、来源标签表

引用时，在条文/案例/观点末尾附加来源标签。复用 `litigation-legal/CLAUDE.md` 中已定义的五个标签，不新造冲突标签。

| 标签 | 含义 | 使用场景 |
|------|------|----------|
| `[法宝]` | 来自北大法宝等法规数据库 | 检索工具获取的全文 |
| `[法规网站]` | 来自政府官方网站 | 全国人大、最高院等官网 |
| `[网络搜索 — 需验证]` | 来自网络搜索结果 | 需对照原始来源核查 |
| `[模型知识 — 需验证]` | 来自训练数据回忆 | 风险较高，须优先核查 |
| `[用户提供]` | 来自用户提供的文件 | 起诉状、仲裁申请、律师函等 |

**标签使用规则：**

1. 每条引用行**必须且只能**附加一个来源标签
2. 禁止剥离或折叠标签
3. `[模型知识 — 需验证]` 和 `[网络搜索 — 需验证]` 的引用在依赖前**必须**对照原始来源检查

## 三、待验证标记规则

以下情形**必须**在引用行末追加 `[待验证]` 并说明验证途径：

| 触发情形 | 说明 | 建议验证途径 |
|---------|------|-------------|
| 来源标签为 `[模型知识 — 需验证]` | 模型回忆的条文号/案号风险高 | 通过法宝、国家法律法规数据库核查 |
| 引用条文距最后更新超过 6 个月 | 法规可能已修订或废止 | 查询官方法规数据库最新版本 |
| 用户口述的条款号或时效期间 | 可能记忆偏差 | 核查合同/裁判文书原文或官方法规全文 |
| 所有案例引用（默认） | 案号易被错误回忆 | 通过中国裁判文书网、法宝等核查 |
| 法规版本未确认 | 无法确认是否为现行有效版本 | 查询国家法律法规数据库 |
| 仲裁法修订状态 | 1994年版是否有后续修订需确认 | 查询全国人大官网 `[待验证]` |

**待验证标记格式：**

```
《民法典》（2020年）第188条 `[模型知识 — 需验证]` `[待验证 — 验证途径：国家法律法规数据库]`
《民事诉讼法》（2023年修正）关于[规则]的规定 `[模型知识 — 需验证]` `[待验证]`
```

**已核实锚点（无需额外标记 `[待验证]`，但仍建议核验）：**

| 锚点 | 法律依据 | 规则摘要 |
|------|---------|---------|
| 诉讼时效一般期间3年 | 《民法典》（2020年）第188条 | 向人民法院请求保护民事权利的诉讼时效期间为三年 |
| 答辩期/举证期限15日 | 《民事诉讼法》（2023年修正）关于普通程序的规定 | 被告应于收到起诉状副本之日起十五日内提出答辩状；举证期限不得少于十五日 `[条号待验证]` |
| 仲裁协议独立性 | 《仲裁法》相应条款 | 仲裁协议独立存在，合同无效不影响仲裁协议效力 `[条号待验证]` |
| 或裁或审原则 | 《仲裁法》相应条款 | 当事人达成仲裁协议，一方向人民法院起诉，人民法院不予受理 `[条号待验证]` |

**验证资源：**

- 国家法律法规数据库：https://flk.npc.gov.cn
- 最高人民法院官网：https://www.court.gov.cn
- 中国裁判文书网：https://wenshu.court.gov.cn
- 北大法宝（如已接入）

## 四、成文法优先与案例定位

### 成文法优先原则

1. **分析基础**：诉讼仲裁风险评估和文书起草的每个结论，必须首先援引现行成文法（法律、行政法规、司法解释）作为规范依据
2. **不得以案例替代法条**：即使存在清晰的指导性案例，也必须同时引用对应法条
3. **法条缺失时**：若无直接法条，须明确说明「无直接成文法规定，参照以下案例/学说」

### 案例的辅助定位

1. **指导性案例**：用于说明法条适用方式，标注「最高院指导，参照意义强」
2. **一般生效判决**：用于说明司法实践倾向，标注「参考，非指导案例」
3. **明确说明非判例法**：引用案例时，须附注「中国采成文法体系，本案例为参考，不具判例法约束力」

### 仲裁裁决的特殊定位

仲裁裁决一裁终局，不对外公开，**不得用于引用法律依据**。仲裁法律问题的分析依据是仲裁法及司法解释，而非具体裁决内容。

## 五、法院层级与管辖标注规则

### 管辖判断链

```
1. 先看是否有仲裁协议（有效仲裁协议 → 仲裁，排除法院管辖）
2. 无仲裁协议 → 确定级别管辖（标的额/案件类型）
3. 确定地域管辖（原则：被告住所地/合同履行地/侵权行为地等）
4. 检查是否有专属管辖（房地产纠纷 → 不动产所在地）
5. 检查是否有专门法院管辖（海事 / 知识产权 / 金融）
```

**级别管辖一般规则（`[待验证]`，具体金额标准以当地法院规定为准）：**

| 法院级别 | 适用情形（一般参考） |
|---------|-----------------|
| 基层法院 | 一般民事纠纷（小额标的） |
| 中级法院 | 标的额较大、涉外、重大疑难 |
| 高级法院 | 在本省有重大影响的案件 |
| 最高院 | 在全国有重大影响的案件 |

> 注：具体金额划分以各省市法院的级别管辖规定为准，不在本文件硬写数字以免误导。

| 类别 | 最小格式 | 示例 |
|------|---------|------|
| 法条 | 《法律名称》（版本年份）第X条[第X款] [来源标签] | 《民法典》（2020年）第188条 `[模型知识 — 需验证]` |
| 司法解释 | 《司法解释名称》（文号）第X条 [来源标签] `[待验证]` | 《最高法关于民事诉讼证据的若干规定》（2020年修正）第X条 `[待验证]` |
| 指导案例 | 最高人民法院指导案例第X号 — [裁判要点] — [法院层级] `[待验证]` | 最高人民法院指导案例第X号 — [要点] — 最高院 `[待验证]` |
| 仲裁规则 | 《[仲裁机构]仲裁规则》（年份版）第X条 [来源标签] `[待验证]` | 《CIETAC仲裁规则》（202X年版）第X条 `[待验证]` |
| 学说 | [作者]，《[著作]》，[出处] — "[观点]" [争议性] | 张某，《民事诉讼法论》，2020年 — "..." 有争议 |
