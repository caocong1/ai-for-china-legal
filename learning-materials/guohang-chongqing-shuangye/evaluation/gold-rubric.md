# 诉讼方案评估标准 — Gold Rubric

本 rubric 用于评估 AI 生成的诉讼方案是否达到执业律师可接受的标准。适用于国航重庆分公司诉重庆双业担保公司测试案例，以及所有涉及保函/担保公司的案件。

## 评分体系

| 等级 | 含义 |
|------|------|
| ✅ 通过 | 达到执业律师最低可接受标准 |
| ⚠️ 可接受但需改进 | 基本可用，但有明显改进空间 |
| ❌ 不通过 | 未达到最低标准，存在法律判断错误 |

## R1: 案由锁定

| 标准 | 评分 | 说明 |
|------|------|------|
| 首选案由唯一明确（"保证合同纠纷"） | ✅ | |
| 首选案由为"独立保函纠纷" | ❌ | 融资担保公司无权开立独立保函 |
| 以"保证合同纠纷或担保合同纠纷"未决收尾 | ❌ | 必须锁定唯一首选 |
| 有首选+备选，论证充分 | ✅ | 最佳实践 |

## R2: 主体资格核验

| 标准 | 评分 | 说明 |
|------|------|------|
| 明确指出融资担保公司不属于"银行或非银行金融机构" | ✅ | |
| 引用了法释〔2016〕24号第1条 | ✅ | |
| 引用了金融监管总局令〔2023〕第3号或国务院令第683号 | ✅ | |
| 未核验出函主体资格 | ❌ | 根本性遗漏 |

## R3: 保证方式分析

| 标准 | 评分 | 说明 |
|------|------|------|
| 对"见索即付"/"无条件支付"分析了连带责任保证路径 | ✅ | |
| 引用了民法典第688条和担保制度解释第25条 | ✅ | |
| 仅警告"可能被认定为一般保证"而不分析连带路径 | ❌ | 偏向性错误 |
| 未分析保证方式 | ❌ | 根本性遗漏 |

## R4: 法律依据完整性

| 标准 | 评分 | 说明 |
|------|------|------|
| 独立保函排除：法释〔2016〕24号第1条 | ✅ | 必引 |
| 主体资格：金融监管总局令〔2023〕第3号 | ✅ | 必引 |
| 保证方式：民法典第686条+第688条 | ✅ | 必引 |
| 保证方式认定：担保制度解释第25条 | ✅ | 必引 |
| 管辖：民事诉讼法第24条 | ✅ | 必引 |
| 管辖：担保制度解释第21条 | ✅ | 必引 |
| 任一必引法条缺失 | ⚠️ | 可接受但需改进 |
| 缺失 3 条以上 | ❌ | 不通过 |

## R5: 诉讼请求措辞

| 标准 | 评分 | 说明 |
|------|------|------|
| 诉请显式包含"连带清偿责任"/"连带责任保证项下的支付责任" | ✅ | |
| 利息/损失写出起算日、止算日、计息标准 | ✅ | |
| 仅写"判令被告支付X元" | ❌ | 缺保证性质表述 |
| 缺少诉讼费/保全费请求 | ⚠️ | 可接受但需改进 |

## R6: 证据目录

| 标准 | 评分 | 说明 |
|------|------|------|
| 按 3 大类分组（基础交易/保函法律/违约事实） | ✅ | |
| 每类至少 1 项证据 | ✅ | |
| 证据为 flat list（不分组） | ❌ | 不满足闸门 |
| 缺任一大类 | ❌ | 不满足闸门 |

## R7: 风险段覆盖

| 标准 | 评分 | 说明 |
|------|------|------|
| 损失认定风险（成渝地区隐藏路径） | ✅ | 必含 |
| 主观故意认定风险 | ✅ | 必含 |
| 保函有效期/索赔时效风险 | ✅ | 必含 |
| 保证方式被改判风险 | ✅ | 必含 |
| 执行风险 | ✅ | 必含 |
| 缺少 1-2 项 | ⚠️ | 可接受但需改进 |
| 缺少 3 项以上 | ❌ | 不通过 |

## R8: 起诉策略对比

| 标准 | 评分 | 说明 |
|------|------|------|
| 明确对比三种策略（单告担保人/单告主债务人/同时起诉） | ✅ | |
| 列出各策略的管辖、举证、保全、执行复杂度 | ✅ | |
| 直接跳到结论而无对比 | ❌ | 不通过 |
| 仅对比两种策略 | ⚠️ | 可接受但需改进 |

## R9: 反向依据检索

| 标准 | 评分 | 说明 |
|------|------|------|
| 检索了不利案例或反例路径 | ✅ | |
| 列出成渝地区反例预警（损失认定、基础合同无效等） | ✅ | |
| 未做任何反向检索 | ❌ | 不通过 |
| 声明"未发现反例"并列出搜索词 | ✅ | 可接受 |

## R10: 格式与落款

| 标准 | 评分 | 说明 |
|------|------|------|
| 诉讼方案文末有律所/律师署名 + 日期位 | ✅ | |
| 落款整段缺失 | ❌ | 不通过 |
| 文书结构按模板 13 段排列 | ✅ | |
| 结构严重偏离模板 | ⚠️ | 可接受但需改进 |

## 总体评分

- **全部 ✅**：优秀，可直接交付律师使用
- **1-2 个 ⚠️，无 ❌**：可接受，建议改进
- **任何 ❌**：不通过，必须修改后重测
