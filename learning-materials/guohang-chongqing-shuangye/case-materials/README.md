## 测试用途

本材料用于测试诉讼方案生成技能（commercial-legal 插件）能否正确：

1. 识别出函主体为融资担保公司（非金融机构）
2. 排除独立保函纠纷案由，选择保证合同纠纷
3. 分析"见索即付"条款在保证合同框架下构成连带责任保证
4. 按 3 大类分组列出证据目录
5. 覆盖全部 5 类风险段
6. 正确引用法释〔2016〕24号、民法典第686/688条等核心法条

## 注意事项

- 本材料为**合成测试数据**，基于真实案件类型构造，当事人名称和关键信息已脱敏处理
- 保函扫描件 OCR 相关测试：保函编号、担保金额、有效期截止日等字段需与备忘录交叉核验
- 管辖约定在保函第六条（"担保人住所地"），但根据担保制度解释第21条，合并起诉时应按主合同确定管辖
