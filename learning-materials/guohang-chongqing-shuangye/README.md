# 真实测试案例：国航重庆分公司诉重庆双业担保公司

## 案例背景

**案件核心争点**：融资担保公司出具载有"见索即付"、"无条件支付"等表述的投标保函时，案由和请求权基础应如何选择。

### 初始失败样本

早期生成的诉讼方案出现以下关键错误：

1. 过度依赖保函文本里的"见索即付"、"无条件支付"等表述
2. 没有核验融资担保公司是否属于可开立独立保函的主体
3. 没有主动检索融资担保公司主体资格、独立保函效力转换、保证合同纠纷类案
4. 案由方向错误，将案件定为独立保函纠纷

### 正确方向

通过补充法规和案例后，正确判断为：

1. 融资担保公司通常不属于独立保函司法解释所称"银行或者非银行金融机构"
2. 其出具的"见索即付"投标保函不宜当然按独立保函处理
3. 如保函具备担保债务履行的意思表示和基础关系，应按保证合同关系主张
4. 若保函约定收到索赔后固定期限内无条件支付、无先诉抗辩安排，需重点论证连带责任保证
5. 诉讼方案应锁定一个首选案由，不能以"独立保函纠纷或保证合同纠纷"未决方式收尾
